Step 6: Game Controller (Blockly)

Add a controller object to display instructions and allow restarting.

Create the Controller Object

  1. Right-click in the Asset Tree and create a new object: obj_controller
  2. No sprite needed (invisible controller)

Draw Instructions

Open the Blockly editor for obj_controller:

Draw
Draw Text "LUNAR LANDER" at 10, 10
Draw Text "Up: Thrust | Left/Right: Steer" at 10, 30

Find Draw in Events. Add two Draw Text blocks (from Drawing) — one for the title and one for the controls.

Restart Level (Press R)

Keyboard Press: R
Restart Room

This lets the player restart the level at any time by pressing R.

Apply Your Blocks

  1. You should have 2 event blocks (Draw + Keyboard Press R)
  2. Click Apply to Events to save
Remember: Place one instance of obj_controller in your room!
