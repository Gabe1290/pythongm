Step 3: Lander Physics (Blockly)

We need gravity to pull the lander down toward the surface. We'll use Blockly visual blocks to set this up.

Open the Blockly Editor

  1. Double-click obj_lander in the Asset Tree to open the Object Editor
  2. Click the Blockly tab at the top of the editor

Enable Gravity (Create Event)

Drag these blocks from the toolbox and snap them together:

Create
Set Gravity: direction 270, strength 0.05

Find Create in the Events category. Then find Set Gravity in the Movement category. Set direction to 270 (down) and strength to 0.05.

Why 0.05? Lunar gravity is very gentle! On the Moon, gravity is about 1/6th of Earth. A small value like 0.05 gives the lander a slow, floaty feel — just like real space landing.

Make the Ground Solid

  1. Double-click obj_ground in the Asset Tree
  2. In the Object Properties panel, check the Solid checkbox

Also make obj_pad solid so the lander can land on it.

Key Concept: The gravity strength (0.05) is very low compared to the platformer tutorial (0.5). This creates the feeling of low-gravity space flight. The thrust (next step) must be stronger than gravity so the lander can slow down!
