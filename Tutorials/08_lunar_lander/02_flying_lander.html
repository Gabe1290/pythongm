Phase 1: A Flying Lander

In this phase, we create the sprites, objects, and controls to fly a lander with lunar gravity. At the end, you'll have a playable lander you can fly around!

Create the Sprites

spr_lander (32x32) — The spacecraft

  1. Right-click Sprites > Create New Sprite
  2. Name it spr_lander (32x32 pixels)
  3. Draw a small spacecraft (triangle or classic lander shape), or import from Tutorials/08_lunar_lander/assets/

spr_ground (32x32) — Rocky terrain

  1. Create a new sprite named spr_ground (32x32 pixels)
  2. Draw rocky terrain using gray or brown colors, or import from assets

Create the Objects

obj_ground

  1. Right-click Objects > Create New Object
  2. Name it obj_ground, assign spr_ground
  3. Check the Solid checkbox

obj_lander

  1. Create a new object named obj_lander
  2. Assign spr_lander

Program the Lander (Blockly)

Double-click obj_lander and open the Blockly tab.

1. Enable Gravity (Create Event)

Create
Set Gravity: direction 270, strength 0.05

Find Create in the Events category. Then find Set Gravity in the Movement category. Set direction to 270 (down) and strength to 0.05.

Why 0.05? The Moon's gravity is about 1/6th of Earth's. A very low value like 0.05 gives the lander a slow, floaty feel — just like a real lunar landing!

2. Thrust Upward (Keyboard Up Held)

Keyboard Held: Up
Set vspeed -2

Find Keyboard Held in Events, choose Up. Add Set vspeed (from Movement) with value -2. A negative vspeed moves the lander upward (thrust against gravity).

3. Steer Left (Keyboard Left Held)

Keyboard Held: Left
Set hspeed -2

Find Keyboard Held in Events, choose Left. Add Set hspeed (from Movement) with value -2.

4. Steer Right (Keyboard Right Held)

Keyboard Held: Right
Set hspeed 2

Find Keyboard Held in Events, choose Right. Add Set hspeed (from Movement) with value 2.

5. Stop Horizontal When No Key Pressed

No Key
Set hspeed 0

Find No Key in Events. Add Set hspeed with value 0. This stops horizontal movement when no arrow is pressed, but gravity keeps pulling the lander down!

Important: We only reset hspeed to 0, not vspeed. This lets gravity continue to work on the lander when no key is pressed — the lander keeps falling!

Apply Your Blocks

  1. You should have 5 event blocks (Create + 3 keyboard + No key)
  2. Click the Apply to Events button to save

Create the Room

  1. Right-click Rooms > Create New Room
  2. Name it room_game
  3. Set size: 640 x 480
  4. Set background color to black (space!)
  5. Enable "Snap to Grid" (32x32)
  6. Place obj_ground blocks along the entire bottom row
  7. Place obj_lander near the top of the room

Test Your Game!

Press F5 to play!

  • The lander falls slowly toward the ground (lunar gravity!)
  • Up Arrow: Thrust upward to fight gravity
  • Left/Right Arrows: Steer horizontally
  • Release all keys and watch the lander drift down slowly
