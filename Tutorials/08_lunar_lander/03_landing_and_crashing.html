Phase 2: Landing and Crashing

Now we add a landing pad and collision detection. Land on the pad to win — crash into the ground to restart!

Create the Landing Pad Sprite

spr_pad (32x32) — A flat green landing pad

  1. Right-click Sprites > Create New Sprite
  2. Name it spr_pad (32x32 pixels)
  3. Draw a flat green platform (a bright green rectangle), or import from assets

Create the Landing Pad Object

  1. Right-click Objects > Create New Object
  2. Name it obj_pad, assign spr_pad
  3. Check the Solid checkbox

Program Lander Collisions (Blockly)

Open the Blockly editor for obj_lander and add two new collision events.

1. Landing on the Pad (Success!)

When colliding with obj_pad
Stop movement
Show Message "Landing successful!"

Find Collision in Events, type obj_pad. Add Stop Movement (from Movement) to halt the lander. Then add Show Message (from Game) with your victory text.

2. Crashing into the Ground (Failure!)

When colliding with obj_ground
Show Message "Crashed!"
Restart Room

Find Collision in Events, type obj_ground. Add Show Message (from Game) with crash text, then Restart Room (from Room). Hitting the rocky ground resets the level!

Apply Your Blocks

  1. You should now have 7 event blocks total (Create + 3 keyboard + No key + 2 collisions)
  2. Click Apply to Events to save

Update the Room

Open room_game and update the terrain:

  1. Make the ground uneven — stack some obj_ground blocks higher in places to create hills
  2. Remove one or two ground blocks from the bottom row to create a gap
  3. Place obj_pad in the gap (the landing zone!)

Example Level Layout

        L                          <- Lander starts here



GG       GG
GGG     GGGG    PP     GGGGG     GGG
GGGGGGGGGGGG    PP     GGGGGGGGGGGGG

G = Ground (crash!)    L = Lander    P = Landing Pad (safe!)
Tip: Make the landing area small enough to be a challenge, but not impossibly tiny! One or two pad blocks side by side works well.

Test Your Game!

Press F5 to play!

  • Fly toward the green landing pad
  • Land on the pad → "Landing successful!" message appears
  • Crash into the ground → "Crashed!" message, then room restarts
  • Use short bursts of thrust (tap Up) to control your descent
