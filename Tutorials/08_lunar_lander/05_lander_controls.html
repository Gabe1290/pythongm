Step 4: Lander Controls (Blockly)

Add thrust and horizontal movement using keyboard blocks.

Thrust (Up Arrow)

Still in the Blockly editor for obj_lander:

Keyboard: Up (held)
Set vertical speed to -2

Find Keyboard (held) in Events, select Up. Then snap Set Vertical Speed from Movement with value -2 (negative = upward thrust).

Steer Left

Keyboard: Left (held)
Set horizontal speed to -2

Steer Right

Keyboard: Right (held)
Set horizontal speed to 2

Stop Drifting When No Keys

Keyboard: No key
Set horizontal speed to 0

This stops horizontal drift when you release the keys. Note: we only reset horizontal speed — vertical speed is controlled by gravity!

Apply Your Blocks

  1. You should have 5 event blocks total (Create + 3 keyboard held + No key)
  2. Click the Apply to Events button to save
How it works: Gravity constantly pulls the lander down (0.05 per frame). Holding Up fires the thrust, pushing the lander upward. Left/Right let you steer. The challenge is to descend slowly enough to land safely!
