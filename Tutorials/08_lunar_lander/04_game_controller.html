Phase 3: Game Controller

Let's add a game controller object to display a title, instructions, and initialize the score.

Create the Controller Object

  1. Right-click Objects > Create New Object
  2. Name it obj_game_controller
  3. No sprite needed — this is an invisible controller

Program the Controller (Blockly)

Double-click obj_game_controller and open the Blockly tab.

1. Initialize Score (Create Event)

Create
Set Score 0

Find Create in Events. Add Set Score (from Score) with value 0. This resets the score each time the room starts.

2. Draw HUD Text (Draw Event)

Draw
Draw Text "Lunar Lander" at 10, 10
Draw Text "Land on the green pad!" at 10, 30

Find Draw in Events. Add two Draw Text blocks (from Drawing) — one for the game title at position (10, 10) and one for the instructions at position (10, 30).

Apply Your Blocks

  1. You should have 2 event blocks (Create + Draw)
  2. Click Apply to Events to save

Place the Controller in the Room

  1. Open room_game
  2. Place one instance of obj_game_controller anywhere in the room (it has no sprite, so its position doesn't matter)

Test Your Game!

Press F5 to play!

  • The title "Lunar Lander" appears at the top-left
  • Instructions "Land on the green pad!" are shown below
  • Fly your lander, land on the pad to win, crash to restart
  • The full game is now complete!

🚀 Congratulations! You've built a Lunar Lander game! 🌕

Your game has lunar gravity, thrust controls, a landing pad, crash detection, and a HUD display. Well done!

Challenge Ideas

Want to make your game even better? Try these improvements:

  • Fuel system: Add a variable for fuel that decreases when thrusting. If fuel reaches 0, the thrust stops working!
  • Speed-based crash detection: Use the lander's vspeed to check landing speed. If the lander is moving too fast when it touches the pad, it crashes instead of landing safely!
  • Smaller landing pad: Make the pad smaller (fewer blocks) to increase the difficulty
  • Multiple levels: Create additional rooms with different terrain layouts and harder landing spots
  • Score based on fuel remaining: Award more points for landing with more fuel left — encourages efficient flying!
