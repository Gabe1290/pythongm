Phase 1: Paddles and Bouncing Ball

Let's create everything we need for a playable Pong game!

Create the Sprites

Create these 3 sprites (draw them or import images):

SpriteSizeDescription
spr_ball16×16Small white circle
spr_paddle16×64Tall narrow rectangle
spr_wall32×32Gray block

Create the Objects

  1. Create obj_wall — sprite: spr_wall, check Solid
  2. Create obj_paddle_left — sprite: spr_paddle, check Solid
  3. Create obj_paddle_right — sprite: spr_paddle, check Solid
  4. Create obj_ball — sprite: spr_ball, leave Solid unchecked

Program the Left Paddle (Player 1: W/S)

Double-click obj_paddle_left → Blockly tab.

Keyboard: W (held)
Set vertical speed to -8
Keyboard: S (held)
Set vertical speed to 8
Keyboard: No key
Stop movement
When colliding with obj_wall
Stop movement

Click Apply to Events.

Program the Right Paddle (Player 2: Arrows)

Double-click obj_paddle_right → Blockly tab. Same structure but different keys:

Keyboard: Up (held)
Set vertical speed to -8
Keyboard: Down (held)
Set vertical speed to 8
Keyboard: No key
Stop movement
When colliding with obj_wall
Stop movement

Click Apply to Events.

Program the Ball

Double-click obj_ball → Blockly tab.

Create Event
Start moving in direction 45 at speed 6
When colliding with obj_wall
Bounce against solid objects
When colliding with obj_paddle_left
Bounce against solid objects
When colliding with obj_paddle_right
Bounce against solid objects

Click Apply to Events.

Create the Room

  1. Create a new room named room_pong (640×480)
  2. Place obj_wall along the top and bottom edges
  3. Place obj_paddle_left near the left edge, centered vertically
  4. Place obj_paddle_right near the right edge, centered vertically
  5. Place obj_ball in the center of the room

Test Your Game!

Press F5 to play!

  • Player 1: W (up) and S (down)
  • Player 2: Up Arrow and Down Arrow
  • The ball bounces off walls and paddles
  • The ball flies off the sides — we'll add goals next!
