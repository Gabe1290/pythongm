Phase 2: Goals and Scoring

The ball flies off the sides, but nobody scores! Let's add invisible goals behind each paddle.

Create the Goal Sprite

  1. Create a new sprite named spr_goal
  2. Draw a 32×32 rectangle in any color (it will be invisible)

Create the Goal Objects

  1. Create obj_goal_left — sprite: spr_goal, uncheck Visible, check Solid
  2. Create obj_goal_right — sprite: spr_goal, uncheck Visible, check Solid

Add Scoring to the Ball

Double-click obj_ball → Blockly tab. Add two new collision events:

When colliding with obj_goal_left
Set variable global.p2score relative +1
Jump to start position

When the ball reaches the left goal, Player 2 scores and the ball resets.

When colliding with obj_goal_right
Set variable global.p1score relative +1
Jump to start position

When the ball reaches the right goal, Player 1 scores.

Click Apply to Events.

What is global.? The global. prefix means the variable is shared across all objects. Any object can read or modify global.p1score.

Place the Goals in the Room

  1. Open room_pong
  2. Stack obj_goal_left along the left edge (behind the left paddle)
  3. Stack obj_goal_right along the right edge (behind the right paddle)

The goals are invisible, so they won't show during gameplay.

Test Your Game!

Press F5 to play!

  • When the ball passes a paddle, it resets to the center
  • The score is tracked internally (we'll display it next!)
