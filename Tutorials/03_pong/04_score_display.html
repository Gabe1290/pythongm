Phase 3: Score Display

Let's add a game controller to initialize and display the scores on screen.

Create the Score Controller

  1. Create a new object named obj_score
  2. No sprite needed

Initialize Scores

Double-click obj_score → Blockly tab.

Create Event
Set variable global.p1score to 0
Set variable global.p2score to 0

Draw Scores on Screen

Draw Event
Draw text "Player 1:" at x: 10 y: 10
Draw variable global.p1score at x: 100 y: 10
Draw text "Player 2:" at x: 10 y: 30
Draw variable global.p2score at x: 100 y: 30

Click Apply to Events.

Place in the Room

Open room_pong and place obj_score anywhere (it has no sprite, just draws text).

Congratulations! Test Your Final Game!

Press F5 to play!

  • Player 1: W (up) and S (down)
  • Player 2: Up Arrow and Down Arrow
  • Scores are displayed in the top-left corner
  • First to miss the ball gives the other player a point!
Challenge Ideas:
  • Make the ball speed up after each rally
  • End the game when a player reaches 10 points
  • Add a center line for visual flair
  • Make the ball angle change based on where it hits the paddle
