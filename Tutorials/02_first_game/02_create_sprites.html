Step 1: Create the Sprites

Sprites are the images that represent objects in your game. We need two sprites: one for the player and one for the star.

Create the Player Sprite

1 Create a new project

Go to File > New Project and name it "CatchTheStar".

2 Create the player sprite

Right-click on Sprites in the asset tree and select Create New Sprite.

Name it spr_player.

3 Draw or import a player image

Double-click spr_player in the asset tree to open the Sprite Editor.

Draw a simple 32x32 character (a spaceship, person, or basket) and press Ctrl+S to save.

Alternatively, right-click spr_player and select Import Image... to load an existing image file.

Create the Star Sprite

4 Create the star sprite

Right-click on Sprites again and select Create New Sprite.

Name it spr_star.

5 Draw or import a star image

Double-click spr_star to open the Sprite Editor and draw a star, coin, or collectible item.

A 32x32 pixel image works well. Press Ctrl+S to save.

Tip: Use simple, clear images. Stars or coins are easy to recognize and make the game feel rewarding!
Sprites created! Click "Next" to create the game objects.
