Phase 1: A Moving Player

Let's create a player, put it in a room, and make it move right away!

Create a New Project

Go to File > New Project and name it CatchTheStar.

Create the Player Sprite

  1. Right-click on Sprites in the Asset Tree → Create New Sprite
  2. Name it spr_player
  3. Double-click it to open the Sprite Editor and draw a simple 32×32 character (a spaceship, person, or basket). Press Ctrl+S to save.

Create the Player Object

  1. Right-click on Objects → Create New Object
  2. Name it obj_player
  3. Set the sprite to spr_player

Create the Room

  1. Right-click on Rooms → Create New Room
  2. Name it room_game
  3. Open the room and place obj_player near the bottom center

Add Player Movement

  1. Double-click obj_player to open the Object Editor
  2. Click the Blockly tab

Move Left

Keyboard: Left (held)
Set horizontal speed to -5

Move Right

Keyboard: Right (held)
Set horizontal speed to 5

Stop When No Key Pressed

Keyboard: No key
Stop movement

You should have 3 event blocks. Click Apply to Events to save.

Test Your Game!

Press F5 to play!

  • Use Left and Right arrow keys to move the player
  • The player can go off screen — we'll fix that later!
