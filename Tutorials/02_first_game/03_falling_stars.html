Phase 2: Falling Stars

Now let's make stars appear at the top of the screen and fall down!

Create the Star Sprite

  1. Create a new sprite named spr_star
  2. Draw a star or coin (32×32 pixels) and save

Create the Star Object

  1. Create a new object named obj_star
  2. Set the sprite to spr_star

Make Stars Fall

  1. Double-click obj_star → Blockly tab
  2. Add this block:
Create Event
Set vertical speed to 3

Each star falls downward at speed 3 as soon as it appears.

Click Apply to Events to save.

Create a Spawner

We need an invisible object that creates stars at random positions.

  1. Create a new object named obj_spawner
  2. No sprite needed (it will be invisible)

Program the Spawner

  1. Double-click obj_spawner → Blockly tab
  2. Add these blocks:
Create Event
Set Alarm 0 to 60
Alarm 0
Create instance of obj_star at x: random y: 0
Set Alarm 0 to 60

The Create event starts a timer. When the alarm fires (every 2 seconds), it creates a star at a random x position at the top of the screen, then restarts the timer.

Click Apply to Events to save.

Place the Spawner in the Room

  1. Open room_game
  2. Place obj_spawner anywhere (it's invisible, location doesn't matter)

Test Your Game!

Press F5 to play!

  • Stars should fall from random positions at the top
  • Move the player to dodge or reach them
  • Stars pass through the player for now — we'll add catching next!
