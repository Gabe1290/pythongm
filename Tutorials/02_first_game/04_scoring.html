Phase 3: Catching and Scoring

Let's make the player catch stars, earn points, and display the score!

Catch Stars on Collision

  1. Double-click obj_player → Blockly tab
  2. Add this new event block (keep your existing movement blocks):
When colliding with obj_star
Add to score 10
Destroy other instance

When the player touches a star:

  • Add to score 10 (from Score/Lives/Health) — earn 10 points
  • Destroy other instance (from Instance) — the star disappears

Click Apply to Events to save.

Display the Score

We need a game controller to show the score on screen.

  1. Create a new object named obj_game_controller
  2. No sprite needed
  1. Double-click obj_game_controller → Blockly tab
  2. Add these blocks:
Create Event
Set score to 0
Draw Event
Draw score at x: 10 y: 10

Click Apply to Events.

Clean Up Missed Stars

  1. Double-click obj_star → Blockly tab
  2. Add this event (keep the existing Create event):
Outside Room
Destroy this instance

Stars that fall off the bottom of the screen are destroyed so they don't pile up in memory.

Click Apply to Events.

Place the Game Controller

Open room_game and place obj_game_controller anywhere in the room.

Test Your Game!

Press F5 to play!

  • Catch falling stars to earn points
  • The score displays in the top-left corner
  • Stars you miss fall off the screen and disappear
