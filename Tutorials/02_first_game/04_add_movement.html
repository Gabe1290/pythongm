Step 3: Add Player Movement

Now we'll make the player move left and right using the arrow keys.

Open the Player Object

1 Edit obj_player

Double-click on obj_player in the asset tree to open the Object Editor.

Add Keyboard Events (Blockly Mode)

Using Blockly: Make sure you're in Blockly mode (check the toggle at the top of the editor).
2 Add Left Arrow event

Click Add Event and select Keyboard > Left.

This event triggers when the left arrow key is pressed.

3 Add movement action

From the Blockly toolbox, drag a "move in direction" block.

Set the direction to 180 (left) and speed to 5.

4 Add Right Arrow event

Click Add Event and select Keyboard > Right.

5 Add movement action

Drag another "move in direction" block.

Set the direction to 0 (right) and speed to 5.

Test Your Game!

6 Run the game

Press F5 to test your game.

Use the left and right arrow keys to move the player!

Problem? If the player moves off screen, we'll fix that later by adding boundary checks.
