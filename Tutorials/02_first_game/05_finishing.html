Phase 4: Finishing Touches

Let's add some polish to make your game feel complete!

Keep Player on Screen

Open obj_player in the Blockly editor and add a Step event with boundary checks:

  • If x < 0, set x = 0
  • If x > room_width − sprite_width, set x = room_width − sprite_width

This prevents the player from leaving the screen.

Click Apply to Events.

Set a Background Color

Open room_game in the Room Editor and set the background color to a nice sky blue or dark space color.

Congratulations!

Your first game is complete!

Press F5 to play the finished game. Catch as many stars as you can!

Challenge Ideas

Want to go further? Try these:
  • Add lives — lose a life when a star reaches the bottom
  • Make stars fall faster as the score increases
  • Add different colored stars worth different points
  • Add a "bad" object (bomb) that ends the game if caught
  • Add sound effects when catching stars
