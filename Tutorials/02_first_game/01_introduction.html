Catch the Star!

Move your player to catch falling stars and score points!

What We'll Create

In this tutorial, you'll build a complete game from scratch:

  • A player that moves left and right with the arrow keys
  • Stars that fall from the top of the screen
  • A scoring system that tracks how many stars you catch
  • A game over when you miss too many stars

What You'll Learn

  • Creating and importing sprites
  • Creating objects with behaviors
  • Using keyboard events for player control
  • Collision detection between objects
  • Using variables for score tracking
  • Displaying text on screen
Time Required: About 15-20 minutes
