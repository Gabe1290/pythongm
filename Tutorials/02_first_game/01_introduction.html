Catch the Star!

Move your player to catch falling stars and score points!

What We'll Create

In this tutorial, you'll build a complete game step by step. You'll have something playable at each phase!

How This Tutorial Works

Phase 1 — A Moving Player
Create a player that moves left and right. Test it right away!
Phase 2 — Falling Stars
Add stars that fall from the sky. Dodge them or try to reach them!
Phase 3 — Catching and Scoring
Catch stars to earn points and display the score on screen.
Phase 4 — Finishing Touches
Keep the player on screen and add challenges to go further.

What You'll Learn

  • Creating sprites and objects
  • Using keyboard events for player control
  • Creating objects automatically with a spawner
  • Collision detection between objects
  • Score tracking and display
Time Required: About 15–20 minutes
