Step 6: Finishing Touches

Let's add some polish to make your game feel complete!

Keep Player on Screen

1 Add boundary check

In obj_player, add a Step event with these conditions:

  • If x < 0, set x = 0
  • If x > room_width - sprite_width, set x = room_width - sprite_width

This prevents the player from moving off the sides of the screen.

Add a Background Color

2 Set room background

Open room_game in the Room Editor.

In the room properties, set the background color to a nice sky blue or dark space color.

Export Your Game!

3 Export as HTML5

Go to File > Export as HTML5.

Choose a location and click Export.

Open the HTML file in a web browser to play your game!

Congratulations! You've created your first complete game in PyGameMaker!

Ideas to Improve Your Game

Challenges to try:
  • Add lives - lose a life when a star reaches the bottom
  • Make stars fall faster as the score increases
  • Add different colored stars worth different points
  • Add a "bad" object (bomb) that ends the game if caught
  • Add sound effects when catching stars
  • Create a title screen and game over screen
