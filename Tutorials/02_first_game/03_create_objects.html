Step 2: Create the Objects

Objects are the "actors" in your game. They use sprites for their appearance and have behaviors defined by events.

Create the Player Object

1 Create a new object

Right-click on Objects in the asset tree and select Create Object.

Name it obj_player.

2 Assign the sprite

In the Object Editor, find the Sprite dropdown.

Select spr_player from the list.

Create the Star Object

3 Create another object

Right-click on Objects and select Create Object.

Name it obj_star.

4 Assign the star sprite

Select spr_star as the sprite for this object.

Create the Game Room

5 Create a room

Right-click on Rooms and select Create Room.

Name it room_game.

6 Place the player

Double-click the room to open the Room Editor.

Select obj_player from the object list on the left.

Click near the bottom center of the room to place the player there.

Don't place stars yet! We'll make them spawn automatically from the top of the screen.
Objects created! Click "Next" to add player movement.
