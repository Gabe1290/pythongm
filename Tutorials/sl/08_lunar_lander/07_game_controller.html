Korak 6: Krmilnik igre (prikazovalnik)

Prikaži gorivo, hitrost in stanje za pomoč igralcu.

Dogodek Draw

V obj_game_controller:

  1. Dodaj dogodek: Draw > Draw
  2. Dodaj akcijo: Control > Execute Code
draw_set_color(c_white);

// Pridobi informacije o pristajalniku
var lander = instance_find(obj_lander, 0);
if (lander != noone) {
    var spd = sqrt(lander.hsp*lander.hsp + lander.vsp*lander.vsp);

    // Nariši prikazovalnik
    draw_text(10, 10, "LUNIN PRISTAJALNIK");
    draw_text(10, 30, "Gorivo: " + string(floor(lander.fuel)) + "%");
    draw_text(10, 50, "Hitrost: " + string_format(spd, 1, 2));
    draw_text(10, 70, "Varno: < " + string(lander.safe_speed));

    // Opozorilo o hitrosti
    if (spd > lander.safe_speed) {
        draw_set_color(c_red);
        draw_text(10, 90, "PREHITRO!");
    } else {
        draw_set_color(c_lime);
        draw_text(10, 90, "Hitrost OK");
    }

    // Opozorilo o gorivu
    if (lander.fuel <= 0) {
        draw_set_color(c_red);
        draw_text(10, 110, "NI GORIVA!");
    }
}

draw_set_color(c_gray);
draw_text(10, room_height - 20, "GOR: Potisk | LEVO/DESNO: Premik");

Kaj prikazovalnik prikazuje

  • Gorivo: Odstotek preostalega goriva (zelena = OK)
  • Hitrost: Trenutna hitrost (rdeča = nevarno)
  • Opozorila: "PREHITRO!" ali "NI GORIVA!"
  • Nadzor: Opomnik na dnu
Vizualna povratna informacija: Barvno kodiranje pomaga igralcem hitro reagirati!
Klikni "Naprej" za nastavitev sobe.
