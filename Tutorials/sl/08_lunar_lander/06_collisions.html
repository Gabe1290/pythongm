Korak 5: Zaznavanje trkov

Obravnavaj pristajanje na ploščadi in strmoglavljenje v teren.

Trk s pristajalno ploščadjo

V obj_lander:

  1. Dodaj dogodek: Collision > obj_pad
  2. Dodaj akcijo: Control > Execute Code
// Izračunaj skupno hitrost
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    // Varen pristanek!
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("Odličen pristanek! Zmagali ste!");
} else {
    // Prehitro - strmoglavljenje!
    crashed = true;
    show_message("Strmoglavljenje! Prehitro!");
    room_restart();
}

Trk s tlemi (strmoglavljenje)

V obj_lander:

  1. Dodaj dogodek: Collision > obj_ground
  2. Dodaj akcijo: Control > Execute Code
crashed = true;
show_message("Strmoglavljenje v teren!");
room_restart();

Izračun hitrosti

Uporabimo Pitagorov izrek za izračun skupne hitrosti:

total_speed = sqrt(hsp*hsp + vsp*vsp)

To združi vodoravno in navpično hitrost v eno vrednost.

Varna hitrost je 2! Če je total_speed > 2, se strmoglaviš tudi na ploščadi.
Klikni "Naprej" za dodajanje prikazovalnika.
