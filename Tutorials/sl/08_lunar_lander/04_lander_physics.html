Korak 3: Fizika pristajalnika (dogodek Create)

Nastavi spremenljivke za gravitacijo, potisk, gorivo in hitrost.

Inicializacija fizikalnih spremenljivk

V obj_lander:

  1. Dodaj dogodek: Create
  2. Dodaj akcijo: Control > Execute Code
// Fizikalne nastavitve
gravity_force = 0.05;  // Nežna gravitacija
thrust_force = 0.1;    // Moč potiska
max_speed = 5;         // Največja hitrost

// Trenutna hitrost
hsp = 0;  // Vodoravna hitrost
vsp = 0;  // Navpična hitrost

// Sistem goriva
fuel = 100;
fuel_use = 0.5;  // Gorivo na sličico

// Stanje igre
landed = false;
crashed = false;

// Varna hitrost pristanka
safe_speed = 2;

Razumevanje spremenljivk

Spremenljivka Namen
gravity_force Stalna sila navzdol (0.05 je nežno)
thrust_force Sila navzgor ob pritisku tipke GOR
fuel 100% goriva, zmanjšuje se ob potisku
safe_speed Največja hitrost za varen pristanek (2)
Ravnovesje: Potisk (0.1) je močnejši od gravitacije (0.05), zato se lahko upočasniš!
Klikni "Naprej" za dodajanje premikanja in nadzora.
