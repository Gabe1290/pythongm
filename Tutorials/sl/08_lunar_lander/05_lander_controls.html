Korak 4: Nadzor pristajalnika (Dogodek Step)

Dodaj Gravitacijo, Potisk, vodoravno Premikanje in preverjanje meja.

Dodaj Dogodek Step

V obj_lander:

  1. Dodaj Dogodek: Step > Step
  2. Dodaj Akcijo: Control > Execute Code
// Ne posodabljaj, če je igre konec
if (landed || crashed) exit;

// === GRAVITACIJA ===
vsp += gravity_force;

// === POTISK (puščica gor) ===
if (keyboard_check(vk_up) && fuel > 0) {
    vsp -= thrust_force;
    fuel -= fuel_use;
    if (fuel < 0) fuel = 0;
}

// === VODORAVNI NADZOR ===
if (keyboard_check(vk_left)) hsp -= 0.05;
if (keyboard_check(vk_right)) hsp += 0.05;

// === OMEJITEV HITROSTI ===
hsp = clamp(hsp, -max_speed, max_speed);
vsp = clamp(vsp, -max_speed, max_speed);

// === UPORABI PREMIK ===
x += hsp;
y += vsp;

// === MEJE ZASLONA ===
if (x < 16) { x = 16; hsp = 0; }
if (x > room_width - 16) { x = room_width - 16; hsp = 0; }
if (y < 16) { y = 16; vsp = 0; }

Kako deluje

  1. Gravitacija: Vsako sličico doda k vsp (padanje)
  2. Potisk: Odšteje od vsp (dviganje), porabi Gorivo
  3. Clamp: Omeji hitrost na varno območje
  4. Meje: Obdrži pristajalnik na zaslonu
Ključno: Funkcija clamp() omeji vrednost med minimum in maksimum!
Klikni "Naprej" za dodajanje zaznavanja Trkov.
