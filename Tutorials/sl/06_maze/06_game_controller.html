Korak 5: Krmilnik igre

Ustvarite krmilnik za sledenje časa in prikaz rezultata.

Ustvarite Objekt krmilnika

  1. Ustvarite Objekt obj_game_controller
  2. Sprite ni potreben (pustite prazno)

Inicializacija časovnika (Dogodek Create)

  1. Dodajte Dogodek: Create
  2. Dodajte Akcijo: Control > Set Variable
  3. Spremenljivka: global.timer
  4. Vrednost: 0

Posodabljanje časovnika (Dogodek Step)

  1. Dodajte Dogodek: Step
  2. Dodajte Akcijo: Control > Set Variable
  3. Spremenljivka: global.timer
  4. Vrednost: 1/room_speed
  5. Označite "Relative"

Prikaz informacij (Dogodek Draw)

  1. Dodajte Dogodek: Draw
  2. Dodajte Akcijo: Control > Execute Code
draw_set_color(c_white);
draw_text(10, 10, "Rezultat: " + string(score));
draw_text(10, 30, "Čas: " + string(floor(global.timer)) + "s");

var coins = instance_number(obj_coin);
draw_text(10, 50, "Kovanci: " + string(coins));
Opomba: Krmilnik je neviden, a bistven za sledenje stanja igre!
Kliknite "Naprej" za nastavitev vaše Sobe.
