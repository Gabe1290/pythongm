Step 6: Room Setup

Now create your maze level!

Create the Room

  1. Right-click Rooms > Create Room
  2. Name it room_maze
  3. Set size to 640x480
  4. Enable Snap to Grid: 32x32

Build Your Maze

  1. Create a border of walls around the edges
  2. Add internal walls to create corridors
  3. Place coins along the paths
  4. Place the exit at the end of the maze
  5. Place the player near the entrance
  6. Add the game controller (anywhere)

Congratulations! You've built a maze game!

Press F5 to play:
  • Use arrow keys to move
  • Collect coins for points
  • Find the exit to win!
