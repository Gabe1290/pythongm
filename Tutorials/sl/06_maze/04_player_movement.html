Korak 3: Premikanje Igralca

Igralec potrebuje gladko Premikanje z zaznavanjem Trkov.

Inicializacija hitrosti (Dogodek Create)

V obj_player:

  1. Dodajte Dogodek: Create
  2. Dodajte Akcijo: Control > Set Variable
  3. Spremenljivka: move_speed
  4. Vrednost: 4

Premikanje s Trki (Dogodek Step)

  1. Dodajte Dogodek: Step
  2. Dodajte Akcijo: Control > Execute Code
// Vodoravno Premikanje
var hspd = 0;
if (keyboard_check(vk_right)) hspd = move_speed;
if (keyboard_check(vk_left)) hspd = -move_speed;

// Navpično Premikanje
var vspd = 0;
if (keyboard_check(vk_down)) vspd = move_speed;
if (keyboard_check(vk_up)) vspd = -move_speed;

// Preveri vodoravni Trk
if (!place_meeting(x + hspd, y, obj_wall)) {
    x += hspd;
}

// Preveri navpični Trk
if (!place_meeting(x, y + vspd, obj_wall)) {
    y += vspd;
}
Kako deluje: Preverimo, ali bi ciljna pozicija trčila s Steno. Če ne, se premaknemo tja!
Kliknite "Naprej" za dodajanje zbiranja Kovancev.
