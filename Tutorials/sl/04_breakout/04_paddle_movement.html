Korak 3: Dodaj premikanje loparja

Lopar se premika levo in desno s puščičnimi tipkami.

Premik levo (leva puščica)

  1. Odpri obj_paddle
  2. Dodaj dogodek: Keyboard > Press Left
  3. Dodaj akcijo: Move > Set Horizontal Speed
  4. Nastavi hitrost na -8 (negativno = levo)

Ustavi premikanje (spusti levo)

  1. Dodaj dogodek: Keyboard > Release Left
  2. Dodaj akcijo: Move > Set Horizontal Speed
  3. Nastavi hitrost na 0

Premik desno (desna puščica)

  1. Dodaj dogodek: Keyboard > Press Right
  2. Dodaj akcijo: Move > Set Horizontal Speed
  3. Nastavi hitrost na 8 (pozitivno = desno)

Ustavi premikanje (spusti desno)

  1. Dodaj dogodek: Keyboard > Release Right
  2. Dodaj akcijo: Move > Set Horizontal Speed
  3. Nastavi hitrost na 0

Trk s steno

  1. Dodaj dogodek: Collision > obj_wall
  2. Dodaj akcijo: Move > Bounce Against Objects
  3. Izberi "Against solid objects"
Opomba: Trk s steno preprečuje, da bi lopar zapustil zaslon!
Klikni "Naprej" za dodajanje premikanja žoge.
