Step 5: Add the Lives System

The player loses a life when the ball falls off the bottom of the screen.

Create a Death Zone

  1. Create a sprite spr_death_zone (32x32, any color)
  2. Create object obj_death_zone
  3. Uncheck "Visible" - it should be invisible
  4. Check "Solid"

Ball Falls into Death Zone

In obj_ball:

  1. Add Event: Collision > obj_death_zone
  2. Add Action: Score > Subtract from Lives
  3. Value: 1
  4. Add Action: Move > Jump to Start Position

Create a Game Controller

  1. Create object obj_game_controller
  2. No sprite needed

Initialize Lives (Create Event)

In obj_game_controller:

  1. Add Event: Create
  2. Add Action: Score > Set Lives
  3. Value: 3
  4. Add Action: Score > Set Score
  5. Value: 0

Check for Game Over (Step Event)

  1. Add Event: Step
  2. Add Action: Control > Test Lives
  3. Check if lives are less than or equal to 0
  4. Add Action: Main2 > Show Message
  5. Message: Game Over!
  6. Add Action: Main2 > Restart Room
Remember: Place obj_game_controller in your room!
