Korak 5: Dodaj sistem Življenj

Igralec izgubi Življenje, ko žoga pade z dna zaslona.

Ustvari smrtno cono

  1. Ustvari Sprite spr_death_zone (32x32, poljubna barva)
  2. Ustvari Objekt obj_death_zone
  3. Odznači "Visible" - mora biti neviden
  4. Označi "Solid"

Žoga pade v smrtno cono

V obj_ball:

  1. Dodaj Dogodek: Trk > obj_death_zone
  2. Dodaj Akcijo: Score > Subtract from Lives
  3. Vrednost: 1
  4. Dodaj Akcijo: Move > Jump to Start Position

Ustvari krmilnik igre

  1. Ustvari Objekt obj_game_controller
  2. Sprite ni potreben

Inicializiraj Življenja (Dogodek Create)

V obj_game_controller:

  1. Dodaj Dogodek: Create
  2. Dodaj Akcijo: Score > Set Lives
  3. Vrednost: 3
  4. Dodaj Akcijo: Score > Set Score
  5. Vrednost: 0

Preveri konec igre (Dogodek Step)

  1. Dodaj Dogodek: Step
  2. Dodaj Akcijo: Control > Test Lives
  3. Preveri, ali so Življenja manjša ali enaka 0
  4. Dodaj Akcijo: Main2 > Show Message
  5. Sporočilo: Konec igre!
  6. Dodaj Akcijo: Main2 > Restart Room
Zapomni si: Postavi obj_game_controller v svojo Sobo!
Klikni "Naprej" za dodajanje prikaza rezultata.
