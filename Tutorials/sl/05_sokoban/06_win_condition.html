Korak 5: Pogoj za zmago

Igralec zmaga, ko so vsi Zaboji na Ciljnih mestih.

Ustvari krmilnik igre

  1. Ustvarite Objekt obj_game_controller
  2. Sprite ni potreben

Preštej Cilje (Dogodek Create)

V obj_game_controller:

  1. Dodajte Dogodek: Create
  2. Dodajte Akcijo: Control > Execute Code
  3. Vnesite: global.total_targets = instance_number(obj_target);

Preveri pogoj za zmago (Dogodek Step)

  1. Dodajte Dogodek: Step
  2. Dodajte Akcijo: Control > Execute Code
  3. Vnesite to kodo:
// Preštej Zaboje na Ciljih
var crates_on_targets = 0;
with (obj_crate) {
    if (place_meeting(x, y, obj_target)) {
        crates_on_targets += 1;
    }
}

// Preveri, ali imajo vsi Cilji Zaboje
if (crates_on_targets >= global.total_targets
    && global.total_targets > 0) {
    show_message("Nivo končan!");
    room_restart();
}

Izriši navodila (Dogodek Draw)

  1. Dodajte Dogodek: Draw
  2. Dodajte Akcijo: Draw > Draw Text
  3. Besedilo: Potisni Zaboje na Cilje!
  4. X: 10, Y: 10
Namig: Poskrbite, da je število Zabojev enako številu Ciljev v vašem nivoju!
Kliknite "Naprej" za nastavitev Sobe igre.
