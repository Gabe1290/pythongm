Step 1: Create the Sprites

All sprites should be the same size (32x32 pixels) to create a proper grid.

Create the Player Sprite

  1. Right-click on Sprites in the Resource Tree
  2. Select Create Sprite
  3. Name it spr_player
  4. Click Edit Sprite
  5. Draw a simple character (32x32 pixels)
  6. Use a distinct color like blue
  7. Click OK to save

Create the Crate Sprites

Normal Crate:

  1. Create sprite spr_crate
  2. Draw a wooden box shape (32x32 pixels)
  3. Use brown or orange color

Crate on Target:

  1. Create sprite spr_crate_ok
  2. Draw the same crate but with green color
  3. This shows when a crate is correctly placed

Create the Wall Sprite

  1. Create sprite spr_wall
  2. Draw a solid brick or stone pattern (32x32 pixels)
  3. Use gray or dark color

Create the Target Sprite

  1. Create sprite spr_target
  2. Draw an X mark or goal indicator (32x32 pixels)
  3. Use a bright color like red
Important: Keep ALL sprites exactly 32x32 pixels for proper grid alignment!
Click "Next" to create the game objects.
