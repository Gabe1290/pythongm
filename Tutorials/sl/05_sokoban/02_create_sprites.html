Korak 1: Ustvari sprite

Vsi spriti morajo biti enake velikosti (32x32 slikovnih pik) za pravilno mrežo.

Ustvari sprite igralca

  1. Z desnim klikom kliknite na Sprites v drevesu virov
  2. Izberite Create Sprite
  3. Poimenujte ga spr_player
  4. Kliknite Edit Sprite
  5. Narišite preprost lik (32x32 slikovnih pik)
  6. Uporabite izrazito barvo, kot je modra
  7. Kliknite OK za shranitev

Ustvari sprite zabojev

Navaden zaboj:

  1. Ustvarite sprite spr_crate
  2. Narišite obliko lesenega zaboja (32x32 slikovnih pik)
  3. Uporabite rjavo ali oranžno barvo

Zaboj na cilju:

  1. Ustvarite sprite spr_crate_ok
  2. Narišite enak zaboj, vendar z zeleno barvo
  3. To prikazuje, kdaj je zaboj pravilno postavljen

Ustvari sprite stene

  1. Ustvarite sprite spr_wall
  2. Narišite vzorec opeke ali kamna (32x32 slikovnih pik)
  3. Uporabite sivo ali temno barvo

Ustvari sprite cilja

  1. Ustvarite sprite spr_target
  2. Narišite oznako X ali kazalnik cilja (32x32 slikovnih pik)
  3. Uporabite svetlo barvo, kot je rdeča
Pomembno: Ohranite VSE sprite natančno 32x32 slikovnih pik za pravilno poravnavo mreže!
Kliknite "Naprej" za ustvarjanje objektov igre.
