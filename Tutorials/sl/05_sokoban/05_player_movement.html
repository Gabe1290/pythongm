Korak 4: Premikanje igralca

Igralec se premika v fiksnih korakih po 32 slikovnih pik (premikanje po mreži).

Pomembno: V igri Sokoban mora premikanje preveriti:
1. Ali je tam stena? (Ne more se premakniti)
2. Ali je tam zaboj? (Ali ga lahko potisnemo?)
3. Ali je za zabojem stena/zaboj? (Ne moremo potisniti)

Premik desno (puščica desno)

V obj_player:

  1. Dodajte dogodek: Keyboard > Press Right
  2. Dodajte akcijo: Control > Execute Code
  3. Vnesite to kodo:
// Preveri, ali se lahko premaknemo desno
if (!place_meeting(x + 32, y, obj_wall)) {
    var crate = instance_place(x + 32, y, obj_crate);
    if (crate != noone) {
        // Tam je zaboj - ali ga lahko potisnemo?
        if (!place_meeting(x + 64, y, obj_wall) &&
            !place_meeting(x + 64, y, obj_crate)) {
            crate.x += 32;
            x += 32;
        }
    } else {
        // Ni zaboja, samo se premakni
        x += 32;
    }
}

Premik levo (puščica levo)

Enaka koda, vendar spremenite:

  • + 32 postane - 32
  • + 64 postane - 64

Premik gor (puščica gor)

Enaka koda, vendar uporabite y namesto x:

  • x + 32 postane y - 32
  • x + 64 postane y - 64
  • crate.x postane crate.y

Premik dol (puščica dol)

Enako kot gor, vendar z + 32 in + 64.

Kliknite "Naprej" za dodajanje zaznavanja pogoja za zmago.
