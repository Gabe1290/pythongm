Step 3: Crate Behavior

The crate needs to change appearance when it's on a target spot.

Check Target Position (Step Event)

In obj_crate:

  1. Add Event: Step
  2. Add Action: Control > Test Instance Position
  3. Object: obj_target
  4. X: 0, Y: 0 (relative)
  5. Check if collision exists

Change Sprite When on Target

If collision exists (crate is on target):

  1. Add Action: Main1 > Change Sprite
  2. Sprite: spr_crate_ok

Change Sprite When Not on Target

  1. Add Action: Control > Else
  2. Add Action: Main1 > Change Sprite
  3. Sprite: spr_crate
Visual Feedback: When the crate turns green, players know it's correctly placed!
