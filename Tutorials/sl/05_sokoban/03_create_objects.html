Step 2: Create the Objects

We need objects for the player, walls, crates, and targets.

Create the Wall Object

  1. Right-click on Objects and select Create Object
  2. Name it obj_wall
  3. Set the sprite to spr_wall
  4. Check "Solid"
  5. No events needed

Create the Target Object

  1. Create object obj_target
  2. Set sprite to spr_target
  3. Leave "Solid" unchecked (player and crates can be on top)
  4. No events needed - it's just a marker

Create the Crate Object

  1. Create object obj_crate
  2. Set sprite to spr_crate
  3. Check "Solid"

Create the Player Object

  1. Create object obj_player
  2. Set sprite to spr_player
  3. Leave "Solid" unchecked
Note: We'll add events to obj_crate and obj_player in the next steps.
