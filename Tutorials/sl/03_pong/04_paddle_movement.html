Step 3: Add Paddle Movement

We'll set up two-player controls using different keys for each paddle.

Control Scheme

Player Move Up Move Down
Player 1 (Left) W key S key
Player 2 (Right) Up Arrow Down Arrow

Left Paddle Controls (obj_paddle_left)

Move Up (W key):

  1. Open obj_paddle_left
  2. Add Event: Keyboard > Press W
  3. Add Action: Move > Set Vertical Speed
  4. Set speed to -8 (negative = up)

Stop Moving (Release W):

  1. Add Event: Keyboard > Release W
  2. Add Action: Move > Set Vertical Speed
  3. Set speed to 0

Move Down (S key):

  1. Add Event: Keyboard > Press S
  2. Add Action: Move > Set Vertical Speed
  3. Set speed to 8 (positive = down)

Stop Moving (Release S):

  1. Add Event: Keyboard > Release S
  2. Add Action: Move > Set Vertical Speed
  3. Set speed to 0

Wall Collision:

  1. Add Event: Collision > obj_wall
  2. Add Action: Move > Bounce Against Objects
  3. Select "Against solid objects"
Now do the same for obj_paddle_right, but use Up Arrow and Down Arrow instead of W and S!
