Korak 6: Krmilnik igre

Krmilnik igre prikazuje točke na zaslonu.

Ustvari objekt krmilnika

  1. Ustvari objekt obj_game_controller
  2. Sprite ni potreben (neviden krmilnik)

Nariši točke

V obj_game_controller:

  1. Dodaj dogodek: Draw > Draw
  2. Dodaj akcijo: Control > Execute Code
// Nastavi barvo besedila
draw_set_color(c_white);

// Nariši točke zgoraj levo
draw_text(10, 10, "Točke: " + string(score));

// Neobvezno: Nariši navodila
draw_text(10, 30, "Puščice: Premik  |  Presledek: Skok");

Alternativa: Uporaba akcij

Namesto kode lahko uporabiš:

  1. Dodaj akcijo: Draw > Set Color (Bela)
  2. Dodaj akcijo: Score > Draw Score
    • X: 10
    • Y: 10
    • Caption: "Točke: "
Zapomni si: Postavi krmilnik igre v svojo sobo! Sam se ne bo pojavil.
Klikni "Naprej", da oblikuješ svoj nivo.
