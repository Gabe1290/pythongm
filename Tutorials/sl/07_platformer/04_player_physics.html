Korak 3: Fizika igralca (dogodek Create)

Nastaviti moramo spremenljivke za gravitacijo, skakanje in premikanje.

Inicializacija spremenljivk fizike

V obj_player:

  1. Dodaj dogodek: Create
  2. Dodaj akcijo: Control > Execute Code
// Nastavitve premikanja
hspeed_max = 4;      // Največja vodoravna hitrost
vspeed_max = 10;     // Največja hitrost padanja
jump_force = -10;    // Moč skoka (negativno = navzgor)
gravity_force = 0.5; // Kako hitro padamo

// Trenutne hitrosti
hsp = 0;
vsp = 0;

// Stanje
on_ground = false;

Razumevanje spremenljivk

Spremenljivka Namen
hsp Vodoravna hitrost (levo/desno)
vsp Navpična hitrost (gor/dol)
gravity_force Konstantna sila, ki vleče navzdol
jump_force Negativna vrednost za premik navzgor
on_ground True, ko stojimo na tleh
Ključni koncept: Moč skoka je negativna, ker se Y v igrah povečuje navzdol. Negativna hitrost Y premakne igralca navzgor!
Klikni "Naprej", da dodaš premikanje in obravnavo trkov.
