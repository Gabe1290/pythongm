Korak 4: Premikanje Igralca (Dogodek Step)

To je jedro naše ploščadne igre: Premikanje, Gravitacija, Skakanje in Trki.

Dodaj Dogodek Step

V obj_player:

  1. Dodaj Dogodek: Step > Step
  2. Dodaj Akcijo: Control > Execute Code
// === VODORAVNO PREMIKANJE ===
var move_input = keyboard_check(vk_right) - keyboard_check(vk_left);
hsp = move_input * hspeed_max;

// === GRAVITACIJA ===
vsp += gravity_force;
if (vsp > vspeed_max) vsp = vspeed_max;

// === PREVERJANJE TAL ===
on_ground = place_meeting(x, y + 1, obj_ground);

// === SKAKANJE ===
if (on_ground && (keyboard_check_pressed(vk_up) || keyboard_check_pressed(vk_space))) {
    vsp = jump_force;
}

// === VODORAVNI TRK ===
if (place_meeting(x + hsp, y, obj_ground)) {
    while (!place_meeting(x + sign(hsp), y, obj_ground)) {
        x += sign(hsp);
    }
    hsp = 0;
}
x += hsp;

// === NAVPIČNI TRK ===
if (place_meeting(x, y + vsp, obj_ground)) {
    while (!place_meeting(x, y + sign(vsp), obj_ground)) {
        y += sign(vsp);
    }
    vsp = 0;
}
y += vsp;

Kako deluje

  1. Vnos: Desna puščica = 1, Leva = -1, Nič = 0
  2. Gravitacija: Vsak okvir doda vrednost k vsp (hitrejše padanje)
  3. Preverjanje tal: Išče tla 1 piksel spodaj
  4. Skok: Deluje samo, ko je on_ground true
  5. Trk: Premik piksel za pikslom do Stene
Čarovnija: Zanka while nas premika piksel za pikslom do roba Stene in preprečuje, da bi se zagozdili!
Klikni "Naprej", da dodaš Kovance, bodice in zastavo.
