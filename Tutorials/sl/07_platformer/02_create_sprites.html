Step 1: Create Sprites

We need sprites for all our game elements.

Sprite Name Size Description
spr_player 32x48 Simple character (rectangle with face)
spr_ground 32x32 Grass/dirt platform tile
spr_platform 64x16 Floating platform (wood/stone)
spr_coin 16x16 Yellow/gold circle
spr_spike 32x32 Triangular spikes pointing up
spr_flag 32x64 Flag on a pole (goal)

Create Player Sprite

  1. Right-click Sprites > Create Sprite
  2. Name it spr_player
  3. Click Edit Sprite
  4. Draw a character (32x48 pixels)
  5. Use a bright color like blue or red

Create Ground Sprite

  1. Create sprite named spr_ground
  2. Size: 32x32 pixels
  3. Draw a grass/dirt tile (green top, brown bottom)
Tip: The player is taller than wide (32x48) to look like a character!
Click "Next" to create the game objects.
