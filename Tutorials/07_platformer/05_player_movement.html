Step 4: Player Movement (Blockly)

Now we add left/right movement, jumping, and collision handling — all with Blockly blocks.

Reminder: We already set gravity in the Create event (previous step), so the player will fall automatically. Now we just need movement and collisions.

Move Left

Still in the Blockly editor for obj_player:

Keyboard: Left (held)
Set horizontal speed to -4

Find Keyboard (held) in Events, select Left. Then snap Set Horizontal Speed from Movement with value -4 (negative = left).

Move Right

Keyboard: Right (held)
Set horizontal speed to 4

Stop When No Key Pressed

Keyboard: No key
Set horizontal speed to 0

This stops horizontal movement when the player releases all keys. We set horizontal speed to 0 (not "Stop Movement") so gravity still works!

Jump

Keyboard Press: Up
Set vertical speed to -10

Find Keyboard Press in Events, select Up. Set vertical speed to -10 (negative = upward). The Press event fires only once, giving a single jump impulse.

Collide with Ground

When colliding with obj_ground
Stop movement

Find Collision in Events, type obj_ground. Snap Stop Movement inside. This prevents the player from falling through the floor and stops them when hitting walls.

Apply Your Blocks

  1. You should have 6 event blocks total (Create + 3 keyboard held/press + No key + Collision)
  2. Click the Apply to Events button to save
How it works: Gravity pulls the player down each frame. Holding Left/Right sets horizontal speed. Pressing Up gives a jump impulse. The ground collision stops all movement so the player lands on platforms. It's that simple!
