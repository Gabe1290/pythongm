Phase 1: A Jumping Player

In this phase, we create the player and ground, set up gravity and movement, and build a room with platforms. At the end, you'll be able to run and jump!

Create Sprites

  1. Right-click Sprites > Create New Sprite
  2. Name it spr_player, size 32x32
  3. Draw a character (or import from Tutorials/07_platformer/assets/)
  4. Create another sprite: spr_ground, size 32x32
  5. Draw a ground tile (green top, brown bottom) or import it

Create Objects

  1. Right-click Objects > Create New Object
  2. Create obj_ground, assign sprite spr_ground
  3. Check the Solid checkbox for obj_ground
  4. Create obj_player, assign sprite spr_player
Why Solid? Marking obj_ground as Solid tells the engine that other objects should not pass through it. This is essential for platforms!

Program obj_player with Blockly

  1. Double-click obj_player to open the Object Editor
  2. Click the Blockly tab

Enable Gravity (Create Event)

Create
Set Gravity: direction 270, strength 0.5

Find Create in the Events category. Then find Set Gravity in the Movement category. Set direction to 270 (down) and strength to 0.5.

Direction 270 = down. Angles start at 0 (right) and go counter-clockwise: 90 = up, 180 = left, 270 = down.

Move Left

Keyboard: Left (held)
Set horizontal speed to -4

Find Keyboard (held) in Events, select Left. Then snap Set Horizontal Speed from Movement with value -4 (negative = left).

Move Right

Keyboard: Right (held)
Set horizontal speed to 4

Stop When No Key Pressed

Keyboard: No key
Set horizontal speed to 0

This stops horizontal movement when no key is pressed. We use Set horizontal speed to 0 (NOT "Stop Movement") so that gravity keeps working!

Important: If you use "Stop Movement" here, it would also cancel the vertical speed and gravity would stop. By only setting horizontal speed to 0, the player still falls normally.

Jump

Keyboard Press: Up
Set vertical speed to -10

Find Keyboard Press in Events, select Up. Set vertical speed to -10 (negative = upward). The Press event fires only once, giving a single jump impulse.

Collide with Ground

When colliding with obj_ground
Stop movement

Find Collision in Events, type obj_ground. Snap Stop Movement inside. This prevents the player from falling through the ground and stops them when hitting walls.

Apply Your Blocks

  1. You should have 6 event blocks in your workspace (Create + Left + Right + No key + Press Up + Collision)
  2. Click the Apply to Events button to save

Create the Room

  1. Right-click Rooms > Create New Room
  2. Name it room_level1
  3. Set size: 800 x 480
  4. Set background color to light blue (sky)
  5. Enable "Snap to Grid" and set grid to 32x32
  6. Place obj_ground along the bottom row
  7. Create platforms at different heights by placing more obj_ground blocks in the air
  8. Place one obj_player on the left side, on top of the ground

Test Your Game!

Press F5 to play!

  • The player falls with gravity
  • Left/Right arrows move the player
  • Up arrow makes the player jump
  • The player lands on platforms and ground
