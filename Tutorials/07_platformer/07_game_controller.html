Step 6: Game Controller (Blockly)

The game controller displays the score on screen.

Create the Controller Object

  1. Right-click in the Asset Tree and create a new object: obj_game_controller
  2. No sprite needed (invisible controller)

Draw the Score

Open the Blockly editor for obj_game_controller:

Draw
Draw Score at 10, 10 caption "Score: "

Find Draw in the Events category. Then find Draw Score in the Score category. Set X to 10, Y to 10, and caption to Score: .

Apply Your Blocks

  1. You should have 1 event block (Draw) in your workspace
  2. Click Apply to Events to save
Remember: Place one instance of obj_game_controller in your room! It won't appear by itself. Since it has no sprite, it's invisible — it just draws the score.
