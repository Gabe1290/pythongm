Step 3: Player Physics (Blockly)

We need to set up gravity so the player falls and can jump. We'll use Blockly visual blocks to program this.

Open the Blockly Editor

  1. Double-click obj_player in the Asset Tree to open the Object Editor
  2. Click the Blockly tab at the top of the editor

Enable Gravity (Create Event)

Drag these blocks from the toolbox and snap them together:

Create
Set Gravity: direction 270, strength 0.5

Find Create in the Events category. Then find Set Gravity in the Movement category. Set direction to 270 (down) and strength to 0.5.

How Gravity works: Once set, the engine automatically applies gravity every frame. Direction 270 means "pull downward". The strength controls how fast the player accelerates.

Make Ground Solid

  1. Double-click obj_ground in the Asset Tree
  2. In the Object Properties panel, check the Solid checkbox

This is important! Solid objects automatically block movement, so the player won't fall through the ground.

Key Concept: Direction 270 = down because angles start at 0 (right) and go counter-clockwise: 90 = up, 180 = left, 270 = down. A strength of 0.5 gives a nice platformer-style gravity feel.
