Step 5: Collectibles and Hazards (Blockly)

Now we add coins to collect, spikes to avoid, and a flag to reach. Each object gets its own Blockly behavior.

Coin — Collect for Points

Open the Blockly editor for obj_coin:

When colliding with obj_player
Add to Score 10
Destroy Instance self

Find Collision in Events and type obj_player. Then snap Add to Score (from Score) with value 10, and Destroy Instance (from Instance) set to Self. The coin disappears and gives 10 points!

Spike — Hurts the Player

Open the Blockly editor for obj_spike:

When colliding with obj_player
Show Message "Ouch! You hit a spike!"
Restart Room

Find Collision in Events, type obj_player. Add Show Message (from Game) with text Ouch! You hit a spike!, then Restart Room (from Room) to reset the level.

Flag — Level Complete

Open the Blockly editor for obj_flag:

When colliding with obj_player
Show Message "Level Complete!"
Restart Room

Same pattern as the spike, but with a victory message. You can use Next Room instead of Restart Room if you create multiple levels!

Apply Your Blocks

For each object (obj_coin, obj_spike, obj_flag):

  1. Build the blocks shown above
  2. Click Apply to Events to save
Tip: Place spikes near pits or on platforms to create challenge! Scatter coins around the level to reward exploration.
