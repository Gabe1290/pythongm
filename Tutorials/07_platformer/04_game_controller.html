Phase 3: Game Controller

The game controller is an invisible object that manages the score and lives display. We also update the spike collision to use a lives system.

Create the Controller Object

  1. Right-click Objects > Create New Object
  2. Name it obj_game_controller
  3. Do NOT assign a sprite — it stays invisible
What is a game controller? It's an invisible object that runs in the background to manage things like the score display and lives. Since it has no sprite, it won't appear in the game — it just draws information on screen.

Program obj_game_controller with Blockly

  1. Double-click obj_game_controller to open the Object Editor
  2. Click the Blockly tab

Initialize Score and Lives (Create Event)

Create
Set score to 0
Set lives to 3

Find Create in Events. From Score/Lives/Health, snap Set Score with value 0 and Set Lives with value 3.

Draw Score and Lives (Draw Event)

Draw
Draw score at x: 10 y: 10
Draw lives at x: 200 y: 10

Find Draw in Events. From Score/Lives/Health, snap Draw Score at position (10, 10) and Draw Lives at position (200, 10). The score and lives will appear in the top-left area of the screen.

Apply Your Blocks

  1. You should have 2 event blocks (Create + Draw)
  2. Click Apply to Events to save

Update obj_player — Spike Collision with Lives

Now open the Blockly editor for obj_player and update the spike collision. Find the existing collision block with obj_spike and add a block before the restart:

When colliding with obj_spike
Add to lives -1
Restart room

Add Add to Lives (from Score/Lives/Health) with value -1 before the Restart Room block. Now each spike hit costs one life!

  1. Click Apply to Events to save
How it works: When the player touches a spike, they lose one life and the room restarts. The game controller displays the current score and remaining lives every frame.

Place the Controller in Your Room

  1. Open room_level1
  2. Place one obj_game_controller anywhere in the room
  3. Since it has no sprite, it will be invisible — position does not matter

Test Your Game!

Press F5 to play!

  • Score and lives appear in the top-left corner
  • Collecting coins increases the score
  • Hitting a spike costs one life and restarts the room
  • Reaching the flag shows "You Win!"

Congratulations! You have built a complete platformer game with gravity, jumping, coins, hazards, score, and lives!

Challenge Ideas

Want to go further? Try these ideas:

  • Moving platforms: Create an object that moves left and right automatically
  • Enemies: Add objects that patrol platforms and hurt the player
  • Double jump: Use a variable to allow jumping once more in the air
  • Multiple levels: Create additional rooms and use "Next Room" on the flag
  • Game over: Check if lives reach 0 and show a "Game Over" message
