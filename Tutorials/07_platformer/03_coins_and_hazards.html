Phase 2: Coins and Hazards

Now we add collectible coins, dangerous spikes, and a flag to reach. We program everything as new collision events on obj_player.

Create Sprites

  1. Create sprite spr_coin, size 32x32 — draw a yellow/gold circle (or import from Tutorials/07_platformer/assets/)
  2. Create sprite spr_spike, size 32x32 — draw triangular spikes pointing up
  3. Create sprite spr_flag, size 32x32 — draw a flag on a pole

Create Objects

  1. Create obj_coin, assign sprite spr_coin
  2. Create obj_spike, assign sprite spr_spike
  3. Create obj_flag, assign sprite spr_flag
No Blockly needed on these objects! We will program all the interactions as collision events on obj_player, since the player is the one who touches coins, spikes, and the flag.

Program obj_player — Add Collision Events

Open the Blockly editor for obj_player (the blocks from Phase 1 are still there). Add these new event blocks:

Collect Coins

When colliding with obj_coin
Add to score 10
Destroy other instance

Find Collision in Events, type obj_coin. Snap Add to Score (from Score/Lives/Health) with value 10. Then snap Destroy Instance (from Instance) set to Other, so the coin disappears when collected.

Hit a Spike

When colliding with obj_spike
Restart room

Find Collision in Events, type obj_spike. Snap Restart Room (from Room) inside. When the player touches a spike, the level resets!

Reach the Flag

When colliding with obj_flag
Show message "You Win!"

Find Collision in Events, type obj_flag. Snap Show Message (from Output) and type You Win!. A popup appears when the player reaches the flag!

Apply Your Blocks

  1. You should now have 9 event blocks total for obj_player (6 from Phase 1 + 3 new collision events)
  2. Click Apply to Events to save

Update Your Room

  1. Open room_level1
  2. Place obj_coin on platforms and hard-to-reach spots
  3. Place obj_spike near pits or on dangerous platforms
  4. Place obj_flag at the end of the level (right side)
                                        F

  GGGGGG      GGGG  GGGG       GGGGGGGGGG

G = Ground    P = Player    F = Flag    C = Coin    X = Spike
Level design tip: Place spikes near gaps to create tension. Put coins on high platforms to reward skilled jumping. Place the flag at the far end so the player must traverse the whole level!

Test Your Game!

Press F5 to play!

  • Collect coins — each one gives 10 points and disappears
  • Touch a spike — the room restarts!
  • Reach the flag — a "You Win!" message appears
