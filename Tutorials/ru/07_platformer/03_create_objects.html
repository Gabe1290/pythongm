Шаг 2: Создание объектов

Теперь создадим объекты для всех элементов игры.

Объект земли

  1. Щёлкните правой кнопкой мыши на Объекты > Создать объект
  2. Назовите его obj_ground
  3. Установите спрайт spr_ground
  4. Отметьте галочку "Твёрдый"

Объект платформы

  1. Создайте obj_platform
  2. Установите спрайт spr_platform
  3. Отметьте галочку "Твёрдый"
  4. Установите родителя obj_ground (опционально)

Объект игрока

  1. Создайте obj_player
  2. Установите спрайт spr_player

Предметы и опасности

  1. Создайте obj_coin со спрайтом spr_coin
  2. Создайте obj_spike со спрайтом spr_spike
  3. Создайте obj_flag со спрайтом spr_flag
Важно: Земля и платформы должны быть "Твёрдыми", чтобы столкновения работали правильно!
Нажмите "Далее", чтобы добавить физику игрока (гравитацию и прыжок).
