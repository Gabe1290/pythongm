Шаг 5: Предметы и опасности

Теперь добавим монеты для сбора, шипы для избегания и флаг для достижения.

Монета - Собирай для очков

В obj_coin:

  1. Добавить событие: Столкновение > obj_player
  2. Добавить действие: Счёт > Установить счёт
    • Новый счёт: 10
    • Отметьте "Относительно"
  3. Добавить действие: Main1 > Уничтожить объект
    • Применить к: Себе

Шип - Ранит игрока

В obj_spike:

  1. Добавить событие: Столкновение > obj_player
  2. Добавить действие: Main2 > Показать сообщение
    • Сообщение: Ой! Вы наткнулись на шип!
  3. Добавить действие: Main1 > Перезапустить комнату

Флаг - Уровень пройден

В obj_flag:

  1. Добавить событие: Столкновение > obj_player
  2. Добавить действие: Main2 > Показать сообщение
    • Сообщение: Уровень пройден!
  3. Добавить действие: Main1 > Перезапустить комнату (или Следующая комната для нескольких уровней)
Совет: Размещайте шипы рядом с пропастями или на платформах для создания сложности!
Нажмите "Далее", чтобы добавить игровой контроллер для отображения счёта.
