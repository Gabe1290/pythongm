Шаг 3: Физика игрока (Событие создания)

Нам нужно настроить переменные для гравитации, прыжка и движения.

Инициализация переменных физики

В obj_player:

  1. Добавить событие: Создать
  2. Добавить действие: Управление > Выполнить код
// Настройки движения
hspeed_max = 4;      // Максимальная горизонтальная скорость
vspeed_max = 10;     // Максимальная скорость падения
jump_force = -10;    // Сила прыжка (отрицательное = вверх)
gravity_force = 0.5; // Как быстро мы падаем

// Текущие скорости
hsp = 0;
vsp = 0;

// Состояние
on_ground = false;

Понимание переменных

Переменная Назначение
hsp Горизонтальная скорость (влево/вправо)
vsp Вертикальная скорость (вверх/вниз)
gravity_force Постоянная сила, тянущая вниз
jump_force Отрицательное значение для движения вверх
on_ground True, когда стоим на земле
Ключевая концепция: Сила прыжка отрицательная, потому что Y увеличивается вниз в играх. Отрицательная скорость по Y перемещает игрока вверх!
Нажмите "Далее", чтобы добавить движение и обработку столкновений.
