Шаг 4: Движение игрока (Событие шага)

Это основа нашего платформера: движение, гравитация, прыжок и столкновения.

Добавление события шага

В obj_player:

  1. Добавить событие: Шаг > Шаг
  2. Добавить действие: Управление > Выполнить код
// === ГОРИЗОНТАЛЬНОЕ ДВИЖЕНИЕ ===
var move_input = keyboard_check(vk_right) - keyboard_check(vk_left);
hsp = move_input * hspeed_max;

// === ГРАВИТАЦИЯ ===
vsp += gravity_force;
if (vsp > vspeed_max) vsp = vspeed_max;

// === ПРОВЕРКА ЗЕМЛИ ===
on_ground = place_meeting(x, y + 1, obj_ground);

// === ПРЫЖОК ===
if (on_ground && (keyboard_check_pressed(vk_up) || keyboard_check_pressed(vk_space))) {
    vsp = jump_force;
}

// === ГОРИЗОНТАЛЬНОЕ СТОЛКНОВЕНИЕ ===
if (place_meeting(x + hsp, y, obj_ground)) {
    while (!place_meeting(x + sign(hsp), y, obj_ground)) {
        x += sign(hsp);
    }
    hsp = 0;
}
x += hsp;

// === ВЕРТИКАЛЬНОЕ СТОЛКНОВЕНИЕ ===
if (place_meeting(x, y + vsp, obj_ground)) {
    while (!place_meeting(x, y + sign(vsp), obj_ground)) {
        y += sign(vsp);
    }
    vsp = 0;
}
y += vsp;

Как это работает

  1. Ввод: Стрелка вправо = 1, Влево = -1, Ничего = 0
  2. Гравитация: Добавляется к vsp каждый кадр (падение ускоряется)
  3. Проверка земли: Проверяет 1 пиксель ниже на наличие земли
  4. Прыжок: Работает только когда on_ground равно true
  5. Столкновение: Движение попиксельно до столкновения со стеной
Магия: Цикл while перемещает нас попиксельно к краю стены, предотвращая застревание!
