Шаг 6: Игровой контроллер

Игровой контроллер отображает счёт на экране.

Создание объекта контроллера

  1. Создайте объект obj_game_controller
  2. Спрайт не нужен (невидимый контроллер)

Отрисовка счёта

В obj_game_controller:

  1. Добавить событие: Рисование > Рисование
  2. Добавить действие: Управление > Выполнить код
// Установить цвет текста
draw_set_color(c_white);

// Рисуем счёт в левом верхнем углу
draw_text(10, 10, "Счёт: " + string(score));

// Опционально: Рисуем инструкции
draw_text(10, 30, "Стрелки: Движение  |  Пробел: Прыжок");

Альтернатива: Использование действий

Вместо кода можно использовать:

  1. Добавить действие: Рисование > Установить цвет (Белый)
  2. Добавить действие: Счёт > Рисовать счёт
    • X: 10
    • Y: 10
    • Подпись: "Счёт: "
Помните: Разместите игровой контроллер в вашей комнате! Он не появится сам по себе.
Нажмите "Далее", чтобы спроектировать ваш уровень.
