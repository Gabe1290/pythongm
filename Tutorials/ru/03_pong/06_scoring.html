Шаг 5: Добавление Ворот для подсчёта очков

Нам нужны невидимые Ворота позади каждой Ракетки для определения момента, когда кто-то забивает гол.

Создание Спрайта Ворот

  1. Создайте новый Спрайт с именем spr_goal
  2. Сделайте его 32x32 пикселя любого цвета
  3. (Он будет невидимым в игре)

Создание Объекта левых Ворот

  1. Создайте Объект obj_goal_left
  2. Установите Спрайт spr_goal
  3. Снимите флажок "Видимый" - Ворота должны быть невидимыми
  4. Установите флажок "Твёрдый"

Создание Объекта правых Ворот

  1. Создайте Объект obj_goal_right
  2. Установите Спрайт spr_goal
  3. Снимите флажок "Видимый"
  4. Установите флажок "Твёрдый"

Мяч сталкивается с левыми Воротами (Игрок 2 забивает)

Вернитесь к obj_ball:

  1. Добавьте Событие: Столкновение > obj_goal_left
  2. Добавьте Действие: Движение > Вернуться на начальную позицию
  3. Добавьте Действие: Счёт > Установить переменную
  4. Переменная: global.p2score
  5. Значение: 1
  6. Установите флажок "Относительно" (добавляет 1 к текущему Счёту)

Мяч сталкивается с правыми Воротами (Игрок 1 забивает)

  1. Добавьте Событие: Столкновение > obj_goal_right
  2. Добавьте Действие: Движение > Вернуться на начальную позицию
  3. Добавьте Действие: Счёт > Установить переменную
  4. Переменная: global.p1score
  5. Значение: 1
  6. Установите флажок "Относительно"
Примечание: Префикс global. означает, что эти переменные доступны из любого Объекта!
Нажмите "Далее", чтобы отобразить Счёт на экране.
