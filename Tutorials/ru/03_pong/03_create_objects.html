Шаг 2: Создание Объектов

Теперь мы создадим игровые Объекты. Каждый Объект использует Спрайт и может иметь поведение.

Создание Объекта Стены

  1. Щёлкните правой кнопкой мыши на Объекты и выберите Создать Объект
  2. Назовите его obj_wall
  3. Установите Спрайт spr_wall
  4. Установите флажок "Твёрдый" - это важно!
  5. События не нужны - Стены просто блокируют Движение

Создание Объекта Мяча

  1. Создайте новый Объект с именем obj_ball
  2. Установите Спрайт spr_ball
  3. Оставьте "Твёрдый" неотмеченным

Создание Объектов Ракеток

Левая Ракетка:

  1. Создайте Объект obj_paddle_left
  2. Установите Спрайт spr_paddle_left
  3. Установите флажок "Твёрдый"

Правая Ракетка:

  1. Создайте Объект obj_paddle_right
  2. Установите Спрайт spr_paddle_right
  3. Установите флажок "Твёрдый"
Важно: Флажок "Твёрдый" крайне важен для правильной работы отскоков!
Нажмите "Далее", чтобы добавить управление Движением Ракеток.
