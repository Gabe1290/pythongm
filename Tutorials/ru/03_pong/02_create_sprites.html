Шаг 1: Создание Спрайтов

Для игры Понг нам нужны четыре Спрайта: Мяч, две Ракетки и Стены.

Создание Спрайта Мяча

  1. Щёлкните правой кнопкой мыши на Спрайты в дереве ресурсов
  2. Выберите Создать Спрайт
  3. Назовите его spr_ball
  4. Нажмите Редактировать Спрайт
  5. Нарисуйте небольшой белый круг (примерно 16x16 пикселей)
  6. Нажмите OK для сохранения

Создание Спрайтов Ракеток

Левая Ракетка (Игрок 1):

  1. Создайте новый Спрайт с именем spr_paddle_left
  2. Нарисуйте высокий узкий прямоугольник (примерно 16x64 пикселей)
  3. Используйте синий цвет для обозначения Игрока 1

Правая Ракетка (Игрок 2):

  1. Создайте новый Спрайт с именем spr_paddle_right
  2. Нарисуйте высокий узкий прямоугольник (примерно 16x64 пикселей)
  3. Используйте красный цвет для обозначения Игрока 2

Создание Спрайта Стены

  1. Создайте новый Спрайт с именем spr_wall
  2. Нарисуйте сплошной серый прямоугольник (32x32 пикселя)
  3. Он будет растянут для формирования верхней и нижней границ
Совет: Делайте все Спрайты простыми! Классический Понг использовал базовые фигуры.
Нажмите "Далее", чтобы создать игровые Объекты.
