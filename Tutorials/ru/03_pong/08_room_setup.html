Шаг 7: Настройка Комнаты

Теперь давайте создадим игровую Комнату и разместим все наши Объекты!

Создание Комнаты

  1. Щёлкните правой кнопкой мыши на Комнаты и выберите Создать Комнату
  2. Назовите её room_pong
  3. Установите размер 640x480 пикселей

Схема Комнаты

+--[СТЕНА СТЕНА СТЕНА СТЕНА СТЕНА]--+
|                                   |
[В]  [И1]                    [И2]  [В]
[О]  [  ]       (o)          [  ]  [О]
[Р]  [  ]                    [  ]  [Р]
[.]                                [.]
|                                   |
+--[СТЕНА СТЕНА СТЕНА СТЕНА СТЕНА]--+

Размещение Объектов

1. Стены (Сверху и снизу)

Разместите экземпляры obj_wall вдоль верхнего и нижнего краёв.

2. Ракетки

  • obj_paddle_left - Около левого края, по центру по вертикали
  • obj_paddle_right - Около правого края, по центру по вертикали

3. Мяч

Разместите obj_ball в центре Комнаты.

4. Ворота

  • obj_goal_left - Расположите вдоль левого края (за Ракеткой)
  • obj_goal_right - Расположите вдоль правого края (за Ракеткой)

5. Отображение Счёта

Разместите obj_score в любом месте (у него нет Спрайта, он только рисует текст).

Поздравляем! Ваша игра Понг готова к запуску!

Тестирование игры

  • Нажмите F5 или кликните Запустить
  • Игрок 1: W (вверх) и S (вниз)
  • Игрок 2: Стрелка вверх и Стрелка вниз
Отлично! Вы создали классическую игру Понг для двух игроков!
