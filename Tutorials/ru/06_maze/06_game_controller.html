Шаг 5: Игровой контроллер

Создайте контроллер для отслеживания времени и отображения счёта.

Создание объекта контроллера

  1. Создайте объект obj_game_controller
  2. Спрайт не нужен (оставьте пустым)

Инициализация таймера (Событие создания)

  1. Добавьте событие: Создать
  2. Добавьте действие: Управление > Установить переменную
  3. Переменная: global.timer
  4. Значение: 0

Обновление таймера (Событие шага)

  1. Добавьте событие: Шаг
  2. Добавьте действие: Управление > Установить переменную
  3. Переменная: global.timer
  4. Значение: 1/room_speed
  5. Отметьте "Относительно"

Отображение информации (Событие рисования)

  1. Добавьте событие: Рисование
  2. Добавьте действие: Управление > Выполнить код
draw_set_color(c_white);
draw_text(10, 10, "Счёт: " + string(score));
draw_text(10, 30, "Время: " + string(floor(global.timer)) + "с");

var coins = instance_number(obj_coin);
draw_text(10, 50, "Монеты: " + string(coins));
Примечание: Контроллер невидим, но необходим для отслеживания состояния игры!
