Шаг 3: Движение игрока

Игроку нужно плавное движение с обнаружением столкновений.

Инициализация скорости (Событие создания)

В obj_player:

  1. Добавьте событие: Создать
  2. Добавьте действие: Управление > Установить переменную
  3. Переменная: move_speed
  4. Значение: 4

Движение со столкновениями (Событие шага)

  1. Добавьте событие: Шаг
  2. Добавьте действие: Управление > Выполнить код
// Горизонтальное движение
var hspd = 0;
if (keyboard_check(vk_right)) hspd = move_speed;
if (keyboard_check(vk_left)) hspd = -move_speed;

// Вертикальное движение
var vspd = 0;
if (keyboard_check(vk_down)) vspd = move_speed;
if (keyboard_check(vk_up)) vspd = -move_speed;

// Проверка горизонтального столкновения
if (!place_meeting(x + hspd, y, obj_wall)) {
    x += hspd;
}

// Проверка вертикального столкновения
if (!place_meeting(x, y + vspd, obj_wall)) {
    y += vspd;
}
Как это работает: Мы проверяем, будет ли целевая позиция сталкиваться со стеной. Если нет, перемещаемся туда!
