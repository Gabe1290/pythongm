ИГРА-ЛАБИРИНТ

Найди путь через лабиринт и собери монеты!

Что мы создадим

В этом уроке вы создадите игру-лабиринт:

  • Игрок, который плавно двигается с помощью клавиш-стрелок
  • Стены, которые блокируют движение игрока
  • Монеты для сбора и получения очков
  • Выход для завершения уровня
  • Таймер для отслеживания времени прохождения

Чему вы научитесь

  • Плавное движение с помощью клавиатуры
  • Обнаружение и обработка столкновений со стенами
  • Создание собираемых предметов
  • Отображение таймера и счёта
  • Дизайн уровней для игр-лабиринтов
Необходимое время: Около 25-30 минут
