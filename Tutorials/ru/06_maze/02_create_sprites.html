Шаг 1: Создание спрайтов

Нам нужны спрайты для игрока, стен, выхода и монет. Стены должны быть 32x32 пикселя для создания сетки.

Создайте спрайт игрока

  1. Щёлкните правой кнопкой мыши на Спрайты и выберите Создать спрайт
  2. Назовите его spr_player
  3. Нажмите Редактировать спрайт
  4. Нарисуйте небольшого персонажа (круг или простую фигурку)
  5. Используйте яркий цвет, например синий или зелёный
  6. Размер: 24x24 пикселя (меньше, чем стены)

Создайте спрайт стены

  1. Создайте новый спрайт с именем spr_wall
  2. Нарисуйте узор кирпича или камня
  3. Используйте серые или тёмные цвета
  4. Размер: 32x32 пикселя

Создайте спрайт выхода

  1. Создайте новый спрайт с именем spr_exit
  2. Нарисуйте дверь, флаг или светящуюся метку
  3. Используйте зелёный или золотой цвет
  4. Размер: 32x32 пикселя

Создайте спрайт монеты

  1. Создайте новый спрайт с именем spr_coin
  2. Нарисуйте маленький жёлтый/золотой круг
  3. Размер: 16x16 пикселей
Совет: Если спрайт игрока меньше стен, это позволяет легче перемещаться по коридорам!
Нажмите "Далее", чтобы создать игровые объекты.
