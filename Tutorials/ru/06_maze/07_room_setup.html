Шаг 6: Настройка комнаты

Теперь создайте свой уровень-лабиринт!

Создание комнаты

  1. Щёлкните правой кнопкой мыши на Комнаты > Создать комнату
  2. Назовите её room_maze
  3. Установите размер 640x480
  4. Включите Привязку к сетке: 32x32

Постройте свой лабиринт

  1. Создайте границу из стен по краям
  2. Добавьте внутренние стены для создания коридоров
  3. Разместите монеты вдоль путей
  4. Поместите выход в конце лабиринта
  5. Разместите игрока рядом с входом
  6. Добавьте игровой контроллер (в любом месте)

Поздравляем! Вы создали игру-лабиринт!

Нажмите F5 для запуска:
  • Используйте клавиши-стрелки для движения
  • Собирайте монеты для получения очков
  • Найдите выход, чтобы победить!
Идеи для улучшений:
  • Добавьте патрулирующих врагов
  • Создайте обратный отсчёт времени
  • Требуйте сбор всех монет перед открытием выхода
  • Добавьте несколько уровней
