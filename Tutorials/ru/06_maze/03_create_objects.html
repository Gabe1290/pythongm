Шаг 2: Создание объектов

Теперь мы создадим объекты для стен, выхода, монет и игрока.

Создайте объект стены

  1. Щёлкните правой кнопкой мыши на Объекты и выберите Создать объект
  2. Назовите его obj_wall
  3. Установите спрайт spr_wall
  4. Отметьте "Сплошной"
  5. События не нужны

Создайте объект выхода

  1. Создайте объект obj_exit
  2. Установите спрайт spr_exit
  3. Оставьте "Сплошной" не отмеченным

Создайте объект монеты

  1. Создайте объект obj_coin
  2. Установите спрайт spr_coin
  3. Оставьте "Сплошной" не отмеченным

Создайте объект игрока

  1. Создайте объект obj_player
  2. Установите спрайт spr_player
  3. Оставьте "Сплошной" не отмеченным
Примечание: Только стена должна быть "Сплошной". Игрок будет проверять столкновения в своём коде.
Нажмите "Далее", чтобы добавить движение игрока.
