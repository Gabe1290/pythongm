Шаг 3: Физика посадочного модуля (Событие создания)

Настройте переменные для гравитации, тяги, топлива и скорости.

Инициализация переменных физики

В obj_lander:

  1. Добавить событие: Создать
  2. Добавить действие: Управление > Выполнить код
// Настройки физики
gravity_force = 0.05;  // Слабая гравитация
thrust_force = 0.1;    // Сила тяги
max_speed = 5;         // Максимальная скорость

// Текущая скорость
hsp = 0;  // Горизонтальная скорость
vsp = 0;  // Вертикальная скорость

// Система топлива
fuel = 100;
fuel_use = 0.5;  // Топливо за кадр

// Состояние игры
landed = false;
crashed = false;

// Безопасная скорость посадки
safe_speed = 2;

Понимание переменных

Переменная Назначение
gravity_force Постоянное притяжение вниз (0.05 - слабое)
thrust_force Сила вверх при нажатии ВВЕРХ
fuel 100% топлива, уменьшается при работе двигателя
safe_speed Максимальная скорость для безопасной посадки (2)
Баланс: Тяга (0.1) сильнее гравитации (0.05), поэтому вы можете замедлиться!
Нажмите "Далее", чтобы добавить движение и управление.
