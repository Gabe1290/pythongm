Добавить событие шага

В obj_lander:

  1. Добавить событие: Шаг > Шаг
  2. Добавить действие: Управление > Выполнить код
// Не обновлять, если игра окончена
if (landed || crashed) exit;

// === ГРАВИТАЦИЯ ===
vsp += gravity_force;

// === ТЯГА (Стрелка вверх) ===
if (keyboard_check(vk_up) && fuel > 0) {
    vsp -= thrust_force;
    fuel -= fuel_use;
    if (fuel < 0) fuel = 0;
}

// === ГОРИЗОНТАЛЬНОЕ УПРАВЛЕНИЕ ===
if (keyboard_check(vk_left)) hsp -= 0.05;
if (keyboard_check(vk_right)) hsp += 0.05;

// === ОГРАНИЧЕНИЕ СКОРОСТИ ===
hsp = clamp(hsp, -max_speed, max_speed);
vsp = clamp(vsp, -max_speed, max_speed);

// === ПРИМЕНИТЬ ДВИЖЕНИЕ ===
x += hsp;
y += vsp;

// === ГРАНИЦЫ ЭКРАНА ===
if (x < 16) { x = 16; hsp = 0; }
if (x > room_width - 16) { x = room_width - 16; hsp = 0; }
if (y < 16) { y = 16; vsp = 0; }

Как это работает

  1. Гравитация: Добавляется к vsp каждый кадр (падение)
  2. Тяга: Вычитается из vsp (подъём), расходует топливо
  3. Clamp: Ограничивает скорость безопасным диапазоном
  4. Границы: Удерживает модуль на экране
Важно: Функция clamp() ограничивает значение между минимумом и максимумом!
