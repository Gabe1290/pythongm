Шаг 6: Игровой контроллер (Интерфейс)

Отображение топлива, скорости и статуса для помощи игроку.

Событие рисования

В obj_game_controller:

  1. Добавить событие: Рисование > Рисование
  2. Добавить действие: Управление > Выполнить код
draw_set_color(c_white);

// Получить информацию о модуле
var lander = instance_find(obj_lander, 0);
if (lander != noone) {
    var spd = sqrt(lander.hsp*lander.hsp + lander.vsp*lander.vsp);

    // Отрисовка интерфейса
    draw_text(10, 10, "ПОСАДОЧНЫЙ МОДУЛЬ");
    draw_text(10, 30, "Топливо: " + string(floor(lander.fuel)) + "%");
    draw_text(10, 50, "Скорость: " + string_format(spd, 1, 2));
    draw_text(10, 70, "Безопасно: < " + string(lander.safe_speed));

    // Предупреждение о скорости
    if (spd > lander.safe_speed) {
        draw_set_color(c_red);
        draw_text(10, 90, "СЛИШКОМ БЫСТРО!");
    } else {
        draw_set_color(c_lime);
        draw_text(10, 90, "Скорость ОК");
    }

    // Предупреждение о топливе
    if (lander.fuel <= 0) {
        draw_set_color(c_red);
        draw_text(10, 110, "НЕТ ТОПЛИВА!");
    }
}

draw_set_color(c_gray);
draw_text(10, room_height - 20, "ВВЕРХ: Тяга | ВЛЕВО/ВПРАВО: Движение");

Что показывает интерфейс

  • Топливо: Оставшийся процент (зелёный = ОК)
  • Скорость: Текущая скорость (красный = опасно)
  • Предупреждения: "СЛИШКОМ БЫСТРО!" или "НЕТ ТОПЛИВА!"
  • Управление: Напоминание внизу
