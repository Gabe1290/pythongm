Шаг 5: Обнаружение столкновений

Обработка посадки на площадку и крушения о местность.

Столкновение с посадочной площадкой

В obj_lander:

  1. Добавить событие: Столкновение > obj_pad
  2. Добавить действие: Управление > Выполнить код
// Вычислить общую скорость
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    // Безопасная посадка!
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("Идеальная посадка! Вы победили!");
} else {
    // Слишком быстро - крушение!
    crashed = true;
    show_message("Крушение! Слишком быстро!");
    room_restart();
}

Расчёт скорости

Мы используем теорему Пифагора для получения общей скорости:

total_speed = sqrt(hsp*hsp + vsp*vsp)

Это объединяет горизонтальную и вертикальную скорость в одно значение.

Безопасная скорость - 2! Если total_speed > 2, вы разобьётесь даже на площадке.
