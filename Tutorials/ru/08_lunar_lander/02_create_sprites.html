Шаг 1: Создание спрайтов

Нам нужны спрайты для посадочного модуля, посадочной площадки и местности.

Имя спрайта Размер Описание
spr_lander 32x32 Космический корабль (треугольник или классическая форма модуля)
spr_pad 64x16 Посадочная платформа с маркировкой "H"
spr_ground 32x32 Скалистая местность (серая/коричневая)

Создание спрайта посадочного модуля

  1. Щёлкните правой кнопкой мыши Спрайты > Создать спрайт
  2. Назовите его spr_lander
  3. Нажмите Редактировать спрайт
  4. Нарисуйте космический корабль (32x32 пикселя)
  5. Установите точку отсчёта по центру снизу для правильной посадки

Создание спрайта посадочной площадки

  1. Создайте спрайт с именем spr_pad
  2. Размер: 64x16 пикселей
  3. Нарисуйте плоскую платформу с маркировкой
  4. Используйте яркие цвета (жёлтый/зелёный)

Создание спрайта земли

  1. Создайте спрайт с именем spr_ground
  2. Размер: 32x32 пикселя
  3. Нарисуйте скалистую, неровную местность
  4. Используйте серые/коричневые цвета
Важно: Установите точку отсчёта посадочного модуля по центру снизу, чтобы он правильно приземлялся на площадку!
Нажмите "Далее", чтобы создать игровые объекты.
