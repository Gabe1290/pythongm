Шаг 2: Создание объектов

Теперь мы создаём объекты для всех игровых элементов.

Объект земли (опасный)

  1. Щёлкните правой кнопкой мыши Объекты > Создать объект
  2. Назовите его obj_ground
  3. Установите спрайт spr_ground
  4. Установите флажок "Твёрдый"

Объект посадочной площадки (безопасный)

  1. Создайте obj_pad
  2. Установите спрайт spr_pad
  3. Установите флажок "Твёрдый"

Объект посадочного модуля (игрок)

  1. Создайте obj_lander
  2. Установите спрайт spr_lander

Объект игрового контроллера

  1. Создайте obj_game_controller
  2. Спрайт не нужен (невидимый)
Ключевое отличие: Посадка на obj_pad = безопасная посадка. Посадка на obj_ground = крушение!
Нажмите "Далее", чтобы добавить физику посадочного модуля.
