Создание первого проекта

Давайте создадим простой проект шаг за шагом.

1 Создайте новый проект

Перейдите в Файл > Новый проект (или нажмите Ctrl+N).

Введите имя проекта и выберите место для сохранения.

2 Создайте Спрайт

Щёлкните правой кнопкой мыши по Спрайты в дереве ресурсов и выберите Создать Спрайт.

Назовите его «spr_player» и импортируйте изображение для персонажа Игрока.

3 Создайте Объект

Щёлкните правой кнопкой мыши по Объекты и выберите Создать Объект.

Назовите его «obj_player» и назначьте только что созданный Спрайт.

4 Создайте Комнату

Щёлкните правой кнопкой мыши по Комнаты и выберите Создать Комнату.

Назовите её «room_game». Это будет ваш первый игровой уровень.

5 Разместите Объект

Дважды щёлкните по Комнате, чтобы открыть редактор Комнат.

Выберите Объект из списка и щёлкните в Комнате, чтобы разместить его.

6 Протестируйте игру

Нажмите F5 или перейдите в Сборка > Тестировать игру.

Откроется окно с вашей игрой и размещённым Объектом!

Поздравляем! Вы создали свой первый проект в PyGameMaker! Пока он мало что делает, но в следующих уроках вы научитесь добавлять поведение.
