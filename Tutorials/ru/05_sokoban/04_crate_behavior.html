Шаг 3: Поведение Ящика

Ящик должен менять внешний вид, когда находится на целевой точке.

Проверка Позиции Цели (Событие Step)

В obj_crate:

  1. Добавить Событие: Step
  2. Добавить Действие: Control > Test Instance Position
  3. Объект: obj_target
  4. X: 0, Y: 0 (относительно)
  5. Проверить, есть ли столкновение

Смена Спрайта на Цели

Если столкновение есть (ящик на цели):

  1. Добавить Действие: Main1 > Change Sprite
  2. Спрайт: spr_crate_ok

Смена Спрайта вне Цели

  1. Добавить Действие: Control > Else
  2. Добавить Действие: Main1 > Change Sprite
  3. Спрайт: spr_crate
Визуальная обратная связь: Когда ящик становится зелёным, игроки понимают, что он правильно размещён!
Нажмите "Далее", чтобы добавить движение игрока.
