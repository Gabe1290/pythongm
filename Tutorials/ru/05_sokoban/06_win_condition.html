Шаг 5: Условие Победы

Игрок побеждает, когда все ящики находятся на целевых точках.

Создание Игрового Контроллера

  1. Создайте объект obj_game_controller
  2. Спрайт не нужен

Подсчёт Целей (Событие Create)

В obj_game_controller:

  1. Добавить Событие: Create
  2. Добавить Действие: Control > Execute Code
  3. Введите: global.total_targets = instance_number(obj_target);

Проверка Условия Победы (Событие Step)

  1. Добавить Событие: Step
  2. Добавить Действие: Control > Execute Code
  3. Введите этот код:
// Подсчёт ящиков на целях
var crates_on_targets = 0;
with (obj_crate) {
    if (place_meeting(x, y, obj_target)) {
        crates_on_targets += 1;
    }
}

// Проверяем, все ли цели заняты ящиками
if (crates_on_targets >= global.total_targets
    && global.total_targets > 0) {
    show_message("Уровень пройден!");
    room_restart();
}

Отрисовка Инструкций (Событие Draw)

  1. Добавить Событие: Draw
  2. Добавить Действие: Draw > Draw Text
  3. Текст: Передвиньте ящики на цели!
  4. X: 10, Y: 10
Совет: Убедитесь, что количество ящиков равно количеству целей на вашем уровне!
Нажмите "Далее", чтобы настроить игровую комнату.
