Шаг 4: Движение Игрока

Игрок перемещается фиксированными шагами по 32 пикселя (движение по сетке).

Важно: В Сокобане движение должно проверять:
1. Есть ли стена? (Нельзя двигаться)
2. Есть ли ящик? (Можем ли мы его толкнуть?)
3. Есть ли стена/ящик за ящиком? (Нельзя толкать)

Движение Вправо (Стрелка Вправо)

В obj_player:

  1. Добавить Событие: Keyboard > Press Right
  2. Добавить Действие: Control > Execute Code
  3. Введите этот код:
// Проверяем, можно ли двигаться вправо
if (!place_meeting(x + 32, y, obj_wall)) {
    var crate = instance_place(x + 32, y, obj_crate);
    if (crate != noone) {
        // Там ящик - можем ли мы его толкнуть?
        if (!place_meeting(x + 64, y, obj_wall) &&
            !place_meeting(x + 64, y, obj_crate)) {
            crate.x += 32;
            x += 32;
        }
    } else {
        // Ящика нет, просто двигаемся
        x += 32;
    }
}

Движение Влево (Стрелка Влево)

Тот же код, но измените:

  • + 32 на - 32
  • + 64 на - 64

Движение Вверх (Стрелка Вверх)

Тот же код, но используйте y вместо x:

  • x + 32 на y - 32
  • x + 64 на y - 64
  • crate.x на crate.y

Движение Вниз (Стрелка Вниз)

То же, что и Вверх, но с + 32 и + 64.

Нажмите "Далее", чтобы добавить определение условия победы.
