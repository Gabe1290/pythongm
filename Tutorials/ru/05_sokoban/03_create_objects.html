Шаг 2: Создание Объектов

Нам нужны объекты для игрока, стен, ящиков и целей.

Создание Объекта Стены

  1. Щелкните правой кнопкой мыши на Объекты и выберите Создать Объект
  2. Назовите его obj_wall
  3. Установите спрайт spr_wall
  4. Отметьте "Твёрдый"
  5. События не нужны

Создание Объекта Цели

  1. Создайте объект obj_target
  2. Установите спрайт spr_target
  3. Оставьте "Твёрдый" неотмеченным (игрок и ящики могут находиться сверху)
  4. События не нужны - это просто маркер

Создание Объекта Ящика

  1. Создайте объект obj_crate
  2. Установите спрайт spr_crate
  3. Отметьте "Твёрдый"

Создание Объекта Игрока

  1. Создайте объект obj_player
  2. Установите спрайт spr_player
  3. Оставьте "Твёрдый" неотмеченным
Примечание: Мы добавим события в obj_crate и obj_player на следующих шагах.
Нажмите "Далее", чтобы добавить поведение ящика.
