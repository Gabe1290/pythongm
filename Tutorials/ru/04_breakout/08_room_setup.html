Шаг 7: Настройка комнаты

Пришло время создать игровую комнату и разместить все объекты!

Создание комнаты

  1. Щёлкните правой кнопкой мыши по Комнаты и выберите Создать комнату
  2. Назовите её room_breakout
  3. Установите размер 640x480 пикселей

Схема комнаты

+--[СТЕНА]--------------------[СТЕНА]--+
|  [СТЕНА]  [К][К][К][К][К]   [СТЕНА]  |
|  [СТЕНА]  [О][О][О][О][О]   [СТЕНА]  |
|  [СТЕНА]  [Ж][Ж][Ж][Ж][Ж]   [СТЕНА]  |
|  [СТЕНА]  [З][З][З][З][З]   [СТЕНА]  |
|  [СТЕНА]                    [СТЕНА]  |
|  [СТЕНА]        (o)         [СТЕНА]  |
|  [СТЕНА]                    [СТЕНА]  |
|  [СТЕНА]     [РАКЕТКА]      [СТЕНА]  |
+--[ЗОНА СМЕРТИ ЗОНА СМЕРТИ ЗОНА]------+

Размещение объектов

1. Стены

Разместите obj_wall вдоль верхнего, левого и правого краёв.

2. Зона смерти

Разместите obj_death_zone вдоль нижнего края (она невидима).

3. Кирпичи

Расположите кирпичи рядами в верхней части:

  • Верхний ряд: красные кирпичи (obj_brick_red)
  • Второй ряд: оранжевые кирпичи
  • Третий ряд: жёлтые кирпичи
  • Четвёртый ряд: зелёные кирпичи

4. Ракетка

Разместите obj_paddle внизу, по центру по горизонтали.

5. Мяч

Разместите obj_ball чуть выше ракетки.

6. Игровой контроллер

Разместите obj_game_controller в любом месте (у него нет спрайта).

Поздравляем! Ваша игра Breakout готова!

Тестирование игры

  • Нажмите F5 или кнопку Запуск
  • Используйте клавиши Влево и Вправо для перемещения
  • Разбейте все кирпичи, чтобы победить!
