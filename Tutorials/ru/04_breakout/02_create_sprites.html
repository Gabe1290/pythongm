Шаг 1: Создание спрайтов

Нам нужны спрайты для ракетки, мяча, кирпичей и стен.

Создание спрайта мяча

  1. Щёлкните правой кнопкой мыши по Спрайты в дереве ресурсов
  2. Выберите Создать спрайт
  3. Назовите его spr_ball
  4. Нажмите Редактировать спрайт
  5. Нарисуйте небольшой белый или жёлтый круг (16x16 пикселей)
  6. Нажмите OK для сохранения

Создание спрайта ракетки

  1. Создайте новый спрайт с именем spr_paddle
  2. Нарисуйте широкий плоский прямоугольник (64x16 пикселей)
  3. Используйте яркий цвет, например синий

Создание спрайтов кирпичей

Создайте разноцветные кирпичи с разной стоимостью в очках:

  1. Создайте spr_brick_red — красный кирпич (32x16 пикселей)
  2. Создайте spr_brick_orange — оранжевый кирпич
  3. Создайте spr_brick_yellow — жёлтый кирпич
  4. Создайте spr_brick_green — зелёный кирпич

Создание спрайта стены

  1. Создайте новый спрайт с именем spr_wall
  2. Нарисуйте сплошной серый прямоугольник (32x32 пикселя)
Совет: Сохраняйте одинаковый размер кирпичей, чтобы они красиво выстраивались в комнате!
Нажмите «Далее», чтобы создать игровые объекты.
