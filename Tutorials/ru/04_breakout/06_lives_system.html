Шаг 5: Добавление системы жизней

Игрок теряет жизнь, когда мяч падает за нижний край экрана.

Создание зоны смерти

  1. Создайте спрайт spr_death_zone (32x32, любой цвет)
  2. Создайте объект obj_death_zone
  3. Снимите отметку «Видимый» — он должен быть невидимым
  4. Отметьте «Твёрдый»

Мяч падает в зону смерти

В obj_ball:

  1. Добавьте событие: Столкновение > obj_death_zone
  2. Добавьте действие: Счёт > Вычесть из жизней
  3. Значение: 1
  4. Добавьте действие: Движение > Вернуться на начальную позицию

Создание игрового контроллера

  1. Создайте объект obj_game_controller
  2. Спрайт не нужен

Инициализация жизней (событие создания)

В obj_game_controller:

  1. Добавьте событие: Создать
  2. Добавьте действие: Счёт > Установить жизни
  3. Значение: 3
  4. Добавьте действие: Счёт > Установить счёт
  5. Значение: 0

Проверка конца игры (событие шага)

  1. Добавьте событие: Шаг
  2. Добавьте действие: Управление > Проверить жизни
  3. Проверьте, меньше или равно ли количество жизней 0
  4. Добавьте действие: Main2 > Показать сообщение
  5. Сообщение: Игра окончена!
  6. Добавьте действие: Main2 > Перезапустить комнату
Помните: Поместите obj_game_controller в вашу комнату!
Нажмите «Далее», чтобы добавить отображение счёта.
