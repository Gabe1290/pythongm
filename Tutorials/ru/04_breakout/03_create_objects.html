Шаг 2: Создание объектов

Теперь мы создадим все игровые объекты.

Создание объекта стены

  1. Щёлкните правой кнопкой мыши по Объекты и выберите Создать объект
  2. Назовите его obj_wall
  3. Установите спрайт spr_wall
  4. Отметьте «Твёрдый»
  5. События не требуются

Создание родительского объекта кирпича

Мы используем родительский объект, чтобы все кирпичи имели одинаковое поведение при столкновении.

  1. Создайте объект obj_brick_parent
  2. Спрайт не нужен (это просто шаблон)
  3. Отметьте «Твёрдый»

Создание цветных объектов кирпичей

Для каждого цвета кирпича:

  1. Создайте obj_brick_red со спрайтом spr_brick_red
  2. Установите Родитель как obj_brick_parent
  3. Отметьте «Твёрдый»

Повторите для obj_brick_orange, obj_brick_yellow и obj_brick_green.

Создание объекта ракетки

  1. Создайте объект obj_paddle
  2. Установите спрайт spr_paddle
  3. Отметьте «Твёрдый»

Создание объекта мяча

  1. Создайте объект obj_ball
  2. Установите спрайт spr_ball
  3. Оставьте «Твёрдый» неотмеченным
Родительские объекты: Когда мяч сталкивается с obj_brick_parent, он будет обнаруживать столкновения со ВСЕМИ типами кирпичей!
Нажмите «Далее», чтобы добавить движение ракетки.
