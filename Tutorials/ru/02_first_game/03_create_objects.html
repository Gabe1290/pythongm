Шаг 2: Создание Объектов

Объекты — это «актёры» в вашей игре. Они используют Спрайты для внешнего вида и имеют поведение, определяемое Событиями.

Создание Объекта Игрока

1 Создайте новый Объект

Щёлкните правой кнопкой мыши по Объекты в дереве ресурсов и выберите Создать Объект.

Назовите его obj_player.

2 Назначьте Спрайт

В редакторе Объектов найдите выпадающий список Спрайт.

Выберите spr_player из списка.

Создание Объекта звезды

3 Создайте ещё один Объект

Щёлкните правой кнопкой мыши по Объекты и выберите Создать Объект.

Назовите его obj_star.

4 Назначьте Спрайт звезды

Выберите spr_star в качестве Спрайта для этого Объекта.

Создание игровой Комнаты

5 Создайте Комнату

Щёлкните правой кнопкой мыши по Комнаты и выберите Создать Комнату.

Назовите её room_game.

6 Разместите Игрока

Дважды щёлкните по Комнате, чтобы открыть редактор Комнат.

Выберите obj_player из списка Объектов слева.

Щёлкните в нижней центральной части Комнаты, чтобы разместить там Игрока.

Пока не размещайте звёзды! Мы сделаем так, чтобы они появлялись автоматически сверху экрана.
Объекты созданы! Нажмите «Далее», чтобы добавить Движение Игрока.
