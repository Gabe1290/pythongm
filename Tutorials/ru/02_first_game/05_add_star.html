Step 4: Make Stars Fall

Now we'll create a spawner that creates falling stars at random positions.

Create a Spawner Object

1 Create obj_spawner

Right-click on Objects and create a new object called obj_spawner.

Don't assign a sprite - this object will be invisible.

2 Add an Alarm event

Open obj_spawner and click Add Event.

Select Alarm > Alarm 0.

3 Create a star

In the Alarm 0 event, add these actions:

  • Create Instance: Create obj_star at position x=random(room_width), y=0
  • Set Alarm: Set Alarm 0 to 60 (creates a new star every 2 seconds)
4 Start the alarm

Add a Create event to obj_spawner.

In this event, add: Set Alarm 0 to 60.

Make Stars Fall Down

5 Edit obj_star

Open obj_star and add a Create event.

6 Set falling speed

Add a "set vspeed" action and set it to 3.

This makes the star move downward at speed 3.

Place the Spawner

7 Add spawner to room

Open room_game in the Room Editor.

Place one obj_spawner anywhere in the room (it's invisible, so location doesn't matter).

Test it! Press F5. Stars should now fall from the top of the screen!
Stars falling! Click "Next" to add collision and scoring.
