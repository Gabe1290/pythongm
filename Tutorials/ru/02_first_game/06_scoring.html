Step 5: Add Scoring

Let's add collision detection so the player can catch stars and earn points!

Catch the Stars

1 Add collision event to player

Open obj_player and add a new event.

Select Collision > obj_star.

2 Add scoring actions

In the collision event, add these actions:

  • Add to Score: Add 10 points
  • Destroy Instance: Destroy the other object (the star)

Display the Score

3 Add Draw event to player

In obj_player, add a Draw > Draw GUI event.

4 Draw the score

Add a Draw Text action:

  • Text: "Score: " + string(score)
  • Position: x=10, y=10

Remove Missed Stars

5 Add Step event to star

Open obj_star and add a Step event.

6 Destroy if off-screen

Add an If condition: if y > room_height

Then: Destroy Instance

This removes stars that fall off the bottom of the screen.

Test it! Press F5. Catch stars to earn points! The score should display in the top-left corner.
Scoring works! Click "Next" to add finishing touches.
