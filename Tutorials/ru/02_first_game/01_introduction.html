Поймай звезду!

Управляйте Игроком, чтобы ловить падающие звёзды и набирать очки!

Что мы создадим

В этом уроке вы создадите полноценную игру с нуля:

  • Игрок, который двигается влево и вправо с помощью клавиш-стрелок
  • Звёзды, падающие сверху экрана
  • Система Счёта, отслеживающая количество пойманных звёзд
  • Конец игры, когда вы пропустите слишком много звёзд

Чему вы научитесь

  • Создание и импорт Спрайтов
  • Создание Объектов с поведением
  • Использование Событий клавиатуры для управления Игроком
  • Обнаружение Столкновений между Объектами
  • Использование переменных для отслеживания Счёта
  • Отображение текста на экране
Необходимое время: Около 15-20 минут
Готовы? Нажмите «Далее», чтобы начать создание Спрайтов для игры!
