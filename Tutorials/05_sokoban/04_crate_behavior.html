Step 3: Crate Behavior (Blockly)

The crate needs to change appearance when it's on a target spot. We'll use Blockly visual blocks to program this.

Open the Blockly Editor

  1. Double-click obj_crate in the Asset Tree to open the Object Editor
  2. Click the Blockly tab at the top of the editor

Check if Crate is on a Target (Step Event)

Drag these blocks from the toolbox and snap them together:

Step
If colliding with obj_target
Set sprite to spr_crate_ok
Else
Set sprite to spr_crate

Find Step in the Events category. Then find Check Collision in the Control category and type obj_target. Snap a Set Sprite block (from Instance) inside for the "then" case with spr_crate_ok. Add an Else section with another Set Sprite block for spr_crate.

Apply Your Blocks

  1. You should have 1 event block (Step) in your workspace
  2. Click the Apply to Events button to save
Visual Feedback: When the crate turns green, players know it's correctly placed!
