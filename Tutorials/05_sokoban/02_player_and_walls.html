Phase 1: Player and Walls

Let's create a player that walks on a grid inside a walled room!

Create the Sprites

  1. Right-click on Sprites in the Asset Tree → Create New Sprite
  2. Name it spr_player (32×32 pixels)
  3. Draw a simple character or import from Tutorials/05_sokoban/assets/
  4. Press Ctrl+S to save
  1. Create another sprite: spr_wall (32×32 pixels)
  2. Draw a dark brick or stone pattern, or import from Tutorials/05_sokoban/assets/
  3. Press Ctrl+S to save

Create the Objects

  1. Right-click on Objects → Create New Object
  2. Name it obj_wall, set its sprite to spr_wall
  3. Check the Solid checkbox
  1. Create another object: obj_player
  2. Set its sprite to spr_player

Program the Player (Blockly)

  1. Double-click obj_player to open the Object Editor
  2. Click the Blockly tab

Move Right

Keyboard Press: Right
Move Grid: Right, 32

Find Keyboard Press in the Events category and select Right. Then find Move Grid in the Movement category, set direction to Right and grid size to 32.

Move Left, Up, Down

Repeat for the other three directions:

Keyboard Press: Left
Move Grid: Left, 32
Keyboard Press: Up
Move Grid: Up, 32
Keyboard Press: Down
Move Grid: Down, 32

Each Keyboard Press event fires once per key press — perfect for grid movement!

Stop at Walls

When colliding with obj_wall
Stop movement

Find Collision in Events and type obj_wall. Snap a Stop Movement block inside. This prevents the player from walking through walls.

Apply Your Blocks

  1. You should have 5 event blocks (4 keyboard + 1 collision)
  2. Click the Apply to Events button to save

Create the Room

  1. Right-click on Rooms → Create New Room
  2. Name it room_sokoban
  3. Set size to 320×320 pixels (a 10×10 grid)
  4. Enable grid snap: 32×32
  1. Place obj_wall all around the edges to form a closed room
  2. Add a few walls inside to create corridors and corners
  3. Place obj_player somewhere inside the room
How it works: Pressing an arrow key moves the player exactly 32 pixels (one grid cell). If the player hits a wall, the collision event fires and stops the movement.

Test Your Game!

Press F5 to play!

  • Use the arrow keys to walk around
  • The player moves exactly one cell per key press
  • Walls block your movement — you can't walk through them
