Step 5: Game Controller (Blockly)

We'll add a controller object to display instructions and let the player restart the level.

Create the Controller Object

  1. Right-click in the Asset Tree and create a new object: obj_controller
  2. No sprite is needed for this object
  3. Place one instance of obj_controller anywhere in your room

Draw Instructions (Draw GUI Event)

Open the Blockly editor for obj_controller and build:

Draw GUI
Draw Text "Push crates onto targets!" at 10, 10

Find Draw GUI in the Events category. Then find Draw Text in the Drawing category. Type Push crates onto targets! as the text, and set X to 10, Y to 10.

Draw GUI vs Draw: The Draw GUI event draws on top of everything and uses screen coordinates. It's perfect for HUD text that shouldn't scroll with the level.

Restart Level (Press R)

Keyboard Press: R
Restart Room

Find Keyboard Press in Events and select R. Then find Restart Room in the Room category. This lets the player reset the puzzle if they get stuck!

Apply Your Blocks

  1. You should have 2 event blocks (Draw GUI + Keyboard Press R)
  2. Click the Apply to Events button to save
How to win: When all crates turn green (showing spr_crate_ok), the puzzle is solved! Remember, the crate behavior from Step 3 already changes the crate's appearance when it's on a target spot.
Tip: Make sure the number of crates equals the number of targets in your level! Press R to restart if you get stuck.
