What We'll Create

In this tutorial, you'll build a classic Sokoban puzzle game step by step using Blockly visual blocks. Instead of building everything before testing, we'll take a progressive approach — you'll have a playable game at each phase!

How This Tutorial Works

Phase 1 — Walking Player in a Room
Create a player and walls. Walk around a room with grid-based movement. Playable right away!
Phase 2 — Crates to Push
Add crates the player can push. Crates stop at walls — the core Sokoban mechanic!
Phase 3 — Targets, Visual Feedback & Game Controller
Add target spots, crate color feedback, on-screen instructions, and a restart key.

What You'll Learn

  • Grid-based movement (moving in fixed 32-pixel steps)
  • Push mechanics — detecting and moving other objects
  • Visual feedback with sprite swapping
  • Drawing text on screen
  • Level design for puzzle games
Time Required: About 20–25 minutes
