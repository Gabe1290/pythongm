Phase 2: Pushing Crates

Now let's add crates the player can push — the core Sokoban mechanic!

Create the Crate Sprite

  1. Right-click on Sprites → Create New Sprite
  2. Name it spr_crate (32×32 pixels)
  3. Draw a brown box or import from Tutorials/05_sokoban/assets/
  4. Press Ctrl+S to save

Create the Crate Object

  1. Right-click on Objects → Create New Object
  2. Name it obj_crate, set its sprite to spr_crate
  3. Check the Solid checkbox

Update the Player: Push Crates

  1. Double-click obj_player to open the Object Editor
  2. Click the Blockly tab
  3. Add a new collision event for obj_crate:
When colliding with obj_crate
If Can Push facing
Push Other Instance 32

Find Collision in Events and type obj_crate. Then find If Can Push in Control — it checks whether the crate can be pushed in the direction the player is moving. Inside the "then" section, snap a Push Other Instance block with distance 32.

How pushing works: When the player collides with a crate, the engine checks if the space behind the crate (in the direction the player is moving) is free. If it is, the crate is pushed one cell. If there's a wall or another crate behind it, the push is blocked.

Program the Crate: Stop at Walls

  1. Double-click obj_crate to open the Object Editor
  2. Click the Blockly tab
  3. Add a collision event for obj_wall:
When colliding with obj_wall
Stop movement

This ensures crates stop when they hit a wall.

Click Apply to Events to save the crate's blocks.

Update the Room

  1. Open room_sokoban
  2. Place 2–3 obj_crate instances inside the room
  3. Make sure each crate has space to be pushed around
Tip: Don't place a crate directly against a corner — it would be stuck immediately!

Test Your Game!

Press F5 to play!

  • Walk up to a crate and push it with the arrow keys
  • Crates move one cell when pushed
  • Crates stop when they hit a wall
  • You cannot push a crate if another crate or wall is behind it
Click "Next" to add targets and finish the game!
