Step 1: Create the Sprites

We need four sprites for our maze game: a player, walls, an exit, and coins. For each sprite you can draw it yourself using the built-in Sprite Editor, or load a ready-made sample image. Both options are explained below.

SpriteSizeDescription
spr_player24×24Small blue character
spr_wall32×32Brick / stone pattern
spr_exit32×32Green door / portal
spr_coin16×16Small gold coin

Option A — Draw it yourself

  1. Double-click spr_player to open the Sprite Editor
  2. Draw a small character (a circle with eyes, or a simple person)
  3. Use a bright color like blue or green
  4. Keep the size at 24×24 pixels (smaller than walls for easier movement)
  5. Save your work with Ctrl+S
