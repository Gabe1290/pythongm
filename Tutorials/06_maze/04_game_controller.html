Phase 3: Game Controller

The game works, but you cannot see your score on screen yet! In this final phase, you will create an invisible game controller object that displays the score and resets it at the start of the game.

What is a game controller? It's an object with no sprite that manages things like the score display. Since it's invisible, it won't interfere with the game — it just runs in the background.

Step 1: Create the Object

  1. Right-click on Objects → Create New Object
  2. Name it obj_game_controller
  3. No sprite needed — leave the sprite field empty
  4. Leave "Solid" unchecked

Step 2: Program the Game Controller (Blockly)

  1. Double-click obj_game_controller to open the Object Editor
  2. Click the Blockly tab

Reset Score at Start

Create Event
Set score to 0

From Events, drag a Create Event block. From Score/Lives/Health, snap Set Score inside and type 0.

This ensures the score starts at zero each time the room loads.

Display Score on Screen

Draw Event
Draw score at x: 10 y: 10

From Events, drag a Draw Event block. From Score/Lives/Health, snap Draw Score inside. Set x to 10 and y to 10 (top-left corner).

Apply Your Blocks

  1. You should have 2 event blocks in your workspace (Create and Draw)
  2. Click the Apply to Events button to save
How it works: The Create Event runs once when the object appears in the room, setting the score to 0. The Draw Event runs every frame and displays "Score: X" in the top-left corner. As the player collects coins, the score updates automatically!

Step 3: Place in the Room

  1. Open room_maze
  2. Select obj_game_controller and place it anywhere in the room
Don't forget! The game controller must be placed in the room for the score to display. Since it has no sprite, you won't see it during gameplay, but it will still be running.

Test Your Game!

Press F5 to play!

  • The score should display as "Score: 0" in the top-left corner
  • Collect coins — the score should increase by 10 for each coin
  • Navigate to the exit — a "You Win!" message appears
  • After the message, the room restarts and the score resets to 0

Congratulations! You have built a complete maze game!

Your game has a player that navigates through corridors, collectible coins, a score display, and a victory exit. Well done!

Challenge Ideas

Want to make your maze game even better? Try these challenges:

  • Multiple levels: Create additional rooms (room_maze2, room_maze3) with harder mazes. Change the exit's Restart room block to Go to next room.
  • Timer: Add a countdown timer using a variable. If time runs out, restart the room!
  • Enemies: Create a patrolling enemy object that moves back and forth. If the player collides with the enemy, restart the room.
  • Keys and doors: Add a key object that must be collected before the exit opens.
