Step 4: Coins and Exit (Blockly)

Now let's make coins collectible and the exit work using Blockly blocks.

Coin Collection (obj_coin)

  1. Double-click obj_coin to open the Object Editor
  2. Click the Blockly tab
  3. Build these blocks:
When colliding with obj_player
Add to score 10
Destroy this instance

From Events, drag a Collision block and type obj_player. From Score/Lives/Health, snap Add to Score inside and type 10. From Instance, snap Destroy Instance below it.

  1. Click Apply to Events
How it works: When the player touches a coin, 10 points are added to the score and the coin disappears.

Exit Behavior (obj_exit)

  1. Double-click obj_exit to open the Object Editor
  2. Click the Blockly tab
  3. Build these blocks:
When colliding with obj_player
Show message "You Win!"
Restart room

From Events, drag a Collision block and type obj_player. From Output, snap Show Message inside and type You Win!. From Room, snap Restart Room below it.

  1. Click Apply to Events
Tip: If you create multiple levels, use Go to next room (from the Room category) instead of Restart room to advance to the next maze!
