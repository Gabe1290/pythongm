Step 2: Create the Objects

Now we'll create objects for walls, exit, coins, and the player.

Create the Wall Object

  1. Right-click on Objects and select Create Object
  2. Name it obj_wall
  3. Set the sprite to spr_wall
  4. Check "Solid"
  5. No events needed

Create the Exit Object

  1. Create object obj_exit
  2. Set sprite to spr_exit
  3. Leave "Solid" unchecked

Create the Coin Object

  1. Create object obj_coin
  2. Set sprite to spr_coin
  3. Leave "Solid" unchecked

Create the Player Object

  1. Create object obj_player
  2. Set sprite to spr_player
  3. Leave "Solid" unchecked

Create the Game Controller

  1. Create object obj_game_controller
  2. No sprite needed (leave it empty)
  3. Leave "Solid" unchecked
Note: Only the wall needs to be "Solid". The game controller is an invisible object that will display the score on screen.
