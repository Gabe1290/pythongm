Phase 1: Player and Maze

In this phase you will create the player and wall sprites, build objects, program movement with Blockly, and design a simple maze. By the end, you can walk around inside your maze!

Step 1: Create the Sprites

spr_player (32x32)

  1. In the Asset Tree, right-click on Sprites and select Create New Sprite
  2. Name it spr_player

Option A — Draw it yourself

  1. Double-click spr_player to open the Sprite Editor
  2. Set the size to 32x32 pixels
  3. Draw a small character (a circle with eyes, or a simple person)
  4. Use a bright color like blue
  5. Save with Ctrl+S

Option B — Load the sample sprite

  1. Right-click on spr_player in the Asset Tree
  2. Select Import Image...
  3. Browse to Tutorials/06_maze/assets/
  4. Select spr_player.png and click Open

spr_wall (32x32)

  1. Create a new sprite named spr_wall

Option A — Draw it yourself

  1. Open the Sprite Editor and set the size to 32x32 pixels
  2. Draw a brick or stone pattern
  3. Use gray or brown colors

Option B — Load the sample sprite

  1. Right-click spr_wall → Import Image...
  2. Select Tutorials/06_maze/assets/spr_wall.png

Step 2: Create the Objects

obj_wall (Solid)

  1. Right-click on Objects → Create New Object
  2. Name it obj_wall
  3. Set the sprite to spr_wall
  4. Check "Solid" — this is essential so the player cannot walk through walls
  5. No events needed for the wall

obj_player

  1. Create a new object named obj_player
  2. Set the sprite to spr_player
  3. Leave "Solid" unchecked

Step 3: Program the Player (Blockly)

How Blockly works: Drag blocks from the toolbox on the left into the workspace. Snap action blocks inside event blocks to define what happens. When you're done, click Apply to Events to save your blocks.
  1. Double-click obj_player in the Asset Tree to open the Object Editor
  2. Click the Blockly tab at the top of the editor

Move Right

Keyboard: Right (held)
Set horizontal speed to 4

Find Keyboard (held) in the Events category and select Right from the dropdown. Then find Set Horizontal Speed in the Movement category and snap it inside. Type 4 in the number field.

Move Left

Keyboard: Left (held)
Set horizontal speed to -4

Same as above, but choose Left and type -4 (negative = left direction).

Move Down

Keyboard: Down (held)
Set vertical speed to 4

Use Set Vertical Speed from Movement. Positive = downward.

Move Up

Keyboard: Up (held)
Set vertical speed to -4

Type -4 for upward movement.

Stop When No Key Pressed

Keyboard: No key
Stop movement

Find Keyboard: No key in Events and Stop Movement in Movement. This prevents the player from sliding forever!

Stop at Walls

When colliding with obj_wall
Stop movement

Find Collision in Events. Type obj_wall in the text field. Snap a Stop Movement block inside.

Apply Your Blocks

  1. You should have 6 event blocks in your workspace
  2. Click the Apply to Events button to save
How it works: Each arrow key sets the speed in that direction. When you release all keys, the player stops. If the player hits a wall, it also stops.

Step 4: Create the Room

  1. Right-click Rooms → Create New Room
  2. Name it room_maze
  3. Set size to 640x480
  4. Enable Snap to Grid: 32x32

Build a Simple Maze

  1. Select obj_wall and click to place walls around the border
  2. Add internal walls to create corridors and paths
  3. Place obj_player near the top-left of the maze
  4. Leave some open space so you can walk around

Test Your Game!

Press F5 to play!

  • Use the arrow keys to move the player
  • The player should stop when hitting walls
  • The player should stop when you release all keys
  • Try navigating through the corridors of your maze
