MAZE GAME

Find your way through the maze and collect coins!

What We'll Create

In this tutorial, you'll build a maze navigation game step by step. The game is playable at the end of each phase, so you can test as you go!

Phase 1 — Walking in a Maze
Create a player that moves with arrow keys and walls that block the path. Build a simple maze and walk around in it.
Phase 2 — Collecting Coins and Exit
Add collectible coins that increase your score and an exit door that displays a victory message.
Phase 3 — Score Display
Add a game controller that shows the score on screen so the player can see their progress.

What You'll Learn

  • Using Blockly visual blocks to program game behavior
  • Smooth movement with keyboard input blocks
  • Wall collision detection and handling
  • Creating collectible items
  • Score display with a game controller object
  • Level design for maze games
Time Required: About 20–25 minutes
