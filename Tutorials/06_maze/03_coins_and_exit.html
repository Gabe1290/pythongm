Phase 2: Coins and Exit

Your maze is already walkable! Now let's add a goal: collect coins for points and reach the exit to win.

Step 1: Create the Sprites

spr_coin (32x32)

  1. Right-click on Sprites → Create New Sprite
  2. Name it spr_coin

Option A — Draw it yourself

  1. Open the Sprite Editor and set the size to 32x32 pixels
  2. Draw a small yellow or gold circle

Option B — Load the sample sprite

  1. Right-click spr_coin → Import Image...
  2. Browse to Tutorials/06_maze/assets/
  3. Select spr_coin.png

spr_exit (32x32)

  1. Create a new sprite named spr_exit

Option A — Draw it yourself

  1. Set the size to 32x32 pixels
  2. Draw a door, flag, or glowing marker
  3. Use green or gold colors so it stands out

Option B — Load the sample sprite

  1. Right-click spr_exit → Import Image...
  2. Select Tutorials/06_maze/assets/spr_exit.png

Step 2: Create the Objects

obj_coin

  1. Right-click on Objects → Create New Object
  2. Name it obj_coin
  3. Set the sprite to spr_coin
  4. Leave "Solid" unchecked

obj_exit

  1. Create a new object named obj_exit
  2. Set the sprite to spr_exit
  3. Leave "Solid" unchecked

Step 3: Program the Coin (Blockly)

  1. Double-click obj_coin in the Asset Tree to open the Object Editor
  2. Click the Blockly tab
  3. Build these blocks:
When colliding with obj_player
Add to score 10
Destroy this instance

From Events, drag a Collision block and type obj_player. From Score/Lives/Health, snap Add to Score inside and type 10. From Instance, snap Destroy Instance below it.

  1. Click Apply to Events
How it works: When the player touches a coin, 10 points are added to the score and the coin disappears.

Step 4: Program the Exit (Blockly)

  1. Double-click obj_exit in the Asset Tree to open the Object Editor
  2. Click the Blockly tab
  3. Build these blocks:
When colliding with obj_player
Show message "You Win!"
Restart room

From Events, drag a Collision block and type obj_player. From Output, snap Show Message inside and type You Win!. From Room, snap Restart Room below it.

  1. Click Apply to Events
Tip: If you later create multiple levels, use Go to next room (from the Room category) instead of Restart room to advance to the next maze!

Step 5: Update Your Room

  1. Open room_maze by double-clicking it
  2. Select obj_coin and scatter several coins along the corridors of your maze
  3. Select obj_exit and place it at the end of the maze (far from the player start)
Design tip: Place coins along different paths to reward exploration. Put the exit in the most distant corner from the player's starting position to make the maze more interesting.

Test Your Game!

Press F5 to play!

  • Navigate through the maze and touch coins — they should disappear
  • Each coin adds 10 points to your score (you will see it on screen in Phase 3)
  • Reach the exit — a "You Win!" message should appear
  • After the message, the room restarts so you can play again
