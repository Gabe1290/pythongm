Давай додамо кілька покращень, щоб твоя гра виглядала завершеною!

Тримати гравця на екрані

1 Додай перевірку меж

У obj_player додай подію Step з такими умовами:

  • Якщо x < 0, встановити x = 0
  • Якщо x > room_width - sprite_width, встановити x = room_width - sprite_width

Це не дозволяє гравцю виходити за бічні краї екрана.

Додати колір фону

2 Встанови фон кімнати

Відкрий room_game у редакторі кімнат.

У властивостях кімнати встанови колір фону на гарний блакитний або темний космічний колір.

Експортуй свою гру!

3 Експортувати як HTML5

Перейди до Файл > Експортувати як HTML5.

Вибери місце збереження і натисни Експортувати.

Відкрий HTML-файл у веб-браузері, щоб пограти в свою гру!

Вітаємо! Ти створив свою першу повноцінну гру в PyGameMaker!

Ідеї для покращення твоєї гри

Виклики для спроби:
  • Додай життя — втрачай життя, коли зірка досягає низу
  • Зроби так, щоб зірки падали швидше, коли рахунок зростає
  • Додай зірки різних кольорів, що дають різну кількість очок
  • Додай "поганий" об'єкт (бомбу), який закінчує гру, якщо його зловити
  • Додай звукові ефекти при ловінні зірок
  • Створи титульний екран та екран закінчення гри
