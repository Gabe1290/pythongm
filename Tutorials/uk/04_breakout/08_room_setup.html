Крок 7: Налаштування кімнати

Час створити ігрову кімнату та розмістити всі об'єкти!

Створення кімнати

  1. Клацніть правою кнопкою миші на Rooms і виберіть Створити кімнату
  2. Назвіть її room_breakout
  3. Встановіть розмір 640x480 пікселів

Схема кімнати

+--[СТІНА]--------------------[СТІНА]--+
|  [СТІНА]  [Ч][Ч][Ч][Ч][Ч]  [СТІНА]  |
|  [СТІНА]  [П][П][П][П][П]  [СТІНА]  |
|  [СТІНА]  [Ж][Ж][Ж][Ж][Ж]  [СТІНА]  |
|  [СТІНА]  [З][З][З][З][З]  [СТІНА]  |
|  [СТІНА]                   [СТІНА]  |
|  [СТІНА]        (o)        [СТІНА]  |
|  [СТІНА]                   [СТІНА]  |
|  [СТІНА]     [РАКЕТКА]     [СТІНА]  |
+--[ЗОНА СМЕРТІ ЗОНА СМЕРТІ ЗОНА]-----+

Розміщення об'єктів

1. Стіни

Розмістіть obj_wall вздовж верхнього, лівого та правого країв.

2. Зона смерті

Розмістіть obj_death_zone вздовж нижнього краю (вона невидима).

3. Цегла

Розташуйте цеглу рядами біля верхньої частини:

  • Верхній ряд: Червона цегла (obj_brick_red)
  • Другий ряд: Помаранчева цегла
  • Третій ряд: Жовта цегла
  • Четвертий ряд: Зелена цегла

4. Ракетка

Розмістіть obj_paddle біля нижньої частини, по центру горизонтально.

5. М'яч

Розмістіть obj_ball трохи вище ракетки.

6. Ігровий контролер

Розмістіть obj_game_controller будь-де (він не має спрайту).

Вітаємо! Ваша гра Breakout готова!

Тестування гри

  • Натисніть F5 або клацніть Run
  • Використовуйте клавіші вліво та вправо для руху
  • Розбийте всю цеглу, щоб перемогти!
