Крок 3: Додавання руху ракетки

Ракетка рухається вліво та вправо за допомогою клавіш зі стрілками.

Рух вліво (стрілка вліво)

  1. Відкрийте obj_paddle
  2. Додайте подію: Keyboard > Press Left
  3. Додайте дію: Move > Set Horizontal Speed
  4. Встановіть швидкість -8 (від'ємне = вліво)

Зупинка руху (відпускання лівої клавіші)

  1. Додайте подію: Keyboard > Release Left
  2. Додайте дію: Move > Set Horizontal Speed
  3. Встановіть швидкість 0

Рух вправо (стрілка вправо)

  1. Додайте подію: Keyboard > Press Right
  2. Додайте дію: Move > Set Horizontal Speed
  3. Встановіть швидкість 8 (додатне = вправо)

Зупинка руху (відпускання правої клавіші)

  1. Додайте подію: Keyboard > Release Right
  2. Додайте дію: Move > Set Horizontal Speed
  3. Встановіть швидкість 0

Зіткнення зі стіною

  1. Додайте подію: Collision > obj_wall
  2. Додайте дію: Move > Bounce Against Objects
  3. Виберіть "Against solid objects"
Примітка: Зіткнення зі стіною запобігає виходу ракетки за межі екрана!
Натисніть "Далі", щоб додати рух м'яча.
