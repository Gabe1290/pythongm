Крок 1: Створення спрайтів

Нам потрібні спрайти для ракетки, м'яча, цегли та стін.

Створення спрайту м'яча

  1. Клацніть правою кнопкою миші на Sprites у дереві ресурсів
  2. Виберіть Створити спрайт
  3. Назвіть його spr_ball
  4. Натисніть Редагувати спрайт
  5. Намалюйте маленьке біле або жовте коло (16x16 пікселів)
  6. Натисніть OK, щоб зберегти

Створення спрайту ракетки

  1. Створіть новий спрайт з назвою spr_paddle
  2. Намалюйте широкий плоский прямокутник (64x16 пікселів)
  3. Використовуйте яскравий колір, наприклад синій

Створення спрайтів цегли

Створіть кольорову цеглу для різних значень балів:

  1. Створіть spr_brick_red - Червона цегла (32x16 пікселів)
  2. Створіть spr_brick_orange - Помаранчева цегла
  3. Створіть spr_brick_yellow - Жовта цегла
  4. Створіть spr_brick_green - Зелена цегла

Створення спрайту стіни

  1. Створіть новий спрайт з назвою spr_wall
  2. Намалюйте суцільний сірий прямокутник (32x32 пікселі)
Порада: Зберігайте однаковий розмір цегли, щоб вона гарно вирівнювалась у кімнаті!
Натисніть "Далі", щоб створити ігрові об'єкти.
