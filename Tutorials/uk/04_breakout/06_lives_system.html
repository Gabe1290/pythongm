Крок 5: Додавання системи життів

Гравець втрачає життя, коли м'яч падає за нижній край екрана.

Створення зони смерті

  1. Створіть спрайт spr_death_zone (32x32, будь-який колір)
  2. Створіть об'єкт obj_death_zone
  3. Зніміть галочку "Visible" - він має бути невидимим
  4. Поставте галочку "Solid"

М'яч потрапляє в зону смерті

В obj_ball:

  1. Додайте подію: Collision > obj_death_zone
  2. Додайте дію: Score > Subtract from Lives
  3. Значення: 1
  4. Додайте дію: Move > Jump to Start Position

Створення ігрового контролера

  1. Створіть об'єкт obj_game_controller
  2. Спрайт не потрібен

Ініціалізація життів (подія Create)

В obj_game_controller:

  1. Додайте подію: Create
  2. Додайте дію: Score > Set Lives
  3. Значення: 3
  4. Додайте дію: Score > Set Score
  5. Значення: 0

Перевірка кінця гри (подія Step)

  1. Додайте подію: Step
  2. Додайте дію: Control > Test Lives
  3. Перевірте, чи кількість життів менша або дорівнює 0
  4. Додайте дію: Main2 > Show Message
  5. Повідомлення: Game Over!
  6. Додайте дію: Main2 > Restart Room
Пам'ятайте: Розмістіть obj_game_controller у вашій кімнаті!
Натисніть "Далі", щоб додати відображення рахунку.
