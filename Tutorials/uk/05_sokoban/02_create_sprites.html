Крок 1: Створення спрайтів

Усі спрайти повинні бути однакового розміру (32x32 пікселі), щоб створити правильну сітку.

Створення спрайта гравця

  1. Клацніть правою кнопкою миші на Спрайти у дереві ресурсів
  2. Виберіть Створити спрайт
  3. Назвіть його spr_player
  4. Натисніть Редагувати спрайт
  5. Намалюйте простого персонажа (32x32 пікселі)
  6. Використовуйте виразний колір, наприклад, синій
  7. Натисніть OK для збереження

Створення спрайтів ящиків

Звичайний ящик:

  1. Створіть спрайт spr_crate
  2. Намалюйте форму дерев'яного ящика (32x32 пікселі)
  3. Використовуйте коричневий або помаранчевий колір

Ящик на цілі:

  1. Створіть спрайт spr_crate_ok
  2. Намалюйте такий самий ящик, але зеленого кольору
  3. Це показує, коли ящик правильно розміщений

Створення спрайта стіни

  1. Створіть спрайт spr_wall
  2. Намалюйте суцільний візерунок цегли або каменю (32x32 пікселі)
  3. Використовуйте сірий або темний колір

Створення спрайта цілі

  1. Створіть спрайт spr_target
  2. Намалюйте позначку X або індикатор цілі (32x32 пікселі)
  3. Використовуйте яскравий колір, наприклад, червоний
Важливо: Зберігайте ВСІ спрайти розміром точно 32x32 пікселі для правильного вирівнювання по сітці!
Натисніть "Далі", щоб створити ігрові об'єкти.
