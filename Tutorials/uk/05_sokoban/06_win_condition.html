Крок 5: Умова перемоги

Гравець перемагає, коли всі ящики знаходяться на цільових місцях.

Створення контролера гри

  1. Створіть об'єкт obj_game_controller
  2. Спрайт не потрібен

Підрахунок цілей (Подія Create)

У obj_game_controller:

  1. Додайте подію: Create
  2. Додайте дію: Control > Execute Code
  3. Введіть: global.total_targets = instance_number(obj_target);

Перевірка умови перемоги (Подія Step)

  1. Додайте подію: Step
  2. Додайте дію: Control > Execute Code
  3. Введіть цей код:
// Підрахунок ящиків на цілях
var crates_on_targets = 0;
with (obj_crate) {
    if (place_meeting(x, y, obj_target)) {
        crates_on_targets += 1;
    }
}

// Перевірка, чи всі цілі мають ящики
if (crates_on_targets >= global.total_targets
    && global.total_targets > 0) {
    show_message("Рівень пройдено!");
    room_restart();
}

Відображення інструкцій (Подія Draw)

  1. Додайте подію: Draw
  2. Додайте дію: Draw > Draw Text
  3. Текст: Штовхайте ящики на цілі!
  4. X: 10, Y: 10
Порада: Переконайтеся, що кількість ящиків дорівнює кількості цілей на вашому рівні!
Натисніть "Далі", щоб налаштувати ігрову кімнату.
