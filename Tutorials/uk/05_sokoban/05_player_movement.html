Крок 4: Рух гравця

Гравець рухається фіксованими кроками по 32 пікселі (рух по сітці).

Важливо: У Сокобані рух повинен перевіряти:
1. Чи є там стіна? (Не можна рухатись)
2. Чи є там ящик? (Чи можемо ми його штовхнути?)
3. Чи є стіна/ящик за ящиком? (Не можна штовхнути)

Рух вправо (Стрілка вправо)

У obj_player:

  1. Додайте подію: Keyboard > Press Right
  2. Додайте дію: Control > Execute Code
  3. Введіть цей код:
// Перевіряємо, чи можемо рухатись вправо
if (!place_meeting(x + 32, y, obj_wall)) {
    var crate = instance_place(x + 32, y, obj_crate);
    if (crate != noone) {
        // Там є ящик - чи можемо ми його штовхнути?
        if (!place_meeting(x + 64, y, obj_wall) &&
            !place_meeting(x + 64, y, obj_crate)) {
            crate.x += 32;
            x += 32;
        }
    } else {
        // Немає ящика, просто рухаємось
        x += 32;
    }
}

Рух вліво (Стрілка вліво)

Той самий код, але змініть:

  • + 32 стає - 32
  • + 64 стає - 64

Рух вгору (Стрілка вгору)

Той самий код, але використовуйте y замість x:

  • x + 32 стає y - 32
  • x + 64 стає y - 64
  • crate.x стає crate.y

Рух вниз (Стрілка вниз)

Те саме, що і вгору, але з + 32 та + 64.

Натисніть "Далі", щоб додати визначення умови перемоги.
