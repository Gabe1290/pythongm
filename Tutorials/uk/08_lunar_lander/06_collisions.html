Крок 5: Виявлення зіткнень

Обробка посадки на майданчик та зіткнення з місцевістю.

Зіткнення з посадковим майданчиком

У obj_lander:

  1. Додайте подію: Collision > obj_pad
  2. Додайте дію: Control > Execute Code
// Обчислення загальної швидкості
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    // Безпечна посадка!
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("Ідеальна посадка! Ви перемогли!");
} else {
    // Занадто швидко - аварія!
    crashed = true;
    show_message("Аварія! Занадто швидко!");
    room_restart();
}

Зіткнення з землею (аварія)

У obj_lander:

  1. Додайте подію: Collision > obj_ground
  2. Додайте дію: Control > Execute Code
crashed = true;
show_message("Зіткнення з місцевістю!");
room_restart();

Обчислення швидкості

Ми використовуємо теорему Піфагора для отримання загальної швидкості:

total_speed = sqrt(hsp*hsp + vsp*vsp)

Це об'єднує горизонтальну та вертикальну швидкість в одне значення.

Безпечна швидкість - 2! Якщо total_speed > 2, ви розіб'єтесь навіть на майданчику.
Натисніть "Далі", щоб додати HUD дисплей.
