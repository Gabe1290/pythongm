Крок 6: Ігровий контролер (HUD)

Відображення палива, швидкості та статусу для допомоги гравцю.

Подія Draw

У obj_game_controller:

  1. Додайте подію: Draw > Draw
  2. Додайте дію: Control > Execute Code
draw_set_color(c_white);

// Отримання інформації про модуль
var lander = instance_find(obj_lander, 0);
if (lander != noone) {
    var spd = sqrt(lander.hsp*lander.hsp + lander.vsp*lander.vsp);

    // Малювання HUD
    draw_text(10, 10, "ПОСАДКОВИЙ МОДУЛЬ");
    draw_text(10, 30, "Паливо: " + string(floor(lander.fuel)) + "%");
    draw_text(10, 50, "Швидкість: " + string_format(spd, 1, 2));
    draw_text(10, 70, "Безпечно: < " + string(lander.safe_speed));

    // Попередження про швидкість
    if (spd > lander.safe_speed) {
        draw_set_color(c_red);
        draw_text(10, 90, "ЗАНАДТО ШВИДКО!");
    } else {
        draw_set_color(c_lime);
        draw_text(10, 90, "Швидкість ОК");
    }

    // Попередження про паливо
    if (lander.fuel <= 0) {
        draw_set_color(c_red);
        draw_text(10, 110, "НЕМАЄ ПАЛИВА!");
    }
}

draw_set_color(c_gray);
draw_text(10, room_height - 20, "ВГОРУ: Тяга | ВЛІВО/ВПРАВО: Рух");

Що показує HUD

  • Паливо: Відсоток залишку (зелений = ОК)
  • Швидкість: Поточна швидкість (червоний = небезпечно)
  • Попередження: "ЗАНАДТО ШВИДКО!" або "НЕМАЄ ПАЛИВА!"
  • Керування: Нагадування внизу
