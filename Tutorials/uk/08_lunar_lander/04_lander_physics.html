Step 3: Lander Physics (Create Event)

Set up variables for gravity, thrust, fuel, and velocity.

Initialize Physics Variables

In obj_lander:

  1. Add Event: Create
  2. Add Action: Control > Execute Code
// Physics settings
gravity_force = 0.05;  // Gentle gravity
thrust_force = 0.1;    // Thrust power
max_speed = 5;         // Maximum velocity

// Current velocity
hsp = 0;  // Horizontal speed
vsp = 0;  // Vertical speed

// Fuel system
fuel = 100;
fuel_use = 0.5;  // Fuel per frame

// Game state
landed = false;
crashed = false;

// Safe landing speed
safe_speed = 2;

Understanding the Variables

Variable Purpose
gravity_force Constant downward pull (0.05 is gentle)
thrust_force Upward force when pressing UP
fuel 100% fuel, decreases when thrusting
safe_speed Max speed for safe landing (2)
Balance: Thrust (0.1) is stronger than gravity (0.05) so you can slow down!
