Крок 3: Фізика посадкового модуля (Подія Create)

Налаштуйте змінні для гравітації, тяги, палива та швидкості.

Ініціалізація змінних фізики

У obj_lander:

  1. Додайте подію: Create
  2. Додайте дію: Control > Execute Code
// Налаштування фізики
gravity_force = 0.05;  // Помірна гравітація
thrust_force = 0.1;    // Сила тяги
max_speed = 5;         // Максимальна швидкість

// Поточна швидкість
hsp = 0;  // Горизонтальна швидкість
vsp = 0;  // Вертикальна швидкість

// Система палива
fuel = 100;
fuel_use = 0.5;  // Паливо за кадр

// Стан гри
landed = false;
crashed = false;

// Безпечна швидкість посадки
safe_speed = 2;

Розуміння змінних

Змінна Призначення
gravity_force Постійне тяжіння вниз (0.05 - помірне)
thrust_force Сила вгору при натисканні ВГОРУ
fuel 100% палива, зменшується при тязі
safe_speed Максимальна швидкість для безпечної посадки (2)
Баланс: Тяга (0.1) сильніша за гравітацію (0.05), тому ви можете сповільнюватись!
Натисніть "Далі", щоб додати рух та керування.
