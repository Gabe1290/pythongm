Крок 1: Створення спрайтів

Нам потрібні спрайти для посадкового модуля, посадкового майданчика та місцевості.

Назва спрайту Розмір Опис
spr_lander 32x32 Космічний корабель (трикутник або класична форма модуля)
spr_pad 64x16 Посадкова платформа з позначкою "H"
spr_ground 32x32 Скелястий рельєф (сірий/коричневий)

Створення спрайту посадкового модуля

  1. Клацніть правою кнопкою Спрайти > Створити спрайт
  2. Назвіть його spr_lander
  3. Натисніть Редагувати спрайт
  4. Намалюйте космічний корабель (32x32 пікселі)
  5. Встановіть початок координат по центру знизу для правильної посадки

Створення спрайту посадкового майданчика

  1. Створіть спрайт з назвою spr_pad
  2. Розмір: 64x16 пікселів
  3. Намалюйте плоску платформу з позначками
  4. Використовуйте яскраві кольори (жовтий/зелений)

Створення спрайту землі

  1. Створіть спрайт з назвою spr_ground
  2. Розмір: 32x32 пікселі
  3. Намалюйте скелястий, нерівний рельєф
  4. Використовуйте сірий/коричневий кольори
Важливо: Встановіть початок координат модуля по центру знизу, щоб він правильно сідав на майданчик!
Натисніть "Далі", щоб створити ігрові об'єкти.
