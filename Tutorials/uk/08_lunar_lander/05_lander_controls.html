Додайте гравітацію, тягу, горизонтальний рух та перевірку меж.

Додавання події Step

У obj_lander:

  1. Додайте подію: Step > Step
  2. Додайте дію: Control > Execute Code
// Не оновлювати, якщо гра закінчена
if (landed || crashed) exit;

// === ГРАВІТАЦІЯ ===
vsp += gravity_force;

// === ТЯГА (Стрілка вгору) ===
if (keyboard_check(vk_up) && fuel > 0) {
    vsp -= thrust_force;
    fuel -= fuel_use;
    if (fuel < 0) fuel = 0;
}

// === ГОРИЗОНТАЛЬНЕ КЕРУВАННЯ ===
if (keyboard_check(vk_left)) hsp -= 0.05;
if (keyboard_check(vk_right)) hsp += 0.05;

// === ОБМЕЖЕННЯ ШВИДКОСТІ ===
hsp = clamp(hsp, -max_speed, max_speed);
vsp = clamp(vsp, -max_speed, max_speed);

// === ЗАСТОСУВАННЯ РУХУ ===
x += hsp;
y += vsp;

// === МЕЖІ ЕКРАНУ ===
if (x < 16) { x = 16; hsp = 0; }
if (x > room_width - 16) { x = room_width - 16; hsp = 0; }
if (y < 16) { y = 16; vsp = 0; }

Як це працює

  1. Гравітація: Додається до vsp кожен кадр (падіння)
  2. Тяга: Віднімається від vsp (підйом), використовує паливо
  3. Clamp: Обмежує швидкість до безпечного діапазону
  4. Межі: Тримає модуль на екрані
Ключове: Функція clamp() обмежує значення між мінімумом та максимумом!
Натисніть "Далі", щоб додати виявлення зіткнень.
