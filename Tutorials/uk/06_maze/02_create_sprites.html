Крок 1: Створення спрайтів

Нам потрібні спрайти для гравця, стін, виходу та монет. Стіни мають бути 32x32 пікселі для створення сітки.

Створення спрайта гравця

  1. Клацніть правою кнопкою миші на Спрайти та виберіть Створити спрайт
  2. Назвіть його spr_player
  3. Натисніть Редагувати спрайт
  4. Намалюйте невеликого персонажа (коло або просту фігурку)
  5. Використовуйте яскравий колір, наприклад синій або зелений
  6. Розмір: 24x24 пікселі (менше за стіни)

Створення спрайта стіни

  1. Створіть новий спрайт з назвою spr_wall
  2. Намалюйте цегляний або кам'яний візерунок
  3. Використовуйте сірий або темні кольори
  4. Розмір: 32x32 пікселі

Створення спрайта виходу

  1. Створіть новий спрайт з назвою spr_exit
  2. Намалюйте двері, прапор або світний маркер
  3. Використовуйте зелений або золотий кольори
  4. Розмір: 32x32 пікселі

Створення спрайта монети

  1. Створіть новий спрайт з назвою spr_coin
  2. Намалюйте маленьке жовте/золоте коло
  3. Розмір: 16x16 пікселів
Порада: Якщо спрайт гравця менший за стіни, легше рухатися коридорами!
Натисніть "Далі", щоб створити ігрові об'єкти.
