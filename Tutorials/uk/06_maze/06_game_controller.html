Крок 5: Контролер гри

Створіть контролер для відстеження часу та відображення рахунку.

Створення об'єкта контролера

  1. Створіть об'єкт obj_game_controller
  2. Спрайт не потрібен (залиште порожнім)

Ініціалізація таймера (Подія Create)

  1. Додати подію: Create
  2. Додати дію: Control > Set Variable
  3. Змінна: global.timer
  4. Значення: 0

Оновлення таймера (Подія Step)

  1. Додати подію: Step
  2. Додати дію: Control > Set Variable
  3. Змінна: global.timer
  4. Значення: 1/room_speed
  5. Поставте галочку "Relative"

Відображення інформації (Подія Draw)

  1. Додати подію: Draw
  2. Додати дію: Control > Execute Code
draw_set_color(c_white);
draw_text(10, 10, "Рахунок: " + string(score));
draw_text(10, 30, "Час: " + string(floor(global.timer)) + "с");

var coins = instance_number(obj_coin);
draw_text(10, 50, "Монети: " + string(coins));
Примітка: Контролер невидимий, але необхідний для відстеження стану гри!
Натисніть "Далі", щоб налаштувати кімнату.
