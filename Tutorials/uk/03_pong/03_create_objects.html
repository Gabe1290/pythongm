Step 2: Create the Objects

Now we'll create the game objects. Each object uses a sprite and can have behaviors.

Create the Wall Object

  1. Right-click on Objects and select Create Object
  2. Name it obj_wall
  3. Set the sprite to spr_wall
  4. Check the "Solid" checkbox - this is important!
  5. No events needed - walls just block movement

Create the Ball Object

  1. Create a new object named obj_ball
  2. Set the sprite to spr_ball
  3. Leave "Solid" unchecked

Create the Paddle Objects

Left Paddle:

  1. Create object obj_paddle_left
  2. Set sprite to spr_paddle_left
  3. Check "Solid"

Right Paddle:

  1. Create object obj_paddle_right
  2. Set sprite to spr_paddle_right
  3. Check "Solid"
Important: The "Solid" checkbox is crucial for bouncing to work correctly!
Click "Next" to add paddle movement controls.
