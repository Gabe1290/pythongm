Крок 1: Створення спрайтів

Нам потрібні чотири спрайти для гри Pong: м'яч, дві ракетки та стіни.

Створення спрайта м'яча

  1. Клацніть правою кнопкою миші на Спрайти в дереві ресурсів
  2. Виберіть Створити спрайт
  3. Назвіть його spr_ball
  4. Натисніть Редагувати спрайт
  5. Намалюйте маленьке біле коло (приблизно 16x16 пікселів)
  6. Натисніть OK, щоб зберегти

Створення спрайтів ракеток

Ліва ракетка (Гравець 1):

  1. Створіть новий спрайт з назвою spr_paddle_left
  2. Намалюйте високий вузький прямокутник (приблизно 16x64 пікселів)
  3. Використовуйте синій колір для ідентифікації Гравця 1

Права ракетка (Гравець 2):

  1. Створіть новий спрайт з назвою spr_paddle_right
  2. Намалюйте високий вузький прямокутник (приблизно 16x64 пікселів)
  3. Використовуйте червоний колір для ідентифікації Гравця 2

Створення спрайта стіни

  1. Створіть новий спрайт з назвою spr_wall
  2. Намалюйте суцільний сірий прямокутник (32x32 пікселів)
  3. Він буде розтягнутий для формування верхньої та нижньої меж
Порада: Робіть усі спрайти простими! Класичний Pong використовував базові форми.
Натисніть "Далі", щоб створити ігрові об'єкти.
