Крок 7: Налаштування кімнати

Тепер створимо ігрову кімнату та розмістимо всі наші об'єкти!

Створення кімнати

  1. Клацніть правою кнопкою миші на Кімнати та виберіть Створити кімнату
  2. Назвіть її room_pong
  3. Встановіть розмір 640x480 пікселів

Схема кімнати

+--[СТІНА СТІНА СТІНА СТІНА СТІНА]--+
|                                   |
[В]  [Г1]                    [Г2]  [В]
[О]  [  ]       (o)          [  ]  [О]
[Р]  [  ]                    [  ]  [Р]
[.]                                [.]
|                                   |
+--[СТІНА СТІНА СТІНА СТІНА СТІНА]--+

Розміщення об'єктів

1. Стіни (Верх та низ)

Розмістіть екземпляри obj_wall вздовж верхнього та нижнього країв.

2. Ракетки

  • obj_paddle_left - Біля лівого краю, по центру вертикально
  • obj_paddle_right - Біля правого краю, по центру вертикально

3. М'яч

Розмістіть obj_ball у центрі кімнати.

4. Ворота

  • obj_goal_left - Розташуйте вздовж лівого краю (позаду ракетки)
  • obj_goal_right - Розташуйте вздовж правого краю (позаду ракетки)

5. Відображення рахунку

Розмістіть obj_score будь-де (він не має спрайта, лише малює текст).

Вітаємо! Ваша гра Pong готова до гри!

Тестування гри

  • Натисніть F5 або клацніть Запустити
  • Гравець 1: W (вгору) та S (вниз)
  • Гравець 2: Стрілка вгору та Стрілка вниз
Чудова робота! Ви створили класичну гру Pong для двох гравців!
