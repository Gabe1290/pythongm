Крок 2: Створення об'єктів

Тепер створимо об'єкти для всіх елементів гри.

Об'єкт землі

  1. Клацніть правою кнопкою Об'єкти > Створити об'єкт
  2. Назвіть його obj_ground
  3. Встановіть спрайт spr_ground
  4. Поставте галочку "Твердий"

Об'єкт платформи

  1. Створіть obj_platform
  2. Встановіть спрайт spr_platform
  3. Поставте галочку "Твердий"
  4. Встановіть Батьківський об'єкт obj_ground (необов'язково)

Об'єкт гравця

  1. Створіть obj_player
  2. Встановіть спрайт spr_player

Предмети для збору та небезпеки

  1. Створіть obj_coin зі спрайтом spr_coin
  2. Створіть obj_spike зі спрайтом spr_spike
  3. Створіть obj_flag зі спрайтом spr_flag
Важливо: Земля та платформи повинні бути "Твердими", щоб зіткнення працювали правильно!
Натисніть "Далі", щоб додати фізику гравця (гравітацію та стрибки).
