Крок 4: Рух гравця (Подія Step)

Це основа нашого платформера: рух, гравітація, стрибки та зіткнення.

Додайте подію Step

В obj_player:

  1. Додайте подію: Step > Step
  2. Додайте дію: Control > Execute Code
// === ГОРИЗОНТАЛЬНИЙ РУХ ===
var move_input = keyboard_check(vk_right) - keyboard_check(vk_left);
hsp = move_input * hspeed_max;

// === ГРАВІТАЦІЯ ===
vsp += gravity_force;
if (vsp > vspeed_max) vsp = vspeed_max;

// === ПЕРЕВІРКА ЗЕМЛІ ===
on_ground = place_meeting(x, y + 1, obj_ground);

// === СТРИБОК ===
if (on_ground && (keyboard_check_pressed(vk_up) || keyboard_check_pressed(vk_space))) {
    vsp = jump_force;
}

// === ГОРИЗОНТАЛЬНЕ ЗІТКНЕННЯ ===
if (place_meeting(x + hsp, y, obj_ground)) {
    while (!place_meeting(x + sign(hsp), y, obj_ground)) {
        x += sign(hsp);
    }
    hsp = 0;
}
x += hsp;

// === ВЕРТИКАЛЬНЕ ЗІТКНЕННЯ ===
if (place_meeting(x, y + vsp, obj_ground)) {
    while (!place_meeting(x, y + sign(vsp), obj_ground)) {
        y += sign(vsp);
    }
    vsp = 0;
}
y += vsp;

Як це працює

  1. Введення: Стрілка вправо = 1, Вліво = -1, Нічого = 0
  2. Гравітація: Додається до vsp кожен кадр (падіння швидше)
  3. Перевірка землі: Перевіряє на 1 піксель нижче наявність землі
  4. Стрибок: Працює тільки коли on_ground дорівнює true
  5. Зіткнення: Рухаємось піксель за пікселем до стіни
Магія: Цикл while переміщує нас піксель за пікселем до краю стіни, запобігаючи застряганню!
Натисніть "Далі", щоб додати монети, шипи та прапор.
