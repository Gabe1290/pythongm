Крок 5: Предмети для збору та небезпеки

Тепер додамо монети для збору, шипи для уникнення та прапор, якого потрібно досягти.

Монета - Збирайте для очок

В obj_coin:

  1. Додайте подію: Collision > obj_player
  2. Додайте дію: Score > Set Score
    • Нові очки: 10
    • Поставте галочку "Relative"
  3. Додайте дію: Main1 > Destroy Instance
    • Застосувати до: Self

Шипи - Шкодять гравцеві

В obj_spike:

  1. Додайте подію: Collision > obj_player
  2. Додайте дію: Main2 > Show Message
    • Повідомлення: Ой! Ви наступили на шипи!
  3. Додайте дію: Main1 > Restart Room

Прапор - Рівень пройдено

В obj_flag:

  1. Додайте подію: Collision > obj_player
  2. Додайте дію: Main2 > Show Message
    • Повідомлення: Рівень пройдено!
  3. Додайте дію: Main1 > Restart Room (або Next Room для кількох рівнів)
Порада: Розміщуйте шипи біля прірв або на платформах, щоб створити виклик!
Натисніть "Далі", щоб додати ігровий контролер для відображення очок.
