Крок 6: Ігровий контролер

Ігровий контролер відображає очки на екрані.

Створення об'єкта контролера

  1. Створіть об'єкт obj_game_controller
  2. Спрайт не потрібен (невидимий контролер)

Малювання очок

В obj_game_controller:

  1. Додайте подію: Draw > Draw
  2. Додайте дію: Control > Execute Code
// Встановити колір тексту
draw_set_color(c_white);

// Малювати очки у верхньому лівому куті
draw_text(10, 10, "Очки: " + string(score));

// Додатково: Малювати інструкції
draw_text(10, 30, "Стрілки: Рух  |  Пробіл: Стрибок");

Альтернатива: Використання дій

Замість коду можна використати:

  1. Додайте дію: Draw > Set Color (Білий)
  2. Додайте дію: Score > Draw Score
    • X: 10
    • Y: 10
    • Підпис: "Очки: "
Пам'ятайте: Розмістіть ігровий контролер у вашій кімнаті! Він не з'явиться сам по собі.
Натисніть "Далі", щоб спроектувати свій рівень.
