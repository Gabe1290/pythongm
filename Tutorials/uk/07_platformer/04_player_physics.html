Крок 3: Фізика гравця (Подія Create)

Нам потрібно налаштувати змінні для гравітації, стрибків та руху.

Ініціалізація змінних фізики

В obj_player:

  1. Додайте подію: Create
  2. Додайте дію: Control > Execute Code
// Налаштування руху
hspeed_max = 4;      // Максимальна горизонтальна швидкість
vspeed_max = 10;     // Максимальна швидкість падіння
jump_force = -10;    // Сила стрибка (від'ємна = вгору)
gravity_force = 0.5; // Швидкість падіння

// Поточні швидкості
hsp = 0;
vsp = 0;

// Стан
on_ground = false;

Розуміння змінних

Змінна Призначення
hsp Горизонтальна швидкість (вліво/вправо)
vsp Вертикальна швидкість (вгору/вниз)
gravity_force Постійна сила, що тягне вниз
jump_force Від'ємне значення для руху вгору
on_ground True, коли стоїмо на землі
Ключова концепція: Сила стрибка від'ємна, тому що Y збільшується донизу в іграх. Від'ємна швидкість Y переміщує гравця вгору!
Натисніть "Далі", щоб додати рух та обробку зіткнень.
