Наступні кроки

Вітаємо! Ви вивчили основи PyGameMaker!

Ось що ви можете досліджувати далі:

Події та дії

Додайте поведінку своїм об'єктам! Об'єкти реагують на події (натискання клавіш, зіткнення, таймери), виконуючи дії (рух, створення об'єктів, відтворення звуків).

Програмування Blockly

Використовуйте візуальні блоки для створення ігрової логіки без написання коду. Перетягуйте блоки для управління об'єктами.

Виявлення зіткнень

Зробіть так, щоб об'єкти взаємодіяли один з одним при дотику. Ідеально для збору предметів, ворогів, стін тощо!

Експорт гри

Експортуйте гру як HTML5 для гри у браузері, або як виконуваний файл Windows для поширення!

Рекомендований посібник: Спробуйте посібник "Твоя перша гра", щоб створити повноцінну гру, де ви ловите зірки!
Приємного створення! Натисніть "Повернутися до списку", щоб побачити всі доступні посібники.
