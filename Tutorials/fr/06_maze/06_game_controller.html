Étape 5 : Contrôleur de Jeu

Créez un contrôleur pour suivre le temps et afficher le score.

Créer l'Objet Contrôleur

  1. Créez l'objet obj_game_controller
  2. Pas de sprite nécessaire (laissez-le vide)

Initialiser le Chronomètre (Événement Create)

  1. Ajoutez un événement : Create
  2. Ajoutez une action : Contrôle > Définir une Variable
  3. Variable : global.timer
  4. Valeur : 0

Mettre à Jour le Chronomètre (Événement Step)

  1. Ajoutez un événement : Step
  2. Ajoutez une action : Contrôle > Définir une Variable
  3. Variable : global.timer
  4. Valeur : 1/room_speed
  5. Cochez « Relatif »

Afficher les Informations (Événement Draw)

  1. Ajoutez un événement : Draw
  2. Ajoutez une action : Contrôle > Exécuter du Code
draw_set_color(c_white);
draw_text(10, 10, "Score : " + string(score));
draw_text(10, 30, "Temps : " + string(floor(global.timer)) + "s");

var coins = instance_number(obj_coin);
draw_text(10, 50, "Pièces : " + string(coins));
Note : Le contrôleur est invisible mais essentiel pour suivre l'état du jeu !
Cliquez sur « Suivant » pour configurer votre room.
