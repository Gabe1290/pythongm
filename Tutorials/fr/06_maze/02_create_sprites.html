Étape 1 : Créer les sprites

Nous avons besoin de sprites pour le joueur, les murs, la sortie et les pièces. Les murs doivent faire 32x32 pixels pour créer une grille.

Créer le sprite du joueur

  1. Faites un clic droit sur Sprites et sélectionnez Créer un nouveau Sprite
  2. Nommez-le spr_player
  3. Double-cliquez sur le sprite dans l'arbre des ressources pour ouvrir l'éditeur de sprites
  4. Cliquez sur le bouton Redimensionner dans la barre d'outils et définissez la taille à 24x24 pixels (plus petit que les murs)
  5. Dessinez un petit personnage (cercle ou personnage simple)
  6. Utilisez une couleur vive comme le bleu ou le vert

Créer le sprite du mur

  1. Créez un nouveau sprite nommé spr_wall
  2. Dessinez un motif de brique ou de pierre
  3. Utilisez des couleurs grises ou foncées
  4. Taille : 32x32 pixels

Créer le sprite de la sortie

  1. Créez un nouveau sprite nommé spr_exit
  2. Dessinez une porte, un drapeau ou un marqueur lumineux
  3. Utilisez des couleurs vertes ou dorées
  4. Taille : 32x32 pixels

Créer le sprite de la pièce

  1. Créez un nouveau sprite nommé spr_coin
  2. Double-cliquez sur le sprite pour ouvrir l'éditeur de sprites
  3. Cliquez sur le bouton Redimensionner dans la barre d'outils et définissez la taille à 16x16 pixels
  4. Dessinez un petit cercle jaune/doré
Astuce : Rendre le sprite du joueur plus petit que les murs permet un mouvement plus facile dans les couloirs !
Cliquez sur « Suivant » pour créer les objets du jeu.
