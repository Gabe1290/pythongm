Phase 1 : Joueur et labyrinthe

Dans cette phase, vous allez créer les sprites du joueur et des murs, construire les objets, programmer le déplacement avec Blockly et concevoir un labyrinthe simple. À la fin, vous pourrez vous déplacer à l'intérieur de votre labyrinthe !

Étape 1 : Créer les Sprites

spr_player (32x32)

  1. Dans l'Arbre des ressources, faites un clic droit sur Sprites et sélectionnez Créer un nouveau Sprite
  2. Nommez-le spr_player

Option A — Dessinez-le vous-même

  1. Double-cliquez sur spr_player pour ouvrir l'éditeur de Sprites
  2. Réglez la taille à 32x32 pixels
  3. Dessinez un petit personnage (un cercle avec des yeux, ou un bonhomme simple)
  4. Utilisez une couleur vive comme le bleu
  5. Enregistrez avec Ctrl+S

Option B — Charger le sprite d'exemple

  1. Faites un clic droit sur spr_player dans l'Arbre des ressources
  2. Sélectionnez Importer une image...
  3. Naviguez vers Tutorials/06_maze/assets/
  4. Sélectionnez spr_player.png et cliquez sur Ouvrir

spr_wall (32x32)

  1. Créez un nouveau sprite nommé spr_wall

Option A — Dessinez-le vous-même

  1. Ouvrez l'éditeur de Sprites et réglez la taille à 32x32 pixels
  2. Dessinez un motif de briques ou de pierres
  3. Utilisez des couleurs grises ou marron

Option B — Charger le sprite d'exemple

  1. Faites un clic droit sur spr_wall → Importer une image...
  2. Sélectionnez Tutorials/06_maze/assets/spr_wall.png

Étape 2 : Créer les Objets

obj_wall (Solide)

  1. Faites un clic droit sur Objets → Créer un nouvel Objet
  2. Nommez-le obj_wall
  3. Définissez le sprite à spr_wall
  4. Cochez « Solide » — c'est essentiel pour que le joueur ne puisse pas traverser les murs
  5. Aucun événement n'est nécessaire pour le mur

obj_player

  1. Créez un nouvel objet nommé obj_player
  2. Définissez le sprite à spr_player
  3. Laissez « Solide » décoché

Étape 3 : Programmer le joueur (Blockly)

Comment fonctionne Blockly : Glissez les blocs depuis la boîte à outils à gauche dans l'espace de travail. Emboîtez les blocs d'action à l'intérieur des blocs d'événements pour définir ce qui se passe. Lorsque vous avez terminé, cliquez sur Appliquer aux événements pour enregistrer vos blocs.
  1. Double-cliquez sur obj_player dans l'Arbre des ressources pour ouvrir l'éditeur d'objets
  2. Cliquez sur l'onglet Blockly en haut de l'éditeur

Déplacement à droite

Keyboard: Right (held)
Set horizontal speed to 4

Trouvez Keyboard (held) dans la catégorie Événements et sélectionnez Right dans le menu déroulant. Puis trouvez Set Horizontal Speed dans la catégorie Mouvement et emboîtez-le à l'intérieur. Tapez 4 dans le champ numérique.

Déplacement à gauche

Keyboard: Left (held)
Set horizontal speed to -4

Même chose que ci-dessus, mais choisissez Left et tapez -4 (négatif = direction gauche).

Déplacement vers le bas

Keyboard: Down (held)
Set vertical speed to 4

Utilisez Set Vertical Speed depuis Mouvement. Positif = vers le bas.

Déplacement vers le haut

Keyboard: Up (held)
Set vertical speed to -4

Tapez -4 pour le mouvement vers le haut.

S'arrêter quand aucune touche n'est appuyée

Keyboard: No key
Stop movement

Trouvez Keyboard: No key dans Événements et Stop Movement dans Mouvement. Cela empêche le joueur de glisser indéfiniment !

S'arrêter contre les murs

When colliding with obj_wall
Stop movement

Trouvez Collision dans Événements. Tapez obj_wall dans le champ texte. Emboîtez un bloc Stop Movement à l'intérieur.

Appliquer vos blocs

  1. Vous devriez avoir 6 blocs d'événements dans votre espace de travail
  2. Cliquez sur le bouton Appliquer aux événements pour enregistrer
Comment ça fonctionne : Chaque touche fléchée règle la vitesse dans cette direction. Lorsque vous relâchez toutes les touches, le joueur s'arrête. Si le joueur touche un mur, il s'arrête également.

Étape 4 : Créer la salle

  1. Faites un clic droit sur Salles → Créer une nouvelle salle
  2. Nommez-la room_maze
  3. Réglez la taille à 640x480
  4. Activez Accrochage à la grille : 32x32

Construire un labyrinthe simple

  1. Sélectionnez obj_wall et cliquez pour placer des murs autour du bord
  2. Ajoutez des murs intérieurs pour créer des couloirs et des chemins
  3. Placez obj_player près du coin supérieur gauche du labyrinthe
  4. Laissez de l'espace libre pour pouvoir vous déplacer
