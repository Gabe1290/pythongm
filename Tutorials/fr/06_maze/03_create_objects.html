Étape 2 : Créer les Objets

Maintenant, nous allons créer des objets pour les murs, la sortie, les pièces et le joueur.

Créer l'Objet Mur

  1. Faites un clic droit sur Objets et sélectionnez Créer un Objet
  2. Nommez-le obj_wall
  3. Définissez le sprite sur spr_wall
  4. Cochez « Solide »
  5. Aucun événement nécessaire

Créer l'Objet Sortie

  1. Créez l'objet obj_exit
  2. Définissez le sprite sur spr_exit
  3. Laissez « Solide » décoché

Créer l'Objet Pièce

  1. Créez l'objet obj_coin
  2. Définissez le sprite sur spr_coin
  3. Laissez « Solide » décoché

Créer l'Objet Joueur

  1. Créez l'objet obj_player
  2. Définissez le sprite sur spr_player
  3. Laissez « Solide » décoché

Créer le Contrôleur de Jeu

  1. Créez l'objet obj_game_controller
  2. Pas de sprite nécessaire (laissez-le vide)
  3. Laissez « Solide » décoché
Note : Seul le mur doit être « Solide ». Le contrôleur de jeu est un objet invisible qui affichera le score à l'écran.
Cliquez sur « Suivant » pour ajouter le mouvement du joueur avec Blockly.
