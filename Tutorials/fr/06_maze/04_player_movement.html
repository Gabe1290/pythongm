Étape 3 : Mouvement du Joueur

Le joueur a besoin d'un mouvement fluide avec détection de collision.

Initialiser la Vitesse (Événement Create)

Dans obj_player :

  1. Ajoutez un événement : Create
  2. Ajoutez une action : Contrôle > Définir une Variable
  3. Variable : move_speed
  4. Valeur : 4

Mouvement avec Collision (Événement Step)

  1. Ajoutez un événement : Step
  2. Ajoutez une action : Contrôle > Exécuter du Code
// Mouvement horizontal
var hspd = 0;
if (keyboard_check(vk_right)) hspd = move_speed;
if (keyboard_check(vk_left)) hspd = -move_speed;

// Mouvement vertical
var vspd = 0;
if (keyboard_check(vk_down)) vspd = move_speed;
if (keyboard_check(vk_up)) vspd = -move_speed;

// Vérifier la collision horizontale
if (!place_meeting(x + hspd, y, obj_wall)) {
    x += hspd;
}

// Vérifier la collision verticale
if (!place_meeting(x, y + vspd, obj_wall)) {
    y += vspd;
}
Comment ça fonctionne : Nous vérifions si la position cible entrerait en collision avec un mur. Si non, nous nous y déplaçons !
