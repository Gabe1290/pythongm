Étape 3 : Mouvement du Joueur (Blockly)

Nous allons utiliser les blocs visuels Blockly pour programmer le mouvement du joueur et les collisions avec les murs.

Ouvrir l'Éditeur Blockly

  1. Double-cliquez sur obj_player dans l'arbre des assets pour ouvrir l'Éditeur d'Objet
  2. Cliquez sur l'onglet Blockly en haut de l'éditeur
Comment fonctionne Blockly : Glissez les blocs depuis la boîte à outils à gauche dans l'espace de travail. Emboîtez les blocs d'action dans les blocs d'événement pour définir ce qui se passe. Quand vous avez terminé, cliquez sur Apply to Events pour sauvegarder vos blocs.

Se Déplacer à Droite

Glissez ces blocs depuis la boîte à outils et emboîtez-les :

Clavier: Flèche droite (maintenue)
définir vitesse horizontale à 4

Trouvez Clavier (maintenue) dans la catégorie Événements, et sélectionnez Flèche droite dans le menu déroulant. Puis trouvez définir vitesse horizontale dans la catégorie Mouvement et emboîtez-le à l'intérieur. Tapez 4 dans le champ numérique.

Se Déplacer à Gauche

Clavier: Flèche gauche (maintenue)
définir vitesse horizontale à -4

Même chose, mais choisissez Flèche gauche et tapez -4 (négatif = direction gauche).

Se Déplacer vers le Bas

Clavier: Flèche bas (maintenue)
définir vitesse verticale à 4

Utilisez définir vitesse verticale depuis Mouvement. Positif = vers le bas.

Se Déplacer vers le Haut

Clavier: Flèche haut (maintenue)
définir vitesse verticale à -4

Tapez -4 pour le mouvement vers le haut.

S'Arrêter Quand Aucune Touche n'Est Pressée

Clavier: Aucune touche
arrêter le mouvement

Trouvez Aucune touche dans Événements et arrêter le mouvement dans Mouvement. Cela empêche le joueur de glisser indéfiniment !

S'Arrêter contre les Murs

Quand collision avec obj_wall
arrêter le mouvement

Trouvez Collision dans Événements. Tapez obj_wall dans le champ texte. Emboîtez un bloc arrêter le mouvement à l'intérieur.

Appliquer Vos Blocs

  1. Vous devez avoir 6 blocs d'événement dans votre espace de travail
  2. Cliquez sur le bouton Appliquer aux événements pour sauvegarder
Comment ça fonctionne : Chaque touche fléchée définit la vitesse dans cette direction. Quand vous relâchez toutes les touches, le joueur s'arrête. Si le joueur touche un mur, il s'arrête aussi.
Cliquez sur « Suivant » pour ajouter la collecte de pièces et la sortie.
