Étape 6 : Touches finales

Ajoutons quelques finitions pour que votre jeu soit complet !

Garder le joueur à l'écran

1 Ajouter une vérification des limites

Dans obj_player, ajoutez un événement Step avec ces conditions :

  • Si x < 0, définir x = 0
  • Si x > room_width - sprite_width, définir x = room_width - sprite_width

Cela empêche le joueur de sortir des côtés de l'écran.

Ajouter une couleur de fond

2 Définir le fond de la salle

Ouvrez room_game dans l'éditeur de salles.

Dans les propriétés de la salle, définissez la couleur de fond en bleu ciel ou en couleur spatiale sombre.

Exporter votre jeu !

3 Exporter en HTML5

Allez dans Fichier > Exporter en HTML5.

Choisissez un emplacement et cliquez sur Exporter.

Ouvrez le fichier HTML dans un navigateur web pour jouer à votre jeu !

Félicitations ! Vous avez créé votre premier jeu complet dans PyGameMaker !

Idées pour améliorer votre jeu

Défis à essayer :
  • Ajouter des vies - perdre une vie quand une étoile atteint le bas
  • Faire tomber les étoiles plus vite à mesure que le score augmente
  • Ajouter des étoiles de différentes couleurs valant des points différents
  • Ajouter un objet "mauvais" (bombe) qui termine le jeu s'il est attrapé
  • Ajouter des effets sonores quand on attrape des étoiles
  • Créer un écran de titre et un écran de game over
