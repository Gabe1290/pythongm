Étape 1 : Créer les sprites

Les sprites sont les images qui représentent les objets dans votre jeu. Nous avons besoin de deux sprites : un pour le joueur et un pour l'étoile.

Créer le sprite du joueur

1 Créer un nouveau projet

Allez dans Fichier > Nouveau projet et nommez-le "AttrapezLetoile".

2 Créer le sprite du joueur

Faites un clic droit sur Sprites dans l'arborescence et sélectionnez Créer un sprite.

Nommez-le spr_player.

3 Ajouter une image

Cliquez sur Charger une image et sélectionnez une image pour votre joueur (un vaisseau spatial, un personnage ou un panier).

Si vous n'avez pas d'image, vous pouvez dessiner un simple carré coloré de 32x32 dans n'importe quel éditeur d'images.

Créer le sprite de l'étoile

4 Créer le sprite de l'étoile

Faites à nouveau un clic droit sur Sprites et sélectionnez Créer un sprite.

Nommez-le spr_star.

5 Ajouter une image d'étoile

Chargez une image d'étoile, de pièce ou de tout objet à collecter.

Une image de 32x32 pixels fonctionne bien.

Conseil : Utilisez des images simples et claires. Les étoiles ou les pièces sont faciles à reconnaître et rendent le jeu plus gratifiant !
Sprites créés ! Cliquez sur "Suivant" pour créer les objets du jeu.
