Phase 2 : Des étoiles qui tombent

Maintenant, faisons apparaître des étoiles en haut de l'écran et faisons-les tomber !

Créer le sprite de l'étoile

  1. Créez un nouveau sprite nommé spr_star
  2. Dessinez une étoile ou une pièce (32×32 pixels) et sauvegardez

Créer l'objet étoile

  1. Créez un nouvel objet nommé obj_star
  2. Définissez le sprite sur spr_star

Faire tomber les étoiles

  1. Double-cliquez sur obj_star → onglet Blockly
  2. Ajoutez ce bloc :
Événement Création
Définir la vitesse verticale à 3

Chaque étoile tombe vers le bas à la vitesse 3 dès qu'elle apparaît.

Cliquez sur Appliquer aux événements pour sauvegarder.

Créer un générateur

Nous avons besoin d'un objet invisible qui crée des étoiles à des positions aléatoires.

  1. Créez un nouvel objet nommé obj_spawner
  2. Pas besoin de sprite (il sera invisible)

Programmer le générateur

  1. Double-cliquez sur obj_spawner → onglet Blockly
  2. Ajoutez ces blocs :
Événement Création
Définir l'alarme 0 à 60
Alarme 0
Créer une instance de obj_star à x: aléatoire y: 0
Définir l'alarme 0 à 60

L'événement Création démarre un minuteur. Quand l'alarme se déclenche (toutes les 2 secondes), elle crée une étoile à une position x aléatoire en haut de l'écran, puis redémarre le minuteur.

Cliquez sur Appliquer aux événements pour sauvegarder.

Placer le générateur dans la salle

  1. Ouvrez room_game
  2. Placez obj_spawner n'importe où (il est invisible, l'emplacement n'a pas d'importance)

Testez votre jeu !

Appuyez sur F5 pour jouer !

  • Les étoiles devraient tomber depuis des positions aléatoires en haut
  • Déplacez le joueur pour les esquiver ou les atteindre
  • Les étoiles passent à travers le joueur pour l'instant — nous ajouterons la capture ensuite !
Cliquez sur « Suivant » pour ajouter les collisions et le score.
