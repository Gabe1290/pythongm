Étape 1 : Créer les Sprites

Nous avons besoin de sprites pour la raquette, la balle, les briques, les murs et une zone de mort. Pour chaque sprite, vous pouvez le dessiner vous-même avec l'éditeur de sprites intégré, ou charger une image d'exemple. Les deux options sont expliquées ci-dessous.

spr_ball (16×16)

  1. Dans l'arbre des ressources, faites un clic droit sur Sprites et sélectionnez Créer un Sprite
  2. Nommez-le spr_ball

Option A — Dessiner vous-même

  1. Double-cliquez sur spr_ball pour ouvrir l'éditeur de sprites
  2. Dessinez un petit cercle
  3. Utilisez une couleur vive comme le jaune ou le blanc
  4. Gardez la taille à 16×16 pixels
  5. Enregistrez avec Ctrl+S

Option B — Charger le sprite d'exemple

  1. Faites un clic droit sur spr_ball dans l'arbre des ressources
  2. Sélectionnez Importer une image...
  3. Naviguez vers Tutorials/04_breakout/assets/ dans le dossier PyGameMaker
  4. Sélectionnez spr_ball.png et cliquez sur Ouvrir

spr_paddle (64×16)

  1. Créez un nouveau sprite nommé spr_paddle

Option A — Dessiner vous-même

  1. Ouvrez l'éditeur de sprites et dessinez un rectangle large et plat
  2. Utilisez une couleur vive comme le bleu
  3. Taille : 64×16 pixels

Option B — Charger le sprite d'exemple

  1. Clic droit sur spr_paddle → Importer une image...
  2. Sélectionnez Tutorials/04_breakout/assets/spr_paddle.png

Sprites des Briques (32×16 chacun)

Créez quatre briques colorées. Chacune suit les mêmes étapes :

  1. Créez un nouveau sprite avec le nom du tableau ci-dessous

Option A — Dessiner vous-même

  1. Ouvrez l'éditeur de sprites et dessinez un rectangle plein
  2. Utilisez la couleur correspondante pour chaque brique
  3. Taille : 32×16 pixels pour toutes les briques

Option B — Charger les sprites d'exemple

Pour chaque brique, clic droit → Importer une image... et sélectionnez le fichier correspondant :

  • spr_brick_red → Tutorials/04_breakout/assets/spr_brick_red.png
  • spr_brick_orange → Tutorials/04_breakout/assets/spr_brick_orange.png
  • spr_brick_yellow → Tutorials/04_breakout/assets/spr_brick_yellow.png
  • spr_brick_green → Tutorials/04_breakout/assets/spr_brick_green.png

spr_wall (32×32)

  1. Créez un nouveau sprite nommé spr_wall

Option A — Dessiner vous-même

  1. Ouvrez l'éditeur de sprites et dessinez un motif de briques ou de pierres
  2. Utilisez des couleurs grises
  3. Taille : 32×32 pixels

Option B — Charger le sprite d'exemple

  1. Clic droit sur spr_wall → Importer une image...
  2. Sélectionnez Tutorials/04_breakout/assets/spr_wall.png

spr_death_zone (32×32)

  1. Créez un nouveau sprite nommé spr_death_zone

Option A — Dessiner vous-même

  1. Dessinez un rectangle plein de n'importe quelle couleur (ex. rouge foncé)
  2. Taille : 32×32 pixels

Option B — Charger le sprite d'exemple

  1. Clic droit sur spr_death_zone → Importer une image...
  2. Sélectionnez Tutorials/04_breakout/assets/spr_death_zone.png
Astuce : Gardez des tailles de briques cohérentes (32×16) pour qu'elles s'alignent bien en rangées ! La zone de mort sera invisible pendant le jeu — elle a juste besoin d'un sprite pour le placement.
Point pédagogique : La fonction Importer une image permet de charger n'importe quel fichier PNG, GIF ou BMP comme sprite. C'est utile quand vous avez des illustrations créées dans un programme externe ou des ressources d'exemple fournies avec un tutoriel.
