Étape 1 : Créer les Sprites

Nous avons besoin de sprites pour la raquette, la balle, les briques et les murs.

Créer le Sprite de la Balle

  1. Faites un clic droit sur Sprites dans l'arbre des ressources
  2. Sélectionnez Créer un Sprite
  3. Nommez-le spr_ball
  4. Cliquez sur Modifier le Sprite
  5. Dessinez un petit cercle blanc ou jaune (16x16 pixels)
  6. Cliquez sur OK pour enregistrer

Créer le Sprite de la Raquette

  1. Créez un nouveau sprite nommé spr_paddle
  2. Dessinez un rectangle large et plat (64x16 pixels)
  3. Utilisez une couleur vive comme le bleu

Créer les Sprites des Briques

Créez des briques colorées pour différentes valeurs de points :

  1. Créez spr_brick_red - Brique rouge (32x16 pixels)
  2. Créez spr_brick_orange - Brique orange
  3. Créez spr_brick_yellow - Brique jaune
  4. Créez spr_brick_green - Brique verte

Créer le Sprite du Mur

  1. Créez un nouveau sprite nommé spr_wall
  2. Dessinez un rectangle gris uni (32x32 pixels)
Astuce : Gardez des tailles de briques cohérentes pour qu'elles s'alignent bien dans la room !
Cliquez sur « Suivant » pour créer les objets du jeu.
