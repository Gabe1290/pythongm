Étape 6 : Afficher le Score et les Vies

Ajoutons un affichage pour le score du joueur et les vies restantes.

Événement Draw dans le Contrôleur de Jeu

Dans obj_game_controller :

  1. Ajoutez un événement : Draw

Dessiner le Score

  1. Ajoutez une action : Dessiner > Dessiner du Texte
  2. Texte : Score :
  3. X : 10, Y : 10
  4. Ajoutez une action : Dessiner > Dessiner le Score
  5. X : 70, Y : 10

Dessiner les Vies

  1. Ajoutez une action : Dessiner > Dessiner du Texte
  2. Texte : Vies :
  3. X : 10, Y : 30
  4. Ajoutez une action : Dessiner > Dessiner les Vies
  5. X : 70, Y : 30
Astuce : Vous pouvez utiliser Dessiner les Vies en Image pour afficher des icônes de cœur au lieu d'un nombre !
Cliquez sur « Suivant » pour configurer la room du jeu.
