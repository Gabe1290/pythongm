Phase 1 : Objets, raquette et balle

Créons les objets du jeu et programmons-les pour pouvoir tester le jeu rapidement !

Créer les Objets

Nous avons besoin de deux types de murs différents pour que la balle sache dans quelle direction rebondir.

  1. Faites un clic droit sur Objets → Créer un nouvel objet
  2. Créez obj_wall_side — sprite : spr_wall, cochez Solide
  3. Créez obj_wall_top — sprite : spr_wall, cochez Solide
  4. Créez obj_paddle — sprite : spr_paddle, cochez Solide
  5. Créez obj_ball — sprite : spr_ball, laissez Solide décoché
  6. Créez obj_death_zone — sprite : spr_death_zone, décochez Visible
Pourquoi deux types de murs ? Quand la balle touche un mur latéral, elle inverse sa direction horizontale. Quand elle touche le mur du haut, elle inverse sa direction verticale. Utiliser deux objets nous permet de programmer des comportements de rebond différents !

Programmer la raquette

  1. Double-cliquez sur obj_paddle pour ouvrir l'éditeur d'objets
  2. Cliquez sur l'onglet Blockly
Comment fonctionne Blockly : Glissez les blocs depuis la boîte à outils à gauche vers l'espace de travail. Emboîtez les blocs d'action dans les blocs d'événements pour définir ce qui se passe. Quand vous avez terminé, cliquez sur Appliquer aux événements pour enregistrer vos blocs.

Déplacement à gauche

Keyboard: Left (held)
Set horizontal speed to -8

Trouvez Keyboard (held) dans Events, sélectionnez Left. Puis emboîtez Set Horizontal Speed depuis Movement à l'intérieur. Tapez -8 (négatif = gauche).

Déplacement à droite

Keyboard: Right (held)
Set horizontal speed to 8

Comme ci-dessus, mais choisissez Right et tapez 8 (positif = droite).

Arrêt quand aucune touche n'est pressée

Keyboard: No key
Stop movement

Cela empêche la raquette de glisser indéfiniment quand vous relâchez la touche.

Arrêt aux murs latéraux

When colliding with obj_wall_side
Stop movement

Cela maintient la raquette à l'intérieur de la zone de jeu.

Vous devriez avoir 4 blocs d'événements. Cliquez sur Appliquer aux événements pour enregistrer.

Programmer la balle

  1. Double-cliquez sur obj_ball pour ouvrir l'éditeur d'objets
  2. Cliquez sur l'onglet Blockly

Lancer la balle en mouvement

Create Event
Set horizontal speed to 3
Set vertical speed to -3

L'Événement Création s'exécute une seule fois au démarrage du jeu. Définir les deux vitesses fait bouger la balle en diagonale vers le haut.

Pourquoi deux vitesses ? La vitesse horizontale (3) déplace la balle latéralement. La vitesse verticale (-3) la déplace vers le haut (négatif = haut). Ensemble, la balle se déplace en diagonale !

Rebond sur les murs latéraux

When colliding with obj_wall_side
Reverse horizontal movement

La balle inverse sa direction gauche/droite lorsqu'elle touche un mur latéral.

Rebond sur le mur du haut

When colliding with obj_wall_top
Reverse vertical movement

La balle rebondit vers le bas lorsqu'elle touche le mur du haut.

Rebond sur la raquette

When colliding with obj_paddle
Reverse vertical movement

Comme pour le mur du haut — la balle rebondit vers le haut.

La balle tombe hors de l'écran

When colliding with obj_death_zone
Jump to x: 320 y: 400

Quand la balle touche la zone de mort en bas, elle réapparaît au-dessus de la raquette. Nous ajouterons les vies plus tard — pour l'instant, la balle revient simplement !

Vous devriez avoir 5 blocs d'événements. Cliquez sur Appliquer aux événements pour enregistrer.

Comment ça marche : La balle se déplace toujours en diagonale. Quand elle touche une surface verticale (mur latéral), on inverse le mouvement horizontal. Quand elle touche une surface horizontale (mur du haut, raquette), on inverse le mouvement vertical. Cela crée un rebond réaliste !
