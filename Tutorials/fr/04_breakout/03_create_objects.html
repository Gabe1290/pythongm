Étape 2 : Créer les Objets

Maintenant, nous allons créer tous les objets du jeu.

Créer l'Objet Mur

  1. Faites un clic droit sur Objets et sélectionnez Créer un Objet
  2. Nommez-le obj_wall
  3. Définissez le sprite sur spr_wall
  4. Cochez « Solide »
  5. Aucun événement nécessaire

Créer l'Objet Parent des Briques

Nous utiliserons un objet parent pour que toutes les briques partagent le même comportement de collision.

  1. Créez l'objet obj_brick_parent
  2. Pas de sprite nécessaire (c'est juste un modèle)
  3. Cochez « Solide »

Créer les Objets Briques Colorées

Pour chaque couleur de brique :

  1. Créez obj_brick_red avec spr_brick_red
  2. Définissez Parent sur obj_brick_parent
  3. Cochez « Solide »

Répétez pour obj_brick_orange, obj_brick_yellow et obj_brick_green.

Créer l'Objet Raquette

  1. Créez l'objet obj_paddle
  2. Définissez le sprite sur spr_paddle
  3. Cochez « Solide »

Créer l'Objet Balle

  1. Créez l'objet obj_ball
  2. Définissez le sprite sur spr_ball
  3. Laissez « Solide » décoché
Objets Parents : Quand la balle entre en collision avec obj_brick_parent, elle détectera les collisions avec TOUS les types de briques !
Cliquez sur « Suivant » pour ajouter le mouvement de la raquette.
