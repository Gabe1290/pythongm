Phase 1 : Créer les Sprites

Nous allons commencer avec juste ce qu'il faut pour un jeu de balle rebondissante — pas encore de briques ! Pour chaque sprite, vous pouvez le dessiner vous-même ou charger une image d'exemple.

SpriteTailleDescription
spr_ball16×16Petit cercle jaune
spr_paddle64×16Large rectangle bleu
spr_wall32×32Motif de briques grises
spr_death_zone32×32Rectangle sombre (invisible dans le jeu)

spr_ball (16×16)

  1. Dans l'Arbre des ressources, faites un clic droit sur Sprites et sélectionnez Créer un nouveau sprite
  2. Nommez-le spr_ball

Option A — Dessinez-le vous-même

  1. Double-cliquez sur spr_ball pour ouvrir l'éditeur de sprites
  2. Dessinez un petit cercle dans une couleur vive comme le jaune ou le blanc
  3. Gardez la taille à 16×16 pixels
  4. Enregistrez votre travail avec Ctrl+S

Option B — Charger le sprite d'exemple

  1. Faites un clic droit sur spr_ball dans l'Arbre des ressources
  2. Sélectionnez Importer une image...
  3. Naviguez vers Tutorials/04_breakout/assets/ dans le dossier PyGameMaker
  4. Sélectionnez spr_ball.png et cliquez sur Ouvrir

spr_paddle (64×16)

  1. Créez un nouveau sprite nommé spr_paddle

Option A — Dessinez-le vous-même

  1. Ouvrez l'éditeur de sprites et dessinez un rectangle large et plat
  2. Utilisez une couleur vive comme le bleu
  3. Taille : 64×16 pixels

Option B — Charger le sprite d'exemple

  1. Clic droit sur spr_paddle → Importer une image...
  2. Sélectionnez Tutorials/04_breakout/assets/spr_paddle.png

spr_wall (32×32)

  1. Créez un nouveau sprite nommé spr_wall

Option A — Dessinez-le vous-même

  1. Dessinez un motif de briques ou de pierres en utilisant des couleurs grises
  2. Taille : 32×32 pixels

Option B — Charger le sprite d'exemple

  1. Clic droit sur spr_wall → Importer une image...
  2. Sélectionnez Tutorials/04_breakout/assets/spr_wall.png

spr_death_zone (32×32)

  1. Créez un nouveau sprite nommé spr_death_zone

Option A — Dessinez-le vous-même

  1. Dessinez un rectangle plein dans n'importe quelle couleur (par ex. rouge foncé)
  2. Taille : 32×32 pixels

Option B — Charger le sprite d'exemple

  1. Clic droit sur spr_death_zone → Importer une image...
  2. Sélectionnez Tutorials/04_breakout/assets/spr_death_zone.png
Point d'apprentissage : La fonction Importer une image permet de charger n'importe quel fichier PNG, GIF ou BMP comme sprite. C'est utile lorsque vous avez des illustrations créées dans un programme externe ou des ressources d'exemple fournies avec un tutoriel.
C'est tout pour l'instant ! Nous n'avons besoin que de 4 sprites pour commencer. Nous ajouterons les sprites de briques plus tard quand nous en aurons besoin.
Cliquez sur « Suivant » pour créer les objets et programmer la raquette et la balle.
