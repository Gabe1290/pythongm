Étape 5 : Ajouter le Système de Vies

Le joueur perd une vie quand la balle tombe en bas de l'écran.

Créer une Zone de Mort

  1. Créez un sprite spr_death_zone (32x32, n'importe quelle couleur)
  2. Créez l'objet obj_death_zone
  3. Décochez « Visible » - il doit être invisible
  4. Cochez « Solide »

La Balle Tombe dans la Zone de Mort

Dans obj_ball :

  1. Ajoutez un événement : Collision > obj_death_zone
  2. Ajoutez une action : Score > Soustraire des Vies
  3. Valeur : 1
  4. Ajoutez une action : Mouvement > Retourner à la Position de Départ

Créer un Contrôleur de Jeu

  1. Créez l'objet obj_game_controller
  2. Pas de sprite nécessaire

Initialiser les Vies (Événement Create)

Dans obj_game_controller :

  1. Ajoutez un événement : Create
  2. Ajoutez une action : Score > Définir les Vies
  3. Valeur : 3
  4. Ajoutez une action : Score > Définir le Score
  5. Valeur : 0

Vérifier le Game Over (Événement Step)

  1. Ajoutez un événement : Step
  2. Ajoutez une action : Contrôle > Tester les Vies
  3. Vérifiez si les vies sont inférieures ou égales à 0
  4. Ajoutez une action : Main2 > Afficher un Message
  5. Message : Game Over !
  6. Ajoutez une action : Main2 > Redémarrer la Room
N'oubliez pas : Placez obj_game_controller dans votre room !
Cliquez sur « Suivant » pour ajouter l'affichage du score.
