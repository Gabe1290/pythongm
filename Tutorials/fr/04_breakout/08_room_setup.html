Étape 7 : Configurer la Room

Il est temps de créer la room du jeu et de placer tous les objets !

Créer la Room

  1. Faites un clic droit sur Rooms et sélectionnez Créer une Room
  2. Nommez-la room_breakout
  3. Définissez la taille sur 640x480 pixels

Disposition de la Room

+--[MUR]----------------------[MUR]--+
|  [MUR]  [R][R][R][R][R]    [MUR]  |
|  [MUR]  [O][O][O][O][O]    [MUR]  |
|  [MUR]  [J][J][J][J][J]    [MUR]  |
|  [MUR]  [V][V][V][V][V]    [MUR]  |
|  [MUR]                     [MUR]  |
|  [MUR]        (o)          [MUR]  |
|  [MUR]                     [MUR]  |
|  [MUR]     [RAQUETTE]      [MUR]  |
+--[ZONE DE MORT ZONE DE MORT]------+

Placer les Objets

1. Murs

Placez obj_wall le long des bords supérieur, gauche et droit.

2. Zone de Mort

Placez obj_death_zone le long du bord inférieur (elle est invisible).

3. Briques

Disposez les briques en rangées près du haut :

  • Première rangée : Briques rouges (obj_brick_red)
  • Deuxième rangée : Briques orange
  • Troisième rangée : Briques jaunes
  • Quatrième rangée : Briques vertes

4. Raquette

Placez obj_paddle près du bas, centrée horizontalement.

5. Balle

Placez obj_ball juste au-dessus de la raquette.

6. Contrôleur de Jeu

Placez obj_game_controller n'importe où (il n'a pas de sprite).

Félicitations ! Votre jeu Breakout est prêt !

Testez Votre Jeu

  • Appuyez sur F5 ou cliquez sur Exécuter
  • Utilisez les touches Gauche et Droite pour vous déplacer
  • Cassez toutes les briques pour gagner !
Bravo ! Vous avez créé un jeu Breakout classique !
