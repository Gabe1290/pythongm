Phase 2 : Première rangée de briques

Maintenant, ajoutons des briques que la balle pourra casser ! Nous allons utiliser une astuce appelée objet parent pour que plus tard, quand nous ajouterons d'autres couleurs de briques, nous n'ayons pas à réécrire le code de collision.

Créer le sprite de brique

  1. Créez un nouveau sprite nommé spr_brick_red

Option A — Dessinez-le vous-même

  1. Ouvrez l'éditeur de sprites et dessinez un rectangle plein rouge
  2. Taille : 32×16 pixels

Option B — Charger le sprite d'exemple

  1. Clic droit sur spr_brick_red → Importer une image...
  2. Sélectionnez Tutorials/04_breakout/assets/spr_brick_red.png

Créer l'objet parent de brique

Un objet parent est un modèle. Tout objet qui l'utilise comme parent héritera automatiquement de son comportement de collision. Ainsi, nous n'écrivons le code « la balle touche une brique » qu'une seule fois.

  1. Créez un nouvel objet nommé obj_brick_parent
  2. Pas de sprite nécessaire — c'est juste un modèle, il n'apparaîtra pas dans le jeu
  3. Cochez Solide

Créer l'objet brique rouge

  1. Créez un nouvel objet nommé obj_brick_red
  2. Définissez le sprite à spr_brick_red
  3. Définissez le Parent à obj_brick_parent
  4. Cochez Solide
Que fait le choix d'un parent ? Quand la balle vérifie une collision avec obj_brick_parent, elle détectera aussi les collisions avec obj_brick_red et tout autre enfant que nous ajouterons plus tard. Un seul événement de collision couvre toutes les couleurs de briques !

Ajouter la collision avec les briques à la balle

Nous devons indiquer à la balle ce qu'elle doit faire quand elle touche une brique.

  1. Double-cliquez sur obj_ball pour ouvrir l'éditeur d'objets
  2. Cliquez sur l'onglet Blockly — vous devriez voir vos 5 blocs existants
  3. Ajoutez ce nouveau bloc d'événement :
Collision avec obj_brick_parent
Inverser le mouvement vertical
Détruire l'autre instance
Ajouter au score 10

Emboîtez trois blocs à l'intérieur :

  • Inverser le mouvement vertical (depuis Mouvement) — la balle rebondit
  • Détruire l'autre instance (depuis Instance) — détruit la brique, pas la balle !
  • Ajouter au score 10 (depuis Score/Vies/Santé) — gagnez des points

Vous devriez maintenant avoir 6 blocs d'événements pour la balle. Cliquez sur Appliquer aux événements.

Point clé : La collision est avec obj_brick_parent, et non obj_brick_red. Cela signifie qu'elle fonctionnera automatiquement avec chaque couleur de brique que nous ajouterons plus tard !

Ajouter des briques à la salle

  1. Ouvrez room_breakout
  2. Placez une rangée de obj_brick_red près du haut, entre les murs latéraux
  3. Activez Aligner sur la grille (32×16) pour un alignement facile
  4. Remplissez une rangée complète de briques rouges

Testez votre jeu !

Appuyez sur F5 pour jouer !

  • La balle devrait rebondir sur les briques et les détruire
  • Chaque brique ajoute 10 points au score
  • Essayez de casser toutes les briques !
Cliquez sur « Suivant » pour ajouter davantage de rangées de briques colorées.
