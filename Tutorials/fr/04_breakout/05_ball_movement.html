Étape 4 : Mouvement et Rebonds de la Balle

La balle doit se déplacer et rebondir sur les murs, la raquette et les briques.

Démarrer le Mouvement de la Balle (Événement Create)

  1. Ouvrez obj_ball
  2. Ajoutez un événement : Create
  3. Ajoutez une action : Mouvement > Commencer à se Déplacer dans une Direction
  4. Choisissez une direction diagonale vers le haut
  5. Définissez la vitesse sur 5

Rebondir sur les Murs

  1. Ajoutez un événement : Collision > obj_wall
  2. Ajoutez une action : Mouvement > Rebondir Contre les Objets
  3. Sélectionnez « Contre les objets solides »

Rebondir sur la Raquette

  1. Ajoutez un événement : Collision > obj_paddle
  2. Ajoutez une action : Mouvement > Rebondir Contre les Objets
  3. Sélectionnez « Contre les objets solides »

Rebondir sur les Briques (et les Détruire !)

  1. Ajoutez un événement : Collision > obj_brick_parent
  2. Ajoutez une action : Mouvement > Rebondir Contre les Objets
  3. Sélectionnez « Contre les objets solides »
  4. Ajoutez une action : Main1 > Détruire l'Instance
  5. Sélectionnez « Autre » (détruit la brique, pas la balle)
  6. Ajoutez une action : Score > Ajouter au Score
  7. Valeur : 10
Point clé : Utiliser obj_brick_parent signifie que cette collision fonctionne pour TOUTES les couleurs de briques !
Cliquez sur « Suivant » pour ajouter le système de vies.
