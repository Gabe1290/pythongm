Étape 3 : Ajouter le Mouvement de la Raquette

La raquette se déplace à gauche et à droite avec les touches fléchées.

Déplacer à Gauche (Flèche Gauche)

  1. Ouvrez obj_paddle
  2. Ajoutez un événement : Clavier > Appuyer Gauche
  3. Ajoutez une action : Mouvement > Définir la Vitesse Horizontale
  4. Définissez la vitesse sur -8 (négatif = gauche)

Arrêter le Mouvement (Relâcher Gauche)

  1. Ajoutez un événement : Clavier > Relâcher Gauche
  2. Ajoutez une action : Mouvement > Définir la Vitesse Horizontale
  3. Définissez la vitesse sur 0

Déplacer à Droite (Flèche Droite)

  1. Ajoutez un événement : Clavier > Appuyer Droite
  2. Ajoutez une action : Mouvement > Définir la Vitesse Horizontale
  3. Définissez la vitesse sur 8 (positif = droite)

Arrêter le Mouvement (Relâcher Droite)

  1. Ajoutez un événement : Clavier > Relâcher Droite
  2. Ajoutez une action : Mouvement > Définir la Vitesse Horizontale
  3. Définissez la vitesse sur 0

Collision avec le Mur

  1. Ajoutez un événement : Collision > obj_wall
  2. Ajoutez une action : Mouvement > Rebondir Contre les Objets
  3. Sélectionnez « Contre les objets solides »
Note : La collision avec le mur empêche la raquette de sortir de l'écran !
Cliquez sur « Suivant » pour ajouter le mouvement de la balle.
