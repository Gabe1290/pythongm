Étape 5 : Ajouter les Buts de Score

Nous avons besoin de buts invisibles derrière chaque raquette pour détecter quand quelqu'un marque.

Créer le Sprite du But

  1. Créez un nouveau sprite nommé spr_goal
  2. Faites-le de 32x32 pixels avec n'importe quelle couleur
  3. (Il sera invisible dans le jeu)

Créer l'Objet But Gauche

  1. Créez l'objet obj_goal_left
  2. Définissez le sprite sur spr_goal
  3. Décochez « Visible » - le but doit être invisible
  4. Cochez « Solide »

Créer l'Objet But Droit

  1. Créez l'objet obj_goal_right
  2. Définissez le sprite sur spr_goal
  3. Décochez « Visible »
  4. Cochez « Solide »

La Balle entre en Collision avec le But Gauche (Le Joueur 2 Marque)

Retournez dans obj_ball :

  1. Ajoutez un événement : Collision > obj_goal_left
  2. Ajoutez une action : Mouvement > Retourner à la Position de Départ
  3. Ajoutez une action : Score > Définir une Variable
  4. Variable : global.p2score
  5. Valeur : 1
  6. Cochez « Relatif » (ajoute 1 au score actuel)

La Balle entre en Collision avec le But Droit (Le Joueur 1 Marque)

  1. Ajoutez un événement : Collision > obj_goal_right
  2. Ajoutez une action : Mouvement > Retourner à la Position de Départ
  3. Ajoutez une action : Score > Définir une Variable
  4. Variable : global.p1score
  5. Valeur : 1
  6. Cochez « Relatif »
Note : Le préfixe global. signifie que ces variables sont accessibles depuis n'importe quel objet !
Cliquez sur « Suivant » pour afficher les scores à l'écran.
