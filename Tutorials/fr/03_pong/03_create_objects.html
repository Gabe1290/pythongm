Étape 2 : Créer les Objets

Maintenant, nous allons créer les objets du jeu. Chaque objet utilise un sprite et peut avoir des comportements.

Créer l'Objet Mur

  1. Faites un clic droit sur Objets et sélectionnez Créer un Objet
  2. Nommez-le obj_wall
  3. Définissez le sprite sur spr_wall
  4. Cochez la case « Solide » - c'est important !
  5. Aucun événement nécessaire - les murs bloquent simplement le mouvement

Créer l'Objet Balle

  1. Créez un nouvel objet nommé obj_ball
  2. Définissez le sprite sur spr_ball
  3. Laissez « Solide » décoché

Créer les Objets Raquettes

Raquette Gauche :

  1. Créez l'objet obj_paddle_left
  2. Définissez le sprite sur spr_paddle_left
  3. Cochez « Solide »

Raquette Droite :

  1. Créez l'objet obj_paddle_right
  2. Définissez le sprite sur spr_paddle_right
  3. Cochez « Solide »
Important : La case « Solide » est cruciale pour que les rebonds fonctionnent correctement !
Cliquez sur « Suivant » pour ajouter les contrôles de mouvement des raquettes.
