Step 4: Ball Movement and Bouncing

The ball needs to move automatically and bounce off walls and paddles.

Start Ball Moving (Create Event)

  1. Open obj_ball
  2. Add Event: Create
  3. Add Action: Move > Start Moving in Direction
  4. Choose a diagonal direction (not straight up/down)
  5. Set speed to 6

Bounce Off Walls

  1. Add Event: Collision > obj_wall
  2. Add Action: Move > Bounce Against Objects
  3. Select "Against solid objects"

Bounce Off Left Paddle

  1. Add Event: Collision > obj_paddle_left
  2. Add Action: Move > Bounce Against Objects
  3. Select "Against solid objects"

Bounce Off Right Paddle

  1. Add Event: Collision > obj_paddle_right
  2. Add Action: Move > Bounce Against Objects
  3. Select "Against solid objects"
