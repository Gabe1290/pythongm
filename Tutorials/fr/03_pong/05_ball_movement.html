Étape 4 : Mouvement et Rebonds de la Balle

La balle doit se déplacer automatiquement et rebondir sur les murs et les raquettes.

Démarrer le Mouvement de la Balle (Événement Create)

  1. Ouvrez obj_ball
  2. Ajoutez un événement : Create
  3. Ajoutez une action : Mouvement > Commencer à se Déplacer dans une Direction
  4. Choisissez une direction diagonale (pas tout droit vers le haut/bas)
  5. Définissez la vitesse sur 6

Rebondir sur les Murs

  1. Ajoutez un événement : Collision > obj_wall
  2. Ajoutez une action : Mouvement > Rebondir Contre les Objets
  3. Sélectionnez « Contre les objets solides »

Rebondir sur la Raquette Gauche

  1. Ajoutez un événement : Collision > obj_paddle_left
  2. Ajoutez une action : Mouvement > Rebondir Contre les Objets
  3. Sélectionnez « Contre les objets solides »

Rebondir sur la Raquette Droite

  1. Ajoutez un événement : Collision > obj_paddle_right
  2. Ajoutez une action : Mouvement > Rebondir Contre les Objets
  3. Sélectionnez « Contre les objets solides »
Conseil de test : Vous pouvez tester votre jeu maintenant ! La balle devrait rebondir partout, mais le score ne fonctionnera pas encore.
Cliquez sur « Suivant » pour ajouter les buts de score.
