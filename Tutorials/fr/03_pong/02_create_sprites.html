Étape 1 : Créer les Sprites

Nous avons besoin de quatre sprites pour notre jeu Pong : une balle, deux raquettes et des murs.

Créer le Sprite de la Balle

  1. Faites un clic droit sur Sprites dans l'arbre des ressources
  2. Sélectionnez Créer un Sprite
  3. Nommez-le spr_ball
  4. Cliquez sur Modifier le Sprite
  5. Dessinez un petit cercle blanc (environ 16x16 pixels)
  6. Cliquez sur OK pour enregistrer

Créer les Sprites des Raquettes

Raquette Gauche (Joueur 1) :

  1. Créez un nouveau sprite nommé spr_paddle_left
  2. Dessinez un rectangle haut et fin (environ 16x64 pixels)
  3. Utilisez la couleur bleue pour identifier le Joueur 1

Raquette Droite (Joueur 2) :

  1. Créez un nouveau sprite nommé spr_paddle_right
  2. Dessinez un rectangle haut et fin (environ 16x64 pixels)
  3. Utilisez la couleur rouge pour identifier le Joueur 2

Créer le Sprite du Mur

  1. Créez un nouveau sprite nommé spr_wall
  2. Dessinez un rectangle gris uni (32x32 pixels)
  3. Il sera étiré pour former les bordures supérieure et inférieure
Astuce : Gardez tous les sprites simples ! Le Pong classique utilisait des formes basiques.
Cliquez sur « Suivant » pour créer les objets du jeu.
