Step 1: Create the Sprites

We need four sprites for our Pong game: a ball, two paddles, and walls.

Create the Ball Sprite

  1. Right-click on Sprites in the Resource Tree
  2. Select Create Sprite
  3. Name it spr_ball
  4. Click Edit Sprite
  5. Draw a small white circle (about 16x16 pixels)
  6. Click OK to save

Create the Paddle Sprites

Left Paddle (Player 1):

  1. Create a new sprite named spr_paddle_left
  2. Draw a tall, thin rectangle (about 16x64 pixels)
  3. Use blue color to identify Player 1

Right Paddle (Player 2):

  1. Create a new sprite named spr_paddle_right
  2. Draw a tall, thin rectangle (about 16x64 pixels)
  3. Use red color to identify Player 2

Create the Wall Sprite

  1. Create a new sprite named spr_wall
  2. Draw a solid gray rectangle (32x32 pixels)
  3. This will be stretched to form the top and bottom boundaries
Tip: Keep all sprites simple! Classic Pong used basic shapes.
