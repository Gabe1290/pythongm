Phase 1 : Raquettes et balle rebondissante

Créons tout ce qu'il faut pour un jeu de Pong jouable !

Créer les sprites

Créez ces 3 sprites (dessinez-les ou importez des images) :

SpriteTailleDescription
spr_ball16×16Petit cercle blanc
spr_paddle16×64Rectangle étroit et haut
spr_wall32×32Bloc gris

Créer les objets

  1. Créez obj_wall — sprite : spr_wall, cochez Solide
  2. Créez obj_paddle_left — sprite : spr_paddle, cochez Solide
  3. Créez obj_paddle_right — sprite : spr_paddle, cochez Solide
  4. Créez obj_ball — sprite : spr_ball, laissez Solide décoché

Programmer la raquette gauche (Joueur 1 : W/S)

Double-cliquez sur obj_paddle_left → onglet Blockly.

Clavier : W (maintenu)
Définir la vitesse verticale à -8
Clavier : S (maintenu)
Définir la vitesse verticale à 8
Clavier : Aucune touche
Arrêter le mouvement
En collision avec obj_wall
Arrêter le mouvement

Cliquez sur Appliquer aux événements.

Programmer la raquette droite (Joueur 2 : Flèches)

Double-cliquez sur obj_paddle_right → onglet Blockly. Même structure mais avec des touches différentes :

Clavier : Haut (maintenu)
Définir la vitesse verticale à -8
Clavier : Bas (maintenu)
Définir la vitesse verticale à 8
Clavier : Aucune touche
Arrêter le mouvement
En collision avec obj_wall
Arrêter le mouvement

Cliquez sur Appliquer aux événements.

Programmer la balle

Double-cliquez sur obj_ball → onglet Blockly.

Événement Création
Commencer à se déplacer dans la direction 45 à la vitesse 6
En collision avec obj_wall
Rebondir contre les objets solides
En collision avec obj_paddle_left
Rebondir contre les objets solides
En collision avec obj_paddle_right
Rebondir contre les objets solides

Cliquez sur Appliquer aux événements.

Créer la salle

  1. Créez une nouvelle salle nommée room_pong (640×480)
  2. Placez obj_wall le long des bords haut et bas
  3. Placez obj_paddle_left près du bord gauche, centré verticalement
  4. Placez obj_paddle_right près du bord droit, centré verticalement
  5. Placez obj_ball au centre de la salle
+--[WALL WALL WALL WALL WALL WALL]--+
|                                     |
|  [P1]                        [P2]  |
|  [  ]         (o)            [  ]  |
|  [  ]                        [  ]  |
|                                     |
+--[WALL WALL WALL WALL WALL WALL]--+

Testez votre jeu !

Appuyez sur F5 pour jouer !

  • Joueur 1 : W (haut) et S (bas)
  • Joueur 2 : Flèche Haut et Flèche Bas
  • La balle rebondit sur les murs et les raquettes
  • La balle s'envole par les côtés — nous ajouterons les buts ensuite !
Cliquez sur « Suivant » pour ajouter les buts et le score.
