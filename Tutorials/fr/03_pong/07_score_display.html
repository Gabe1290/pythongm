Étape 6 : Afficher les Scores

Nous avons besoin d'un objet contrôleur pour initialiser les scores et les afficher.

Créer l'Objet Contrôleur de Score

  1. Créez un nouvel objet nommé obj_score
  2. Pas de sprite nécessaire (laissez-le vide)

Initialiser les Scores (Événement Create)

  1. Ajoutez un événement : Create
  2. Ajoutez une action : Score > Définir une Variable
  3. Variable : global.p1score, Valeur : 0
  4. Ajoutez une autre action Définir une Variable
  5. Variable : global.p2score, Valeur : 0

Dessiner les Scores (Événement Draw)

  1. Ajoutez un événement : Draw
  2. Ajoutez une action : Dessiner > Dessiner du Texte
  3. Texte : Joueur 1 :
  4. X : 10, Y : 10
  1. Ajoutez une action : Dessiner > Dessiner une Variable
  2. Variable : global.p1score
  3. X : 100, Y : 10
  1. Ajoutez une action : Dessiner > Dessiner du Texte
  2. Texte : Joueur 2 :
  3. X : 10, Y : 30
  1. Ajoutez une action : Dessiner > Dessiner une Variable
  2. Variable : global.p2score
  3. X : 100, Y : 30
N'oubliez pas : N'oubliez pas de placer obj_score dans votre room !
Cliquez sur « Suivant » pour configurer la room du jeu.
