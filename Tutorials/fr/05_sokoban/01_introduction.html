Sokoban : Puzzle de Poussée de Caisses

SOKOBAN

Poussez toutes les caisses sur les emplacements cibles !

Ce que nous allons créer

Dans ce tutoriel, vous allez construire un jeu de puzzle Sokoban classique :

  • Un joueur qui se déplace sur une grille (un pas à la fois)
  • Des caisses que le joueur peut pousser (mais pas tirer)
  • Des emplacements cibles où les caisses doivent être placées
  • Une condition de victoire quand toutes les caisses sont sur les cibles

Ce que vous apprendrez

  • Le mouvement basé sur une grille (déplacement par pas fixes)
  • Les mécaniques de poussée - détecter et déplacer d'autres objets
  • La vérification de conditions multiples avant de se déplacer
  • La détection de condition de victoire
  • La conception de niveaux pour les jeux de puzzle
Temps requis : Environ 30-35 minutes
