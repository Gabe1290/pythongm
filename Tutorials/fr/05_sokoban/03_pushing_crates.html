Phase 2 : Pousser des caisses

Ajoutons maintenant des caisses que le joueur peut pousser — la mécanique principale du Sokoban !

Créer le sprite de la caisse

  1. Faites un clic droit sur Sprites → Créer un nouveau Sprite
  2. Nommez-le spr_crate (32×32 pixels)
  3. Dessinez une boîte marron ou importez-la depuis Tutorials/05_sokoban/assets/
  4. Appuyez sur Ctrl+S pour enregistrer

Créer l'objet caisse

  1. Faites un clic droit sur Objets → Créer un nouvel Objet
  2. Nommez-le obj_crate, définissez son sprite à spr_crate
  3. Cochez la case Solide

Mettre à jour le joueur : pousser les caisses

  1. Double-cliquez sur obj_player pour ouvrir l'éditeur d'objets
  2. Cliquez sur l'onglet Blockly
  3. Ajoutez un nouvel événement de collision pour obj_crate :
Collision avec obj_crate
Si peut pousser facing
Pousser l'autre instance 32

Trouvez Collision dans Événements et tapez obj_crate. Puis trouvez Si peut pousser dans Contrôle — ce bloc vérifie si la caisse peut être poussée dans la direction où le joueur se déplace. Dans la section « alors », insérez un bloc Pousser l'autre instance avec une distance de 32.

Comment fonctionne la poussée : Lorsque le joueur entre en collision avec une caisse, le moteur vérifie si l'espace derrière la caisse (dans la direction du déplacement du joueur) est libre. Si c'est le cas, la caisse est poussée d'une cellule. S'il y a un mur ou une autre caisse derrière, la poussée est bloquée.

Programmer la caisse : s'arrêter contre les murs

  1. Double-cliquez sur obj_crate pour ouvrir l'éditeur d'objets
  2. Cliquez sur l'onglet Blockly
  3. Ajoutez un événement de collision pour obj_wall :
Collision avec obj_wall
Arrêter le mouvement

Cela garantit que les caisses s'arrêtent lorsqu'elles touchent un mur.

Cliquez sur Appliquer aux événements pour enregistrer les blocs de la caisse.

Mettre à jour la salle

  1. Ouvrez room_sokoban
  2. Placez 2–3 instances de obj_crate à l'intérieur de la salle
  3. Assurez-vous que chaque caisse a de l'espace pour être poussée
Astuce : Ne placez pas une caisse directement dans un coin — elle serait coincée immédiatement !

Testez votre jeu !

Appuyez sur F5 pour jouer !

  • Approchez-vous d'une caisse et poussez-la avec les touches fléchées
  • Les caisses se déplacent d'une cellule lorsqu'elles sont poussées
  • Les caisses s'arrêtent lorsqu'elles touchent un mur
  • Vous ne pouvez pas pousser une caisse si une autre caisse ou un mur se trouve derrière
Cliquez sur « Suivant » pour ajouter les cibles et terminer le jeu !
