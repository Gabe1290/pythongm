Étape 3 : Comportement des Caisses

La caisse doit changer d'apparence quand elle est sur un emplacement cible.

Vérifier la Position sur la Cible (Événement Step)

Dans obj_crate :

  1. Ajoutez un événement : Step
  2. Ajoutez une action : Contrôle > Tester la Position d'Instance
  3. Objet : obj_target
  4. X : 0, Y : 0 (relatif)
  5. Vérifiez si une collision existe

Changer le Sprite quand sur la Cible

Si une collision existe (la caisse est sur la cible) :

  1. Ajoutez une action : Main1 > Changer le Sprite
  2. Sprite : spr_crate_ok

Changer le Sprite quand pas sur la Cible

  1. Ajoutez une action : Contrôle > Sinon
  2. Ajoutez une action : Main1 > Changer le Sprite
  3. Sprite : spr_crate
Retour Visuel : Quand la caisse devient verte, les joueurs savent qu'elle est correctement placée !
