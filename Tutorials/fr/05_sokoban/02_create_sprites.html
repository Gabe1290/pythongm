Étape 1 : Créer les Sprites

Tous les sprites doivent avoir la même taille (32x32 pixels) pour créer une grille correcte.

Créer le Sprite du Joueur

  1. Faites un clic droit sur Sprites dans l'arbre des ressources
  2. Sélectionnez Créer un Sprite
  3. Nommez-le spr_player
  4. Cliquez sur Modifier le Sprite
  5. Dessinez un personnage simple (32x32 pixels)
  6. Utilisez une couleur distincte comme le bleu
  7. Cliquez sur OK pour enregistrer

Créer les Sprites des Caisses

Caisse Normale :

  1. Créez le sprite spr_crate
  2. Dessinez une forme de boîte en bois (32x32 pixels)
  3. Utilisez une couleur marron ou orange

Caisse sur Cible :

  1. Créez le sprite spr_crate_ok
  2. Dessinez la même caisse mais en couleur verte
  3. Cela indique quand une caisse est correctement placée

Créer le Sprite du Mur

  1. Créez le sprite spr_wall
  2. Dessinez un motif de brique ou de pierre solide (32x32 pixels)
  3. Utilisez une couleur grise ou foncée

Créer le Sprite de la Cible

  1. Créez le sprite spr_target
  2. Dessinez une marque X ou un indicateur d'objectif (32x32 pixels)
  3. Utilisez une couleur vive comme le rouge
Important : Gardez TOUS les sprites exactement à 32x32 pixels pour un alignement correct de la grille !
Cliquez sur « Suivant » pour créer les objets du jeu.
