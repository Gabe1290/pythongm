Étape 4 : Mouvement du Joueur

Le joueur se déplace par pas fixes de 32 pixels (mouvement basé sur une grille).

Important : Dans Sokoban, le mouvement doit vérifier :
1. Y a-t-il un mur ? (Impossible de se déplacer)
2. Y a-t-il une caisse ? (Peut-on la pousser ?)
3. Y a-t-il un mur/une caisse derrière la caisse ? (Impossible de pousser)

Se Déplacer à Droite (Flèche Droite)

Dans obj_player :

  1. Ajoutez un événement : Clavier > Appuyer Droite
  2. Ajoutez une action : Contrôle > Exécuter du Code
  3. Entrez ce code :
// Vérifier si on peut se déplacer à droite
if (!place_meeting(x + 32, y, obj_wall)) {
    var crate = instance_place(x + 32, y, obj_crate);
    if (crate != noone) {
        // Il y a une caisse - peut-on la pousser ?
        if (!place_meeting(x + 64, y, obj_wall) &&
            !place_meeting(x + 64, y, obj_crate)) {
            crate.x += 32;
            x += 32;
        }
    } else {
        // Pas de caisse, on se déplace simplement
        x += 32;
    }
}

Se Déplacer à Gauche (Flèche Gauche)

Même code mais changez :

  • + 32 devient - 32
  • + 64 devient - 64

Se Déplacer vers le Haut (Flèche Haut)

Même code mais utilisez y au lieu de x :

  • x + 32 devient y - 32
  • x + 64 devient y - 64
  • crate.x devient crate.y

Se Déplacer vers le Bas (Flèche Bas)

Comme pour le Haut mais avec + 32 et + 64.

Cliquez sur « Suivant » pour ajouter la détection de condition de victoire.
