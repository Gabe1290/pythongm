Étape 2 : Créer les Objets

Nous avons besoin d'objets pour le joueur, les murs, les caisses et les cibles.

Créer l'Objet Mur

  1. Faites un clic droit sur Objets et sélectionnez Créer un Objet
  2. Nommez-le obj_wall
  3. Définissez le sprite sur spr_wall
  4. Cochez « Solide »
  5. Aucun événement nécessaire

Créer l'Objet Cible

  1. Créez l'objet obj_target
  2. Définissez le sprite sur spr_target
  3. Laissez « Solide » décoché (le joueur et les caisses peuvent être dessus)
  4. Aucun événement nécessaire - c'est juste un marqueur

Créer l'Objet Caisse

  1. Créez l'objet obj_crate
  2. Définissez le sprite sur spr_crate
  3. Cochez « Solide »

Créer l'Objet Joueur

  1. Créez l'objet obj_player
  2. Définissez le sprite sur spr_player
  3. Laissez « Solide » décoché
Note : Nous ajouterons des événements à obj_crate et obj_player dans les prochaines étapes.
Cliquez sur « Suivant » pour ajouter le comportement des caisses.
