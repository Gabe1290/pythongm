Étape 5 : Condition de Victoire

Le joueur gagne quand toutes les caisses sont sur les emplacements cibles.

Créer le Contrôleur de Jeu

  1. Créez l'objet obj_game_controller
  2. Pas de sprite nécessaire

Compter les Cibles (Événement Create)

Dans obj_game_controller :

  1. Ajoutez un événement : Create
  2. Ajoutez une action : Contrôle > Exécuter du Code
  3. Entrez : global.total_targets = instance_number(obj_target);

Vérifier la Condition de Victoire (Événement Step)

  1. Ajoutez un événement : Step
  2. Ajoutez une action : Contrôle > Exécuter du Code
  3. Entrez ce code :
// Compter les caisses sur les cibles
var crates_on_targets = 0;
with (obj_crate) {
    if (place_meeting(x, y, obj_target)) {
        crates_on_targets += 1;
    }
}

// Vérifier si toutes les cibles ont des caisses
if (crates_on_targets >= global.total_targets
    && global.total_targets > 0) {
    show_message("Niveau Terminé !");
    room_restart();
}

Afficher les Instructions (Événement Draw)

  1. Ajoutez un événement : Draw
  2. Ajoutez une action : Dessiner > Dessiner du Texte
  3. Texte : Poussez les caisses sur les cibles !
  4. X : 10, Y : 10
Astuce : Assurez-vous que le nombre de caisses est égal au nombre de cibles dans votre niveau !
Cliquez sur « Suivant » pour configurer la room du jeu.
