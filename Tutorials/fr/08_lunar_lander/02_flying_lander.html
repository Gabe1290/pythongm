Phase 1 : Un atterrisseur volant

Dans cette phase, nous créons les sprites, les objets et les contrôles pour piloter un atterrisseur avec la gravité lunaire. À la fin, vous aurez un atterrisseur jouable que vous pourrez piloter !

Créer les Sprites

spr_lander (32x32) — Le vaisseau spatial

  1. Clic droit sur Sprites > Créer un nouveau Sprite
  2. Nommez-le spr_lander (32x32 pixels)
  3. Dessinez un petit vaisseau spatial (triangle ou forme classique d'atterrisseur), ou importez depuis Tutorials/08_lunar_lander/assets/

spr_ground (32x32) — Terrain rocheux

  1. Créez un nouveau sprite nommé spr_ground (32x32 pixels)
  2. Dessinez un terrain rocheux avec des couleurs grises ou marron, ou importez depuis les ressources

Créer les Objets

obj_ground

  1. Clic droit sur Objets > Créer un nouvel Objet
  2. Nommez-le obj_ground, assignez spr_ground
  3. Cochez la case Solide

obj_lander

  1. Créez un nouvel objet nommé obj_lander
  2. Assignez spr_lander

Programmer l'atterrisseur (Blockly)

Double-cliquez sur obj_lander et ouvrez l'onglet Blockly.

1. Activer la gravité (Événement Création)

Create
Set Gravity: direction 270, strength 0.05

Trouvez Create dans la catégorie Events. Puis trouvez Définir la gravité dans la catégorie Movement. Réglez la direction à 270 (vers le bas) et la force à 0.05.

Pourquoi 0.05 ? La gravité de la Lune est environ 1/6e de celle de la Terre. Une valeur très faible comme 0.05 donne à l'atterrisseur une sensation lente et flottante — comme un vrai alunissage !

2. Poussée vers le haut (Clavier Haut maintenu)

Keyboard Held: Up
Set vspeed -2

Trouvez Clavier (maintenu) dans Events, choisissez Up. Ajoutez Set vspeed (depuis Movement) avec la valeur -2. Un vspeed négatif déplace l'atterrisseur vers le haut (poussée contre la gravité).

3. Diriger à gauche (Clavier Gauche maintenu)

Keyboard Held: Left
Set hspeed -2

Trouvez Clavier (maintenu) dans Events, choisissez Left. Ajoutez Set hspeed (depuis Movement) avec la valeur -2.

4. Diriger à droite (Clavier Droite maintenu)

Keyboard Held: Right
Set hspeed 2

Trouvez Clavier (maintenu) dans Events, choisissez Right. Ajoutez Set hspeed (depuis Movement) avec la valeur 2.

5. Arrêter le mouvement horizontal quand aucune touche n'est pressée

No Key
Set hspeed 0

Trouvez Aucune touche dans Events. Ajoutez Set hspeed avec la valeur 0. Cela arrête le mouvement horizontal quand aucune flèche n'est pressée, mais la gravité continue d'attirer l'atterrisseur vers le bas !

Important : Nous ne remettons à 0 que hspeed, pas vspeed. Cela permet à la gravité de continuer à agir sur l'atterrisseur quand aucune touche n'est pressée — l'atterrisseur continue de tomber !

Appliquer vos blocs

  1. Vous devriez avoir 5 blocs d'événements (Create + 3 clavier + No key)
  2. Cliquez sur le bouton Appliquer aux événements pour sauvegarder

Créer la salle

  1. Clic droit sur Rooms > Créer une nouvelle salle
  2. Nommez-la room_game
  3. Définissez la taille : 640 x 480
  4. Définissez la couleur de fond en noir (l'espace !)
  5. Activez « Aligner sur la grille » (32x32)
  6. Placez des blocs obj_ground le long de toute la rangée du bas
  7. Placez obj_lander près du haut de la salle

Testez votre jeu !

Appuyez sur F5 pour jouer !

  • L'atterrisseur tombe lentement vers le sol (gravité lunaire !)
  • Flèche Haut : Poussée vers le haut pour lutter contre la gravité
  • Flèches Gauche/Droite : Déplacement horizontal
  • Relâchez toutes les touches et observez l'atterrisseur dériver lentement vers le bas
Cliquez sur « Suivant » pour ajouter l'atterrissage et le crash !
