Étape 3 : Physique de l'Alunisseur (Événement Create)

Configurez les variables pour la gravité, la poussée, le carburant et la vélocité.

Initialiser les Variables Physiques

Dans obj_lander :

  1. Ajoutez un événement : Create
  2. Ajoutez une action : Contrôle > Exécuter du Code
// Paramètres physiques
gravity_force = 0.05;  // Gravité douce
thrust_force = 0.1;    // Puissance de poussée
max_speed = 5;         // Vélocité maximale

// Vélocité actuelle
hsp = 0;  // Vitesse horizontale
vsp = 0;  // Vitesse verticale

// Système de carburant
fuel = 100;
fuel_use = 0.5;  // Carburant par frame

// État du jeu
landed = false;
crashed = false;

// Vitesse d'atterrissage sûre
safe_speed = 2;

Comprendre les Variables

Variable Fonction
gravity_force Force descendante constante (0.05 est douce)
thrust_force Force ascendante en appuyant sur HAUT
fuel 100% de carburant, diminue avec la poussée
safe_speed Vitesse max pour un atterrissage sûr (2)
Équilibre : La poussée (0.1) est plus forte que la gravité (0.05) pour pouvoir ralentir !
Cliquez sur « Suivant » pour ajouter le mouvement et les contrôles.
