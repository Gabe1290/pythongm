Étape 1 : Créer les Sprites

Nous avons besoin de sprites pour l'alunisseur, la plateforme d'atterrissage et le terrain.

Nom du Sprite Taille Description
spr_lander 32x32 Vaisseau spatial (triangle ou forme d'alunisseur classique)
spr_pad 64x16 Plateforme d'atterrissage avec marquage « H »
spr_ground 32x32 Terrain rocheux (gris/marron)

Créer le Sprite de l'Alunisseur

  1. Faites un clic droit sur Sprites > Créer un Sprite
  2. Nommez-le spr_lander
  3. Cliquez sur Modifier le Sprite
  4. Dessinez un vaisseau spatial (32x32 pixels)
  5. Définissez l'origine au centre-bas pour un atterrissage correct

Créer le Sprite de la Plateforme

  1. Créez un sprite nommé spr_pad
  2. Taille : 64x16 pixels
  3. Dessinez une plateforme plate avec des marquages
  4. Utilisez des couleurs vives (jaune/vert)

Créer le Sprite du Sol

  1. Créez un sprite nommé spr_ground
  2. Taille : 32x32 pixels
  3. Dessinez un terrain rocheux et accidenté
  4. Utilisez des couleurs gris/marron
Important : Définissez l'origine de l'alunisseur au centre-bas pour qu'il atterrisse correctement sur la plateforme !
Cliquez sur « Suivant » pour créer les objets du jeu.
