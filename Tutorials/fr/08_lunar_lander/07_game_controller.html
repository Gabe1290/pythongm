Étape 6 : Contrôleur de Jeu (HUD)

Affichez le carburant, la vitesse et l'état pour aider le joueur.

Événement Draw

Dans obj_game_controller :

  1. Ajoutez un événement : Draw > Draw
  2. Ajoutez une action : Contrôle > Exécuter du Code
draw_set_color(c_white);

// Obtenir les infos de l'alunisseur
var lander = instance_find(obj_lander, 0);
if (lander != noone) {
    var spd = sqrt(lander.hsp*lander.hsp + lander.vsp*lander.vsp);

    // Afficher le HUD
    draw_text(10, 10, "ALUNISSEUR");
    draw_text(10, 30, "Carburant : " + string(floor(lander.fuel)) + "%");
    draw_text(10, 50, "Vitesse : " + string_format(spd, 1, 2));
    draw_text(10, 70, "Sûre : < " + string(lander.safe_speed));

    // Avertissement de vitesse
    if (spd > lander.safe_speed) {
        draw_set_color(c_red);
        draw_text(10, 90, "TROP RAPIDE !");
    } else {
        draw_set_color(c_lime);
        draw_text(10, 90, "Vitesse OK");
    }

    // Avertissement de carburant
    if (lander.fuel <= 0) {
        draw_set_color(c_red);
        draw_text(10, 110, "PLUS DE CARBURANT !");
    }
}

draw_set_color(c_gray);
draw_text(10, room_height - 20, "HAUT : Poussée | GAUCHE/DROITE : Bouger");

Ce Que le HUD Affiche

  • Carburant : Pourcentage restant (vert = OK)
  • Vitesse : Vélocité actuelle (rouge = dangereux)
  • Avertissements : « TROP RAPIDE ! » ou « PLUS DE CARBURANT ! »
  • Contrôles : Rappel en bas
