Phase 2 : Atterrissage et crash

Maintenant nous ajoutons une aire d'atterrissage et la détection de collision. Atterrissez sur l'aire pour gagner — écrasez-vous sur le sol pour recommencer !

Créer le sprite de l'aire d'atterrissage

spr_pad (32x32) — Une aire d'atterrissage verte et plate

  1. Clic droit sur Sprites > Créer un nouveau Sprite
  2. Nommez-le spr_pad (32x32 pixels)
  3. Dessinez une plateforme verte et plate (un rectangle vert vif), ou importez depuis les ressources

Créer l'objet aire d'atterrissage

  1. Clic droit sur Objets > Créer un nouvel Objet
  2. Nommez-le obj_pad, assignez spr_pad
  3. Cochez la case Solide

Programmer les collisions de l'atterrisseur (Blockly)

Ouvrez l'éditeur Blockly pour obj_lander et ajoutez deux nouveaux événements de collision.

1. Atterrir sur l'aire (Succès !)

Collision avec obj_pad
Arrêter le mouvement
Afficher un message "Landing successful!"

Trouvez Collision dans Événements, tapez obj_pad. Ajoutez Arrêter le mouvement (depuis Mouvement) pour stopper l'atterrisseur. Puis ajoutez Afficher un message (depuis Jeu) avec votre texte de victoire.

2. S'écraser sur le sol (Échec !)

Collision avec obj_ground
Afficher un message "Crashed!"
Recommencer la salle

Trouvez Collision dans Événements, tapez obj_ground. Ajoutez Afficher un message (depuis Jeu) avec le texte de crash, puis Recommencer la salle (depuis Salle). Toucher le sol rocheux réinitialise le niveau !

Appliquer vos blocs

  1. Vous devriez maintenant avoir 7 blocs d'événements au total (Créer + 3 clavier + Aucune touche + 2 collisions)
  2. Cliquez sur Appliquer aux événements pour sauvegarder

Mettre à jour la salle

Ouvrez room_game et mettez à jour le terrain :

  1. Rendez le sol irrégulier — empilez des blocs obj_ground plus haut à certains endroits pour créer des collines
  2. Retirez un ou deux blocs de sol de la rangée du bas pour créer un espace
  3. Placez obj_pad dans l'espace (la zone d'atterrissage !)

Exemple de disposition de niveau

        L                          <- L'atterrisseur commence ici



GG       GG
GGG     GGGG    PP     GGGGG     GGG
GGGGGGGGGGGG    PP     GGGGGGGGGGGGG

G = Sol (crash !)    L = Atterrisseur    P = Aire d'atterrissage (sûr !)
Conseil : Rendez la zone d'atterrissage assez petite pour que ce soit un défi, mais pas trop minuscule ! Un ou deux blocs d'aire côte à côte fonctionnent bien.

Testez votre jeu !

Appuyez sur F5 pour jouer !

  • Volez vers l'aire d'atterrissage verte
  • Atterrissez sur l'aire → le message « Landing successful! » apparaît
  • Écrasez-vous sur le sol → le message « Crashed! » apparaît, puis la salle redémarre
  • Utilisez de courtes impulsions de poussée (tapotez Haut) pour contrôler votre descente
