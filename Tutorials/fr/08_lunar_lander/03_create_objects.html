Étape 2 : Créer les Objets

Maintenant, nous créons des objets pour tous les éléments du jeu.

Objet Sol (Dangereux)

  1. Faites un clic droit sur Objets > Créer un Objet
  2. Nommez-le obj_ground
  3. Définissez le sprite sur spr_ground
  4. Cochez la case « Solide »

Objet Plateforme d'Atterrissage (Sûr)

  1. Créez obj_pad
  2. Définissez le sprite sur spr_pad
  3. Cochez la case « Solide »

Objet Alunisseur (Joueur)

  1. Créez obj_lander
  2. Définissez le sprite sur spr_lander

Objet Contrôleur de Jeu

  1. Créez obj_game_controller
  2. Pas de sprite nécessaire (invisible)
Différence Clé : Atterrir sur obj_pad = atterrissage réussi. Atterrir sur obj_ground = crash !
Cliquez sur « Suivant » pour ajouter la physique de l'alunisseur.
