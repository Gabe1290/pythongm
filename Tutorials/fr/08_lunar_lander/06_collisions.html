Étape 5 : Détection des Collisions

Gérez l'atterrissage sur la plateforme et le crash sur le terrain.

Collision avec la Plateforme d'Atterrissage

Dans obj_lander :

  1. Ajoutez un événement : Collision > obj_pad
  2. Ajoutez une action : Contrôle > Exécuter du Code
// Calculer la vitesse totale
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    // Atterrissage réussi !
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("Atterrissage Parfait ! Vous Avez Gagné !");
} else {
    // Trop rapide - crash !
    crashed = true;
    show_message("Crash ! Trop rapide !");
    room_restart();
}

Collision avec le Sol (Crash)

Dans obj_lander :

  1. Ajoutez un événement : Collision > obj_ground
  2. Ajoutez une action : Contrôle > Exécuter du Code
crashed = true;
show_message("Crash sur le terrain !");
room_restart();

Calcul de la Vitesse

Nous utilisons le théorème de Pythagore pour obtenir la vitesse totale :

total_speed = sqrt(hsp*hsp + vsp*vsp)

Cela combine la vitesse horizontale et verticale en une seule valeur.

La vitesse sûre est 2 ! Si total_speed > 2, vous crashez même sur la plateforme.
