Étape 4 : Contrôles de l'Alunisseur (Événement Step)

Ajoutez la gravité, la poussée, le mouvement horizontal et les vérifications de limites.

Ajouter l'Événement Step

Dans obj_lander :

  1. Ajoutez un événement : Step > Step
  2. Ajoutez une action : Contrôle > Exécuter du Code
// Ne pas mettre à jour si le jeu est terminé
if (landed || crashed) exit;

// === GRAVITÉ ===
vsp += gravity_force;

// === POUSSÉE (Flèche Haut) ===
if (keyboard_check(vk_up) && fuel > 0) {
    vsp -= thrust_force;
    fuel -= fuel_use;
    if (fuel < 0) fuel = 0;
}

// === CONTRÔLE HORIZONTAL ===
if (keyboard_check(vk_left)) hsp -= 0.05;
if (keyboard_check(vk_right)) hsp += 0.05;

// === LIMITER LA VITESSE ===
hsp = clamp(hsp, -max_speed, max_speed);
vsp = clamp(vsp, -max_speed, max_speed);

// === APPLIQUER LE MOUVEMENT ===
x += hsp;
y += vsp;

// === LIMITES DE L'ÉCRAN ===
if (x < 16) { x = 16; hsp = 0; }
if (x > room_width - 16) { x = room_width - 16; hsp = 0; }
if (y < 16) { y = 16; vsp = 0; }

Comment Ça Fonctionne

  1. Gravité : S'ajoute à vsp à chaque frame (chute)
  2. Poussée : Soustrait de vsp (montée), utilise du carburant
  3. Clamp : Limite la vitesse à une plage sûre
  4. Limites : Garde l'alunisseur à l'écran
Clé : La fonction clamp() limite une valeur entre un minimum et un maximum !
