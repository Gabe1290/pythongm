Étape 1 : Créer les Sprites

Nous avons besoin de sprites pour tous les éléments de notre jeu.

Nom du Sprite Taille Description
spr_player 32x48 Personnage simple (rectangle avec visage)
spr_ground 32x32 Tuile de plateforme herbe/terre
spr_platform 64x16 Plateforme flottante (bois/pierre)
spr_coin 16x16 Cercle jaune/doré
spr_spike 32x32 Piques triangulaires pointant vers le haut
spr_flag 32x64 Drapeau sur un mât (objectif)

Créer le Sprite du Joueur

  1. Faites un clic droit sur Sprites > Créer un Sprite
  2. Nommez-le spr_player
  3. Cliquez sur Modifier le Sprite
  4. Dessinez un personnage (32x48 pixels)
  5. Utilisez une couleur vive comme le bleu ou le rouge

Créer le Sprite du Sol

  1. Créez un sprite nommé spr_ground
  2. Taille : 32x32 pixels
  3. Dessinez une tuile herbe/terre (vert en haut, marron en bas)
Astuce : Le joueur est plus haut que large (32x48) pour ressembler à un personnage !
Cliquez sur « Suivant » pour créer les objets du jeu.
