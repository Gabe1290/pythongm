Étape 3 : Physique du Joueur (Événement Create)

Nous devons configurer des variables pour la gravité, le saut et le mouvement.

Initialiser les Variables Physiques

Dans obj_player :

  1. Ajoutez un événement : Create
  2. Ajoutez une action : Contrôle > Exécuter du Code
// Paramètres de mouvement
hspeed_max = 4;      // Vitesse horizontale maximale
vspeed_max = 10;     // Vitesse de chute maximale
jump_force = -10;    // Force de saut (négatif = vers le haut)
gravity_force = 0.5; // Vitesse de chute

// Vitesses actuelles
hsp = 0;
vsp = 0;

// État
on_ground = false;

Comprendre les Variables

Variable Fonction
hsp Vitesse horizontale (gauche/droite)
vsp Vitesse verticale (haut/bas)
gravity_force Force constante tirant vers le bas
jump_force Valeur négative pour aller vers le haut
on_ground Vrai quand on est debout sur le sol
Concept Clé : La force de saut est négative car Y augmente vers le bas dans les jeux. Une vélocité Y négative déplace le joueur vers le haut !
