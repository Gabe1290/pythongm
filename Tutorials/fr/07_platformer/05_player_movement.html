Étape 4 : Mouvement du Joueur (Événement Step)

C'est le cœur de notre jeu de plateformes : mouvement, gravité, saut et collision.

Ajouter l'Événement Step

Dans obj_player :

  1. Ajoutez un événement : Step > Step
  2. Ajoutez une action : Contrôle > Exécuter du Code
// === MOUVEMENT HORIZONTAL ===
var move_input = keyboard_check(vk_right) - keyboard_check(vk_left);
hsp = move_input * hspeed_max;

// === GRAVITÉ ===
vsp += gravity_force;
if (vsp > vspeed_max) vsp = vspeed_max;

// === VÉRIFICATION DU SOL ===
on_ground = place_meeting(x, y + 1, obj_ground);

// === SAUT ===
if (on_ground && (keyboard_check_pressed(vk_up) || keyboard_check_pressed(vk_space))) {
    vsp = jump_force;
}

// === COLLISION HORIZONTALE ===
if (place_meeting(x + hsp, y, obj_ground)) {
    while (!place_meeting(x + sign(hsp), y, obj_ground)) {
        x += sign(hsp);
    }
    hsp = 0;
}
x += hsp;

// === COLLISION VERTICALE ===
if (place_meeting(x, y + vsp, obj_ground)) {
    while (!place_meeting(x, y + sign(vsp), obj_ground)) {
        y += sign(vsp);
    }
    vsp = 0;
}
y += vsp;

Comment Ça Fonctionne

  1. Entrée : Flèche droite = 1, Gauche = -1, Rien = 0
  2. Gravité : S'ajoute à vsp à chaque frame (chute plus rapide)
  3. Vérification du Sol : Regarde 1 pixel en dessous pour le sol
  4. Saut : Fonctionne seulement quand on_ground est vrai
  5. Collision : Se déplace pixel par pixel jusqu'à toucher le mur
La Magie : La boucle while nous déplace pixel par pixel jusqu'au bord du mur, nous empêchant de rester coincé !
