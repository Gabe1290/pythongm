Étape 6 : Contrôleur de Jeu

Le contrôleur de jeu affiche le score à l'écran.

Créer l'Objet Contrôleur

  1. Créez l'objet obj_game_controller
  2. Pas de sprite nécessaire (contrôleur invisible)

Afficher le Score

Dans obj_game_controller :

  1. Ajoutez un événement : Draw > Draw
  2. Ajoutez une action : Contrôle > Exécuter du Code
// Définir la couleur du texte
draw_set_color(c_white);

// Afficher le score en haut à gauche
draw_text(10, 10, "Score : " + string(score));

// Optionnel : Afficher les instructions
draw_text(10, 30, "Flèches : Bouger  |  Espace : Sauter");

Alternative : Utiliser les Actions

Au lieu du code, vous pouvez utiliser :

  1. Ajoutez une action : Draw > Définir la Couleur (Blanc)
  2. Ajoutez une action : Score > Afficher le Score
    • X : 10
    • Y : 10
    • Légende : « Score : »
Rappel : Placez le contrôleur de jeu dans votre room ! Il n'apparaîtra pas tout seul.
