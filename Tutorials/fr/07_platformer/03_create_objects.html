Étape 2 : Créer les Objets

Maintenant, nous créons des objets pour tous les éléments de notre jeu.

Objet Sol

  1. Faites un clic droit sur Objets > Créer un Objet
  2. Nommez-le obj_ground
  3. Définissez le sprite sur spr_ground
  4. Cochez la case « Solide »

Objet Plateforme

  1. Créez obj_platform
  2. Définissez le sprite sur spr_platform
  3. Cochez la case « Solide »
  4. Définissez le Parent sur obj_ground (optionnel)

Objet Joueur

  1. Créez obj_player
  2. Définissez le sprite sur spr_player

Objets à Collecter et Dangers

  1. Créez obj_coin avec spr_coin
  2. Créez obj_spike avec spr_spike
  3. Créez obj_flag avec spr_flag
Important : Le sol et les plateformes doivent être « Solides » pour que les collisions fonctionnent correctement !
Cliquez sur « Suivant » pour ajouter la physique du joueur (gravité et saut).
