Phase 2 : Pièces et dangers

Maintenant nous ajoutons des pièces à collecter, des pics dangereux et un drapeau à atteindre. Nous programmons tout sous forme de nouveaux événements de collision sur obj_player.

Créer les Sprites

  1. Créez le sprite spr_coin, taille 32x32 — dessinez un cercle jaune/doré (ou importez depuis Tutorials/07_platformer/assets/)
  2. Créez le sprite spr_spike, taille 32x32 — dessinez des pics triangulaires pointant vers le haut
  3. Créez le sprite spr_flag, taille 32x32 — dessinez un drapeau sur un mât

Créer les Objets

  1. Créez obj_coin, assignez le sprite spr_coin
  2. Créez obj_spike, assignez le sprite spr_spike
  3. Créez obj_flag, assignez le sprite spr_flag
Pas besoin de Blockly sur ces objets ! Nous allons programmer toutes les interactions sous forme d'événements de collision sur obj_player, puisque c'est le joueur qui touche les pièces, les pics et le drapeau.

Programmer obj_player — Ajouter des événements de collision

Ouvrez l'éditeur Blockly pour obj_player (les blocs de la Phase 1 sont toujours là). Ajoutez ces nouveaux blocs d'événements :

Collecter les pièces

When colliding with obj_coin
Add to score 10
Destroy other instance

Trouvez Collision dans Events, tapez obj_coin. Ajoutez Ajouter au score (depuis Score/Lives/Health) avec la valeur 10. Puis ajoutez Détruire l'instance (depuis Instance) réglé sur Other, pour que la pièce disparaisse quand elle est collectée.

Toucher un pic

When colliding with obj_spike
Restart room

Trouvez Collision dans Events, tapez obj_spike. Ajoutez Redémarrer la salle (depuis Room) à l'intérieur. Quand le joueur touche un pic, le niveau se réinitialise !

Atteindre le drapeau

When colliding with obj_flag
Show message "You Win!"

Trouvez Collision dans Events, tapez obj_flag. Ajoutez Afficher un message (depuis Output) et tapez You Win!. Un message apparaît quand le joueur atteint le drapeau !

Appliquer vos blocs

  1. Vous devriez maintenant avoir 9 blocs d'événements au total pour obj_player (6 de la Phase 1 + 3 nouveaux événements de collision)
  2. Cliquez sur Appliquer aux événements pour sauvegarder

Mettre à jour votre salle

  1. Ouvrez room_level1
  2. Placez obj_coin sur les plateformes et les endroits difficiles d'accès
  3. Placez obj_spike près des gouffres ou sur les plateformes dangereuses
  4. Placez obj_flag à la fin du niveau (côté droit)
                                        F

                  C           C
                ====        ====
    C                               C
  ====  X     X         X     ====
    P               C
  ====  ====  ====  ====  ====  ====  ====
  GGGGGG      GGGG  GGGG       GGGGGGGGGG

G = Sol    P = Joueur    F = Drapeau    C = Pièce    X = Pic
Conseil de conception de niveau : Placez les pics près des trous pour créer de la tension. Mettez les pièces sur les plateformes hautes pour récompenser les sauts habiles. Placez le drapeau tout au bout pour que le joueur doive traverser tout le niveau !

Testez votre jeu !

Appuyez sur F5 pour jouer !

  • Collectez les pièces — chacune donne 10 points et disparaît
  • Touchez un pic — la salle redémarre !
  • Atteignez le drapeau — un message « You Win! » apparaît
Cliquez sur « Suivant » pour ajouter un contrôleur de jeu qui affiche le score et les vies !
