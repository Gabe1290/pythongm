Step 5: Collectibles and Hazards

Now we add coins to collect, spikes to avoid, and a flag to reach.

Coin - Collect for Points

In obj_coin:

  1. Add Event: Collision > obj_player
  2. Add Action: Score > Set Score
    • New Score: 10
    • Check "Relative"
  3. Add Action: Main1 > Destroy Instance
    • Applies to: Self

Spike - Hurts the Player

In obj_spike:

  1. Add Event: Collision > obj_player
  2. Add Action: Main2 > Show Message
    • Message: Ouch! You hit a spike!
  3. Add Action: Main1 > Restart Room

Flag - Level Complete

In obj_flag:

  1. Add Event: Collision > obj_player
  2. Add Action: Main2 > Show Message
    • Message: Level Complete!
  3. Add Action: Main1 > Restart Room (or Next Room for multiple levels)
Tip: Place spikes near pits or on platforms to create challenge!
Click "Next" to add the game controller for score display.
