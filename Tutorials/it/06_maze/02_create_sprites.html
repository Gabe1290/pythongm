Step 1: Create the Sprites

We need sprites for the player, walls, exit, and coins. Walls should be 32x32 pixels to create a grid.

Create the Player Sprite

  1. Right-click on Sprites and select Create Sprite
  2. Name it spr_player
  3. Click Edit Sprite
  4. Draw a small character (circle or simple person)
  5. Use a bright color like blue or green
  6. Size: 24x24 pixels (smaller than walls)

Create the Wall Sprite

  1. Create a new sprite named spr_wall
  2. Draw a brick or stone pattern
  3. Use gray or dark colors
  4. Size: 32x32 pixels

Create the Exit Sprite

  1. Create a new sprite named spr_exit
  2. Draw a door, flag, or glowing marker
  3. Use green or gold colors
  4. Size: 32x32 pixels

Create the Coin Sprite

  1. Create a new sprite named spr_coin
  2. Draw a small yellow/gold circle
  3. Size: 16x16 pixels
Tip: Making the player sprite smaller than the walls allows easier movement through corridors!
Click "Next" to create the game objects.
