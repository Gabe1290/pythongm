Passo 3: Movimento del Giocatore

Il giocatore ha bisogno di un movimento fluido con rilevamento delle collisioni.

Inizializza la Velocita (Evento Create)

In obj_player:

  1. Aggiungi Evento: Create
  2. Aggiungi Azione: Controllo > Imposta Variabile
  3. Variabile: move_speed
  4. Valore: 4

Movimento con Collisione (Evento Step)

  1. Aggiungi Evento: Step
  2. Aggiungi Azione: Controllo > Esegui Codice
// Movimento orizzontale
var hspd = 0;
if (keyboard_check(vk_right)) hspd = move_speed;
if (keyboard_check(vk_left)) hspd = -move_speed;

// Movimento verticale
var vspd = 0;
if (keyboard_check(vk_down)) vspd = move_speed;
if (keyboard_check(vk_up)) vspd = -move_speed;

// Controlla collisione orizzontale
if (!place_meeting(x + hspd, y, obj_wall)) {
    x += hspd;
}

// Controlla collisione verticale
if (!place_meeting(x, y + vspd, obj_wall)) {
    y += vspd;
}
Come funziona: Controlliamo se la posizione di destinazione colliderebbe con un muro. Se no, ci spostiamo li!
Clicca "Avanti" per aggiungere la raccolta delle monete.
