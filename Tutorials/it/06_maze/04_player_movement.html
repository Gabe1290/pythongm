Step 3: Player Movement

The player needs smooth movement with collision detection.

Initialize Speed (Create Event)

In obj_player:

  1. Add Event: Create
  2. Add Action: Control > Set Variable
  3. Variable: move_speed
  4. Value: 4

Movement with Collision (Step Event)

  1. Add Event: Step
  2. Add Action: Control > Execute Code
// Horizontal movement
var hspd = 0;
if (keyboard_check(vk_right)) hspd = move_speed;
if (keyboard_check(vk_left)) hspd = -move_speed;

// Vertical movement
var vspd = 0;
if (keyboard_check(vk_down)) vspd = move_speed;
if (keyboard_check(vk_up)) vspd = -move_speed;

// Check horizontal collision
if (!place_meeting(x + hspd, y, obj_wall)) {
    x += hspd;
}

// Check vertical collision
if (!place_meeting(x, y + vspd, obj_wall)) {
    y += vspd;
}
How it works: We check if the target position would collide with a wall. If not, we move there!
