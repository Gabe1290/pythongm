Step 5: Game Controller

Create a controller to track time and display score.

Create Controller Object

  1. Create object obj_game_controller
  2. No sprite needed (leave it empty)

Initialize Timer (Create Event)

  1. Add Event: Create
  2. Add Action: Control > Set Variable
  3. Variable: global.timer
  4. Value: 0

Update Timer (Step Event)

  1. Add Event: Step
  2. Add Action: Control > Set Variable
  3. Variable: global.timer
  4. Value: 1/room_speed
  5. Check "Relative"

Display Info (Draw Event)

  1. Add Event: Draw
  2. Add Action: Control > Execute Code
draw_set_color(c_white);
draw_text(10, 10, "Score: " + string(score));
draw_text(10, 30, "Time: " + string(floor(global.timer)) + "s");

var coins = instance_number(obj_coin);
draw_text(10, 50, "Coins: " + string(coins));
Note: The controller is invisible but essential for tracking game state!
