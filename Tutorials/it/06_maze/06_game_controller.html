Passo 5: Controller di Gioco

Crea un controller per tenere traccia del tempo e visualizzare il punteggio.

Crea l'Oggetto Controller

  1. Crea l'oggetto obj_game_controller
  2. Non serve sprite (lascialo vuoto)

Inizializza il Timer (Evento Create)

  1. Aggiungi Evento: Create
  2. Aggiungi Azione: Controllo > Imposta Variabile
  3. Variabile: global.timer
  4. Valore: 0

Aggiorna il Timer (Evento Step)

  1. Aggiungi Evento: Step
  2. Aggiungi Azione: Controllo > Imposta Variabile
  3. Variabile: global.timer
  4. Valore: 1/room_speed
  5. Spunta "Relativo"

Visualizza Informazioni (Evento Draw)

  1. Aggiungi Evento: Draw
  2. Aggiungi Azione: Controllo > Esegui Codice
draw_set_color(c_white);
draw_text(10, 10, "Punteggio: " + string(score));
draw_text(10, 30, "Tempo: " + string(floor(global.timer)) + "s");

var coins = instance_number(obj_coin);
draw_text(10, 50, "Monete: " + string(coins));
Nota: Il controller e invisibile ma essenziale per tenere traccia dello stato del gioco!
Clicca "Avanti" per configurare la tua stanza.
