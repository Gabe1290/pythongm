Passo 5: Aggiungi il Sistema di Vite

Il giocatore perde una vita quando la palla cade oltre il bordo inferiore dello schermo.

Crea una Zona di Morte

  1. Crea uno sprite spr_death_zone (32x32, qualsiasi colore)
  2. Crea l'oggetto obj_death_zone
  3. Deseleziona "Visibile" - deve essere invisibile
  4. Spunta "Solido"

La Palla Cade nella Zona di Morte

In obj_ball:

  1. Aggiungi Evento: Collisione > obj_death_zone
  2. Aggiungi Azione: Punteggio > Sottrai dalle Vite
  3. Valore: 1
  4. Aggiungi Azione: Movimento > Salta alla Posizione Iniziale

Crea un Controllore di Gioco

  1. Crea l'oggetto obj_game_controller
  2. Non serve uno sprite

Inizializza le Vite (Evento Creazione)

In obj_game_controller:

  1. Aggiungi Evento: Creazione
  2. Aggiungi Azione: Punteggio > Imposta Vite
  3. Valore: 3
  4. Aggiungi Azione: Punteggio > Imposta Punteggio
  5. Valore: 0

Controlla il Game Over (Evento Step)

  1. Aggiungi Evento: Step
  2. Aggiungi Azione: Controllo > Verifica Vite
  3. Controlla se le vite sono minori o uguali a 0
  4. Aggiungi Azione: Main2 > Mostra Messaggio
  5. Messaggio: Game Over!
  6. Aggiungi Azione: Main2 > Riavvia Stanza
Ricorda: Posiziona obj_game_controller nella tua stanza!
Clicca "Avanti" per aggiungere la visualizzazione del punteggio.
