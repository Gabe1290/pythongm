Step 7: Set Up the Room

Time to create the game room and place all the objects!

Create the Room

  1. Right-click on Rooms and select Create Room
  2. Name it room_breakout
  3. Set size to 640x480 pixels

Room Layout

+--[WALL]--------------------[WALL]--+
|  [WALL]  [R][R][R][R][R]  [WALL]  |
|  [WALL]  [O][O][O][O][O]  [WALL]  |
|  [WALL]  [Y][Y][Y][Y][Y]  [WALL]  |
|  [WALL]  [G][G][G][G][G]  [WALL]  |
|  [WALL]                   [WALL]  |
|  [WALL]        (o)        [WALL]  |
|  [WALL]                   [WALL]  |
|  [WALL]      [PADDLE]     [WALL]  |
+--[DEATH ZONE DEATH ZONE DEATH]----+

Place the Objects

1. Walls

Place obj_wall along the top, left, and right edges.

2. Death Zone

Place obj_death_zone along the bottom edge (it's invisible).

3. Bricks

Arrange bricks in rows near the top:

  • Top row: Red bricks (obj_brick_red)
  • Second row: Orange bricks
  • Third row: Yellow bricks
  • Fourth row: Green bricks

4. Paddle

Place obj_paddle near the bottom, centered horizontally.

5. Ball

Place obj_ball just above the paddle.

6. Game Controller

Place obj_game_controller anywhere (it has no sprite).

Congratulations! Your Breakout game is ready!

Test Your Game

  • Press F5 or click Run
  • Use Left and Right arrow keys to move
  • Break all the bricks to win!
Great job! You've created a classic Breakout game!
