Passo 7: Configura la Stanza

E ora di creare la stanza di gioco e posizionare tutti gli oggetti!

Crea la Stanza

  1. Clicca col tasto destro su Rooms e seleziona Crea Stanza
  2. Chiamala room_breakout
  3. Imposta le dimensioni a 640x480 pixel

Layout della Stanza

+--[MURO]---------------------[MURO]--+
|  [MURO]  [R][R][R][R][R]   [MURO]  |
|  [MURO]  [A][A][A][A][A]   [MURO]  |
|  [MURO]  [G][G][G][G][G]   [MURO]  |
|  [MURO]  [V][V][V][V][V]   [MURO]  |
|  [MURO]                    [MURO]  |
|  [MURO]        (o)         [MURO]  |
|  [MURO]                    [MURO]  |
|  [MURO]    [RACCHETTA]     [MURO]  |
+--[ZONA MORTE ZONA MORTE ZONA]------+

Posiziona gli Oggetti

1. Muri

Posiziona obj_wall lungo i bordi superiore, sinistro e destro.

2. Zona di Morte

Posiziona obj_death_zone lungo il bordo inferiore (e invisibile).

3. Mattoni

Disponi i mattoni in file vicino alla parte superiore:

  • Prima fila: Mattoni rossi (obj_brick_red)
  • Seconda fila: Mattoni arancioni
  • Terza fila: Mattoni gialli
  • Quarta fila: Mattoni verdi

4. Racchetta

Posiziona obj_paddle vicino alla parte inferiore, centrata orizzontalmente.

5. Palla

Posiziona obj_ball appena sopra la racchetta.

6. Controllore di Gioco

Posiziona obj_game_controller ovunque (non ha sprite).

Congratulazioni! Il tuo gioco Breakout e pronto!

Testa il Tuo Gioco

  • Premi F5 o clicca Esegui
  • Usa i tasti freccia Sinistra e Destra per muoverti
  • Rompi tutti i mattoni per vincere!
Ottimo lavoro! Hai creato un classico gioco Breakout!
