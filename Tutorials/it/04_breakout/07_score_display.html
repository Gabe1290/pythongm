Step 6: Display Score and Lives

Let's add a display for the player's score and remaining lives.

Draw Event in Game Controller

In obj_game_controller:

  1. Add Event: Draw

Draw Score

  1. Add Action: Draw > Draw Text
  2. Text: Score:
  3. X: 10, Y: 10
  4. Add Action: Draw > Draw Score
  5. X: 70, Y: 10

Draw Lives

  1. Add Action: Draw > Draw Text
  2. Text: Lives:
  3. X: 10, Y: 30
  4. Add Action: Draw > Draw Lives
  5. X: 70, Y: 30
Tip: You can use Draw Lives as Image to show heart icons instead of a number!
