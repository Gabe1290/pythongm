Passo 4: Movimento del Giocatore

Il giocatore si muove a passi fissi di 32 pixel (movimento basato su griglia).

Importante: In Sokoban, il movimento deve controllare:
1. C'e un muro? (Non ci si puo muovere)
2. C'e una cassa? (Possiamo spingerla?)
3. C'e un muro/cassa dietro la cassa? (Non si puo spingere)

Muovi a Destra (Freccia Destra)

In obj_player:

  1. Aggiungi Evento: Keyboard > Press Right
  2. Aggiungi Azione: Control > Execute Code
  3. Inserisci questo codice:
// Controlla se possiamo muoverci a destra
if (!place_meeting(x + 32, y, obj_wall)) {
    var crate = instance_place(x + 32, y, obj_crate);
    if (crate != noone) {
        // C'e una cassa - possiamo spingerla?
        if (!place_meeting(x + 64, y, obj_wall) &&
            !place_meeting(x + 64, y, obj_crate)) {
            crate.x += 32;
            x += 32;
        }
    } else {
        // Nessuna cassa, muoviti e basta
        x += 32;
    }
}

Muovi a Sinistra (Freccia Sinistra)

Stesso codice ma cambia:

  • + 32 diventa - 32
  • + 64 diventa - 64

Muovi in Alto (Freccia Su)

Stesso codice ma usa y invece di x:

  • x + 32 diventa y - 32
  • x + 64 diventa y - 64
  • crate.x diventa crate.y

Muovi in Basso (Freccia Giu)

Come per Su ma con + 32 e + 64.

Clicca "Avanti" per aggiungere il rilevamento della condizione di vittoria.
