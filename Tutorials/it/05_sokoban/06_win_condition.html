Passo 5: Condizione di Vittoria

Il giocatore vince quando tutte le casse sono sui bersagli.

Crea il Controller di Gioco

  1. Crea l'oggetto obj_game_controller
  2. Non serve uno sprite

Conta i Bersagli (Evento Create)

In obj_game_controller:

  1. Aggiungi Evento: Create
  2. Aggiungi Azione: Control > Execute Code
  3. Inserisci: global.total_targets = instance_number(obj_target);

Controlla la Condizione di Vittoria (Evento Step)

  1. Aggiungi Evento: Step
  2. Aggiungi Azione: Control > Execute Code
  3. Inserisci questo codice:
// Conta le casse sui bersagli
var crates_on_targets = 0;
with (obj_crate) {
    if (place_meeting(x, y, obj_target)) {
        crates_on_targets += 1;
    }
}

// Controlla se tutti i bersagli hanno le casse
if (crates_on_targets >= global.total_targets
    && global.total_targets > 0) {
    show_message("Livello Completato!");
    room_restart();
}

Disegna le Istruzioni (Evento Draw)

  1. Aggiungi Evento: Draw
  2. Aggiungi Azione: Draw > Draw Text
  3. Testo: Spingi le casse sui bersagli!
  4. X: 10, Y: 10
Suggerimento: Assicurati che il numero di casse sia uguale al numero di bersagli nel tuo livello!
Clicca "Avanti" per configurare la stanza di gioco.
