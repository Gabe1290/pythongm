Passo 6: Controller di Gioco (HUD)

Visualizza carburante, velocita e stato per aiutare il giocatore.

Evento Draw

In obj_game_controller:

  1. Aggiungi Evento: Draw > Draw
  2. Aggiungi Azione: Controllo > Esegui Codice
draw_set_color(c_white);

// Ottieni info del lander
var lander = instance_find(obj_lander, 0);
if (lander != noone) {
    var spd = sqrt(lander.hsp*lander.hsp + lander.vsp*lander.vsp);

    // Disegna HUD
    draw_text(10, 10, "LUNAR LANDER");
    draw_text(10, 30, "Carburante: " + string(floor(lander.fuel)) + "%");
    draw_text(10, 50, "Velocita: " + string_format(spd, 1, 2));
    draw_text(10, 70, "Sicura: < " + string(lander.safe_speed));

    // Avviso velocita
    if (spd > lander.safe_speed) {
        draw_set_color(c_red);
        draw_text(10, 90, "TROPPO VELOCE!");
    } else {
        draw_set_color(c_lime);
        draw_text(10, 90, "Velocita OK");
    }

    // Avviso carburante
    if (lander.fuel <= 0) {
        draw_set_color(c_red);
        draw_text(10, 110, "CARBURANTE ESAURITO!");
    }
}

draw_set_color(c_gray);
draw_text(10, room_height - 20, "SU: Spinta | SINISTRA/DESTRA: Muovi");

Cosa Mostra l'HUD

  • Carburante: Percentuale rimanente (verde = OK)
  • Velocita: Velocita attuale (rosso = pericoloso)
  • Avvisi: "TROPPO VELOCE!" o "CARBURANTE ESAURITO!"
  • Controlli: Promemoria in basso
