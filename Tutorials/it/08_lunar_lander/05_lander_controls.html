Passo 4: Controlli del Lander (Evento Step)

Aggiungi gravita, spinta, movimento orizzontale e controlli dei bordi.

Aggiungi Evento Step

In obj_lander:

  1. Aggiungi Evento: Step > Step
  2. Aggiungi Azione: Controllo > Esegui Codice
// Non aggiornare se il gioco e finito
if (landed || crashed) exit;

// === GRAVITA ===
vsp += gravity_force;

// === SPINTA (Freccia Su) ===
if (keyboard_check(vk_up) && fuel > 0) {
    vsp -= thrust_force;
    fuel -= fuel_use;
    if (fuel < 0) fuel = 0;
}

// === CONTROLLO ORIZZONTALE ===
if (keyboard_check(vk_left)) hsp -= 0.05;
if (keyboard_check(vk_right)) hsp += 0.05;

// === LIMITA VELOCITA ===
hsp = clamp(hsp, -max_speed, max_speed);
vsp = clamp(vsp, -max_speed, max_speed);

// === APPLICA MOVIMENTO ===
x += hsp;
y += vsp;

// === BORDI DELLO SCHERMO ===
if (x < 16) { x = 16; hsp = 0; }
if (x > room_width - 16) { x = room_width - 16; hsp = 0; }
if (y < 16) { y = 16; vsp = 0; }

Come Funziona

  1. Gravita: Aggiunge a vsp ogni frame (caduta)
  2. Spinta: Sottrae da vsp (salita), consuma carburante
  3. Clamp: Limita la velocita a un intervallo sicuro
  4. Bordi: Mantiene il lander sullo schermo
Importante: La funzione clamp() limita un valore tra minimo e massimo!
Clicca "Avanti" per aggiungere il rilevamento delle collisioni.
