Passo 5: Rilevamento Collisioni

Gestisci l'atterraggio sulla piattaforma e lo schianto contro il terreno.

Collisione con la Piattaforma di Atterraggio

In obj_lander:

  1. Aggiungi Evento: Collisione > obj_pad
  2. Aggiungi Azione: Controllo > Esegui Codice
// Calcola velocita totale
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    // Atterraggio sicuro!
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("Atterraggio Perfetto! Hai Vinto!");
} else {
    // Troppo veloce - schianto!
    crashed = true;
    show_message("Schianto! Troppo veloce!");
    room_restart();
}

Collisione con il Terreno (Schianto)

In obj_lander:

  1. Aggiungi Evento: Collisione > obj_ground
  2. Aggiungi Azione: Controllo > Esegui Codice
crashed = true;
show_message("Schianto contro il terreno!");
room_restart();

Calcolo della Velocita

Usiamo il teorema di Pitagora per ottenere la velocita totale:

total_speed = sqrt(hsp*hsp + vsp*vsp)

Questo combina la velocita orizzontale e verticale in un unico valore.

La velocita sicura e 2! Se total_speed > 2, ti schianti anche sulla piattaforma.
