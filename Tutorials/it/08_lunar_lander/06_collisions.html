Step 5: Collision Detection

Handle landing on the pad and crashing into terrain.

Collision with Landing Pad

In obj_lander:

  1. Add Event: Collision > obj_pad
  2. Add Action: Control > Execute Code
// Calculate total speed
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    // Safe landing!
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("Perfect Landing! You Win!");
} else {
    // Too fast - crash!
    crashed = true;
    show_message("Crashed! Too fast!");
    room_restart();
}

Collision with Ground (Crash)

In obj_lander:

  1. Add Event: Collision > obj_ground
  2. Add Action: Control > Execute Code
crashed = true;
show_message("Crashed into terrain!");
room_restart();

Speed Calculation

We use the Pythagorean theorem to get total speed:

total_speed = sqrt(hsp*hsp + vsp*vsp)

This combines horizontal and vertical speed into one value.

Safe speed is 2! If total_speed > 2, you crash even on the pad.
