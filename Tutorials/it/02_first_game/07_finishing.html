Passo 6: Ritocchi Finali

Aggiungiamo qualche rifinitura per rendere il tuo gioco completo!

Mantenere il Giocatore sullo Schermo

1 Aggiungi controllo dei bordi

In obj_player, aggiungi un evento Step con queste condizioni:

  • Se x < 0, imposta x = 0
  • Se x > room_width - sprite_width, imposta x = room_width - sprite_width

Questo impedisce al giocatore di uscire dai lati dello schermo.

Aggiungere un Colore di Sfondo

2 Imposta lo sfondo della stanza

Apri room_game nell'Editor Stanze.

Nelle proprietà della stanza, imposta il colore di sfondo su un bel celeste o un colore spazio scuro.

Esporta il Tuo Gioco!

3 Esporta come HTML5

Vai su File > Esporta come HTML5.

Scegli una posizione e clicca Esporta.

Apri il file HTML in un browser web per giocare al tuo gioco!

Congratulazioni! Hai creato il tuo primo gioco completo in PyGameMaker!

Idee per Migliorare il Tuo Gioco

Sfide da provare:
  • Aggiungi vite - perdi una vita quando una stella raggiunge il fondo
  • Fai cadere le stelle più velocemente quando il punteggio aumenta
  • Aggiungi stelle di colori diversi che valgono punti diversi
  • Aggiungi un oggetto "cattivo" (bomba) che termina il gioco se catturato
  • Aggiungi effetti sonori quando catturi le stelle
  • Crea una schermata del titolo e una schermata game over
