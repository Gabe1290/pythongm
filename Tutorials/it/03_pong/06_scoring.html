Step 5: Add Scoring Goals

We need invisible goals behind each paddle to detect when someone scores.

Create Goal Sprite

  1. Create a new sprite named spr_goal
  2. Make it 32x32 pixels with any color
  3. (It will be invisible in the game)

Create Left Goal Object

  1. Create object obj_goal_left
  2. Set sprite to spr_goal
  3. Uncheck "Visible" - goal should be invisible
  4. Check "Solid"

Create Right Goal Object

  1. Create object obj_goal_right
  2. Set sprite to spr_goal
  3. Uncheck "Visible"
  4. Check "Solid"

Ball Collides with Left Goal (Player 2 Scores)

Go back to obj_ball:

  1. Add Event: Collision > obj_goal_left
  2. Add Action: Move > Jump to Start Position
  3. Add Action: Score > Set Variable
  4. Variable: global.p2score
  5. Value: 1
  6. Check "Relative" (adds 1 to current score)

Ball Collides with Right Goal (Player 1 Scores)

  1. Add Event: Collision > obj_goal_right
  2. Add Action: Move > Jump to Start Position
  3. Add Action: Score > Set Variable
  4. Variable: global.p1score
  5. Value: 1
  6. Check "Relative"
Note: The global. prefix means these variables are accessible from any object!
