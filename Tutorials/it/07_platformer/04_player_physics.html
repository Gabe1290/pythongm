Passo 3: Fisica del Giocatore (Evento Creazione)

Dobbiamo impostare le variabili per gravita, salto e movimento.

Inizializzare le Variabili Fisiche

In obj_player:

  1. Aggiungi Evento: Creazione
  2. Aggiungi Azione: Controllo > Esegui Codice
// Impostazioni movimento
hspeed_max = 4;      // Velocita orizzontale massima
vspeed_max = 10;     // Velocita di caduta massima
jump_force = -10;    // Forza del salto (negativo = su)
gravity_force = 0.5; // Quanto velocemente cadiamo

// Velocita correnti
hsp = 0;
vsp = 0;

// Stato
on_ground = false;

Comprendere le Variabili

Variabile Scopo
hsp Velocita orizzontale (sinistra/destra)
vsp Velocita verticale (su/giu)
gravity_force Forza costante che tira verso il basso
jump_force Valore negativo per andare verso l'alto
on_ground Vero quando si sta in piedi sul terreno
Concetto Chiave: La forza del salto e negativa perche Y aumenta verso il basso nei giochi. Una velocita Y negativa muove il giocatore verso l'alto!
Clicca "Avanti" per aggiungere il movimento e la gestione delle collisioni.
