Step 6: Game Controller

The game controller displays the score on screen.

Create the Controller Object

  1. Create object obj_game_controller
  2. No sprite needed (invisible controller)

Draw the Score

In obj_game_controller:

  1. Add Event: Draw > Draw
  2. Add Action: Control > Execute Code
// Set text color
draw_set_color(c_white);

// Draw score at top-left
draw_text(10, 10, "Score: " + string(score));

// Optional: Draw instructions
draw_text(10, 30, "Arrows: Move  |  Space: Jump");

Alternative: Using Actions

Instead of code, you can use:

  1. Add Action: Draw > Set Color (White)
  2. Add Action: Score > Draw Score
    • X: 10
    • Y: 10
    • Caption: "Score: "
Remember: Place the game controller in your room! It won't appear by itself.
