Passo 4: Movimento del Giocatore (Evento Step)

Questo e il cuore del nostro platformer: movimento, gravita, salto e collisioni.

Aggiungere l'Evento Step

In obj_player:

  1. Aggiungi Evento: Step > Step
  2. Aggiungi Azione: Controllo > Esegui Codice
// === MOVIMENTO ORIZZONTALE ===
var move_input = keyboard_check(vk_right) - keyboard_check(vk_left);
hsp = move_input * hspeed_max;

// === GRAVITA ===
vsp += gravity_force;
if (vsp > vspeed_max) vsp = vspeed_max;

// === CONTROLLO TERRENO ===
on_ground = place_meeting(x, y + 1, obj_ground);

// === SALTO ===
if (on_ground && (keyboard_check_pressed(vk_up) || keyboard_check_pressed(vk_space))) {
    vsp = jump_force;
}

// === COLLISIONE ORIZZONTALE ===
if (place_meeting(x + hsp, y, obj_ground)) {
    while (!place_meeting(x + sign(hsp), y, obj_ground)) {
        x += sign(hsp);
    }
    hsp = 0;
}
x += hsp;

// === COLLISIONE VERTICALE ===
if (place_meeting(x, y + vsp, obj_ground)) {
    while (!place_meeting(x, y + sign(vsp), obj_ground)) {
        y += sign(vsp);
    }
    vsp = 0;
}
y += vsp;

Come Funziona

  1. Input: Freccia destra = 1, Sinistra = -1, Nessuna = 0
  2. Gravita: Aggiunge a vsp ogni frame (caduta accelerata)
  3. Controllo Terreno: Cerca il terreno 1 pixel sotto
  4. Salto: Funziona solo quando on_ground e vero
  5. Collisione: Muove pixel per pixel fino a colpire il muro
La Magia: Il ciclo while ci muove pixel per pixel fino al bordo del muro, impedendoci di rimanere bloccati!
Clicca "Avanti" per aggiungere monete, punte e la bandiera.
