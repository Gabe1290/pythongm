Phase 1: A Moving Player

Let's start by creating a player and making it move left and right with the arrow keys.

Create a New Project

  1. Go to File > New Project
  2. Name it CatchTheCoins
  3. For the Template, choose "With Game Over Screen"

This template gives you a ready-made Game Over screen that we'll connect later, so you don't have to build it yourself.

Create the Player Sprite

  1. Right-click Sprites in the Asset Tree → Create New Sprite
  2. Name it spr_player
  3. Double-click it to open the Sprite Editor and draw a 32×32 character (a paddle, a basket, or just a green square works fine). Press Ctrl+S to save.

Create the Player Object

  1. Right-click Objects → Create New Object
  2. Name it obj_player
  3. Set the sprite to spr_player

Create the Game Room

  1. Right-click Rooms → Create New Room
  2. Name it room_main
  3. Open the room and place obj_player near the bottom center

Add Player Movement

  1. Double-click obj_player to open the Object Editor
  2. Click the Blockly tab

Move Left

Keyboard: Left Arrow (held)
Set horizontal speed to -5

Move Right

Keyboard: Right Arrow (held)
Set horizontal speed to 5

Stop When No Key Pressed

Keyboard: No key
Stop movement

You should have 3 event blocks.

Test Your Game!

Press F5 to play!

  • Use Left and Right arrow keys to move the player
  • The player can drift off screen for now — coins are coming next!
Click "Next" to add coins and the enemy.
