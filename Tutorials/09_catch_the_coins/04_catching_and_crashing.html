Phase 3: Catching and Crashing

Time to make the coins and the enemy actually do something. We'll add three rules:

  • Touching a coin: +1 score, the coin disappears
  • Touching the enemy: game over
  • The score is shown on screen

Open the Player Object

  1. Double-click obj_player to open the Object Editor
  2. Click the Blockly tab

Reset the Score When the Game Starts

Add this event so the score begins at zero each time:

When created
Set score to 0

Catch a Coin (Score and Destroy It)

Add a Collision event with obj_coin:

Collision with obj_coin
Add to score 1
Destroy other instance

Destroy other instance means “destroy the coin we just touched” — not the player.

Crash Into the Enemy (Game Over)

Add another Collision event, this time with obj_enemy:

Collision with obj_enemy
Go to room room_gameover

The room_gameover room came from the "With Game Over Screen" template — it already shows "GAME OVER" and restarts when the player presses SPACE.

Show the Score on Screen

Add a Draw event so the score is visible while you play:

When drawing
Draw Score at x:10 y:10

Your obj_player now has: When created, When drawing, two collisions, and the three keyboard events from Phase 1 — seven event blocks total.

Test Your Game!

Press F5 to play!

  • Catch coins → score goes up, coins disappear
  • Touch the enemy → Game Over screen appears
  • Press SPACE on the Game Over screen to restart
Click "Next" to add the win screen.
