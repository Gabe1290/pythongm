Catch the Coins!

💰

Catch every coin — without bumping the enemy!

Move left and right, collect all the falling coins to win, and avoid the enemy or it's game over.

What We'll Build

You'll create a complete game with a clear win and lose state. The game will be playable at the end of every phase, so you can stop and try it whenever you want.

How This Tutorial Works

Phase 1 — A Moving Player
Create a player that moves left and right. Test it right away!
Phase 2 — Coins and an Enemy
Add coins that fall from the sky, plus one enemy. Place them in the room.
Phase 3 — Catching and Crashing
Catch coins for points, hit the enemy and the game ends.
Phase 4 — Winning the Game
Add a win screen that appears when every coin has been caught.

What You'll Learn

  • Creating sprites, objects, and rooms
  • Using keyboard events for player control
  • Detecting collisions between objects
  • Tracking and displaying a score
  • Using room transitions for game-over and win screens
  • Using the If Condition block to check how many objects remain
Time Required: About 15–20 minutes
