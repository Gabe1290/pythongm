Phase 2: Coins and an Enemy

Now we'll add the coins to collect and one enemy to avoid. Both fall straight down.

Create the Coin Sprite

  1. Right-click Sprites → Create New Sprite
  2. Name it spr_coin
  3. Open the Sprite Editor and draw a 24×24 yellow circle. Save with Ctrl+S.

Create the Enemy Sprite

  1. Repeat the steps above for a sprite called spr_enemy
  2. Draw a 32×32 red square so the enemy is easy to spot

Create the Coin Object

  1. Right-click Objects → Create New Object
  2. Name it obj_coin
  3. Set the sprite to spr_coin
  4. Open the Blockly tab and add this:
When created
Set vertical speed to 3

Create the Enemy Object

  1. Create another object called obj_enemy
  2. Set its sprite to spr_enemy
  3. Add the same When created event so it falls too:
When created
Set vertical speed to 3

Place Them in the Room

  1. Open room_main
  2. Place 5 to 8 coins across the top of the room, spread out horizontally
  3. Place 1 enemy somewhere along the top, between the coins
  4. Make sure obj_player is still at the bottom center
Tip: The number of coins is the score the player needs to reach to win. Start with 5 for a quick demo — you can always add more later.

Test Your Game!

Press F5 to play!

  • Coins and the enemy should fall from the top of the room
  • The player can move left and right under them
  • Touching them does nothing yet — that's the next phase!
