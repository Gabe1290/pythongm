Welcome to PyGameMaker!

PyGameMaker is a visual game development environment inspired by GameMaker Studio. It allows you to create 2D games using either visual programming (Blockly blocks) or traditional event-based programming.

What You'll Learn

In this tutorial series, you will learn how to:

  • Navigate the PyGameMaker interface
  • Create and manage game assets (sprites, objects, rooms)
  • Add behaviors to game objects using events and actions
  • Test and export your games
Tip: You can close this tutorial panel at any time by clicking the "Close" button at the top. To return to the tutorials, select Help > Tutorials from the menu.

Prerequisites

No programming experience is required! PyGameMaker's visual programming system makes it easy for beginners to get started with game development.
