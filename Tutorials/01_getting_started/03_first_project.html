Creating Your First Project

Let's create a simple project step by step.

1 Create a New Project

Go to File > New Project (or press Ctrl+N).

Enter a name for your project and choose where to save it.

2 Create a Sprite

Right-click on Sprites in the asset tree and select Create Sprite.

Name it "spr_player" and import an image file for your player character.

3 Create an Object

Right-click on Objects and select Create Object.

Name it "obj_player" and assign the sprite you just created.

4 Create a Room

Right-click on Rooms and select Create Room.

Name it "room_game". This will be your first game level.

5 Place Your Object

Double-click your room to open the Room Editor.

Select your object from the list and click in the room to place it.

6 Test Your Game

Press F5 or go to Build > Test Game.

A window will open showing your game with the object you placed!

Congratulations! You've created your first PyGameMaker project! It doesn't do much yet, but you'll learn how to add behaviors in the next tutorials.
