The PyGameMaker Interface

The PyGameMaker window is divided into three main areas:

1. Asset Tree (Left Panel)

This panel shows all the assets in your project organized by type:

  • Sprites - Images and animations for your game
  • Sounds - Sound effects and music
  • Objects - Game entities with behaviors
  • Rooms - Game levels and screens

Right-click on any category to create new assets.

2. Editor Area (Center)

This is where you edit your assets. Different editors open in tabs:

  • Sprite Editor - View and configure sprites
  • Object Editor - Define object behaviors with events
  • Room Editor - Design your game levels

Double-click an asset in the tree to open its editor.

3. Properties Panel (Right Panel)

Shows properties of the currently selected asset or object. You can modify settings here when editing rooms or objects.

Key Menus

  • File - Create, open, and save projects
  • Assets - Create and import game assets
  • Build - Test and export your game
  • Help - Access tutorials and documentation
Shortcut: Press F5 to quickly test your game!
