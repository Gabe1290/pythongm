Step 5: Game Controller — Lives and Score (Blockly)

The game controller is an invisible object that sets up the lives and score at the start of the game.

What is a game controller? It's an object with no sprite that manages things like score and lives. Since it's invisible, it doesn't interfere with the game — it just works in the background.

Initialize Lives and Score

  1. Double-click obj_game_controller to open the Object Editor
  2. Click the Blockly tab
  3. Build these blocks:
Create Event
Set lives to 3
Set score to 0

From Events, drag a Create Event block. From Score/Lives/Health, snap Set lives to 3 and Set score to 0 inside.

How it works: The Create Event runs once when the game starts. It gives the player 3 lives and sets the score to zero. When the ball hits the death zone (set up in the previous step), it subtracts one life.
  1. Click Apply to Events to save
Remember: Place obj_game_controller in your room! It has no sprite, so you won't see it during the game, but it must be in the room for lives and score to work.
