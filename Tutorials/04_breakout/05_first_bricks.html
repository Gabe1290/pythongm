Phase 2: First Row of Bricks

Now let's add bricks for the ball to break! We'll use a clever trick called a parent object so that later, when we add more brick colors, we won't have to rewrite the collision code.

Create the Brick Sprite

  1. Create a new sprite named spr_brick_red

Option A — Draw it yourself

  1. Open the Sprite Editor and draw a solid red rectangle
  2. Size: 32×16 pixels

Option B — Load the sample sprite

  1. Right-click spr_brick_red → Import Image...
  2. Select Tutorials/04_breakout/assets/spr_brick_red.png

Create the Parent Brick Object

A parent object is a template. Any object that uses it as a parent will automatically inherit its collision behavior. This way, we only write the "ball hits brick" code once.

  1. Create a new object named obj_brick_parent
  2. No sprite needed — it's just a template, it won't appear in the game
  3. Check Solid

Create the Red Brick Object

  1. Create a new object named obj_brick_red
  2. Set the sprite to spr_brick_red
  3. Set Parent to obj_brick_parent
  4. Check Solid
What does setting a parent do? When the ball checks for a collision with obj_brick_parent, it will also detect collisions with obj_brick_red and any other child brick we add later. One collision event covers all brick colors!

Add Brick Collision to the Ball

We need to tell the ball what to do when it hits a brick.

  1. Double-click obj_ball to open the Object Editor
  2. Click the Blockly tab — you should see your 5 existing blocks
  3. Add this new event block:
When colliding with obj_brick_parent
Reverse vertical movement
Destroy other instance
Add to score 10

Snap three blocks inside:

  • Reverse vertical movement (from Movement) — ball bounces back
  • Destroy other instance (from Instance) — destroys the brick, not the ball!
  • Add to score 10 (from Score/Lives/Health) — earn points

You should now have 6 event blocks for the ball. Click Apply to Events.

Key Point: The collision is with obj_brick_parent, not obj_brick_red. This means it will automatically work with every brick color we add later!

Add Bricks to the Room

  1. Open room_breakout
  2. Place a row of obj_brick_red near the top, between the side walls
  3. Enable Snap to Grid (32×16) for easy alignment
  4. Fill one complete row with red bricks

Test Your Game!

Press F5 to play!

  • The ball should bounce off bricks and destroy them
  • Each brick adds 10 points to the score
  • Try to break all the bricks!
