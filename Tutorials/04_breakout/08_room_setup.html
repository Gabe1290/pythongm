Step 7: Set Up the Room

Time to create the game room and place all the objects!

Create the Room

  1. Right-click on Rooms and select Create Room
  2. Name it room_breakout
  3. Set size to 640x480 pixels

Room Layout

S T T T T T T T T T T T T T T T T T S
S                                     S
S   [R][R][R][R][R][R][R][R][R][R]    S
S   [O][O][O][O][O][O][O][O][O][O]    S
S   [Y][Y][Y][Y][Y][Y][Y][Y][Y][Y]    S
S   [G][G][G][G][G][G][G][G][G][G]    S
S                                     S
S                                     S
S                (o)                  S
S                                     S
S            [==PADDLE==]             S
D D D D D D D D D D D D D D D D D D D

S = Side Wall (obj_wall_side)
T = Top Wall (obj_wall_top)
D = Death Zone (obj_death_zone, invisible)
R/O/Y/G = Bricks   (o) = Ball

Place the Objects

1. Side Walls

Place obj_wall_side along the left and right edges, from top to bottom.

2. Top Wall

Place obj_wall_top along the top edge, between the side walls.

3. Death Zone

Place obj_death_zone along the bottom edge. It will be invisible during gameplay.

Important: There is NO wall at the bottom! The death zone catches the ball when it falls past the paddle.

4. Bricks

Arrange bricks in rows near the top:

  • Top row: Red bricks (obj_brick_red)
  • Second row: Orange bricks (obj_brick_orange)
  • Third row: Yellow bricks (obj_brick_yellow)
  • Fourth row: Green bricks (obj_brick_green)

Enable Snap to Grid (32x16) for easy alignment.

5. Paddle

Place obj_paddle near the bottom, centered horizontally.

6. Ball

Place obj_ball just above the paddle, near the center of the room.

7. Game Controller

Place obj_game_controller anywhere in the room (it has no sprite, so location doesn't matter).

Congratulations! Your Breakout game is ready!

Press F5 to play:
  • Use Left and Right arrow keys to move the paddle
  • Break all the bricks to win!
  • Don't let the ball fall past the paddle!
  • Watch your score and lives in the top-left corner
Challenge Ideas:
  • Give different colored bricks different point values
  • Make the ball speed up as bricks are destroyed
  • Add a second level with a different brick layout
  • Add power-ups that fall from destroyed bricks
