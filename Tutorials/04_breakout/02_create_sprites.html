Step 1: Create the Sprites

We need sprites for the paddle, ball, bricks, walls, and a death zone. For each sprite you can draw it yourself using the built-in Sprite Editor, or load a ready-made sample image. Both options are explained below.

SpriteSizeDescription
spr_ball16×16Small yellow circle
spr_paddle64×16Wide blue rectangle
spr_brick_red32×16Red brick
spr_brick_orange32×16Orange brick
spr_brick_yellow32×16Yellow brick
spr_brick_green32×16Green brick
spr_wall32×32Gray brick pattern
spr_death_zone32×32Dark rectangle (invisible in game)

spr_ball (16×16)

  1. In the Asset Tree, right-click on Sprites and select Create New Sprite
  2. Name it spr_ball

Option A — Draw it yourself

  1. Double-click spr_ball to open the Sprite Editor
  2. Draw a small circle
  3. Use a bright color like yellow or white
  4. Keep the size at 16×16 pixels
  5. Save your work with Ctrl+S

Option B — Load the sample sprite

  1. Right-click on spr_ball in the Asset Tree
  2. Select Import Image...
  3. Browse to Tutorials/04_breakout/assets/ inside the PyGameMaker folder
  4. Select spr_ball.png and click Open

spr_paddle (64×16)

  1. Create a new sprite named spr_paddle

Option A — Draw it yourself

  1. Open the Sprite Editor and draw a wide, flat rectangle
  2. Use a bright color like blue
  3. Size: 64×16 pixels

Option B — Load the sample sprite

  1. Right-click spr_paddle → Import Image...
  2. Select Tutorials/04_breakout/assets/spr_paddle.png

Brick Sprites (32×16 each)

Create four colorful bricks. Each one follows the same steps:

  1. Create a new sprite with the name from the table below

Option A — Draw them yourself

  1. Open the Sprite Editor and draw a solid rectangle
  2. Use the matching color for each brick
  3. Size: 32×16 pixels for all bricks

Option B — Load the sample sprites

For each brick, right-click → Import Image... and select the matching file:

  • spr_brick_red → Tutorials/04_breakout/assets/spr_brick_red.png
  • spr_brick_orange → Tutorials/04_breakout/assets/spr_brick_orange.png
  • spr_brick_yellow → Tutorials/04_breakout/assets/spr_brick_yellow.png
  • spr_brick_green → Tutorials/04_breakout/assets/spr_brick_green.png

spr_wall (32×32)

  1. Create a new sprite named spr_wall

Option A — Draw it yourself

  1. Open the Sprite Editor and draw a brick or stone pattern
  2. Use gray colors
  3. Size: 32×32 pixels

Option B — Load the sample sprite

  1. Right-click spr_wall → Import Image...
  2. Select Tutorials/04_breakout/assets/spr_wall.png

spr_death_zone (32×32)

  1. Create a new sprite named spr_death_zone

Option A — Draw it yourself

  1. Draw a solid rectangle in any color (e.g. dark red)
  2. Size: 32×32 pixels

Option B — Load the sample sprite

  1. Right-click spr_death_zone → Import Image...
  2. Select Tutorials/04_breakout/assets/spr_death_zone.png
Tip: Keep brick sizes consistent (32×16) so they align nicely in rows! The death zone will be invisible during gameplay — it just needs a sprite for placement.
Learning point: The Import Image feature lets you load any PNG, GIF or BMP file as a sprite. This is useful when you have artwork created in an external program or sample assets provided with a tutorial.
