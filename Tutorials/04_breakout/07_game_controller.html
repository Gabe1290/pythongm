Phase 4: Game Controller

The final step! We'll add an invisible game controller that manages lives, displays the score on screen, and makes the ball lose a life when it falls.

Create the Game Controller Object

  1. Create a new object named obj_game_controller
  2. No sprite needed (leave it empty)
  3. Leave Solid unchecked
What is a game controller? It's an invisible object that manages things like score and lives. Since it has no sprite, it doesn't appear in the game — it just works in the background.

Initialize Lives and Score

  1. Double-click obj_game_controller to open the Object Editor
  2. Click the Blockly tab
  3. Build these blocks:
Create Event
Set lives to 3
Set score to 0

The Create Event runs once when the game starts. It gives the player 3 lives and resets the score to zero.

Display Score and Lives

Add a second event block to show the score and lives on screen:

Draw Event
Draw score at x: 10 y: 10
Draw lives at x: 200 y: 10

The Draw Event runs every frame. It paints the score and lives text on screen.

You should have 2 event blocks. Click Apply to Events.

Update the Ball: Lose a Life

Currently, the ball just respawns when it falls. Let's make it subtract a life too.

  1. Double-click obj_ball to open the Object Editor
  2. Click the Blockly tab
  3. Find the collision with obj_death_zone event block
  4. Add a new block before the jump block:
When colliding with obj_death_zone
Add to lives -1
Jump to x: 320 y: 100

Add to lives -1 (from Score/Lives/Health) subtracts one life each time the ball falls past the paddle.

Click Apply to Events to save the updated ball.

Place the Game Controller in the Room

  1. Open room_breakout
  2. Place obj_game_controller anywhere in the room (it has no sprite, so location doesn't matter)
Don't forget! The game controller must be in the room for lives and score to work. Even though it's invisible, it needs to exist in the room.

Congratulations! Test Your Final Game!

Press F5 to play!

  • Use Left and Right arrow keys to move the paddle
  • Break all the bricks to win!
  • Don't let the ball fall past the paddle — you only have 3 lives!
  • Watch your score and lives in the top-left corner
Challenge Ideas:
  • Give different colored bricks different point values (red = 40, orange = 30, yellow = 20, green = 10)
  • Make the ball speed up as bricks are destroyed
  • Add a second level with a different brick layout
  • End the game when lives reach zero
