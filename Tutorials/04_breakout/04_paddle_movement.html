Step 3: Add Paddle Movement

The paddle moves left and right using the arrow keys.

Move Left (Left Arrow)

  1. Open obj_paddle
  2. Add Event: Keyboard > Press Left
  3. Add Action: Move > Set Horizontal Speed
  4. Set speed to -8 (negative = left)

Stop Moving (Release Left)

  1. Add Event: Keyboard > Release Left
  2. Add Action: Move > Set Horizontal Speed
  3. Set speed to 0

Move Right (Right Arrow)

  1. Add Event: Keyboard > Press Right
  2. Add Action: Move > Set Horizontal Speed
  3. Set speed to 8 (positive = right)

Stop Moving (Release Right)

  1. Add Event: Keyboard > Release Right
  2. Add Action: Move > Set Horizontal Speed
  3. Set speed to 0

Wall Collision

  1. Add Event: Collision > obj_wall
  2. Add Action: Move > Bounce Against Objects
  3. Select "Against solid objects"
Note: The wall collision prevents the paddle from going off-screen!
