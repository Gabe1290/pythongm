Step 3: Paddle Movement (Blockly)

We'll use Blockly visual blocks to program the paddle's left and right movement.

Open the Blockly Editor

  1. Double-click obj_paddle in the Asset Tree to open the Object Editor
  2. Click the Blockly tab at the top of the editor
How Blockly works: Drag blocks from the toolbox on the left into the workspace. Snap action blocks inside event blocks to define what happens. When you're done, click Apply to Events to save your blocks.

Move Left

Drag these blocks from the toolbox and snap them together:

Keyboard: Left (held)
Set horizontal speed to -8

Find Keyboard (held) in the Events category, and select Left from the dropdown. Then find Set Horizontal Speed in the Movement category and snap it inside. Type -8 (negative = left direction).

Move Right

Keyboard: Right (held)
Set horizontal speed to 8

Same as above, but choose Right and type 8 (positive = right direction).

Stop When No Key Pressed

Keyboard: No key
Stop movement

Find No Key in Events and Stop Movement in Movement. This prevents the paddle from sliding forever when you let go of the key!

Stop at Side Walls

When colliding with obj_wall_side
Stop movement

Find Collision in Events. Type obj_wall_side in the text field. Snap a Stop Movement block inside. This keeps the paddle on screen!

Apply Your Blocks

  1. You should have 4 event blocks in your workspace
  2. Click the Apply to Events button to save
How it works: Holding Left or Right sets the paddle's speed. When you release all keys, the paddle stops. If the paddle hits a side wall, it also stops.
