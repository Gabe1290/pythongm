Step 4: Ball Movement and Bouncing

The ball needs to move and bounce off walls, paddle, and bricks.

Start Ball Moving (Create Event)

  1. Open obj_ball
  2. Add Event: Create
  3. Add Action: Move > Start Moving in Direction
  4. Choose an upward diagonal direction
  5. Set speed to 5

Bounce Off Walls

  1. Add Event: Collision > obj_wall
  2. Add Action: Move > Bounce Against Objects
  3. Select "Against solid objects"

Bounce Off Paddle

  1. Add Event: Collision > obj_paddle
  2. Add Action: Move > Bounce Against Objects
  3. Select "Against solid objects"

Bounce Off Bricks (and Destroy Them!)

  1. Add Event: Collision > obj_brick_parent
  2. Add Action: Move > Bounce Against Objects
  3. Select "Against solid objects"
  4. Add Action: Main1 > Destroy Instance
  5. Select "Other" (destroys the brick, not the ball)
  6. Add Action: Score > Add to Score
  7. Value: 10
Key Point: Using obj_brick_parent means this collision works for ALL brick colors!
