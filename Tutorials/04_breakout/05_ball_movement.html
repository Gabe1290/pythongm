Step 4: Ball Movement and Bouncing (Blockly)

The ball needs to move diagonally and bounce off walls, the paddle, and bricks. We'll program all of this with Blockly blocks.

Open the Blockly Editor

  1. Double-click obj_ball in the Asset Tree to open the Object Editor
  2. Click the Blockly tab

Start the Ball Moving (Create Event)

Create Event
Set horizontal speed to 3
Set vertical speed to -3

From Events, drag a Create Event block. From Movement, snap two speed blocks inside: Set horizontal speed to 3 and Set vertical speed to -3. This makes the ball move diagonally upward when the game starts.

Why two speeds? The horizontal speed (3) moves the ball sideways. The vertical speed (-3) moves it upward (negative = up). Together, the ball moves diagonally!

Bounce Off Side Walls

When colliding with obj_wall_side
Reverse horizontal movement

When the ball hits a left or right wall, it reverses its horizontal direction. Find Reverse horizontal movement in the Movement category.

Bounce Off Top Wall

When colliding with obj_wall_top
Reverse vertical movement

When the ball hits the top wall, it reverses its vertical direction and bounces downward.

Bounce Off Paddle

When colliding with obj_paddle
Reverse vertical movement

Same as the top wall — the ball bounces back upward when it hits the paddle.

Hit Bricks (Bounce + Destroy + Score!)

When colliding with obj_brick_parent
Reverse vertical movement
Destroy other instance
Add to score 10

From Events, add a Collision with obj_brick_parent. Snap three blocks inside:

  • Reverse vertical movement (from Movement) — ball bounces back
  • Destroy other instance (from Instance) — destroys the brick, not the ball
  • Add to score 10 (from Score/Lives/Health) — earn points!
Key Point: Using obj_brick_parent means this collision works for ALL brick colors! Destroy other destroys the object we collided with (the brick), not the ball.

Ball Falls Off Screen (Lose a Life)

When colliding with obj_death_zone
Add to lives -1
Jump to x: 320 y: 400

When the ball falls off the bottom and hits the death zone:

  • Add to lives -1 (from Score/Lives/Health) — lose one life
  • Jump to x: 320 y: 400 (from Movement) — respawn the ball above the paddle

Apply Your Blocks

  1. You should have 6 event blocks in your workspace
  2. Click the Apply to Events button to save
How bouncing works: The ball always moves diagonally. When it hits a vertical surface (side wall), we reverse horizontal movement. When it hits a horizontal surface (top wall, paddle, bricks), we reverse vertical movement. This creates realistic bouncing!
