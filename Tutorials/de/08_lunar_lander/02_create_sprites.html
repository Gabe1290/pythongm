Schritt 1: Sprites erstellen

Wir benoetigen Sprites fuer den Lander, die Landeplattform und das Gelaende.

Sprite-Name Groesse Beschreibung
spr_lander 32x32 Raumschiff (Dreieck oder klassische Lander-Form)
spr_pad 64x16 Landeplattform mit "H"-Markierung
spr_ground 32x32 Felsiges Gelaende (grau/braun)

Lander-Sprite erstellen

  1. Rechtsklick auf Sprites > Sprite erstellen
  2. Benenne es spr_lander
  3. Klicke auf Sprite bearbeiten
  4. Zeichne ein Raumschiff (32x32 Pixel)
  5. Setze den Ursprung auf Mitte-Unten fuer korrekte Landung

Landeplattform-Sprite erstellen

  1. Erstelle ein Sprite namens spr_pad
  2. Groesse: 64x16 Pixel
  3. Zeichne eine flache Plattform mit Markierungen
  4. Verwende helle Farben (gelb/gruen)

Boden-Sprite erstellen

  1. Erstelle ein Sprite namens spr_ground
  2. Groesse: 32x32 Pixel
  3. Zeichne felsiges, raues Gelaende
  4. Verwende graue/braune Farben
Wichtig: Setze den Ursprung des Landers auf Mitte-Unten, damit er korrekt auf der Plattform landet!
Klicke auf "Weiter", um die Spielobjekte zu erstellen.
