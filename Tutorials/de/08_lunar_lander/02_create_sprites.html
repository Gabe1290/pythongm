Step 1: Create Sprites

We need sprites for the lander, landing pad, and terrain.

Sprite Name Size Description
spr_lander 32x32 Spacecraft (triangle or classic lander shape)
spr_pad 64x16 Landing platform with "H" marking
spr_ground 32x32 Rocky terrain (gray/brown)

Create Lander Sprite

  1. Right-click Sprites > Create Sprite
  2. Name it spr_lander
  3. Click Edit Sprite
  4. Draw a spacecraft (32x32 pixels)
  5. Set origin to center-bottom for proper landing

Create Landing Pad Sprite

  1. Create sprite named spr_pad
  2. Size: 64x16 pixels
  3. Draw a flat platform with markings
  4. Use bright colors (yellow/green)

Create Ground Sprite

  1. Create sprite named spr_ground
  2. Size: 32x32 pixels
  3. Draw rocky, rough terrain
  4. Use gray/brown colors
Important: Set the lander's origin to center-bottom so it lands correctly on the pad!
Click "Next" to create the game objects.
