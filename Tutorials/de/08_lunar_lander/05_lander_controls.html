Schritt 4: Lander-Steuerung (Step-Event)

Fuege Schwerkraft, Schub, horizontale Bewegung und Randpruefungen hinzu.

Step-Event hinzufuegen

In obj_lander:

  1. Event hinzufuegen: Step > Step
  2. Aktion hinzufuegen: Control > Execute Code
// Nicht aktualisieren, wenn Spiel vorbei ist
if (landed || crashed) exit;

// === SCHWERKRAFT ===
vsp += gravity_force;

// === SCHUB (Pfeil nach oben) ===
if (keyboard_check(vk_up) && fuel > 0) {
    vsp -= thrust_force;
    fuel -= fuel_use;
    if (fuel < 0) fuel = 0;
}

// === HORIZONTALE STEUERUNG ===
if (keyboard_check(vk_left)) hsp -= 0.05;
if (keyboard_check(vk_right)) hsp += 0.05;

// === GESCHWINDIGKEIT BEGRENZEN ===
hsp = clamp(hsp, -max_speed, max_speed);
vsp = clamp(vsp, -max_speed, max_speed);

// === BEWEGUNG ANWENDEN ===
x += hsp;
y += vsp;

// === BILDSCHIRMRAENDER ===
if (x < 16) { x = 16; hsp = 0; }
if (x > room_width - 16) { x = room_width - 16; hsp = 0; }
if (y < 16) { y = 16; vsp = 0; }

Wie es funktioniert

  1. Schwerkraft: Addiert jeden Frame zu vsp (Fallen)
  2. Schub: Subtrahiert von vsp (Steigen), verbraucht Treibstoff
  3. Clamp: Begrenzt Geschwindigkeit auf sicheren Bereich
  4. Raender: Haelt den Lander auf dem Bildschirm
Wichtig: Die clamp()-Funktion begrenzt einen Wert zwischen Minimum und Maximum!
