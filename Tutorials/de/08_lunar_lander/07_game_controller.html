Schritt 6: Spielsteuerung (HUD)

Zeige Treibstoff, Geschwindigkeit und Status an, um dem Spieler zu helfen.

Draw-Event

In obj_game_controller:

  1. Event hinzufuegen: Draw > Draw
  2. Aktion hinzufuegen: Control > Execute Code
draw_set_color(c_white);

// Lander-Informationen abrufen
var lander = instance_find(obj_lander, 0);
if (lander != noone) {
    var spd = sqrt(lander.hsp*lander.hsp + lander.vsp*lander.vsp);

    // HUD zeichnen
    draw_text(10, 10, "LUNAR LANDER");
    draw_text(10, 30, "Treibstoff: " + string(floor(lander.fuel)) + "%");
    draw_text(10, 50, "Tempo: " + string_format(spd, 1, 2));
    draw_text(10, 70, "Sicher: < " + string(lander.safe_speed));

    // Geschwindigkeitswarnung
    if (spd > lander.safe_speed) {
        draw_set_color(c_red);
        draw_text(10, 90, "ZU SCHNELL!");
    } else {
        draw_set_color(c_lime);
        draw_text(10, 90, "Tempo OK");
    }

    // Treibstoffwarnung
    if (lander.fuel <= 0) {
        draw_set_color(c_red);
        draw_text(10, 110, "KEIN TREIBSTOFF!");
    }
}

draw_set_color(c_gray);
draw_text(10, room_height - 20, "HOCH: Schub | LINKS/RECHTS: Bewegen");

Was das HUD anzeigt

  • Treibstoff: Verbleibender Prozentsatz (gruen = OK)
  • Geschwindigkeit: Aktuelle Geschwindigkeit (rot = gefaehrlich)
  • Warnungen: "ZU SCHNELL!" oder "KEIN TREIBSTOFF!"
  • Steuerung: Erinnerung am unteren Rand
Visuelles Feedback: Farbkodierung hilft Spielern, schnell zu reagieren!
