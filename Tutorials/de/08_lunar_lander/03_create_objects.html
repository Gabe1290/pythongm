Step 2: Create Objects

Now we create objects for all game elements.

Ground Object (Dangerous)

  1. Right-click Objects > Create Object
  2. Name it obj_ground
  3. Set sprite to spr_ground
  4. Check the "Solid" checkbox

Landing Pad Object (Safe)

  1. Create obj_pad
  2. Set sprite to spr_pad
  3. Check the "Solid" checkbox

Lander Object (Player)

  1. Create obj_lander
  2. Set sprite to spr_lander

Game Controller Object

  1. Create obj_game_controller
  2. No sprite needed (invisible)
Key Difference: Landing on obj_pad = safe landing. Landing on obj_ground = crash!
Click "Next" to add lander physics.
