Schritt 2: Objekte erstellen

Jetzt erstellen wir Objekte fuer alle Spielelemente.

Boden-Objekt (Gefaehrlich)

  1. Rechtsklick auf Objects > Objekt erstellen
  2. Benenne es obj_ground
  3. Setze das Sprite auf spr_ground
  4. Aktiviere das Kontrollkaestchen "Solid"

Landeplattform-Objekt (Sicher)

  1. Erstelle obj_pad
  2. Setze das Sprite auf spr_pad
  3. Aktiviere das Kontrollkaestchen "Solid"

Lander-Objekt (Spieler)

  1. Erstelle obj_lander
  2. Setze das Sprite auf spr_lander

Spielsteuerungs-Objekt

  1. Erstelle obj_game_controller
  2. Kein Sprite erforderlich (unsichtbar)
Wichtiger Unterschied: Landung auf obj_pad = sichere Landung. Landung auf obj_ground = Absturz!
Klicke auf "Weiter", um die Lander-Physik hinzuzufuegen.
