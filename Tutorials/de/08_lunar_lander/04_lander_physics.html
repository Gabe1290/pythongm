Schritt 3: Lander-Physik (Create-Event)

Richte Variablen fuer Schwerkraft, Schub, Treibstoff und Geschwindigkeit ein.

Physik-Variablen initialisieren

In obj_lander:

  1. Event hinzufuegen: Create
  2. Aktion hinzufuegen: Control > Execute Code
// Physik-Einstellungen
gravity_force = 0.05;  // Sanfte Schwerkraft
thrust_force = 0.1;    // Schubkraft
max_speed = 5;         // Maximale Geschwindigkeit

// Aktuelle Geschwindigkeit
hsp = 0;  // Horizontale Geschwindigkeit
vsp = 0;  // Vertikale Geschwindigkeit

// Treibstoff-System
fuel = 100;
fuel_use = 0.5;  // Treibstoff pro Frame

// Spielzustand
landed = false;
crashed = false;

// Sichere Landegeschwindigkeit
safe_speed = 2;

Die Variablen verstehen

Variable Zweck
gravity_force Konstante Abwaertskraft (0.05 ist sanft)
thrust_force Aufwaertskraft beim Druecken von HOCH
fuel 100% Treibstoff, verringert sich beim Schub
safe_speed Maximale Geschwindigkeit fuer sichere Landung (2)
Balance: Schub (0.1) ist staerker als Schwerkraft (0.05), damit du abbremsen kannst!
Klicke auf "Weiter", um Bewegung und Steuerung hinzuzufuegen.
