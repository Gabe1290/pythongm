Schritt 5: Kollisionserkennung

Behandle das Landen auf der Plattform und das Abstuerzen ins Gelaende.

Kollision mit Landeplattform

In obj_lander:

  1. Event hinzufuegen: Collision > obj_pad
  2. Aktion hinzufuegen: Control > Execute Code
// Gesamtgeschwindigkeit berechnen
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    // Sichere Landung!
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("Perfekte Landung! Du hast gewonnen!");
} else {
    // Zu schnell - Absturz!
    crashed = true;
    show_message("Abgestuerzt! Zu schnell!");
    room_restart();
}

Kollision mit Boden (Absturz)

In obj_lander:

  1. Event hinzufuegen: Collision > obj_ground
  2. Aktion hinzufuegen: Control > Execute Code
crashed = true;
show_message("Im Gelaende abgestuerzt!");
room_restart();

Geschwindigkeitsberechnung

Wir verwenden den Satz des Pythagoras, um die Gesamtgeschwindigkeit zu berechnen:

total_speed = sqrt(hsp*hsp + vsp*vsp)

Dies kombiniert horizontale und vertikale Geschwindigkeit zu einem Wert.

Sichere Geschwindigkeit ist 2! Wenn total_speed > 2, stuerzt du sogar auf der Plattform ab.
