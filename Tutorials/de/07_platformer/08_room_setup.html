Step 7: Room Setup

Time to design your platformer level!

Create the Room

  1. Right-click Rooms > Create Room
  2. Name it room_level1
  3. Set size: 800 x 480
  4. Set background color to light blue (sky)
  5. Enable "Snap to Grid" and set to 32x32

Example Level Layout

                                        F
                                      ===
                          C       C
                        =====   =====
            C                           C
          ===== X     X         X     =====
    P                   C
  ====== === ===   ===   === === ===== ======
  GGGGGG     GGG   GGG   GGG         GGGGGGGG

G = Ground    P = Player    F = Flag    C = Coin
X = Spike     === = Platform

Building Your Level

  1. Ground: Place obj_ground along the bottom
  2. Gaps: Leave spaces for pits (death traps)
  3. Platforms: Place obj_platform in the air
  4. Coins: Scatter on platforms and hard-to-reach spots
  5. Spikes: Place near pits or on platforms
  6. Flag: Place at the end of the level
  7. Player: Place at the start (left side)
  8. Controller: Place obj_game_controller anywhere

Test Your Game!

Press F5 or click Run!

  • Left/Right Arrows: Move
  • Up Arrow or Space: Jump
  • Collect coins for points
  • Avoid spikes!
  • Reach the flag to win!
Congratulations! You've created a platformer game with gravity, jumping, and collision!
Challenge: Add more levels, moving platforms, or enemies!
