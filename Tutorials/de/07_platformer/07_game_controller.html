Schritt 6: Spiel-Controller

Der Spiel-Controller zeigt die Punkte auf dem Bildschirm an.

Das Controller-Objekt erstellen

  1. Erstelle das Objekt obj_game_controller
  2. Kein Sprite noetig (unsichtbarer Controller)

Die Punkte zeichnen

In obj_game_controller:

  1. Event hinzufuegen: Draw > Draw
  2. Aktion hinzufuegen: Steuerung > Code ausfuehren
// Textfarbe setzen
draw_set_color(c_white);

// Punkte oben links zeichnen
draw_text(10, 10, "Punkte: " + string(score));

// Optional: Anleitung zeichnen
draw_text(10, 30, "Pfeiltasten: Bewegen  |  Leertaste: Springen");

Alternative: Mit Aktionen

Anstatt Code kannst du auch verwenden:

  1. Aktion hinzufuegen: Draw > Farbe setzen (Weiss)
  2. Aktion hinzufuegen: Punkte > Punkte zeichnen
    • X: 10
    • Y: 10
    • Beschriftung: "Punkte: "
Denke daran: Platziere den Spiel-Controller in deinem Raum! Er erscheint nicht von selbst.
Klicke auf "Weiter", um dein Level zu gestalten.
