Schritt 4: Spielerbewegung (Step-Event)

Dies ist der Kern unseres Plattformers: Bewegung, Schwerkraft, Springen und Kollision.

Step-Event hinzufuegen

In obj_player:

  1. Event hinzufuegen: Step > Step
  2. Aktion hinzufuegen: Steuerung > Code ausfuehren
// === HORIZONTALE BEWEGUNG ===
var move_input = keyboard_check(vk_right) - keyboard_check(vk_left);
hsp = move_input * hspeed_max;

// === SCHWERKRAFT ===
vsp += gravity_force;
if (vsp > vspeed_max) vsp = vspeed_max;

// === BODENPRUEFUNG ===
on_ground = place_meeting(x, y + 1, obj_ground);

// === SPRINGEN ===
if (on_ground && (keyboard_check_pressed(vk_up) || keyboard_check_pressed(vk_space))) {
    vsp = jump_force;
}

// === HORIZONTALE KOLLISION ===
if (place_meeting(x + hsp, y, obj_ground)) {
    while (!place_meeting(x + sign(hsp), y, obj_ground)) {
        x += sign(hsp);
    }
    hsp = 0;
}
x += hsp;

// === VERTIKALE KOLLISION ===
if (place_meeting(x, y + vsp, obj_ground)) {
    while (!place_meeting(x, y + sign(vsp), obj_ground)) {
        y += sign(vsp);
    }
    vsp = 0;
}
y += vsp;

Wie es funktioniert

  1. Eingabe: Rechte Pfeiltaste = 1, Links = -1, Keine = 0
  2. Schwerkraft: Addiert jeden Frame zu vsp (faellt schneller)
  3. Bodenpruefung: Schaut 1 Pixel unterhalb nach Boden
  4. Sprung: Funktioniert nur, wenn on_ground wahr ist
  5. Kollision: Bewegt sich Pixel fuer Pixel bis zur Wand
Der Trick: Die while-Schleife bewegt uns Pixel fuer Pixel bis zum Rand der Wand und verhindert so, dass wir stecken bleiben!
