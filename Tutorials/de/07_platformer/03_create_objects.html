Step 2: Create Objects

Now we create objects for all our game elements.

Ground Object

  1. Right-click Objects > Create Object
  2. Name it obj_ground
  3. Set sprite to spr_ground
  4. Check the "Solid" checkbox

Platform Object

  1. Create obj_platform
  2. Set sprite to spr_platform
  3. Check the "Solid" checkbox
  4. Set Parent to obj_ground (optional)

Player Object

  1. Create obj_player
  2. Set sprite to spr_player

Collectibles and Hazards

  1. Create obj_coin with spr_coin
  2. Create obj_spike with spr_spike
  3. Create obj_flag with spr_flag
Important: Ground and platforms must be "Solid" for collision to work properly!
Click "Next" to add player physics (gravity and jumping).
