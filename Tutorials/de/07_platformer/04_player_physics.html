Step 3: Player Physics (Create Event)

We need to set up variables for gravity, jumping, and movement.

Initialize Physics Variables

In obj_player:

  1. Add Event: Create
  2. Add Action: Control > Execute Code
// Movement settings
hspeed_max = 4;      // Maximum horizontal speed
vspeed_max = 10;     // Maximum fall speed
jump_force = -10;    // Jump strength (negative = up)
gravity_force = 0.5; // How fast we fall

// Current speeds
hsp = 0;
vsp = 0;

// State
on_ground = false;

Understanding the Variables

Variable Purpose
hsp Horizontal speed (left/right)
vsp Vertical speed (up/down)
gravity_force Constant force pulling down
jump_force Negative value to go upward
on_ground True when standing on ground
Key Concept: Jump force is negative because Y increases downward in games. A negative Y velocity moves the player up!
