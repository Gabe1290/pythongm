Schritt 3: Spielerphysik (Create-Event)

Wir muessen Variablen fuer Schwerkraft, Springen und Bewegung einrichten.

Physik-Variablen initialisieren

In obj_player:

  1. Event hinzufuegen: Create
  2. Aktion hinzufuegen: Steuerung > Code ausfuehren
// Bewegungseinstellungen
hspeed_max = 4;      // Maximale horizontale Geschwindigkeit
vspeed_max = 10;     // Maximale Fallgeschwindigkeit
jump_force = -10;    // Sprungstaerke (negativ = nach oben)
gravity_force = 0.5; // Wie schnell wir fallen

// Aktuelle Geschwindigkeiten
hsp = 0;
vsp = 0;

// Zustand
on_ground = false;

Die Variablen verstehen

Variable Zweck
hsp Horizontale Geschwindigkeit (links/rechts)
vsp Vertikale Geschwindigkeit (oben/unten)
gravity_force Konstante Kraft, die nach unten zieht
jump_force Negativer Wert, um nach oben zu gehen
on_ground Wahr, wenn auf dem Boden stehend
Schluesselkonzept: Die Sprungkraft ist negativ, weil Y in Spielen nach unten zunimmt. Eine negative Y-Geschwindigkeit bewegt den Spieler nach oben!
