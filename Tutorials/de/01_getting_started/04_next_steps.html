Nächste Schritte

Jetzt, da Sie die Grundlagen kennen, sind hier einige Themen zum Erkunden:

Bewegung hinzufügen

Öffnen Sie Ihr Objekt und fügen Sie ein Tastatur-Ereignis hinzu. Verwenden Sie Aktionen wie "In Richtung bewegen" oder "Geschwindigkeit setzen", um Ihr Objekt auf Tastendrücke reagieren zu lassen.

Kollisionserkennung

Erstellen Sie ein weiteres Objekt (wie eine Wand) und fügen Sie ein Kollisions-Ereignis zu Ihrem Spieler hinzu. Verwenden Sie Aktionen, um die Bewegung zu stoppen oder die Richtung zu ändern, wenn Objekte kollidieren.

Visuelle Programmierung mit Blockly

PyGameMaker unterstützt visuelle Blockly-Programmierung. Im Objekt-Editor wechseln Sie in den Blockly-Modus, um Verhaltensweisen durch das Verbinden von Blöcken anstelle von Code zu erstellen.

Mehrere Räume

Erstellen Sie weitere Räume für verschiedene Level. Verwenden Sie die Aktion "Zum Raum gehen", um zwischen ihnen zu wechseln, wenn der Spieler ein Ziel oder eine Tür erreicht.

Spiel exportieren

Wenn Ihr Spiel fertig ist, gehen Sie zu Datei > Als HTML5 exportieren, um eine Web-Version zu erstellen, die in jedem Browser gespielt werden kann!
