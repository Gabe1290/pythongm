Schritt 2: Objekte erstellen

Jetzt erstellen wir Objekte fuer Waende, Ausgang, Muenzen und den Spieler.

Wand-Objekt erstellen

  1. Rechtsklicke auf Objekte und waehle Objekt erstellen
  2. Benenne es obj_wall
  3. Setze das Sprite auf spr_wall
  4. Aktiviere "Solid"
  5. Keine Events erforderlich

Ausgang-Objekt erstellen

  1. Erstelle Objekt obj_exit
  2. Setze Sprite auf spr_exit
  3. Lasse "Solid" deaktiviert

Muenzen-Objekt erstellen

  1. Erstelle Objekt obj_coin
  2. Setze Sprite auf spr_coin
  3. Lasse "Solid" deaktiviert

Spieler-Objekt erstellen

  1. Erstelle Objekt obj_player
  2. Setze Sprite auf spr_player
  3. Lasse "Solid" deaktiviert
Hinweis: Nur die Wand muss "Solid" sein. Der Spieler prueft Kollisionen in seinem Code.
Klicke auf "Weiter", um die Spielerbewegung hinzuzufuegen.
