Schritt 3: Spielerbewegung

Der Spieler braucht fluessige Bewegung mit Kollisionserkennung.

Geschwindigkeit initialisieren (Create-Event)

In obj_player:

  1. Event hinzufuegen: Create
  2. Aktion hinzufuegen: Control > Set Variable
  3. Variable: move_speed
  4. Wert: 4

Bewegung mit Kollision (Step-Event)

  1. Event hinzufuegen: Step
  2. Aktion hinzufuegen: Control > Execute Code
// Horizontale Bewegung
var hspd = 0;
if (keyboard_check(vk_right)) hspd = move_speed;
if (keyboard_check(vk_left)) hspd = -move_speed;

// Vertikale Bewegung
var vspd = 0;
if (keyboard_check(vk_down)) vspd = move_speed;
if (keyboard_check(vk_up)) vspd = -move_speed;

// Horizontale Kollision pruefen
if (!place_meeting(x + hspd, y, obj_wall)) {
    x += hspd;
}

// Vertikale Kollision pruefen
if (!place_meeting(x, y + vspd, obj_wall)) {
    y += vspd;
}
So funktioniert es: Wir pruefen, ob die Zielposition mit einer Wand kollidieren wuerde. Falls nicht, bewegen wir uns dorthin!
