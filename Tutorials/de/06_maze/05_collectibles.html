Step 4: Coins and Exit

Now let's make coins collectible and the exit work.

Coin Collection

In obj_coin:

  1. Add Event: Collision > obj_player
  2. Add Action: Score > Set Score
  3. New Score: 10
  4. Check "Relative" to add 10 points
  5. Add Action: Main1 > Destroy Instance
  6. Applies to: Self

Exit Behavior

In obj_exit:

  1. Add Event: Collision > obj_player
  2. Add Action: Main2 > Show Message
  3. Message: You Win!
  4. Add Action: Main1 > Restart Room
Tip: You can use Next Room instead of Restart Room if you create multiple levels!
