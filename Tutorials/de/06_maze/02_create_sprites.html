Schritt 1: Sprites erstellen

Wir brauchen Sprites fuer den Spieler, Waende, Ausgang und Muenzen. Waende sollten 32x32 Pixel gross sein, um ein Raster zu erstellen.

Spieler-Sprite erstellen

  1. Rechtsklicke auf Sprites und waehle Sprite erstellen
  2. Benenne es spr_player
  3. Klicke auf Sprite bearbeiten
  4. Zeichne eine kleine Figur (Kreis oder einfache Person)
  5. Verwende eine helle Farbe wie Blau oder Gruen
  6. Groesse: 24x24 Pixel (kleiner als die Waende)

Wand-Sprite erstellen

  1. Erstelle ein neues Sprite namens spr_wall
  2. Zeichne ein Ziegel- oder Steinmuster
  3. Verwende graue oder dunkle Farben
  4. Groesse: 32x32 Pixel

Ausgang-Sprite erstellen

  1. Erstelle ein neues Sprite namens spr_exit
  2. Zeichne eine Tuer, Flagge oder leuchtende Markierung
  3. Verwende gruene oder goldene Farben
  4. Groesse: 32x32 Pixel

Muenzen-Sprite erstellen

  1. Erstelle ein neues Sprite namens spr_coin
  2. Zeichne einen kleinen gelben/goldenen Kreis
  3. Groesse: 16x16 Pixel
Tipp: Wenn das Spieler-Sprite kleiner als die Waende ist, erleichtert das die Bewegung durch Gaenge!
Klicke auf "Weiter", um die Spielobjekte zu erstellen.
