Schritt 5: Spielcontroller

Erstelle einen Controller, um die Zeit zu verfolgen und den Punktestand anzuzeigen.

Controller-Objekt erstellen

  1. Erstelle Objekt obj_game_controller
  2. Kein Sprite erforderlich (leer lassen)

Timer initialisieren (Create-Event)

  1. Event hinzufuegen: Create
  2. Aktion hinzufuegen: Control > Set Variable
  3. Variable: global.timer
  4. Wert: 0

Timer aktualisieren (Step-Event)

  1. Event hinzufuegen: Step
  2. Aktion hinzufuegen: Control > Set Variable
  3. Variable: global.timer
  4. Wert: 1/room_speed
  5. Aktiviere "Relative"

Informationen anzeigen (Draw-Event)

  1. Event hinzufuegen: Draw
  2. Aktion hinzufuegen: Control > Execute Code
draw_set_color(c_white);
draw_text(10, 10, "Punkte: " + string(score));
draw_text(10, 30, "Zeit: " + string(floor(global.timer)) + "s");

var coins = instance_number(obj_coin);
draw_text(10, 50, "Muenzen: " + string(coins));
