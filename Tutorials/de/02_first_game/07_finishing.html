Schritt 6: Feinschliff

Lass uns ein paar Verbesserungen hinzufügen, damit sich dein Spiel vollständig anfühlt!

Spieler auf dem Bildschirm halten

1 Grenzprüfung hinzufügen

In obj_player füge ein Step-Ereignis mit diesen Bedingungen hinzu:

  • Wenn x < 0, setze x = 0
  • Wenn x > room_width - sprite_width, setze x = room_width - sprite_width

Das verhindert, dass der Spieler über die Seiten des Bildschirms hinausbewegt.

Eine Hintergrundfarbe hinzufügen

2 Raumhintergrund setzen

Öffne room_game im Raum-Editor.

In den Raumeigenschaften setze die Hintergrundfarbe auf ein schönes Himmelblau oder eine dunkle Weltraumfarbe.

Dein Spiel exportieren!

3 Als HTML5 exportieren

Gehe zu Datei > Als HTML5 exportieren.

Wähle einen Speicherort und klicke auf Exportieren.

Öffne die HTML-Datei in einem Webbrowser, um dein Spiel zu spielen!

Herzlichen Glückwunsch! Du hast dein erstes vollständiges Spiel in PyGameMaker erstellt!

Ideen zur Verbesserung deines Spiels

Herausforderungen zum Ausprobieren:
  • Leben hinzufügen - verliere ein Leben, wenn ein Stern den Boden erreicht
  • Sterne schneller fallen lassen, wenn der Punktestand steigt
  • Verschiedenfarbige Sterne hinzufügen, die unterschiedliche Punkte wert sind
  • Ein "schlechtes" Objekt (Bombe) hinzufügen, das das Spiel beendet, wenn es gefangen wird
  • Soundeffekte beim Fangen von Sternen hinzufügen
  • Einen Titelbildschirm und einen Game-Over-Bildschirm erstellen
