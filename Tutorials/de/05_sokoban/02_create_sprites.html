Schritt 1: Sprites erstellen

Alle Sprites sollten die gleiche Groesse haben (32x32 Pixel), um ein ordentliches Raster zu erstellen.

Spieler-Sprite erstellen

  1. Rechtsklicke auf Sprites im Ressourcen-Baum
  2. Waehle Sprite erstellen
  3. Benenne es spr_player
  4. Klicke auf Sprite bearbeiten
  5. Zeichne eine einfache Figur (32x32 Pixel)
  6. Verwende eine auffaellige Farbe wie Blau
  7. Klicke auf OK zum Speichern

Kisten-Sprites erstellen

Normale Kiste:

  1. Erstelle Sprite spr_crate
  2. Zeichne eine Holzkisten-Form (32x32 Pixel)
  3. Verwende Braun oder Orange als Farbe

Kiste auf Zielfeld:

  1. Erstelle Sprite spr_crate_ok
  2. Zeichne die gleiche Kiste, aber in Gruen
  3. Dies zeigt an, wenn eine Kiste richtig platziert ist

Wand-Sprite erstellen

  1. Erstelle Sprite spr_wall
  2. Zeichne ein solides Ziegel- oder Steinmuster (32x32 Pixel)
  3. Verwende Grau oder eine dunkle Farbe

Zielfeld-Sprite erstellen

  1. Erstelle Sprite spr_target
  2. Zeichne ein X-Zeichen oder einen Ziel-Indikator (32x32 Pixel)
  3. Verwende eine helle Farbe wie Rot
Wichtig: Halte ALLE Sprites genau bei 32x32 Pixel fuer die korrekte Rasterausrichtung!
Klicke auf "Weiter", um die Spielobjekte zu erstellen.
