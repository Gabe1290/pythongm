Schritt 3: Kistenverhalten

Die Kiste muss ihr Aussehen aendern, wenn sie auf einem Zielfeld steht.

Zielposition pruefen (Step-Event)

In obj_crate:

  1. Event hinzufuegen: Step
  2. Aktion hinzufuegen: Control > Test Instance Position
  3. Object: obj_target
  4. X: 0, Y: 0 (relativ)
  5. Pruefen, ob eine Kollision existiert

Sprite aendern, wenn auf Zielfeld

Wenn Kollision existiert (Kiste ist auf Zielfeld):

  1. Aktion hinzufuegen: Main1 > Change Sprite
  2. Sprite: spr_crate_ok

Sprite aendern, wenn nicht auf Zielfeld

  1. Aktion hinzufuegen: Control > Else
  2. Aktion hinzufuegen: Main1 > Change Sprite
  3. Sprite: spr_crate
Visuelles Feedback: Wenn die Kiste gruen wird, wissen die Spieler, dass sie richtig platziert ist!
Klicke auf "Weiter", um die Spielerbewegung hinzuzufuegen.
