Schritt 2: Objekte erstellen

Wir brauchen Objekte fuer den Spieler, Waende, Kisten und Zielfelder.

Wand-Objekt erstellen

  1. Rechtsklicke auf Objects und waehle Objekt erstellen
  2. Benenne es obj_wall
  3. Setze das Sprite auf spr_wall
  4. Aktiviere "Solid"
  5. Keine Events benoetigt

Zielfeld-Objekt erstellen

  1. Erstelle Objekt obj_target
  2. Setze Sprite auf spr_target
  3. Lasse "Solid" deaktiviert (Spieler und Kisten koennen darueber stehen)
  4. Keine Events benoetigt - es ist nur eine Markierung

Kisten-Objekt erstellen

  1. Erstelle Objekt obj_crate
  2. Setze Sprite auf spr_crate
  3. Aktiviere "Solid"

Spieler-Objekt erstellen

  1. Erstelle Objekt obj_player
  2. Setze Sprite auf spr_player
  3. Lasse "Solid" deaktiviert
Hinweis: Wir werden Events zu obj_crate und obj_player in den naechsten Schritten hinzufuegen.
Klicke auf "Weiter", um das Kistenverhalten hinzuzufuegen.
