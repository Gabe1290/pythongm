Schritt 5: Siegbedingung

Der Spieler gewinnt, wenn alle Kisten auf Zielfeldern stehen.

Spiel-Controller erstellen

  1. Erstelle Objekt obj_game_controller
  2. Kein Sprite benoetigt

Zielfelder zaehlen (Create-Event)

In obj_game_controller:

  1. Event hinzufuegen: Create
  2. Aktion hinzufuegen: Control > Execute Code
  3. Eingeben: global.total_targets = instance_number(obj_target);

Siegbedingung pruefen (Step-Event)

  1. Event hinzufuegen: Step
  2. Aktion hinzufuegen: Control > Execute Code
  3. Diesen Code eingeben:
// Kisten auf Zielfeldern zaehlen
var crates_on_targets = 0;
with (obj_crate) {
    if (place_meeting(x, y, obj_target)) {
        crates_on_targets += 1;
    }
}

// Pruefen, ob alle Zielfelder Kisten haben
if (crates_on_targets >= global.total_targets
    && global.total_targets > 0) {
    show_message("Level geschafft!");
    room_restart();
}

Anweisungen zeichnen (Draw-Event)

  1. Event hinzufuegen: Draw
  2. Aktion hinzufuegen: Draw > Draw Text
  3. Text: Schiebe Kisten auf die Zielfelder!
  4. X: 10, Y: 10
Tipp: Stelle sicher, dass die Anzahl der Kisten gleich der Anzahl der Zielfelder in deinem Level ist!
Klicke auf "Weiter", um den Spielraum einzurichten.
