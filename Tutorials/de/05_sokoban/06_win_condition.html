Step 5: Win Condition

The player wins when all crates are on target spots.

Create Game Controller

  1. Create object obj_game_controller
  2. No sprite needed

Count Targets (Create Event)

In obj_game_controller:

  1. Add Event: Create
  2. Add Action: Control > Execute Code
  3. Enter: global.total_targets = instance_number(obj_target);

Check Win Condition (Step Event)

  1. Add Event: Step
  2. Add Action: Control > Execute Code
  3. Enter this code:
// Count crates on targets
var crates_on_targets = 0;
with (obj_crate) {
    if (place_meeting(x, y, obj_target)) {
        crates_on_targets += 1;
    }
}

// Check if all targets have crates
if (crates_on_targets >= global.total_targets
    && global.total_targets > 0) {
    show_message("Level Complete!");
    room_restart();
}

Draw Instructions (Draw Event)

  1. Add Event: Draw
  2. Add Action: Draw > Draw Text
  3. Text: Push crates onto targets!
  4. X: 10, Y: 10
Tip: Make sure the number of crates equals the number of targets in your level!
