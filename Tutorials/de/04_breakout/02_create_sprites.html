Schritt 1: Erstelle die Sprites

Wir brauchen Sprites fuer den Schlaeger, Ball, Steine und Waende.

Erstelle das Ball-Sprite

  1. Rechtsklicke auf Sprites im Ressourcenbaum
  2. Waehle Sprite erstellen
  3. Benenne es spr_ball
  4. Klicke auf Sprite bearbeiten
  5. Zeichne einen kleinen weissen oder gelben Kreis (16x16 Pixel)
  6. Klicke auf OK zum Speichern

Erstelle das Schlaeger-Sprite

  1. Erstelle ein neues Sprite namens spr_paddle
  2. Zeichne ein breites, flaches Rechteck (64x16 Pixel)
  3. Verwende eine helle Farbe wie Blau

Erstelle Stein-Sprites

Erstelle bunte Steine fuer verschiedene Punktwerte:

  1. Erstelle spr_brick_red - Roter Stein (32x16 Pixel)
  2. Erstelle spr_brick_orange - Oranger Stein
  3. Erstelle spr_brick_yellow - Gelber Stein
  4. Erstelle spr_brick_green - Gruener Stein

Erstelle das Wand-Sprite

  1. Erstelle ein neues Sprite namens spr_wall
  2. Zeichne ein solides graues Rechteck (32x32 Pixel)
Tipp: Halte die Steingroessen einheitlich, damit sie im Raum schoen ausgerichtet sind!
Klicke auf "Weiter", um die Spielobjekte zu erstellen.
