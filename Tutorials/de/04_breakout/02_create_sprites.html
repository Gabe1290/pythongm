Step 1: Create the Sprites

We need sprites for the paddle, ball, bricks, and walls.

Create the Ball Sprite

  1. Right-click on Sprites in the Resource Tree
  2. Select Create Sprite
  3. Name it spr_ball
  4. Click Edit Sprite
  5. Draw a small white or yellow circle (16x16 pixels)
  6. Click OK to save

Create the Paddle Sprite

  1. Create a new sprite named spr_paddle
  2. Draw a wide, flat rectangle (64x16 pixels)
  3. Use a bright color like blue

Create Brick Sprites

Create colorful bricks for different point values:

  1. Create spr_brick_red - Red brick (32x16 pixels)
  2. Create spr_brick_orange - Orange brick
  3. Create spr_brick_yellow - Yellow brick
  4. Create spr_brick_green - Green brick

Create the Wall Sprite

  1. Create a new sprite named spr_wall
  2. Draw a solid gray rectangle (32x32 pixels)
Tip: Keep brick sizes consistent so they align nicely in the room!
Click "Next" to create the game objects.
