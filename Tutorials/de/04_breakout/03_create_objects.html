Schritt 2: Erstelle die Objekte

Jetzt erstellen wir alle Spielobjekte.

Erstelle das Wand-Objekt

  1. Rechtsklicke auf Objekte und waehle Objekt erstellen
  2. Benenne es obj_wall
  3. Setze das Sprite auf spr_wall
  4. Aktiviere "Solide"
  5. Keine Ereignisse benoetigt

Erstelle das Eltern-Stein-Objekt

Wir verwenden ein Elternobjekt, damit alle Steine das gleiche Kollisionsverhalten teilen.

  1. Erstelle Objekt obj_brick_parent
  2. Kein Sprite benoetigt (es ist nur eine Vorlage)
  3. Aktiviere "Solide"

Erstelle die farbigen Stein-Objekte

Fuer jede Steinfarbe:

  1. Erstelle obj_brick_red mit spr_brick_red
  2. Setze Elternobjekt auf obj_brick_parent
  3. Aktiviere "Solide"

Wiederhole fuer obj_brick_orange, obj_brick_yellow und obj_brick_green.

Erstelle das Schlaeger-Objekt

  1. Erstelle Objekt obj_paddle
  2. Setze Sprite auf spr_paddle
  3. Aktiviere "Solide"

Erstelle das Ball-Objekt

  1. Erstelle Objekt obj_ball
  2. Setze Sprite auf spr_ball
  3. Lasse "Solide" deaktiviert
Elternobjekte: Wenn der Ball mit obj_brick_parent kollidiert, erkennt er Kollisionen mit ALLEN Steintypen!
Klicke auf "Weiter", um Schlaegerbewegung hinzuzufuegen.
