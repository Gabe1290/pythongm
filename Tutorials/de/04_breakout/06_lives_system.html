Schritt 5: Fuege das Leben-System hinzu

Der Spieler verliert ein Leben, wenn der Ball vom unteren Bildschirmrand faellt.

Erstelle eine Todeszone

  1. Erstelle ein Sprite spr_death_zone (32x32, beliebige Farbe)
  2. Erstelle Objekt obj_death_zone
  3. Deaktiviere "Sichtbar" - es soll unsichtbar sein
  4. Aktiviere "Solide"

Ball faellt in die Todeszone

In obj_ball:

  1. Fuege Ereignis hinzu: Kollision > obj_death_zone
  2. Fuege Aktion hinzu: Punkte > Von Leben abziehen
  3. Wert: 1
  4. Fuege Aktion hinzu: Bewegung > Zur Startposition springen

Erstelle einen Spiel-Controller

  1. Erstelle Objekt obj_game_controller
  2. Kein Sprite benoetigt

Leben initialisieren (Erstellen-Ereignis)

In obj_game_controller:

  1. Fuege Ereignis hinzu: Erstellen
  2. Fuege Aktion hinzu: Punkte > Leben setzen
  3. Wert: 3
  4. Fuege Aktion hinzu: Punkte > Punktzahl setzen
  5. Wert: 0

Auf Spielende pruefen (Schritt-Ereignis)

  1. Fuege Ereignis hinzu: Schritt
  2. Fuege Aktion hinzu: Steuerung > Leben testen
  3. Pruefe, ob Leben kleiner oder gleich 0 sind
  4. Fuege Aktion hinzu: Main2 > Nachricht anzeigen
  5. Nachricht: Spiel vorbei!
  6. Fuege Aktion hinzu: Main2 > Raum neu starten
Denke daran: Platziere obj_game_controller in deinem Raum!
Klicke auf "Weiter", um die Punkteanzeige hinzuzufuegen.
