Step 7: Set Up the Room

Now let's create the game room and place all our objects!

Create the Room

  1. Right-click on Rooms and select Create Room
  2. Name it room_pong
  3. Set size to 640x480 pixels

Room Layout

+--[WALL WALL WALL WALL WALL WALL]--+
|                                   |
[G]  [P1]                    [P2]  [G]
[O]  [  ]       (o)          [  ]  [O]
[A]  [  ]                    [  ]  [A]
[L]                                [L]
|                                   |
+--[WALL WALL WALL WALL WALL WALL]--+

Place the Objects

1. Walls (Top and Bottom)

Place obj_wall instances along the top and bottom edges.

2. Paddles

  • obj_paddle_left - Near left edge, centered vertically
  • obj_paddle_right - Near right edge, centered vertically

3. Ball

Place obj_ball in the center of the room.

4. Goals

  • obj_goal_left - Stack along the left edge (behind paddle)
  • obj_goal_right - Stack along the right edge (behind paddle)

5. Score Display

Place obj_score anywhere (it has no sprite, just draws text).

Congratulations! Your Pong game is ready to play!

Test Your Game

  • Press F5 or click Run
  • Player 1: W (up) and S (down)
  • Player 2: Up Arrow and Down Arrow
Well done! You've created a classic two-player Pong game!
