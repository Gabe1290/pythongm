Schritt 2: Objekte erstellen

Jetzt erstellen wir die Spielobjekte. Jedes Objekt verwendet ein Sprite und kann Verhaltensweisen haben.

Wand-Objekt erstellen

  1. Rechtsklick auf Objekte und wähle Objekt erstellen
  2. Benenne es obj_wall
  3. Setze das Sprite auf spr_wall
  4. Aktiviere das Kontrollkästchen „Solid" - das ist wichtig!
  5. Keine Ereignisse nötig - Wände blockieren nur die Bewegung

Ball-Objekt erstellen

  1. Erstelle ein neues Objekt namens obj_ball
  2. Setze das Sprite auf spr_ball
  3. Lasse „Solid" deaktiviert

Schläger-Objekte erstellen

Linker Schläger:

  1. Erstelle Objekt obj_paddle_left
  2. Setze Sprite auf spr_paddle_left
  3. Aktiviere „Solid"

Rechter Schläger:

  1. Erstelle Objekt obj_paddle_right
  2. Setze Sprite auf spr_paddle_right
  3. Aktiviere „Solid"
Wichtig: Das Kontrollkästchen „Solid" ist entscheidend, damit das Abprallen korrekt funktioniert!
Klicke auf „Weiter", um die Schlägersteuerung hinzuzufügen.
