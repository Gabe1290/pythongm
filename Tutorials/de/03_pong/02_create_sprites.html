Schritt 1: Sprites erstellen

Wir benötigen vier Sprites für unser Pong-Spiel: einen Ball, zwei Schläger und Wände.

Ball-Sprite erstellen

  1. Rechtsklick auf Sprites im Ressourcenbaum
  2. Wähle Sprite erstellen
  3. Benenne es spr_ball
  4. Klicke auf Sprite bearbeiten
  5. Zeichne einen kleinen weißen Kreis (etwa 16x16 Pixel)
  6. Klicke auf OK zum Speichern

Schläger-Sprites erstellen

Linker Schläger (Spieler 1):

  1. Erstelle ein neues Sprite namens spr_paddle_left
  2. Zeichne ein hohes, schmales Rechteck (etwa 16x64 Pixel)
  3. Verwende blaue Farbe zur Kennzeichnung von Spieler 1

Rechter Schläger (Spieler 2):

  1. Erstelle ein neues Sprite namens spr_paddle_right
  2. Zeichne ein hohes, schmales Rechteck (etwa 16x64 Pixel)
  3. Verwende rote Farbe zur Kennzeichnung von Spieler 2

Wand-Sprite erstellen

  1. Erstelle ein neues Sprite namens spr_wall
  2. Zeichne ein graues Rechteck (32x32 Pixel)
  3. Dies wird gestreckt, um die oberen und unteren Begrenzungen zu bilden
Tipp: Halte alle Sprites einfach! Das klassische Pong verwendete einfache Formen.
Klicke auf „Weiter", um die Spielobjekte zu erstellen.
