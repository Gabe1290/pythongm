Schritt 6: Punkteanzeige

Wir benötigen ein Controller-Objekt, um die Punkte zu initialisieren und anzuzeigen.

Punktzahl-Controller-Objekt erstellen

  1. Erstelle ein neues Objekt namens obj_score
  2. Kein Sprite erforderlich (leer lassen)

Punkte initialisieren (Create-Ereignis)

  1. Ereignis hinzufügen: Create
  2. Aktion hinzufügen: Punktzahl > Variable setzen
  3. Variable: global.p1score, Wert: 0
  4. Füge eine weitere Variable setzen-Aktion hinzu
  5. Variable: global.p2score, Wert: 0

Punkte zeichnen (Draw-Ereignis)

  1. Ereignis hinzufügen: Draw
  2. Aktion hinzufügen: Zeichnen > Text zeichnen
  3. Text: Spieler 1:
  4. X: 10, Y: 10
  1. Aktion hinzufügen: Zeichnen > Variable zeichnen
  2. Variable: global.p1score
  3. X: 100, Y: 10
  1. Aktion hinzufügen: Zeichnen > Text zeichnen
  2. Text: Spieler 2:
  3. X: 10, Y: 30
  1. Aktion hinzufügen: Zeichnen > Variable zeichnen
  2. Variable: global.p2score
  3. X: 100, Y: 30
Denke daran: Vergiss nicht, obj_score in deinem Raum zu platzieren!
Klicke auf „Weiter", um den Spielraum einzurichten.
