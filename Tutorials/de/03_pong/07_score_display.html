Step 6: Display the Scores

We need a controller object to initialize scores and display them.

Create Score Controller Object

  1. Create a new object named obj_score
  2. No sprite needed (leave it empty)

Initialize Scores (Create Event)

  1. Add Event: Create
  2. Add Action: Score > Set Variable
  3. Variable: global.p1score, Value: 0
  4. Add another Set Variable action
  5. Variable: global.p2score, Value: 0

Draw Scores (Draw Event)

  1. Add Event: Draw
  2. Add Action: Draw > Draw Text
  3. Text: Player 1:
  4. X: 10, Y: 10
  1. Add Action: Draw > Draw Variable
  2. Variable: global.p1score
  3. X: 100, Y: 10
  1. Add Action: Draw > Draw Text
  2. Text: Player 2:
  3. X: 10, Y: 30
  1. Add Action: Draw > Draw Variable
  2. Variable: global.p2score
  3. X: 100, Y: 30
Remember: Don't forget to place obj_score in your room!
