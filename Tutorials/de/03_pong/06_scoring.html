Schritt 5: Tore hinzufügen

Wir benötigen unsichtbare Tore hinter jedem Schläger, um zu erkennen, wenn jemand punktet.

Tor-Sprite erstellen

  1. Erstelle ein neues Sprite namens spr_goal
  2. Mache es 32x32 Pixel mit beliebiger Farbe
  3. (Es wird im Spiel unsichtbar sein)

Linkes Tor-Objekt erstellen

  1. Erstelle Objekt obj_goal_left
  2. Setze Sprite auf spr_goal
  3. Deaktiviere „Sichtbar" - das Tor soll unsichtbar sein
  4. Aktiviere „Solid"

Rechtes Tor-Objekt erstellen

  1. Erstelle Objekt obj_goal_right
  2. Setze Sprite auf spr_goal
  3. Deaktiviere „Sichtbar"
  4. Aktiviere „Solid"

Ball trifft linkes Tor (Spieler 2 punktet)

Gehe zurück zu obj_ball:

  1. Ereignis hinzufügen: Kollision > obj_goal_left
  2. Aktion hinzufügen: Bewegung > Zur Startposition springen
  3. Aktion hinzufügen: Punktzahl > Variable setzen
  4. Variable: global.p2score
  5. Wert: 1
  6. Aktiviere „Relativ" (addiert 1 zum aktuellen Punktestand)

Ball trifft rechtes Tor (Spieler 1 punktet)

  1. Ereignis hinzufügen: Kollision > obj_goal_right
  2. Aktion hinzufügen: Bewegung > Zur Startposition springen
  3. Aktion hinzufügen: Punktzahl > Variable setzen
  4. Variable: global.p1score
  5. Wert: 1
  6. Aktiviere „Relativ"
Hinweis: Das Präfix global. bedeutet, dass diese Variablen von jedem Objekt aus zugänglich sind!
Klicke auf „Weiter", um die Punkteanzeige einzurichten.
