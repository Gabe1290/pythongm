Paso 3: Movimiento del Jugador

El jugador necesita movimiento suave con deteccion de colisiones.

Inicializar Velocidad (Evento Crear)

En obj_player:

  1. Agregar Evento: Crear
  2. Agregar Accion: Control > Establecer Variable
  3. Variable: move_speed
  4. Valor: 4

Movimiento con Colision (Evento Step)

  1. Agregar Evento: Step
  2. Agregar Accion: Control > Ejecutar Codigo
// Movimiento horizontal
var hspd = 0;
if (keyboard_check(vk_right)) hspd = move_speed;
if (keyboard_check(vk_left)) hspd = -move_speed;

// Movimiento vertical
var vspd = 0;
if (keyboard_check(vk_down)) vspd = move_speed;
if (keyboard_check(vk_up)) vspd = -move_speed;

// Verificar colision horizontal
if (!place_meeting(x + hspd, y, obj_wall)) {
    x += hspd;
}

// Verificar colision vertical
if (!place_meeting(x, y + vspd, obj_wall)) {
    y += vspd;
}
Como funciona: Verificamos si la posicion objetivo colisionaria con una pared. Si no, nos movemos ahi!
