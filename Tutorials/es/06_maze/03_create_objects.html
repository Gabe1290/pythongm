Paso 2: Crear los Objetos

Ahora crearemos objetos para las paredes, salida, monedas y el jugador.

Crear el Objeto Pared

  1. Haz clic derecho en Objetos y selecciona Crear Objeto
  2. Nombralo obj_wall
  3. Asigna el sprite spr_wall
  4. Marca "Solido"
  5. No se necesitan eventos

Crear el Objeto Salida

  1. Crea el objeto obj_exit
  2. Asigna el sprite spr_exit
  3. Deja "Solido" sin marcar

Crear el Objeto Moneda

  1. Crea el objeto obj_coin
  2. Asigna el sprite spr_coin
  3. Deja "Solido" sin marcar

Crear el Objeto Jugador

  1. Crea el objeto obj_player
  2. Asigna el sprite spr_player
  3. Deja "Solido" sin marcar
Nota: Solo la pared necesita ser "Solido". El jugador verificara las colisiones en su codigo.
Haz clic en "Siguiente" para agregar el movimiento del jugador.
