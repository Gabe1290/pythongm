Paso 5: Controlador del Juego

Crea un controlador para registrar el tiempo y mostrar la puntuacion.

Crear el Objeto Controlador

  1. Crea el objeto obj_game_controller
  2. No se necesita sprite (dejalo vacio)

Inicializar Temporizador (Evento Crear)

  1. Agregar Evento: Crear
  2. Agregar Accion: Control > Establecer Variable
  3. Variable: global.timer
  4. Valor: 0

Actualizar Temporizador (Evento Step)

  1. Agregar Evento: Step
  2. Agregar Accion: Control > Establecer Variable
  3. Variable: global.timer
  4. Valor: 1/room_speed
  5. Marca "Relativo"

Mostrar Informacion (Evento Draw)

  1. Agregar Evento: Draw
  2. Agregar Accion: Control > Ejecutar Codigo
draw_set_color(c_white);
draw_text(10, 10, "Puntuacion: " + string(score));
draw_text(10, 30, "Tiempo: " + string(floor(global.timer)) + "s");

var coins = instance_number(obj_coin);
draw_text(10, 50, "Monedas: " + string(coins));
Nota: El controlador es invisible pero esencial para registrar el estado del juego!
