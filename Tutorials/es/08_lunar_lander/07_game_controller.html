Paso 6: Controlador del Juego (HUD)

Muestra el Combustible, velocidad y estado para ayudar al jugador.

Evento Draw

En obj_game_controller:

  1. Añadir Evento: Draw > Draw
  2. Añadir Acción: Control > Ejecutar Código
draw_set_color(c_white);

// Obtener información del módulo lunar
var lander = instance_find(obj_lander, 0);
if (lander != noone) {
    var spd = sqrt(lander.hsp*lander.hsp + lander.vsp*lander.vsp);

    // Dibujar HUD
    draw_text(10, 10, "MODULO LUNAR");
    draw_text(10, 30, "Combustible: " + string(floor(lander.fuel)) + "%");
    draw_text(10, 50, "Velocidad: " + string_format(spd, 1, 2));
    draw_text(10, 70, "Seguro: < " + string(lander.safe_speed));

    // Advertencia de velocidad
    if (spd > lander.safe_speed) {
        draw_set_color(c_red);
        draw_text(10, 90, "¡DEMASIADO RAPIDO!");
    } else {
        draw_set_color(c_lime);
        draw_text(10, 90, "Velocidad OK");
    }

    // Advertencia de Combustible
    if (lander.fuel <= 0) {
        draw_set_color(c_red);
        draw_text(10, 110, "¡SIN COMBUSTIBLE!");
    }
}

draw_set_color(c_gray);
draw_text(10, room_height - 20, "ARRIBA: Propulsion | IZQ/DER: Mover");

Lo Que Muestra el HUD

  • Combustible: Porcentaje restante (verde = OK)
  • Velocidad: Velocidad actual (rojo = peligroso)
  • Advertencias: "¡DEMASIADO RAPIDO!" o "¡SIN COMBUSTIBLE!"
  • Controles: Recordatorio en la parte inferior
