Paso 4: Controles del Módulo Lunar (Evento Step)

Añade Gravedad, Propulsión, Movimiento horizontal y verificación de límites.

Añadir Evento Step

En obj_lander:

  1. Añadir Evento: Step > Step
  2. Añadir Acción: Control > Ejecutar Código
// No actualizar si el juego terminó
if (landed || crashed) exit;

// === GRAVEDAD ===
vsp += gravity_force;

// === PROPULSION (Flecha Arriba) ===
if (keyboard_check(vk_up) && fuel > 0) {
    vsp -= thrust_force;
    fuel -= fuel_use;
    if (fuel < 0) fuel = 0;
}

// === CONTROL HORIZONTAL ===
if (keyboard_check(vk_left)) hsp -= 0.05;
if (keyboard_check(vk_right)) hsp += 0.05;

// === LIMITAR VELOCIDAD ===
hsp = clamp(hsp, -max_speed, max_speed);
vsp = clamp(vsp, -max_speed, max_speed);

// === APLICAR MOVIMIENTO ===
x += hsp;
y += vsp;

// === LIMITES DE PANTALLA ===
if (x < 16) { x = 16; hsp = 0; }
if (x > room_width - 16) { x = room_width - 16; hsp = 0; }
if (y < 16) { y = 16; vsp = 0; }

Cómo Funciona

  1. Gravedad: Se añade a vsp cada fotograma (caída)
  2. Propulsión: Resta de vsp (ascenso), usa Combustible
  3. Clamp: Limita la velocidad a un rango seguro
  4. Límites: Mantiene el módulo lunar en pantalla
Clave: ¡La función clamp() limita un valor entre un mínimo y un máximo!
