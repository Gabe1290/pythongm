Paso 3: Física del Módulo Lunar (Evento Crear)

Configura las variables para la Gravedad, Propulsión, Combustible y velocidad.

Inicializar Variables de Física

En obj_lander:

  1. Añadir Evento: Crear
  2. Añadir Acción: Control > Ejecutar Código
// Configuración de física
gravity_force = 0.05;  // Gravedad suave
thrust_force = 0.1;    // Potencia de Propulsión
max_speed = 5;         // Velocidad máxima

// Velocidad actual
hsp = 0;  // Velocidad horizontal
vsp = 0;  // Velocidad vertical

// Sistema de Combustible
fuel = 100;
fuel_use = 0.5;  // Combustible por fotograma

// Estado del juego
landed = false;
crashed = false;

// Velocidad segura de aterrizaje
safe_speed = 2;

Entendiendo las Variables

Variable Propósito
gravity_force Fuerza constante hacia abajo (0.05 es suave)
thrust_force Fuerza hacia arriba al presionar ARRIBA
fuel 100% de Combustible, disminuye al usar Propulsión
safe_speed Velocidad máxima para aterrizaje seguro (2)
Equilibrio: ¡La Propulsión (0.1) es más fuerte que la Gravedad (0.05) para que puedas frenar!
