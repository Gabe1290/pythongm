Paso 1: Crear Sprites

Necesitamos Sprites para el Módulo lunar, la Plataforma de aterrizaje y el terreno.

Nombre del Sprite Tamaño Descripción
spr_lander 32x32 Nave espacial (triángulo o forma clásica de módulo lunar)
spr_pad 64x16 Plataforma de aterrizaje con marca "H"
spr_ground 32x32 Terreno rocoso (gris/marrón)

Crear el Sprite del Módulo Lunar

  1. Clic derecho en Sprites > Crear Sprite
  2. Nómbralo spr_lander
  3. Haz clic en Editar Sprite
  4. Dibuja una nave espacial (32x32 píxeles)
  5. Establece el origen en centro-inferior para un aterrizaje correcto

Crear el Sprite de la Plataforma de Aterrizaje

  1. Crea un Sprite llamado spr_pad
  2. Tamaño: 64x16 píxeles
  3. Dibuja una plataforma plana con marcas
  4. Usa colores brillantes (amarillo/verde)

Crear el Sprite del Suelo

  1. Crea un Sprite llamado spr_ground
  2. Tamaño: 32x32 píxeles
  3. Dibuja terreno rocoso y áspero
  4. Usa colores gris/marrón
Importante: ¡Establece el origen del módulo lunar en centro-inferior para que aterrice correctamente en la plataforma!
Haz clic en "Siguiente" para crear los Objetos del juego.
