Paso 5: Detección de Colisión

Maneja el aterrizaje en la plataforma y el Choque contra el terreno.

Colisión con la Plataforma de Aterrizaje

En obj_lander:

  1. Añadir Evento: Colisión > obj_pad
  2. Añadir Acción: Control > Ejecutar Código
// Calcular velocidad total
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    // ¡Aterrizaje seguro!
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("¡Aterrizaje Perfecto! ¡Ganaste!");
} else {
    // Demasiado rápido - ¡Choque!
    crashed = true;
    show_message("¡Choque! ¡Demasiado rápido!");
    room_restart();
}

Colisión con el Suelo (Choque)

En obj_lander:

  1. Añadir Evento: Colisión > obj_ground
  2. Añadir Acción: Control > Ejecutar Código
crashed = true;
show_message("¡Choque contra el terreno!");
room_restart();

Cálculo de Velocidad

Usamos el teorema de Pitágoras para obtener la velocidad total:

total_speed = sqrt(hsp*hsp + vsp*vsp)

Esto combina la velocidad horizontal y vertical en un solo valor.

¡La velocidad segura es 2! Si total_speed > 2, te estrellas incluso en la plataforma.
