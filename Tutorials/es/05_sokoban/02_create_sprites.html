Paso 1: Crear los Sprites

Todos los sprites deben tener el mismo tamaño (32x32 píxeles) para crear una cuadrícula adecuada.

Crear el Sprite del Jugador

  1. Haz clic derecho en Sprites en el Árbol de Recursos
  2. Selecciona Crear Sprite
  3. Nómbralo spr_player
  4. Haz clic en Editar Sprite
  5. Dibuja un personaje simple (32x32 píxeles)
  6. Usa un color distintivo como azul
  7. Haz clic en OK para guardar

Crear los Sprites de la Caja

Caja Normal:

  1. Crea el sprite spr_crate
  2. Dibuja una forma de caja de madera (32x32 píxeles)
  3. Usa color marrón o naranja

Caja en el Objetivo:

  1. Crea el sprite spr_crate_ok
  2. Dibuja la misma caja pero con color verde
  3. Esto indica cuando una caja está correctamente colocada

Crear el Sprite de la Pared

  1. Crea el sprite spr_wall
  2. Dibuja un patrón sólido de ladrillo o piedra (32x32 píxeles)
  3. Usa color gris u oscuro

Crear el Sprite del Objetivo

  1. Crea el sprite spr_target
  2. Dibuja una marca X o indicador de meta (32x32 píxeles)
  3. Usa un color brillante como rojo
Importante: ¡Mantén TODOS los sprites exactamente en 32x32 píxeles para una alineación de cuadrícula adecuada!
Haz clic en "Siguiente" para crear los objetos del juego.
