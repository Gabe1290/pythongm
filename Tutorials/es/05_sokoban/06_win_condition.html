Paso 5: Condición de Victoria

El jugador gana cuando todas las cajas están en los objetivos.

Crear Controlador del Juego

  1. Crea el objeto obj_game_controller
  2. No se necesita sprite

Contar Objetivos (Evento Crear)

En obj_game_controller:

  1. Agregar Evento: Crear
  2. Agregar Acción: Control > Ejecutar Código
  3. Ingresa: global.total_targets = instance_number(obj_target);

Verificar Condición de Victoria (Evento Step)

  1. Agregar Evento: Step
  2. Agregar Acción: Control > Ejecutar Código
  3. Ingresa este código:
// Contar cajas en los objetivos
var crates_on_targets = 0;
with (obj_crate) {
    if (place_meeting(x, y, obj_target)) {
        crates_on_targets += 1;
    }
}

// Verificar si todos los objetivos tienen cajas
if (crates_on_targets >= global.total_targets
    && global.total_targets > 0) {
    show_message("¡Nivel Completado!");
    room_restart();
}

Dibujar Instrucciones (Evento Dibujar)

  1. Agregar Evento: Dibujar
  2. Agregar Acción: Dibujar > Dibujar Texto
  3. Texto: ¡Empuja las cajas hacia los objetivos!
  4. X: 10, Y: 10
Consejo: ¡Asegúrate de que el número de cajas sea igual al número de objetivos en tu nivel!
Haz clic en "Siguiente" para configurar la habitación del juego.
