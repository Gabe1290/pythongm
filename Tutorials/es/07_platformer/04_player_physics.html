Paso 3: Fisica del Jugador (Evento Crear)

Necesitamos configurar variables para la gravedad, el salto y el movimiento.

Inicializar Variables de Fisica

En obj_player:

  1. Agregar Evento: Crear
  2. Agregar Accion: Control > Ejecutar Codigo
// Configuracion de movimiento
hspeed_max = 4;      // Velocidad horizontal maxima
vspeed_max = 10;     // Velocidad maxima de caida
jump_force = -10;    // Fuerza de salto (negativo = arriba)
gravity_force = 0.5; // Que tan rapido caemos

// Velocidades actuales
hsp = 0;
vsp = 0;

// Estado
on_ground = false;

Entendiendo las Variables

Variable Proposito
hsp Velocidad horizontal (izquierda/derecha)
vsp Velocidad vertical (arriba/abajo)
gravity_force Fuerza constante que jala hacia abajo
jump_force Valor negativo para ir hacia arriba
on_ground Verdadero cuando esta parado en el suelo
Concepto Clave: La fuerza de salto es negativa porque Y aumenta hacia abajo en los juegos. Una velocidad Y negativa mueve al jugador hacia arriba!
Haz clic en "Siguiente" para agregar movimiento y manejo de colisiones.
