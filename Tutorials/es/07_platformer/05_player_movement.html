Paso 4: Movimiento del Jugador (Evento Step)

Este es el nucleo de nuestro juego de plataforma: movimiento, gravedad, salto y colision.

Agregar Evento Step

En obj_player:

  1. Agregar Evento: Step > Step
  2. Agregar Accion: Control > Ejecutar Codigo
// === MOVIMIENTO HORIZONTAL ===
var move_input = keyboard_check(vk_right) - keyboard_check(vk_left);
hsp = move_input * hspeed_max;

// === GRAVEDAD ===
vsp += gravity_force;
if (vsp > vspeed_max) vsp = vspeed_max;

// === VERIFICACION DE SUELO ===
on_ground = place_meeting(x, y + 1, obj_ground);

// === SALTO ===
if (on_ground && (keyboard_check_pressed(vk_up) || keyboard_check_pressed(vk_space))) {
    vsp = jump_force;
}

// === COLISION HORIZONTAL ===
if (place_meeting(x + hsp, y, obj_ground)) {
    while (!place_meeting(x + sign(hsp), y, obj_ground)) {
        x += sign(hsp);
    }
    hsp = 0;
}
x += hsp;

// === COLISION VERTICAL ===
if (place_meeting(x, y + vsp, obj_ground)) {
    while (!place_meeting(x, y + sign(vsp), obj_ground)) {
        y += sign(vsp);
    }
    vsp = 0;
}
y += vsp;

Como Funciona

  1. Entrada: Flecha derecha = 1, Izquierda = -1, Ninguna = 0
  2. Gravedad: Se suma a vsp cada frame (cayendo mas rapido)
  3. Verificacion de Suelo: Busca 1 pixel abajo por suelo
  4. Salto: Solo funciona cuando on_ground es verdadero
  5. Colision: Moverse pixel por pixel hasta chocar con pared
La Magia: El bucle while nos mueve pixel por pixel hasta el borde de la pared, evitando que nos quedemos atascados!
