Paso 2: Crear los Objetos

Ahora crearemos los objetos del juego. Cada objeto usa un sprite y puede tener comportamientos.

Crear el Objeto Pared

  1. Haz clic derecho en Objetos y selecciona Crear Objeto
  2. Nombralo obj_wall
  3. Establece el sprite a spr_wall
  4. Marca la casilla "Solido" - esto es importante!
  5. No se necesitan eventos - las paredes solo bloquean el movimiento

Crear el Objeto Pelota

  1. Crea un nuevo objeto llamado obj_ball
  2. Establece el sprite a spr_ball
  3. Deja "Solido" sin marcar

Crear los Objetos Paleta

Paleta Izquierda:

  1. Crea el objeto obj_paddle_left
  2. Establece el sprite a spr_paddle_left
  3. Marca "Solido"

Paleta Derecha:

  1. Crea el objeto obj_paddle_right
  2. Establece el sprite a spr_paddle_right
  3. Marca "Solido"
Importante: La casilla "Solido" es crucial para que el rebote funcione correctamente!
Haz clic en "Siguiente" para agregar controles de movimiento a las paletas.
