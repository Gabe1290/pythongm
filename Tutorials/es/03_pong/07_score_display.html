Paso 6: Mostrar las Puntuaciones

Necesitamos un objeto controlador para inicializar las puntuaciones y mostrarlas.

Crear Objeto Controlador de Puntuacion

  1. Crea un nuevo objeto llamado obj_score
  2. No se necesita sprite (dejalo vacio)

Inicializar Puntuaciones (Evento Crear)

  1. Agrega Evento: Crear
  2. Agrega Accion: Puntuacion > Establecer Variable
  3. Variable: global.p1score, Valor: 0
  4. Agrega otra accion Establecer Variable
  5. Variable: global.p2score, Valor: 0

Dibujar Puntuaciones (Evento Dibujar)

  1. Agrega Evento: Dibujar
  2. Agrega Accion: Dibujar > Dibujar Texto
  3. Texto: Jugador 1:
  4. X: 10, Y: 10
  1. Agrega Accion: Dibujar > Dibujar Variable
  2. Variable: global.p1score
  3. X: 100, Y: 10
  1. Agrega Accion: Dibujar > Dibujar Texto
  2. Texto: Jugador 2:
  3. X: 10, Y: 30
  1. Agrega Accion: Dibujar > Dibujar Variable
  2. Variable: global.p2score
  3. X: 100, Y: 30
Recuerda: No olvides colocar obj_score en tu habitacion!
