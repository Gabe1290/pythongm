Paso 2: Crear los Objetos

Ahora crearemos todos los objetos del juego.

Crear el Objeto Pared

  1. Haz clic derecho en Objetos y selecciona Crear Objeto
  2. Nómbralo obj_wall
  3. Establece el sprite a spr_wall
  4. Marca "Sólido"
  5. No se necesitan eventos

Crear el Objeto Padre de Ladrillos

Usaremos un objeto padre para que todos los ladrillos compartan el mismo comportamiento de colisión.

  1. Crea el objeto obj_brick_parent
  2. No se necesita sprite (es solo una plantilla)
  3. Marca "Sólido"

Crear los Objetos de Ladrillos de Colores

Para cada color de ladrillo:

  1. Crea obj_brick_red con spr_brick_red
  2. Establece Padre a obj_brick_parent
  3. Marca "Sólido"

Repite para obj_brick_orange, obj_brick_yellow y obj_brick_green.

Crear el Objeto Paleta

  1. Crea el objeto obj_paddle
  2. Establece el sprite a spr_paddle
  3. Marca "Sólido"

Crear el Objeto Pelota

  1. Crea el objeto obj_ball
  2. Establece el sprite a spr_ball
  3. Deja "Sólido" desmarcado
Objetos Padre: Cuando la pelota colisiona con obj_brick_parent, ¡detectará colisiones con TODOS los tipos de ladrillos!
Haz clic en "Siguiente" para agregar el movimiento de la paleta.
