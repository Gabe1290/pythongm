Proximos pasos

Ahora que conoces lo basico, aqui hay algunos temas para explorar:

Agregar movimiento

Abre tu objeto y agrega un evento de Teclado. Usa acciones como "Mover en direccion" o "Establecer velocidad" para hacer que tu objeto responda a las pulsaciones de teclas.

Deteccion de colisiones

Crea otro objeto (como una pared) y agrega un evento de Colision a tu jugador. Usa acciones para detener el movimiento o cambiar de direccion cuando los objetos colisionen.

Programacion visual con Blockly

PyGameMaker soporta programacion visual con Blockly. En el Editor de Objetos, cambia al Modo Blockly para crear comportamientos conectando bloques en lugar de escribir codigo.

Multiples habitaciones

Crea mas habitaciones para diferentes niveles. Usa la accion "Ir a habitacion" para cambiar entre ellas cuando el jugador alcance una meta o puerta.

Exportar tu juego

Cuando tu juego este listo, ve a Archivo > Exportar como HTML5 para crear una version web que se pueda jugar en cualquier navegador.
