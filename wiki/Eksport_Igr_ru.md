Узнайте, как экспортировать ваши игры pyGM для различных платформ.

## Обзор

pyGM позволяет экспорт для:
- Windows (EXE)
- macOS (APP)
- Linux
- Веб (HTML5)

## Подготовка к экспорту

### Перед экспортом проверьте
1. **Все ресурсы на месте**: Спрайты, звуки и т.д.
2. **Игра протестирована**: Без критических ошибок
3. **Настройки оптимизированы**: Разрешение, полноэкранный режим

### Настройки проекта
- **Название игры**: Отображаемое название
- **Версия**: Номер версии
- **Иконка**: Иконка приложения
- **Начальный экран**: Splash Screen

## Экспорт для Windows

### Предварительные требования
- pyinstaller установлен
- Система Windows или кросс-компиляция

### Шаги
1. Перейдите в Файл > Экспорт
2. Выберите "Windows Executable"
3. Настройте опции:
   - Файл иконки
   - Скрыть окно консоли
   - Один EXE файл
4. Нажмите "Экспорт"

### Выход
- Один EXE файл
- Или папка с зависимостями

## Экспорт для macOS

### Предварительные требования
- Система macOS рекомендована
- py2app или pyinstaller

### Шаги
1. Файл > Экспорт > macOS
2. Введите название App Bundle
3. Выберите иконку (формат ICNS)
4. Экспорт

## Экспорт для Linux

### Опции
- AppImage (рекомендуется)
- Пакет Debian
- Исполняемый файл

### Шаги
1. Файл > Экспорт > Linux
2. Выберите формат
3. Экспорт

## Веб-экспорт (HTML5)

### Преимущества
- Работает в браузере
- Легко делиться
- Не нужна установка

### Шаги
1. Файл > Экспорт > Веб
2. Настройте:
   - Размер холста
   - Экран загрузки
   - Сжатие
3. Экспорт

### Выход
- HTML файл
- JavaScript файлы
- Папка ресурсов

### Хостинг
- Загрузите на веб-сервер
- Используйте itch.io
- Используйте GitHub Pages

## Опции экспорта

### Общие
- **Сжатие**: Уменьшить размер файла
- **Режим отладки**: Оставить для тестов
- **Встроить ресурсы**: Всё в один файл

### Специфические для платформы
- **Windows**: UAC манифест
- **macOS**: Подписывание кода
- **Веб**: Версия WebGL

## Решение проблем

### Распространённые проблемы

**Экспорт не удаётся**
- Проверьте сообщения об ошибках
- Обновите pyinstaller

**Отсутствуют ресурсы**
- Проверьте пути
- Повторно включите ресурсы

**Игра не запускается**
- Тестируйте в режиме отладки
- Проверьте зависимости

## Оптимизация

1. **Оптимизируйте размеры изображений**: Сожмите спрайты
2. **Сожмите аудио**: MP3 вместо WAV
3. **Удалите неиспользуемые ресурсы**
4. **Оптимизируйте код**: Улучшите эффективность
