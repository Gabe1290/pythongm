## Вступ

У цьому навчанні ви створите класичну гру **Pong** - одну з перших відеоігор, коли-небудь створених! Pong - це гра для двох гравців, де кожен гравець керує ракеткою і намагаєтьсяударити м'яч повз ракетку супротивника, щоб набрати очки.

**Що ви дізнаєтесь:**
- Створення спрайтів для ракеток, м'яча та стін
- Обробка введення з клавіатури для двох гравців
- Відскоки об'єктів один від одного
- Відстеження та відображення очок обох гравців
- Використання глобальних змінних

**Рівень складності:** Для початківців
**Набір:** Beginner Preset

---

## Крок 1: Планування гри

Перед тим як розпочати, давайте розберемось, що нам потрібно:

| Елемент | Призначення |
|---------|---------|
| **М'яч** | Відскакує між гравцями |
| **Ліва ракетка** | Гравець 1 керує клавішами W/S |
| **Права ракетка** | Гравець 2 керує стрілками вгору/вниз |
| **Стіни** | Верхня та нижня межі гри |
| **Цілі** | Невидимі області позаду кожної ракетки для виявлення забитих голів |
| **Відображення очок** | Показує очки обох гравців |

---

## Крок 2: Створення спрайтів

### 2.1 Спрайт м'яча

1. У **Дереві ресурсів** клацніть правою кнопкою миші на **Спрайти** і виберіть **Створити спрайт**
2. Назвіть його `spr_ball`
3. Клацніть **Редагувати спрайт**, щоб відкрити редактор спрайтів
4. Намалюйте невелике біле коло (близько 16х16 пікселів)
5. Клацніть **ОК**, щоб зберегти

### 2.2 Спрайти ракеток

Ми створимо дві ракетки - по одній для кожного гравця:

**Ліва ракетка (Гравець 1):**
1. Створіть новий спрайт з ім'ям `spr_paddle_left`
2. Намалюйте довгий тонкий прямокутник вигнутої форми, як дужка ")" - рекомендується синій колір
3. Розмір: приблизно 16х64 пікселів

**Права ракетка (Гравець 2):**
1. Створіть новий спрайт з ім'ям `spr_paddle_right`
2. Намалюйте довгий тонкий прямокутник вигнутої форми, як дужка "(" - рекомендується червоний колір
3. Розмір: приблизно 16х64 пікселів

### 2.3 Спрайт стіни

1. Створіть новий спрайт з ім'ям `spr_wall`
2. Намалюйте суцільний прямокутник (сірий або білий)
3. Розмір: 32х32 пікселів (ми розтягнемо його в кімнаті)

### 2.4 Спрайт цілі (невидимий)

1. Створіть новий спрайт з ім'ям `spr_goal`
2. Розмір: 32х32 пікселів
3. Залиште його прозорим або зробіть суцільним кольором (він буде невидимим у грі)

---

## Крок 3: Створення об'єкту стіни

Об'єкт стіни створює межі вгорі та внизу ігрової площі.

1. Клацніть правою кнопкою миші на **Об'єкти** і виберіть **Створити об'єкт**
2. Назвіть його `obj_wall`
3. Встановіть спрайт на `spr_wall`
4. **Встановіть прапорець "Solid"** - це важливо для відскоків!
5. Жодних подій не потрібні - стіна просто перебуває там

---

## Крок 4: Створення об'єктів ракеток

### 4.1 Ліва ракетка (Гравець 1)

1. Створіть новий об'єкт з ім'ям `obj_paddle_left`
2. Встановіть спрайт на `spr_paddle_left`
3. **Встановіть прапорець "Solid"**

**Додавання подій клавіатури для руху:**

**Подія: Натиск W**
1. Додайте подію → Клавіатура → Натиск W
2. Додайте дію: **Рух** → **Встановити вертикальну швидкість**
3. Встановіть вертикальну швидкість на `-8` (рух вгору)

**Подія: Відпускання W**
1. Додайте подію → Клавіатура → Відпускання W
2. Додайте дію: **Рух** → **Встановити вертикальну швидкість**
3. Встановіть вертикальну швидкість на `0` (зупинка руху)

**Подія: Натиск S**
1. Додайте подію → Клавіатура → Натиск S
2. Додайте дію: **Рух** → **Встановити вертикальну швидкість**
3. Встановіть вертикальну швидкість на `8` (рух вниз)

**Подія: Відпускання S**
1. Додайте подію → Клавіатура → Відпускання S
2. Додайте дію: **Рух** → **Встановити вертикальну швидкість**
3. Встановіть вертикальну швидкість на `0` (зупинка руху)

**Подія: Зіткнення з obj_wall**
1. Додайте подію → Зіткнення → obj_wall
2. Додайте дію: **Рух** → **Відскочити від об'єктів**
3. Виберіть "Проти твердих об'єктів"

### 4.2 Права ракетка (Гравець 2)

1. Створіть новий об'єкт з ім'ям `obj_paddle_right`
2. Встановіть спрайт на `spr_paddle_right`
3. **Встановіть прапорець "Solid"**

**Додавання подій клавіатури для руху:**

**Подія: Натиск стрілки вгору**
1. Додайте подію → Клавіатура → Натиск вгору
2. Додайте дію: **Рух** → **Встановити вертикальну швидкість**
3. Встановіть вертикальну швидкість на `-8`

**Подія: Відпускання стрілки вгору**
1. Додайте подію → Клавіатура → Відпускання вгору
2. Додайте дію: **Рух** → **Встановити вертикальну швидкість**
3. Встановіть вертикальну швидкість на `0`

**Подія: Натиск стрілки вниз**
1. Додайте подію → Клавіатура → Натиск вниз
2. Додайте дію: **Рух** → **Встановити вертикальну швидкість**
3. Встановіть вертикальну швидкість на `8`

**Подія: Відпускання стрілки вниз**
1. Додайте подію → Клавіатура → Відпускання вниз
2. Додайте дію: **Рух** → **Встановити вертикальну швидкість**
3. Встановіть вертикальну швидкість на `0`

**Подія: Зіткнення з obj_wall**
1. Додайте подію → Зіткнення → obj_wall
2. Додайте дію: **Рух** → **Відскочити від об'єктів**
3. Виберіть "Проти твердих об'єктів"

## Крок 5: Створення об'єкту м'яча

1. Створіть новий об'єкт з ім'ям `obj_ball`
2. Встановіть спрайт на `spr_ball`

**Подія: Створення**
1. Додайте подію → Створення
2. Додайте дію: **Рух** → **Почати рух у напрямку**
3. Виберіть діагональний напрямок (не прямо вгору або вниз)
4. Встановіть швидкість на `6`

**Подія: Зіткнення з obj_paddle_left**
1. Додайте подію → Зіткнення → obj_paddle_left
2. Додайте дію: **Рух** → **Відскочити від об'єктів**
3. Виберіть "Проти твердих об'єктів"

**Подія: Зіткнення з obj_paddle_right**
1. Додайте подію → Зіткнення → obj_paddle_right
2. Додайте дію: **Рух** → **Відскочити від об'єктів**
3. Виберіть "Проти твердих об'єктів"

**Подія: Зіткнення з obj_wall**
1. Додайте подію → Зіткнення → obj_wall
2. Додайте дію: **Рух** → **Відскочити від об'єктів**
3. Виберіть "Проти твердих об'єктів"

---

## Крок 6: Створення об'єктів цілей

Цілі - це невидимі області позаду кожної ракетки. Коли м'яч потрапляє в ціль, супротивник набирає очко.

### 6.1 Ліва ціль

1. Створіть новий об'єкт з ім'ям `obj_goal_left`
2. Встановіть спрайт на `spr_goal`
3. **Зніміть прапорець "Видимий"** - ціль повинна бути невидимою
4. **Встановіть прапорець "Solid"**

### 6.2 Права ціль

1. Створіть новий об'єкт з ім'ям `obj_goal_right`
2. Встановіть спрайт на `spr_goal`
3. **Зніміть прапорець "Видимий"**
4. **Встановіть прапорець "Solid"**

### 6.3 Додавання подій зіткнення цілей до м'яча

Повертаємось до `obj_ball` і додаємо ці події:

**Подія: Зіткнення з obj_goal_left**
1. Додайте подію → Зіткнення → obj_goal_left
2. Додайте дію: **Рух** → **Перейти до початкової позиції** (перезагружає м'яч)
3. Додайте дію: **Очко** → **Встановити очко**
   - Змінна: `global.p2score`
   - Значення: `1`
   - Встановіть прапорець "Відносне" (додає 1 до поточного очка)

**Подія: Зіткнення з obj_goal_right**
1. Додайте подію → Зіткнення → obj_goal_right
2. Додайте дію: **Рух** → **Перейти до початкової позиції**
3. Додайте дію: **Очко** → **Встановити очко**
   - Змінна: `global.p1score`
   - Значення: `1`
   - Встановіть прапорець "Відносне"

## Крок 7: Створення об'єкту для відображення очок

1. Створіть новий об'єкт з ім'ям `obj_score`
2. Спрайт не потрібен

**Подія: Створення**
1. Додайте подію → Створення
2. Додайте дію: **Очко** → **Встановити очко**
   - Змінна: `global.p1score`
   - Значення: `0`
3. Додайте дію: **Очко** → **Встановити очко**
   - Змінна: `global.p2score`
   - Значення: `0`

**Подія: Малювання**
1. Додайте подію → Малювання
2. Додайте дію: **Малювання** → **Намалювати текст**
   - Текст: `Гравець 1:`
   - X: `10`
   - Y: `10`
3. Додайте дію: **Малювання** → **Намалювати змінну**
   - Змінна: `global.p1score`
   - X: `100`
   - Y: `10`
4. Додайте дію: **Малювання** → **Намалювати текст**
   - Текст: `Гравець 2:`
   - X: `10`
   - Y: `30`
5. Додайте дію: **Малювання** → **Намалювати змінну**
   - Змінна: `global.p2score`
   - X: `100`
   - Y: `30`

## Крок 8: Дизайн кімнати

1. Клацніть правою кнопкою миші на **Кімнати** і виберіть **Створити кімнату**
2. Назвіть її `room_pong`
3. Встановіть розмір кімнати (наприклад, 640х480)

**Розміщення об'єктів:**

1. **Стіни**: Розмістіть екземпляри `obj_wall` по верхньому і нижньому краю кімнати
2. **Ліва ракетка**: Розмістіть `obj_paddle_left` біля лівого краю, по центру вертикально
3. **Права ракетка**: Розмістіть `obj_paddle_right` біля правого краю, по центру вертикально
4. **М'яч**: Розмістіть `obj_ball` у центрі кімнати
5. **Цілі**:
   - Розмістіть екземпляри `obj_goal_left` по лівому краю (позаду, де розташована ракетка)
   - Розмістіть екземпляри `obj_goal_right` по правому краю
6. **Відображення очок**: Розмістіть `obj_score` де-небудь (у неї немає спрайту, вона просто малює текст)

**Приклад макету кімнати:**
```
[СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА]
[ЦІЛЬ]  [РАКЕТКА_Л]            [М'ЯЧ]            [РАКЕТКА_П]  [ЦІЛЬ]
[ЦІЛЬ]  [РАКЕТКА_Л]                              [РАКЕТКА_П]  [ЦІЛЬ]
[ЦІЛЬ]                                                      [ЦІЛЬ]
[СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА СТІНА]
```

## Крок 9: Тестування гри!

1. Клацніть **Запустити** або натисніть **F5**, щоб протестувати гру
2. Гравець 1 використовує **W** (вгору) і **S** (вниз)
3. Гравець 2 використовує **стрілку вгору** та **стрілку вниз**
4. Спробуйте ударити м'яч повз ракетку супротивника!

## Вдосконалення (додатково)

### Збільшення швидкості
Зробіть м'яч швидшим кожного разу, коли він вдаряється об ракетку, додавши до подій зіткнення:
- Після дії відскоку додайте **Рух** → **Встановити швидкість**
- Встановіть швидкість на `speed + 0.5` з прапорцем "Відносне" встановленим

### Звукові ефекти
Додайте звуки коли:
- М'яч вдаряється об ракетку
- М'яч вдаряється об стіну
- Гравець забиває гол

### Умова перемоги
Додайте перевірку в подію малювання:
- Якщо `global.p1score >= 10`, відобразіть "Гравець 1 переміг!"
- Якщо `global.p2score >= 10`, відобразіть "Гравець 2 переміг!"

## Усунення неполадок

| Проблема | Рішення |
|---------|----------|
| М'яч проходить крізь ракетку | Переконайтесь, що ракетки мають встановлений прапорець "Solid" |
| Ракетка не зупиняється у стін | Додайте подію зіткнення з obj_wall |
| Очко не оновлюється | Перевірте, що назви змінних збігаються точно (global.p1score, global.p2score) |
| М'яч не рухається | Перевірте, що подія створення має дію руху |
