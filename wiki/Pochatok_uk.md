## Встановлення

### Передумови
- Python 3.8 або новіший
- pip (менеджер пакетів Python)

### Кроки встановлення

1. **Встановіть pyGM**
   ```bash
   pip install pygm
   ```

2. **Перевірте встановлення**
   ```bash
   pygm --version
   ```

## Запуск pyGM

Після встановлення ви можете запустити pyGM:

```bash
pygm
```

Це відкриє візуальний редактор, де ви можете почати створювати свою гру.

## Інтерфейс користувача

### Основні області
- **Ліва панель**: Управління ресурсами (спрайти, об'єкти, кімнати)
- **Центральна область**: Робоча область редактора
- **Права панель**: Властивості та налаштування

### Базові елементи керування
- **Коліщатко миші**: Масштабування виду
- **Середня кнопка миші**: Переміщення виду
- **Лівий клік**: Вибір/розміщення елементів
- **Правий клік**: Контекстне меню
