## Вступ

У цьому посібнику ви створите **Гру Місячний Посадковий Модуль** - класичну аркадну гру, де ви керуєте космічним кораблем, що спускається на посадкову платформу. Ви повинні керувати тягою, щоб протидіяти гравітації та м'яко приземлитися без аварії. Ця гра ідеально підходить для вивчення фізичних концепцій, таких як гравітація, тяга, швидкість та управління паливом.

**Що ви навчитесь:**
- Фізика гравітації та тяги
- Виявлення посадки на основі швидкості
- Система управління паливом
- Контроль обертання або напрямку
- Безпечні зони посадки

**Складність:** Початківець
**Пресет:** Пресет для Початківців

## Крок 1: Зрозуміти Гру

### Механіки Гри
1. Модуль притягується вниз гравітацією
2. Натискання ВГОРУ застосовує тягу вгору (використовує паливо)
3. ВЛІВО/ВПРАВО керує обертанням або рухом
4. Приземліться м'яко на платформу, щоб виграти
5. Аварія, якщо приземлитесь занадто швидко або промахнетесь
6. Без палива ви не можете сповільнитись!

### Що Нам Потрібно

| Елемент | Призначення |
|---------|-------------|
| **Модуль** | Корабель, яким ви керуєте |
| **Платформа** | Безпечна зона для посадки |
| **Земля** | Місцевість, що спричиняє аварію |
| **Дисплей Палива** | Показує залишок палива |
| **Дисплей Швидкості** | Показує поточну швидкість |

## Крок 5: Створити Об'єкт Модуля

### Подія Create
```gml
gravity_force = 0.05;
thrust_force = 0.1;
max_speed = 5;
hsp = 0;
vsp = 0;
fuel = 100;
fuel_use = 0.5;
landed = false;
crashed = false;
safe_speed = 2;
```

### Подія Step
```gml
if (landed || crashed) exit;

vsp += gravity_force;

if (keyboard_check(vk_up) && fuel > 0) {
    vsp -= thrust_force;
    fuel -= fuel_use;
    if (fuel < 0) fuel = 0;
}

if (keyboard_check(vk_left)) hsp -= 0.05;
if (keyboard_check(vk_right)) hsp += 0.05;

hsp = clamp(hsp, -max_speed, max_speed);
vsp = clamp(vsp, -max_speed, max_speed);

x += hsp;
y += vsp;

if (x < 16) { x = 16; hsp = 0; }
if (x > room_width - 16) { x = room_width - 16; hsp = 0; }
if (y < 16) { y = 16; vsp = 0; }
```

### Зіткнення з obj_pad
```gml
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("Ідеальна Посадка! Ви Виграли!");
} else {
    crashed = true;
    show_message("Аварія! Занадто швидко!");
    room_restart();
}
```

### Зіткнення з obj_ground
```gml
crashed = true;
show_message("Аварія на місцевості!");
room_restart();
```
