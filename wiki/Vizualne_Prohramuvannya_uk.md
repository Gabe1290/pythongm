pyGM пропонує систему візуального програмування для легкої розробки ігор без коду.

## Огляд

З візуальним програмуванням ви можете:
- Створювати ігрову логіку методом перетягування
- З'єднувати блоки для складної поведінки
- Розробляти без знань програмування

## Редактор Blockly

### Інтерфейс
1. **Палітра блоків**: Доступні блоки за категоріями
2. **Робоча область**: Тут ви з'єднуєте блоки
3. **Панель інструментів**: Зберегти, Завантажити, Видалити

### Категорії блоків
- **Логіка**: Якщо/Тоді, порівняння, логічні значення
- **Цикли**: Повторення
- **Математика**: Обчислення
- **Текст**: Операції з текстом
- **Змінні**: Зберігання значень
- **Функції**: Повторно використовувані блоки
- **Гра**: Специфічні дії pyGM

## Використання блоків

### Додавання блока
1. Клацніть на категорію
2. Перетягніть блок на робочу область
3. З'єднайте його з іншими блоками

### З'єднання блоків
- Блоки автоматично з'єднуються
- Звертайте увагу на відповідні форми
- Вкладення блоків можливе

### Налаштування блока
- Заповніть поля введення
- Виберіть опції з випадаючого меню
- Вставте підблоки

## Приклади

### Простий рух
```
Коли [стрілка вправо] натиснута
  Встановити x на (x + 5)
```

### Умовна логіка
```
Якщо <Життя <= 0> тоді
  Показати повідомлення "Game Over"
  Перейти до кімнати [rm_gameover]
```

### Цикл
```
Повторити [10] разів
  Створити екземпляр [obj_moneta] на позиції (Випадково 0-800, Випадково 0-600)
```

## Ігрові блоки

### Рух
- **Перемістити до**: Рух до позиції
- **Встановити швидкість**: Швидкість руху
- **Встановити напрямок**: Напрямок руху

### Екземпляри
- **Створити екземпляр**: Створити новий об'єкт
- **Знищити**: Видалити об'єкт
- **Для всіх**: Всі екземпляри типу

### Змінні
- **Встановити змінну**: Зберегти значення
- **Змінити змінну**: Змінити значення
- **Отримати змінну**: Отримати значення

### Події
- **Коли клавіша**: Введення з клавіатури
- **Коли зіткнення**: Контакт об'єктів
- **Коли таймер**: На основі часу

## Поради

1. **Починайте з малого**: Спочатку прості проекти
2. **Тестуйте**: Регулярно запускайте
3. **Організуйте**: Логічно групуйте блоки
4. **Коментарі**: Додавайте примітки

## Від блоків до коду

Редактор Blockly також може генерувати код:
1. Вивчайте концепції програмування візуально
2. Переглядайте згенерований код
3. Переходьте на Python пізніше
