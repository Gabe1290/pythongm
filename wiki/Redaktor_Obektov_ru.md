Редактор объектов является центральным инструментом для определения поведения игровых элементов.

## Обзор

Объекты являются строительными блоками вашей игры. Они определяют:
- Внешний вид (Спрайт)
- Поведение (События и действия)
- Физические свойства
- Взаимодействия

## Интерфейс редактора

### Основные области
1. **Список объектов**: Все объекты в проекте
2. **Панель свойств**: Базовые настройки
3. **Список событий**: Определённые события
4. **Редактор действий**: Действия для событий

## Свойства объекта

### Общие
- **Название**: Уникальный идентификатор (напр. obj_igrok)
- **Спрайт**: Назначенная графика
- **Видимый**: Отображается ли объект
- **Постоянный**: Сохраняется при смене комнат

### Физика
- **Твёрдый**: Сталкивается с другими объектами
- **Глубина**: Порядок рисования
- **Родительский объект**: Наследование свойств

## Работа с событиями

### Добавление события
1. Нажмите "Добавить событие"
2. Выберите тип события
3. Добавьте действия

### Типы событий
- **Create**: При создании экземпляра
- **Step**: Каждый кадр
- **Draw**: Для рисования
- **Клавиатура**: Ввод с клавиатуры
- **Мышь**: Взаимодействия с мышью
- **Столкновение**: При касании других объектов

## Использование действий

### Добавление действий
1. Выберите событие
2. Перетащите действия из библиотеки
3. Настройте параметры

### Распространённые действия
- Движение в направлении
- Установить переменную
- Создать/уничтожить экземпляр
- Воспроизвести звук
- Сменить комнату

## Лучшие практики

1. **Чёткие названия**: Используйте префиксы как "obj_"
2. **Модульность**: Маленькие, повторно используемые объекты
3. **Используйте наследование**: Родительские объекты для общего поведения
4. **Документация**: Комментарии в сложных событиях
