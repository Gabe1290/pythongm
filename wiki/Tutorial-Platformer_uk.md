## Вступ

У цьому посібнику ви створите **Гру-Платформер** - екшн-гру з боковою прокруткою, де гравець бігає, стрибає та переміщується платформами, уникаючи небезпек та збираючи монети. Цей класичний жанр ідеально підходить для вивчення гравітації, механіки стрибків та зіткнень з платформами.

**Що ви навчитесь:**
- Гравітація та фізика падіння
- Механіка стрибків з виявленням землі
- Зіткнення з платформами (приземлення зверху)
- Рух вліво/вправо
- Предмети для збирання та небезпеки

**Складність:** Початківець
**Пресет:** Пресет для Початківців

## Крок 1: Зрозуміти Гру

### Механіки Гри
1. На гравця діє гравітація, він падає вниз
2. Гравець може рухатися вліво та вправо
3. Гравець може стрибати, коли стоїть на землі
4. Платформи не дають гравцю провалитися
5. Збирайте монети для очок
6. Досягніть прапора, щоб завершити рівень

### Що Нам Потрібно

| Елемент | Призначення |
|---------|-------------|
| **Гравець** | Персонаж, яким ви керуєте |
| **Земля/Платформа** | Тверді поверхні для стояння |
| **Монета** | Предмети для збирання та очок |
| **Шип** | Небезпека, що шкодить гравцю |
| **Прапор** | Ціль, що завершує рівень |

## Крок 2: Створити Спрайти

### Спрайти
- `spr_player` (32x48 пікселів) - простий персонаж
- `spr_ground` (32x32 пікселі) - плитка трави/землі
- `spr_platform` (64x16 пікселів) - плаваюча платформа
- `spr_coin` (16x16 пікселів) - золоте коло
- `spr_spike` (32x32 пікселі) - трикутні шипи
- `spr_flag` (32x64 пікселі) - прапор на палиці

## Крок 5: Створити Об'єкт Гравця

### Подія Create
```gml
hspeed_max = 4;
vspeed_max = 10;
jump_force = -10;
gravity_force = 0.5;
hsp = 0;
vsp = 0;
on_ground = false;
```

### Подія Step
```gml
var move_input = keyboard_check(vk_right) - keyboard_check(vk_left);
hsp = move_input * hspeed_max;

vsp += gravity_force;
if (vsp > vspeed_max) vsp = vspeed_max;

on_ground = place_meeting(x, y + 1, obj_ground);

if (on_ground && (keyboard_check_pressed(vk_up) || keyboard_check_pressed(vk_space))) {
    vsp = jump_force;
}

if (place_meeting(x + hsp, y, obj_ground)) {
    while (!place_meeting(x + sign(hsp), y, obj_ground)) x += sign(hsp);
    hsp = 0;
}
x += hsp;

if (place_meeting(x, y + vsp, obj_ground)) {
    while (!place_meeting(x, y + sign(vsp), obj_ground)) y += sign(vsp);
    vsp = 0;
}
y += vsp;
```

## Крок 6-8: Предмети та Небезпеки

**obj_coin** - Зіткнення з obj_player: Рахунок +10, знищити Self

**obj_spike** - Зіткнення з obj_player: Показати повідомлення, перезапустити кімнату

**obj_flag** - Зіткнення з obj_player: Показати повідомлення, наступна кімната

## Крок 9: Спроектуйте Свій Рівень

1. Створіть `room_level1` (800x480)
