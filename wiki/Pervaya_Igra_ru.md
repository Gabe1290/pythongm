В этом руководстве вы создадите простую игру, чтобы изучить основы pyGM.

## Обзор

Мы создадим простую игру с:
- Персонажем игрока, который может двигаться
- Предметом для сбора
- Системой очков

## Шаг 1: Создайте новый проект

1. Запустите pyGM
2. Выберите "Новый проект"
3. Введите название проекта
4. Выберите место сохранения

## Шаг 2: Создайте спрайты

### Спрайт игрока
1. Правый клик на "Спрайты" в дереве ресурсов
2. Выберите "Новый спрайт"
3. Используйте встроенный редактор или импортируйте изображение
4. Назовите его "spr_igrok"

### Спрайт предмета для сбора
1. Создайте ещё один спрайт
2. Назовите его "spr_moneta"

## Шаг 3: Создайте объекты

### Объект игрока
1. Правый клик на "Объекты"
2. Выберите "Новый объект"
3. Назовите его "obj_igrok"
4. Назначьте "spr_igrok" как спрайт

### Добавьте движение
1. Добавьте событие "Нажатие клавиши"
2. Используйте действия для движения:
   - Стрелка вверх: Движение вверх
   - Стрелка вниз: Движение вниз
   - Стрелка влево: Движение влево
   - Стрелка вправо: Движение вправо

### Объект монеты
1. Создайте "obj_moneta"
2. Назначьте "spr_moneta"
3. Добавьте событие столкновения с игроком
4. Действие: Уничтожить экземпляр и добавить очки

## Шаг 4: Создайте комнату

1. Правый клик на "Комнаты"
2. Выберите "Новая комната"
3. Назовите её "rm_uroven1"
4. Разместите объекты:
   - Одного игрока
   - Несколько монет

## Шаг 5: Протестируйте игру

1. Нажмите "Запустить игру" или нажмите F5
2. Протестируйте движение
3. Собирайте монеты

## Идеи для расширения

- Добавьте препятствия
- Реализуйте систему времени
- Создайте разные уровни
- Добавьте звуковые эффекты
