## Введение

В этом руководстве вы создадите **Игру Лунный Посадочный Модуль** - классическую аркадную игру, где вы управляете космическим кораблём, спускающимся на посадочную платформу. Вы должны управлять тягой, чтобы противодействовать гравитации и мягко приземлиться без аварии. Эта игра идеально подходит для изучения физических концепций, таких как гравитация, тяга, скорость и управление топливом.

**Что вы узнаете:**
- Физика гравитации и тяги
- Обнаружение посадки на основе скорости
- Система управления топливом
- Контроль вращения или направления
- Безопасные зоны посадки

**Сложность:** Начинающий
**Пресет:** Пресет для Начинающих

---

## Шаг 1: Понять Игру

### Механики Игры
1. Модуль притягивается вниз гравитацией
2. Нажатие ВВЕРХ применяет тягу вверх (использует топливо)
3. ВЛЕВО/ВПРАВО управляет вращением или движением
4. Приземлитесь мягко на платформу, чтобы выиграть
5. Авария, если приземлитесь слишком быстро или промахнётесь
6. Без топлива вы не можете замедлиться!

### Что Нам Нужно

| Элемент | Назначение |
|---------|------------|
| **Модуль** | Корабль, которым вы управляете |
| **Платформа** | Безопасная зона для посадки |
| **Земля** | Местность, вызывающая аварию |
| **Дисплей Топлива** | Показывает остаток топлива |
| **Дисплей Скорости** | Показывает текущую скорость |

## Шаг 5: Создать Объект Модуля

### Событие Create
```gml
gravity_force = 0.05;
thrust_force = 0.1;
max_speed = 5;
hsp = 0;
vsp = 0;
fuel = 100;
fuel_use = 0.5;
landed = false;
crashed = false;
safe_speed = 2;
```

### Событие Step
```gml
if (landed || crashed) exit;

vsp += gravity_force;

if (keyboard_check(vk_up) && fuel > 0) {
    vsp -= thrust_force;
    fuel -= fuel_use;
    if (fuel < 0) fuel = 0;
}

if (keyboard_check(vk_left)) hsp -= 0.05;
if (keyboard_check(vk_right)) hsp += 0.05;

hsp = clamp(hsp, -max_speed, max_speed);
vsp = clamp(vsp, -max_speed, max_speed);

x += hsp;
y += vsp;

if (x < 16) { x = 16; hsp = 0; }
if (x > room_width - 16) { x = room_width - 16; hsp = 0; }
if (y < 16) { y = 16; vsp = 0; }
```

### Столкновение с obj_pad
```gml
var total_speed = sqrt(hsp*hsp + vsp*vsp);

if (total_speed <= safe_speed) {
    landed = true;
    hsp = 0;
    vsp = 0;
    show_message("Идеальная Посадка! Вы Выиграли!");
} else {
    crashed = true;
    show_message("Авария! Слишком быстро!");
    room_restart();
}
```

### Столкновение с obj_ground
```gml
crashed = true;
show_message("Авария на местности!");
room_restart();
```
