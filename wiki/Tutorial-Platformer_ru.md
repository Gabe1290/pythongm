## Введение

В этом руководстве вы создадите **Игру-Платформер** - экшн-игру с боковой прокруткой, где игрок бегает, прыгает и перемещается по платформам, избегая опасностей и собирая монеты. Этот классический жанр идеально подходит для изучения гравитации, механики прыжков и столкновений с платформами.

**Что вы узнаете:**
- Гравитация и физика падения
- Механика прыжков с обнаружением земли
- Столкновение с платформами (приземление сверху)
- Движение влево/вправо
- Собираемые предметы и опасности

**Сложность:** Начинающий
**Пресет:** Пресет для Начинающих

---

## Шаг 1: Понять Игру

### Механики Игры
1. На игрока действует гравитация, он падает вниз
2. Игрок может двигаться влево и вправо
3. Игрок может прыгать, когда стоит на земле
4. Платформы не дают игроку провалиться
5. Собирайте монеты для очков
6. Достигните флага, чтобы завершить уровень

### Что Нам Нужно

| Элемент | Назначение |
|---------|------------|
| **Игрок** | Персонаж, которым вы управляете |
| **Земля/Платформа** | Твёрдые поверхности для стояния |
| **Монета** | Собираемые предметы для очков |
| **Шип** | Опасность, которая ранит игрока |
| **Флаг** | Цель, завершающая уровень |

## Шаг 2: Создать Спрайты

### Спрайты
- `spr_player` (32x48 пикселей) - простой персонаж
- `spr_ground` (32x32 пикселя) - плитка травы/земли
- `spr_platform` (64x16 пикселей) - плавающая платформа
- `spr_coin` (16x16 пикселей) - золотой круг
- `spr_spike` (32x32 пикселя) - треугольные шипы
- `spr_flag` (32x64 пикселя) - флаг на шесте

---

## Шаг 3-4: Создать Объекты Земли и Платформы

**obj_ground** и **obj_platform**: Установите спрайт, отметьте "Твёрдый"

---

## Шаг 5: Создать Объект Игрока

### Событие Create
```gml
hspeed_max = 4;
vspeed_max = 10;
jump_force = -10;
gravity_force = 0.5;
hsp = 0;
vsp = 0;
on_ground = false;
```

### Событие Step
```gml
var move_input = keyboard_check(vk_right) - keyboard_check(vk_left);
hsp = move_input * hspeed_max;

vsp += gravity_force;
if (vsp > vspeed_max) vsp = vspeed_max;

on_ground = place_meeting(x, y + 1, obj_ground);

if (on_ground && (keyboard_check_pressed(vk_up) || keyboard_check_pressed(vk_space))) {
    vsp = jump_force;
}

if (place_meeting(x + hsp, y, obj_ground)) {
    while (!place_meeting(x + sign(hsp), y, obj_ground)) x += sign(hsp);
    hsp = 0;
}
x += hsp;

if (place_meeting(x, y + vsp, obj_ground)) {
    while (!place_meeting(x, y + sign(vsp), obj_ground)) y += sign(vsp);
    vsp = 0;
}
y += vsp;
```

## Шаг 6-8: Предметы и Опасности

**obj_coin** - Столкновение с obj_player: Счёт +10, уничтожить Self

**obj_spike** - Столкновение с obj_player: Показать сообщение, перезапустить комнату

**obj_flag** - Столкновение с obj_player: Показать сообщение, следующая комната

---

## Шаг 9: Спроектируйте Свой Уровень

1. Создайте `room_level1` (800x480)
2. Включите привязку к сетке (32x32)
3. Разместите землю внизу, платформы в воздухе
4. Добавьте монеты, шипы
5. Поставьте флаг в конце, игрока в начале

## Что Вы Узнали

- **Физика гравитации** - Постоянная сила вниз

- **Обнаружение земли** - Использование `place_meeting`
