## Общие вопросы

### Что такое pyGM?
pyGM - это визуальный редактор для разработки игр на Python. Он позволяет создавать 2D игры без глубоких знаний программирования.

### pyGM бесплатный?
Да, pyGM является открытым исходным кодом и полностью бесплатен.

### Какой язык программирования используется?
pyGM основан на Python. Вы можете использовать визуальное программирование или писать код Python напрямую.

### Для каких платформ я могу разрабатывать?
- Windows
- macOS
- Linux
- Веб (HTML5)

## Установка

### Как установить pyGM?
```bash
pip install pygm
```

### Какая версия Python нужна?
Python 3.8 или выше.

### pyGM не запускается. Что делать?
1. Проверьте версию Python
2. Переустановите зависимости
3. Запустите из командной строки для просмотра ошибок

## Разработка

### Как создать новый проект?
Запустите pyGM и выберите "Новый проект" или используйте Файл > Новый.

### Как добавить спрайты?
1. Правый клик на "Спрайты" в дереве ресурсов
2. Выберите "Новый спрайт"
3. Импортируйте изображение или создайте его

### Как создать анимации?
1. Откройте спрайт
2. Добавьте несколько кадров
3. Настройте скорость анимации

### Как программировать поведение объектов?
1. Откройте объект
2. Добавьте события (напр. Create, Step)
3. Добавьте действия к событиям
4. Или используйте визуальный редактор Blockly

## Ресурсы

### Какие форматы изображений поддерживаются?
- PNG (рекомендуется)
- JPG
- GIF
- BMP

### Какие аудио форматы поддерживаются?
- WAV
- MP3
- OGG

### Как оптимизировать мои ресурсы?
- Используйте подходящие размеры изображений
- Сжимайте аудио файлы
- Удаляйте неиспользуемые ресурсы

## Геймплей

### Как реализовать обнаружение столкновений?
1. Создайте событие столкновения в объекте
2. Выберите другой объект
3. Добавьте действия для реакции

### Как создать несколько уровней?
1. Создайте несколько комнат
2. Используйте действие "Перейти в комнату"
3. Или "Перейти в следующую комнату"

### Как сохранить прогресс игры?
Используйте встроенные функции сохранения:
- `save_game()`: Сохранить игру
- `load_game()`: Загрузить игру

## Экспорт

### Как экспортировать мою игру?
1. Перейдите в Файл > Экспорт
2. Выберите целевую платформу
3. Настройте опции
4. Нажмите "Экспорт"

### Почему экспортированный файл такой большой?
- Включает Python runtime
- Все ресурсы встроены
- Совет: Оптимизируйте ресурсы

### Могу ли я экспортировать для мобильных устройств?
В настоящее время не поддерживается напрямую. Веб-экспорт работает в мобильных браузерах.

## Решение проблем

### Моя игра медленная
- Уменьшите код в событиях Step
- Оптимизируйте размеры спрайтов
- Избегайте слишком многих экземпляров

### Спрайты не отображаются
- Проверьте путь к спрайту
- Убедитесь, что Видимый=true
- Проверьте порядок рисования (глубина)

### Столкновения не работают
- Проверьте маски столкновений
- Убедитесь, что объекты твёрдые (если нужно)
- Проверьте настройки событий
