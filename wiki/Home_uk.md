**Візуальне середовище розробки ігор, натхненне GameMaker 7.0**

PyGameMaker — це IDE з відкритим кодом, яке робить створення 2D-ігор доступним завдяки візуальному блоковому програмуванню (Google Blockly) та системі подій-дій. Створюйте ігри без глибоких знань програмування, а потім експортуйте їх для Windows, Linux, HTML5 або мобільних платформ.

---

## Оберіть Свій Рівень

PyGameMaker використовує **пресети** для контролю доступних подій та дій. Це допомагає початківцям зосередитися на основних функціях, дозволяючи досвідченим користувачам отримати доступ до повного набору інструментів.

| Пресет | Підходить Для | Функції |
|--------|---------------|---------|
| [**Початківець**](Beginner-Preset_uk) | Новачків у розробці ігор | 4 події, 17 дій - Рух, зіткнення, рахунок, кімнати |
| [**Середній**](Intermediate-Preset_uk) | Деякий досвід | +4 події, +12 дій - Життя, здоров'я, звук, будильники, малювання |
| **Просунутий** | Досвідчених користувачів | Усі 40+ подій та дій доступні |

**Нові користувачі:** Почніть з [Пресету для Початківців](Beginner-Preset_uk), щоб вивчити основи без перевантаження.

Див. [Посібник з Пресетів](Preset-Guide_uk) для повного огляду системи пресетів.

## Початок Роботи

### Системні Вимоги

- **Python** 3.10 або новіший
- **Операційна Система:** Windows, Linux або macOS

### Встановлення

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/Gabe1290/pythongm.git
   cd pythongm
   ```

2. Створіть віртуальне середовище (рекомендовано):
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/macOS
   # або
   venv\Scripts\activate     # Windows
   ```

3. Встановіть залежності:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустіть PyGameMaker:
   ```bash
   python main.py
   ```

## Основні Концепції

### Об'єкти
Ігрові сутності зі спрайтами, властивостями та поведінкою. Кожен об'єкт може мати кілька подій з пов'язаними діями.

### Події
Тригери, що виконують дії при настанні певних умов:
- **Create** - Коли екземпляр створено
- **Step** - Кожен кадр (зазвичай 60 FPS)
- **Draw** - Фаза користувацького рендерингу
- **Destroy** - Коли екземпляр знищено
- **Keyboard** - Клавіша натиснута, відпущена або утримується
- **Mouse** - Кліки, рух, вхід/вихід
- **Collision** - Коли екземпляри торкаються
- **Alarm** - Таймери зворотного відліку (12 доступних)

Див. [Довідник Подій](Event-Reference_uk) для повної документації.

### Дії
Операції, що виконуються при спрацюванні подій. 40+ вбудованих дій для:
- Руху та фізики
- Малювання та спрайтів
- Рахунку, життів та здоров'я
- Звуку та музики
- Управління екземплярами та кімнатами

Див. [Повний Довідник Дій](Full-Action-Reference_uk) для повної документації.

### Кімнати
Ігрові рівні, де ви розміщуєте екземпляри об'єктів, встановлюєте фони та визначаєте ігрову область.

## Візуальне Програмування з Blockly

PyGameMaker інтегрує Google Blockly для візуального програмування. Блоки організовані в категорії:

- **Події** - Create, Step, Draw, Keyboard, Mouse
- **Рух** - Швидкість, напрямок, позиція, гравітація
- **Таймінг** - Будильники та затримки
- **Малювання** - Фігури, текст, спрайти
- **Рахунок/Життя/Здоров'я** - Відстеження стану гри
- **Екземпляр** - Створення та знищення об'єктів
- **Кімната** - Навігація та управління
- **Значення** - Змінні та вирази
- **Звук** - Відтворення аудіо
- **Вивід** - Налагодження та відображення

## Опції Експорту

### Windows EXE
Автономні виконувані файли Windows з використанням PyInstaller. Python не потрібен на цільовій машині.

### HTML5
Веб-ігри в одному файлі, що працюють у будь-якому сучасному браузері. Стиснуті gzip для швидкого завантаження.

### Linux
Нативні виконувані файли Linux для середовищ Python 3.10+.

### Kivy
Кросплатформні додатки для мобільних (iOS/Android) та десктопу через Buildozer.

## Структура Проекту

```
назва_проекту/
├── project.json      # Конфігурація проекту
├── backgrounds/      # Фонові зображення та метадані
├── data/             # Користувацькі файли даних
├── fonts/            # Визначення шрифтів
├── objects/          # Визначення об'єктів (JSON)
├── rooms/            # Макети кімнат (JSON)
├── scripts/          # Користувацькі скрипти
├── sounds/           # Аудіофайли та метадані
├── sprites/          # Зображення спрайтів та метадані
└── thumbnails/       # Згенеровані мініатюри ресурсів
```
