Дізнайтеся, як експортувати ваші ігри pyGM для різних платформ.

## Огляд

pyGM дозволяє експорт для:
- Windows (EXE)
- macOS (APP)
- Linux
- Веб (HTML5)

## Підготовка до експорту

### Перед експортом перевірте
1. **Всі ресурси на місці**: Спрайти, звуки тощо
2. **Гра протестована**: Без критичних помилок
3. **Налаштування оптимізовані**: Роздільна здатність, повноекранний режим

### Налаштування проекту
- **Назва гри**: Відображувана назва
- **Версія**: Номер версії
- **Іконка**: Іконка програми
- **Початковий екран**: Splash Screen

## Експорт для Windows

### Передумови
- pyinstaller встановлено
- Система Windows або крос-компіляція

### Кроки
1. Перейдіть до Файл > Експорт
2. Виберіть "Windows Executable"
3. Налаштуйте опції:
   - Файл іконки
   - Приховати вікно консолі
   - Один EXE файл
4. Натисніть "Експорт"

### Вихід
- Один EXE файл
- Або папка з залежностями

## Експорт для macOS

### Передумови
- Система macOS рекомендована
- py2app або pyinstaller

### Кроки
1. Файл > Експорт > macOS
2. Введіть назву App Bundle
3. Виберіть іконку (формат ICNS)
4. Експорт

## Експорт для Linux

### Опції
- AppImage (рекомендовано)
- Пакет Debian
- Виконуваний файл

### Кроки
1. Файл > Експорт > Linux
2. Виберіть формат
3. Експорт

## Веб-експорт (HTML5)

### Переваги
- Працює в браузері
- Легко ділитися
- Не потрібна установка

### Кроки
1. Файл > Експорт > Веб
2. Налаштуйте:
   - Розмір полотна
   - Екран завантаження
   - Стиснення
3. Експорт

### Вихід
- HTML файл
- JavaScript файли
- Папка ресурсів

### Хостинг
- Завантажте на веб-сервер
- Використовуйте itch.io
- Використовуйте GitHub Pages

## Опції експорту

### Загальні
- **Стиснення**: Зменшити розмір файлу
- **Режим налагодження**: Залишити для тестів
- **Вбудувати ресурси**: Все в один файл

### Специфічні для платформи
- **Windows**: UAC маніфест
- **macOS**: Підписування коду
- **Веб**: Версія WebGL

## Вирішення проблем

### Поширені проблеми

**Експорт не вдається**
- Перевірте повідомлення про помилки
- Оновіть pyinstaller

**Відсутні ресурси**
- Перевірте шляхи
- Повторно включіть ресурси

**Гра не запускається**
- Тестуйте в режимі налагодження
- Перевірте залежності

## Оптимізація

1. **Оптимізуйте розміри зображень**: Стисніть спрайти
2. **Стисніть аудіо**: MP3 замість WAV
3. **Видаліть невикористані ресурси**
4. **Оптимізуйте код**: Покращте ефективність
