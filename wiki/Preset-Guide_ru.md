# Руководство по Пресетам

*[Русский](Preset-Guide_ru) | [Вернуться на Главную](Home_ru)*

PyGameMaker предлагает различные пресеты, которые контролируют, какие события и действия доступны. Это помогает начинающим сосредоточиться на основных функциях, позволяя опытным пользователям получить доступ к полному набору инструментов.

## Выберите Свой Уровень

| Пресет | Подходит Для | Функции |
|--------|--------------|---------|
| [**Начинающий**](Beginner-Preset_ru) | Новичков в разработке игр | 4 события, 17 действий - Движение, столкновения, счёт, комнаты |
| [**Средний**](Intermediate-Preset_ru) | Некоторый опыт | +4 события, +12 действий - Жизни, здоровье, звук, будильники, рисование |
| **Продвинутый** | Опытных пользователей | Все 40+ событий и действий доступны |

## Пример Быстрого Старта

Вот простая игра сбора монет с использованием только функций для Начинающих:

### 1. Создайте Объекты
- `obj_player` - Управляемый персонаж
- `obj_coin` - Коллекционные предметы
- `obj_wall` - Твёрдые препятствия

### 2. Добавьте События к Игроку

**Клавиатура (Клавиши Стрелок):**
```
Стрелка Влево   → Установить Горизонтальную Скорость: -4
Стрелка Вправо  → Установить Горизонтальную Скорость: 4
Стрелка Вверх   → Установить Вертикальную Скорость: -4
Стрелка Вниз    → Установить Вертикальную Скорость: 4
```

**Столкновение с obj_coin:**
```
Добавить Счёт: 10
Уничтожить Экземпляр: other
```

**Столкновение с obj_wall:**
```
Остановить Движение
```

### 3. Создайте Комнату
- Разместите игрока
- Добавьте несколько монет
- Добавьте стены вокруг краёв

### 4. Запустите Игру!
Нажмите кнопку Играть, чтобы протестировать вашу игру.

## Советы для Успеха

1. **Начинайте Просто** - Сначала используйте пресет для Начинающих
2. **Тестируйте Часто** - Часто запускайте игру, чтобы обнаруживать проблемы
3. **Одна Вещь за Раз** - Добавляйте функции постепенно
4. **Используйте Столкновения** - Большинство игровых механик включает события столкновений
5. **Читайте Документацию** - Проверяйте справочные страницы, когда застряли
