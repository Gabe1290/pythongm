## Installazione

### Prerequisiti
- Python 3.8 o superiore
- pip (gestore pacchetti Python)

### Passaggi di installazione

1. **Installare pyGM**
   ```bash
   pip install pygm
   ```

2. **Verificare l'installazione**
   ```bash
   pygm --version
   ```

## Avviare pyGM

Dopo l'installazione, puoi avviare pyGM:

```bash
pygm
```

Questo apre l'editor visuale dove puoi iniziare a creare il tuo gioco.

## Interfaccia utente

### Aree principali
- **Pannello sinistro**: Gestione risorse (sprite, oggetti, stanze)
- **Area centrale**: Area di lavoro dell'editor
- **Pannello destro**: Proprieta e impostazioni

### Controlli di base
- **Rotella del mouse**: Zoom della vista
- **Pulsante centrale del mouse**: Spostare la vista
- **Click sinistro**: Selezionare/posizionare elementi
- **Click destro**: Menu contestuale
