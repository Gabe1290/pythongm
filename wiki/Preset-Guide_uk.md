# Посібник з Пресетів

*[Українська](Preset-Guide_uk) | [Повернутися на Головну](Home_uk)*

PyGameMaker пропонує різні пресети, які контролюють, які події та дії доступні. Це допомагає початківцям зосередитися на основних функціях, дозволяючи досвідченим користувачам отримати доступ до повного набору інструментів.

## Оберіть Свій Рівень

| Пресет | Підходить Для | Функції |
|--------|---------------|---------|
| [**Початківець**](Beginner-Preset_uk) | Новачків у розробці ігор | 4 події, 17 дій - Рух, зіткнення, рахунок, кімнати |
| [**Середній**](Intermediate-Preset_uk) | Деякий досвід | +4 події, +12 дій - Життя, здоров'я, звук, будильники, малювання |
| **Просунутий** | Досвідчених користувачів | Усі 40+ подій та дій доступні |

## Приклад Швидкого Старту

Ось проста гра збору монет з використанням лише функцій для Початківців:

### 1. Створіть Об'єкти
- `obj_player` - Керований персонаж
- `obj_coin` - Колекційні предмети
- `obj_wall` - Тверді перешкоди

### 2. Додайте Події до Гравця

**Клавіатура (Клавіші Стрілок):**
```
Стрілка Вліво   → Встановити Горизонтальну Швидкість: -4
Стрілка Вправо  → Встановити Горизонтальну Швидкість: 4
Стрілка Вгору   → Встановити Вертикальну Швидкість: -4
Стрілка Вниз    → Встановити Вертикальну Швидкість: 4
```

**Зіткнення з obj_coin:**
```
Додати Рахунок: 10
Знищити Екземпляр: other
```

**Зіткнення з obj_wall:**
```
Зупинити Рух
```

### 3. Створіть Кімнату
- Розмістіть гравця
- Додайте кілька монет
- Додайте стіни навколо країв

### 4. Запустіть Гру!
Натисніть кнопку Грати, щоб протестувати вашу гру.

## Поради для Успіху

1. **Починайте Просто** - Спочатку використовуйте пресет для Початківців
2. **Тестуйте Часто** - Часто запускайте гру, щоб виявляти проблеми
3. **Одна Річ за Раз** - Додавайте функції поступово
4. **Використовуйте Зіткнення** - Більшість ігрових механік включає події зіткнень
5. **Читайте Документацію** - Перевіряйте довідкові сторінки, коли застрягли
