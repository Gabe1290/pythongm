# Руководство: Создание игры Breakout

*[Home](Home_ru) | [Beginner Preset](Beginner-Preset_ru) | [English](Tutorial-Breakout)*

Это руководство проведет вас через создание классической игры Breakout. Это идеальный первый проект для изучения PyGameMaker!

![Концепция игры Breakout](https://upload.wikimedia.org/wikipedia/commons/thumb/c/c7/Breakout2600.svg/220px-Breakout2600.svg.png)

---

## Что вы узнаете

- Создание и использование спрайтов
- Настройка игровых объектов с событиями и действиями
- Управление с клавиатуры для движения игрока
- Обнаружение столкновений и отскоки
- Уничтожение объектов при столкновении
- Построение игровой комнаты

---

## Шаг 1: Создание спрайтов

Сначала нам нужно создать визуальные элементы для нашей игры.

### 1.1 Создание спрайта ракетки
1. На панели **Assets** щелкните правой кнопкой на **Sprites** → **Create Sprite**
2. Назовите его `spr_paddle`
3. Нарисуйте горизонтальный прямоугольник (примерно 64x16 пикселей)
4. **Важно:** Нажмите **Center**, чтобы установить начало координат в центре

### 1.2 Создание спрайта мяча
1. Создайте другой спрайт с именем `spr_ball`
2. Нарисуйте маленький круг (примерно 16x16 пикселей)
3. Нажмите **Center**, чтобы установить начало координат

### 1.3 Создание спрайта кирпича
1. Создайте спрайт с именем `spr_brick`
2. Нарисуйте прямоугольник (примерно 48x24 пикселей)
3. Нажмите **Center**, чтобы установить начало координат

### 1.4 Создание спрайта стены
1. Создайте спрайт с именем `spr_wall`
2. Нарисуйте квадрат (примерно 32x32 пикселей) - это будет граница
3. Нажмите **Center**, чтобы установить начало координат

### 1.5 Создание фона (необязательно)
1. Щелкните правой кнопкой на **Backgrounds** → **Create Background**
2. Назовите его `bg_game`
3. Нарисуйте или загрузите изображение фона

---

## Шаг 2: Создание объекта ракетки

Теперь давайте запрограммируем ракетку, которой управляет игрок.

### 2.1 Создание объекта
1. Щелкните правой кнопкой на **Objects** → **Create Object**
2. Назовите его `obj_paddle`
3. Установите **Sprite** как `spr_paddle`
4. Поставьте флажок **Solid**

### 2.2 Добавление движения вправо
1. Нажмите **Add Event** → **Keyboard** → выберите **Right Arrow**
2. Добавьте действие **Set Horizontal Speed**
3. Установите **value** как `5` (или любую скорость по вашему выбору)

### 2.3 Добавление движения влево
1. Нажмите **Add Event** → **Keyboard** → выберите **Left Arrow**
2. Добавьте действие **Set Horizontal Speed**
3. Установите **value** как `-5`

### 2.4 Остановка при отпускании клавиш
Ракетка продолжает двигаться даже после отпускания клавиши! Давайте это исправим.

1. Нажмите **Add Event** → **Keyboard Release** → выберите **Right Arrow**
2. Добавьте действие **Set Horizontal Speed**
3. Установите **value** как `0`

4. Нажмите **Add Event** → **Keyboard Release** → выберите **Left Arrow**
5. Добавьте действие **Set Horizontal Speed**
6. Установите **value** как `0`

Теперь ракетка останавливается, когда вы отпускаете клавиши со стрелками.

---

## Шаг 3: Создание объекта мяча

### 3.1 Создание объекта
1. Создайте новый объект с именем `obj_ball`
2. Установите **Sprite** как `spr_ball`
3. Поставьте флажок **Solid**

### 3.2 Установка начального движения
1. Нажмите **Add Event** → **Create**
2. Добавьте действие **Move in Direction** (или **Set Horizontal/Vertical Speed**)
3. Установите диагональное направление со скоростью `5`
   - Например: **hspeed** = `4`, **vspeed** = `-4`

Это заставляет мяч начать двигаться, когда игра начинается.

### 3.3 Отскок от ракетки
1. Нажмите **Add Event** → **Collision** → выберите `obj_paddle`
2. Добавьте действие **Reverse Vertical** (для отскока)

### 3.4 Отскок от стен
1. Нажмите **Add Event** → **Collision** → выберите `obj_wall`
2. Добавьте действие **Reverse Horizontal** или **Reverse Vertical** по необходимости
   - Или используйте оба для обработки отскоков в углах

---

## Шаг 4: Создание объекта кирпича

### 4.1 Создание объекта
1. Создайте новый объект с именем `obj_brick`
2. Установите **Sprite** как `spr_brick`
3. Поставьте флажок **Solid**

### 4.2 Уничтожение при столкновении с мячом
1. Нажмите **Add Event** → **Collision** → выберите `obj_ball`
2. Добавьте действие **Destroy Instance** с целью **self**

Это уничтожает кирпич, когда мяч ударяет по нему!

### 4.3 Отскок мяча
В том же событии столкновения также добавьте:
1. Добавьте действие **Reverse Vertical** (примененное к **other** - мячу)

Или вернитесь к `obj_ball` и добавьте:
1. **Add Event** → **Collision** → выберите `obj_brick`
2. Добавьте действие **Reverse Vertical**

---

## Шаг 5: Создание объекта стены

### 5.1 Создание объекта
1. Создайте новый объект с именем `obj_wall`
2. Установите **Sprite** как `spr_wall`
3. Поставьте флажок **Solid**

Это все - стена просто должна быть твердой, чтобы мяч отскакивал.

---

## Шаг 6: Создание игровой комнаты

### 6.1 Создание комнаты
1. Щелкните правой кнопкой на **Rooms** → **Create Room**
2. Назовите ее `room_game`

### 6.2 Установка фона (необязательно)
1. В настройках комнаты найдите **Background**
2. Выберите ваш фон `bg_game`
3. Поставьте флажок **Stretch**, если хотите, чтобы он заполнил комнату

### 6.3 Размещение объектов

Теперь разместите ваши объекты в комнате:

1. **Разместите ракетку:** Поместите `obj_paddle` внизу по центру комнаты

2. **Разместите стены:** Поместите экземпляры `obj_wall` по краям:
   - Вдоль верха
   - Вдоль левой стороны
   - Вдоль правой стороны
   - Оставьте низ открытым (здесь мяч может вылететь!)

3. **Разместите мяч:** Поместите `obj_ball` где-нибудь посередине

4. **Разместите кирпичи:** Расположите экземпляры `obj_brick` рядами в верхней части комнаты

---

## Шаг 7: Протестируйте вашу игру!

1. Нажмите кнопку **Play** (зеленая стрелка)
2. Используйте клавиши **Влево** и **Вправо** для перемещения ракетки
3. Попробуйте отбить мяч, чтобы уничтожить все кирпичи!
4. Нажмите **Escape** для выхода

---

## Что дальше?

Ваша базовая игра Breakout завершена! Вот некоторые улучшения, которые можно попробовать:

### Добавить систему жизней
- Добавьте событие **No More Lives**, чтобы показать "Game Over"
- Теряйте жизнь, когда мяч вылетает снизу

### Добавить подсчет очков
- Используйте действие **Add Score** при уничтожении кирпичей
- Отображайте счет с помощью **Draw Score**

### Добавить несколько уровней
- Создайте больше комнат с разным расположением кирпичей
- Используйте **Next Room**, когда все кирпичи уничтожены

### Добавить звуковые эффекты
- Добавьте звуки для отскоков и уничтожения кирпичей
- Используйте действие **Play Sound**

## Сводка объектов

| Объект | Спрайт | Твердый | События |
|--------|--------|---------|---------|
| `obj_paddle` | `spr_paddle` | Да | Keyboard (Left/Right), Keyboard Release |
| `obj_ball` | `spr_ball` | Да | Create, Collision (paddle, wall, brick) |
| `obj_brick` | `spr_brick` | Да | Collision (ball) - Уничтожить self |
| `obj_wall` | `spr_wall` | Да | Не требуется |
