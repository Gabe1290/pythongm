Відповіді на поширені запитання про pyGM.

## Загальні запитання

### Що таке pyGM?
pyGM - це візуальний редактор для розробки ігор на Python. Він дозволяє створювати 2D ігри без глибоких знань програмування.

### pyGM безкоштовний?
Так, pyGM є відкритим вихідним кодом і повністю безкоштовний.

### Яка мова програмування використовується?
pyGM базується на Python. Ви можете використовувати візуальне програмування або писати код Python безпосередньо.

### Для яких платформ я можу розробляти?
- Windows
- macOS
- Linux
- Веб (HTML5)

## Встановлення

### Як встановити pyGM?
```bash
pip install pygm
```

### Яка версія Python потрібна?
Python 3.8 або новіший.

### pyGM не запускається. Що робити?
1. Перевірте версію Python
2. Перевстановіть залежності
3. Запустіть з командного рядка для перегляду помилок

## Розробка

### Як створити новий проект?
Запустіть pyGM і виберіть "Новий проект" або використайте Файл > Новий.

### Як додати спрайти?
1. Правий клік на "Спрайти" в дереві ресурсів
2. Виберіть "Новий спрайт"
3. Імпортуйте зображення або створіть його

### Як створити анімації?
1. Відкрийте спрайт
2. Додайте кілька кадрів
3. Налаштуйте швидкість анімації

### Як програмувати поведінку об'єктів?
1. Відкрийте об'єкт
2. Додайте події (напр. Create, Step)
3. Додайте дії до подій
4. Або використайте візуальний редактор Blockly

## Ресурси

- BMP

### Які аудіо формати підтримуються?

## Геймплей

### Як реалізувати виявлення зіткнень?
1. Створіть подію зіткнення в об'єкті
2. Виберіть інший об'єкт
3. Додайте дії для реакції

### Як створити кілька рівнів?
1. Створіть кілька кімнат
2. Використайте дію "Перейти до кімнати"
3. Або "Перейти до наступної кімнати"

### Як зберегти прогрес гри?
Використовуйте вбудовані функції збереження:
- `save_game()`: Зберегти гру
- `load_game()`: Завантажити гру

## Експорт

### Як експортувати мою гру?
1. Перейдіть до Файл > Експорт
2. Виберіть цільову платформу
3. Налаштуйте опції
4. Натисніть "Експорт"

### Чому експортований файл такий великий?
- Включає Python runtime
- Всі ресурси вбудовані
- Порада: Оптимізуйте ресурси

### Чи можу я експортувати для мобільних пристроїв?
Наразі не підтримується безпосередньо. Веб-експорт працює в мобільних браузерах.

## Вирішення проблем

### Моя гра повільна
- Зменшіть код у подіях Step
- Оптимізуйте розміри спрайтів
- Уникайте занадто багатьох екземплярів

### Спрайти не відображаються
- Перевірте шлях до спрайта
- Переконайтеся, що Видимий=true
- Перевірте порядок малювання (глибина)

### Зіткнення не працюють
- Перевірте маски зіткнень
- Переконайтеся, що об'єкти тверді (якщо потрібно)
- Перевірте налаштування подій
