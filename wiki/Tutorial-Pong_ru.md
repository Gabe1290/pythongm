## Введение

В этом учебнике вы создадите классическую игру **Pong** - одну из первых видеоигр! Pong - это игра для двух игроков, где каждый игрок управляет ракеткой и старается забить мяч мимо ракетки противника, чтобы заработать очки.

**Что вы научитесь:**
- Создавать спрайты для ракеток, мяча и стен
- Обрабатывать ввод с клавиатуры для двух игроков
- Заставлять объекты отскакивать друг от друга
- Отслеживать и отображать счёт обоих игроков
- Использовать глобальные переменные

**Сложность:** Начинающий
**Пресет:** Beginner Preset

## Шаг 1: Спланируйте вашу игру

Перед началом давайте разберёмся, что нам нужно:

| Элемент | Назначение |
|---------|---------|
| **Мяч** | Отскакивает между игроками |
| **Левая ракетка** | Игрок 1 управляет клавишами W/S |
| **Правая ракетка** | Игрок 2 управляет стрелками вверх/вниз |
| **Стены** | Верхние и нижние границы поля |
| **Зоны гола** | Невидимые области за каждой ракеткой для обнаружения забитых голов |
| **Отображение счёта** | Показывает счёт обоих игроков |

## Шаг 2: Создайте спрайты

### 2.1 Спрайт мяча

1. В **Дереве ресурсов** щелкните правой кнопкой мыши на **Sprites** и выберите **Create Sprite**
2. Назовите его `spr_ball`
3. Нажмите **Edit Sprite**, чтобы открыть редактор спрайтов
4. Нарисуйте маленький белый круг (примерно 16x16 пикселей)
5. Нажмите **OK** для сохранения

### 2.2 Спрайты ракеток

Мы создадим две ракетки - по одной для каждого игрока:

**Левая ракетка (Игрок 1):**
1. Создайте новый спрайт с именем `spr_paddle_left`
2. Нарисуйте высокий тонкий прямоугольник, изогнутый как скобка ")" - рекомендуется синий цвет
3. Размер: примерно 16x64 пикселей

**Правая ракетка (Игрок 2):**
1. Создайте новый спрайт с именем `spr_paddle_right`
2. Нарисуйте высокий тонкий прямоугольник, изогнутый как скобка "(" - рекомендуется красный цвет
3. Размер: примерно 16x64 пикселей

### 2.3 Спрайт стены

1. Создайте новый спрайт с именем `spr_wall`
2. Нарисуйте сплошной прямоугольник (серый или белый)
3. Размер: 32x32 пикселя (мы растянем его в комнате)

### 2.4 Спрайт зоны гола (невидимый)

1. Создайте новый спрайт с именем `spr_goal`
2. Размер: 32x32 пикселя
3. Оставьте его прозрачным или сделайте сплошным цветом (он будет невидимым в игре)

---

## Шаг 3: Создайте объект стены

Объект стены создаёт границы в верхней и нижней части игрового поля.

1. Щелкните правой кнопкой мыши на **Objects** и выберите **Create Object**
2. Назовите его `obj_wall`
3. Установите спрайт на `spr_wall`
4. **Отметьте флажок "Solid"** - это важно для отскока!
5. События не требуются - стена просто стоит на месте

---

## Шаг 4: Создайте объекты ракеток

### 4.1 Левая ракетка (Игрок 1)

1. Создайте новый объект с именем `obj_paddle_left`
2. Установите спрайт на `spr_paddle_left`
3. **Отметьте флажок "Solid"**

**Добавьте события клавиатуры для движения:**

**Событие: Нажата клавиша W**
1. Добавьте событие → Keyboard → Press W
2. Добавьте действие: **Move** → **Set Vertical Speed**
3. Установите вертикальную скорость на `-8` (движение вверх)

**Событие: Отпущена клавиша W**
1. Добавьте событие → Keyboard → Release W
2. Добавьте действие: **Move** → **Set Vertical Speed**
3. Установите вертикальную скорость на `0` (остановка)

**Событие: Нажата клавиша S**
1. Добавьте событие → Keyboard → Press S
2. Добавьте действие: **Move** → **Set Vertical Speed**
3. Установите вертикальную скорость на `8` (движение вниз)

**Событие: Отпущена клавиша S**
1. Добавьте событие → Keyboard → Release S
2. Добавьте действие: **Move** → **Set Vertical Speed**
3. Установите вертикальную скорость на `0` (остановка)

**Событие: Столкновение с obj_wall**
1. Добавьте событие → Collision → obj_wall
2. Добавьте действие: **Move** → **Bounce Against Objects**
3. Выберите "Against solid objects"

### 4.2 Правая ракетка (Игрок 2)

1. Создайте новый объект с именем `obj_paddle_right`
2. Установите спрайт на `spr_paddle_right`
3. **Отметьте флажок "Solid"**

**Добавьте события клавиатуры для движения:**

**Событие: Нажата стрелка вверх**
1. Добавьте событие → Keyboard → Press Up
2. Добавьте действие: **Move** → **Set Vertical Speed**
3. Установите вертикальную скорость на `-8`

**Событие: Отпущена стрелка вверх**
1. Добавьте событие → Keyboard → Release Up
2. Добавьте действие: **Move** → **Set Vertical Speed**
3. Установите вертикальную скорость на `0`

**Событие: Нажата стрелка вниз**
1. Добавьте событие → Keyboard → Press Down
2. Добавьте действие: **Move** → **Set Vertical Speed**
3. Установите вертикальную скорость на `8`

**Событие: Отпущена стрелка вниз**
1. Добавьте событие → Keyboard → Release Down
2. Добавьте действие: **Move** → **Set Vertical Speed**
3. Установите вертикальную скорость на `0`

**Событие: Столкновение с obj_wall**
1. Добавьте событие → Collision → obj_wall
2. Добавьте действие: **Move** → **Bounce Against Objects**
3. Выберите "Against solid objects"

## Шаг 5: Создайте объект мяча

1. Создайте новый объект с именем `obj_ball`
2. Установите спрайт на `spr_ball`

**Событие: Create**
1. Добавьте событие → Create
2. Добавьте действие: **Move** → **Start Moving in Direction**
3. Выберите диагональное направление (не прямо вверх или вниз)
4. Установите скорость на `6`

**Событие: Столкновение с obj_paddle_left**
1. Добавьте событие → Collision → obj_paddle_left
2. Добавьте действие: **Move** → **Bounce Against Objects**
3. Выберите "Against solid objects"

**Событие: Столкновение с obj_paddle_right**
1. Добавьте событие → Collision → obj_paddle_right
2. Добавьте действие: **Move** → **Bounce Against Objects**
3. Выберите "Against solid objects"

**Событие: Столкновение с obj_wall**
1. Добавьте событие → Collision → obj_wall
2. Добавьте действие: **Move** → **Bounce Against Objects**
3. Выберите "Against solid objects"

## Шаг 6: Создайте объекты зон голов

Зоны голов - это невидимые области за каждой ракеткой. Когда мяч попадает в зону гола, противоположный игрок забивает.

### 6.1 Левая зона гола

1. Создайте новый объект с именем `obj_goal_left`
2. Установите спрайт на `spr_goal`
3. **Снимите флажок "Visible"** - зона гола должна быть невидимой
4. **Отметьте флажок "Solid"**

### 6.2 Правая зона гола

1. Создайте новый объект с именем `obj_goal_right`
2. Установите спрайт на `spr_goal`
3. **Снимите флажок "Visible"**
4. **Отметьте флажок "Solid"**

### 6.3 Добавьте события столкновения мяча с зонами голов

Вернитесь к `obj_ball` и добавьте эти события:

**Событие: Столкновение с obj_goal_left**
1. Добавьте событие → Collision → obj_goal_left
2. Добавьте действие: **Move** → **Jump to Start Position** (сбрасывает мяч)
3. Добавьте действие: **Score** → **Set Score**
   - Переменная: `global.p2score`
   - Значение: `1`
   - Отметьте "Relative" (добавляет 1 к текущему счёту)

**Событие: Столкновение с obj_goal_right**
1. Добавьте событие → Collision → obj_goal_right
2. Добавьте действие: **Move** → **Jump to Start Position**
3. Добавьте действие: **Score** → **Set Score**
   - Переменная: `global.p1score`
   - Значение: `1`
   - Отметьте "Relative"

## Шаг 7: Создайте объект отображения счёта

1. Создайте новый объект с именем `obj_score`
2. Спрайт не требуется

**Событие: Create**
1. Добавьте событие → Create
2. Добавьте действие: **Score** → **Set Score**
   - Переменная: `global.p1score`
   - Значение: `0`
3. Добавьте действие: **Score** → **Set Score**
   - Переменная: `global.p2score`
   - Значение: `0`

**Событие: Draw**
1. Добавьте событие → Draw
2. Добавьте действие: **Draw** → **Draw Text**
   - Текст: `Player 1:`
   - X: `10`
   - Y: `10`
3. Добавьте действие: **Draw** → **Draw Variable**
   - Переменная: `global.p1score`
   - X: `100`
   - Y: `10`
4. Добавьте действие: **Draw** → **Draw Text**
   - Текст: `Player 2:`
   - X: `10`
   - Y: `30`
5. Добавьте действие: **Draw** → **Draw Variable**
   - Переменная: `global.p2score`
   - X: `100`
   - Y: `30`

## Шаг 8: Спроектируйте комнату

1. Щелкните правой кнопкой мыши на **Rooms** и выберите **Create Room**
2. Назовите её `room_pong`
3. Установите размер комнаты (например, 640x480)

**Расположите объекты:**

1. **Стены**: Поместите экземпляры `obj_wall` вдоль верхнего и нижнего краёв комнаты
2. **Левая ракетка**: Поместите `obj_paddle_left` рядом с левым краем, по центру по вертикали
3. **Правая ракетка**: Поместите `obj_paddle_right` рядом с правым краем, по центру по вертикали
4. **Мяч**: Поместите `obj_ball` в центр комнаты
5. **Зоны голов:**
   - Поместите экземпляры `obj_goal_left` вдоль левого края (позади того места, где расположена ракетка)
   - Поместите экземпляры `obj_goal_right` вдоль правого края
6. **Отображение счёта**: Поместите `obj_score` где-нибудь (у него нет спрайта, он просто рисует текст)

**Пример макета комнаты:**
```
[WALL WALL WALL WALL WALL WALL WALL WALL WALL WALL]
[GOAL]  [PADDLE_L]            [BALL]            [PADDLE_R]  [GOAL]
[GOAL]  [PADDLE_L]                              [PADDLE_R]  [GOAL]
[GOAL]                                                      [GOAL]
[WALL WALL WALL WALL WALL WALL WALL WALL WALL WALL]
```

## Улучшения (Необязательно)

### Увеличение скорости
Сделайте мяч быстрее каждый раз, когда он ударяется о ракетку, добавив в события столкновения:
- После действия отскока добавьте **Move** → **Set Speed**
- Установите скорость на `speed + 0.5` с отмеченным "Relative"

### Звуковые эффекты
Добавьте звуки когда:
- Мяч ударяется о ракетку
- Мяч ударяется о стену
- Игрок забивает

### Условие победы
Добавьте проверку в событие Draw:
- Если `global.p1score >= 10`, отобразите "Player 1 Wins!"
- Если `global.p2score >= 10`, отобразите "Player 2 Wins!"

## Устранение проблем

| Проблема | Решение |
|---------|----------|
| Мяч проходит сквозь ракетку | Убедитесь, что ракетки имеют отмеченный флажок "Solid" |
| Ракетка не останавливается у стен | Добавьте событие столкновения с obj_wall |
| Счёт не обновляется | Проверьте, что названия переменных совпадают точно (global.p1score, global.p2score) |
| Мяч не движется | Проверьте, что событие Create содержит действие движения |
