Добро пожаловать в pyGM! Это руководство поможет вам начать разработку игр.

## Установка

### Предварительные требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Шаги установки

1. **Установите pyGM**
   ```bash
   pip install pygm
   ```

2. **Проверьте установку**
   ```bash
   pygm --version
   ```

## Запуск pyGM

После установки вы можете запустить pyGM:

```bash
pygm
```

Это откроет визуальный редактор, где вы можете начать создавать свою игру.

## Интерфейс пользователя

### Основные области
- **Левая панель**: Управление ресурсами (спрайты, объекты, комнаты)
- **Центральная область**: Рабочая область редактора
- **Правая панель**: Свойства и настройки

### Базовые элементы управления
- **Колёсико мыши**: Масштабирование вида
- **Средняя кнопка мыши**: Перемещение вида
- **Левый клик**: Выбор/размещение элементов
- **Правый клик**: Контекстное меню
