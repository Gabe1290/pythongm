# PyGameMaker IDE

### Create Amazing 2D Games — No Coding Required!

---

**PyGameMaker** is a powerful, free, open-source game development environment inspired by the legendary GameMaker. Written entirely in Python, it runs natively on **Linux, Windows, and macOS** — your game projects are saved in a universal format that works identically across all platforms. Whether you're a student taking your first steps into game design or an educator looking for the perfect teaching tool, PyGameMaker makes game creation fun, visual, and accessible to everyone.

---

## What Makes PyGameMaker Special?

**Drag-and-Drop Game Design** — Build complete games using an intuitive visual event/action system. Choose from over 80 built-in actions covering movement, collisions, scoring, sound, drawing, and more. No programming knowledge needed!

**Built-In Sprite Editor** — Draw and animate your game characters right inside the IDE. Full-featured pixel art tools, animation timeline, color palette, and GIF import support.

**Room Editor** — Design your game levels visually. Place objects, paint tiles, configure scrolling backgrounds, and set up camera views — all with drag-and-drop simplicity.

**Real-Time Testing** — Hit F5 and your game is running in seconds. Instant feedback means faster iteration and more fun experimenting.

## Learn While You Create

PyGameMaker is built for learning. Toggle **View Code** to see how your visual actions translate into real Python code. Explore the open-source codebase to understand how a game engine works from the inside out.

**8 Built-In Tutorials** guide you from your very first project through classic games like Pong, Breakout, Sokoban, and Platformer.

---

## Blockly Visual Programming

Prefer code blocks? PyGameMaker integrates **Google Blockly** with 173 snap-together blocks across 19 categories. Choose from presets like Beginner, Platformer, or RPG — or build your own custom block palette.

---

## Thymio Robot Simulator

Bring your code into the real world! PyGameMaker includes a full **Thymio robot simulator** with motors, proximity sensors, ground sensors, LEDs, and buttons. Test your robot programs on screen, then export to a real Thymio via Aseba.
