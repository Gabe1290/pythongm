# PyGameMaker IDE

### Creez des jeux 2D incroyables — sans écrire une seule ligne de code !


**PyGameMaker** est un environnement de développement de jeux puissant, gratuit et open-source, inspire du légendaire GameMaker. Entièrement écrit en Python, il fonctionne nativement sur **Linux, Windows et macOS** — vos projets de jeux sont sauvegardes dans un format universel qui fonctionne de manière identique sur toutes les plateformes. Que vous soyez un étudiant faisant ses premiers pas dans la conception de jeux ou un enseignant a la recherche de l'outil pédagogique idéal, PyGameMaker rend la création de jeux amusante, visuelle et accessible a tous.


## Qu'est-ce qui rend PyGameMaker spécial ?

**Conception de jeux par glisser-déposer** — Construisez des jeux complets grâce a un système visuel intuitif d’extensivement et d'actions. Choisissez parmi plus de 80 actions intégrées couvrant le mouvement, les collisions, le score, le son, le dessin et bien plus encore. Aucune connaissance en programmation requise !

**Éditeur de sprites intégré** — Dessinez et animez vos personnages de jeu directement dans l'IDE. Outils complets de pixel art, timeline d'animation, palette de couleurs et support d'importation GIF.

**Editeur de salles** — Concevez vos niveaux de jeu visuellement. Placez des objets, peignez des tuiles, configurez les arrière-plans défilants et paramétrez les vues de camera — le tout par simple glisser-déposer.

**Test en temps reel** — Appuyez sur F5 et votre jeu se lance en quelques secondes. Un retour instantané signifie des itérations plus rapides et plus de plaisir a expérimenter.

## Apprenez en créant

PyGameMaker est conçu pour l'apprentissage. Activez **Voir le code** pour découvrir comment vos actions visuelles se traduisent en véritable code Python. Explorez le code source ouvert pour comprendre le fonctionnement interne d'un moteur de jeu.

**8 tutoriels integres** vous guident depuis votre tout premier projet jusqu’à des jeux classiques comme Pong, Breakout, Sokoban et Platformer.


## Programmation visuelle Blockly

Vous preferez les blocs de code ? PyGameMaker intègre **Google Blockly** avec 173 blocs a assembler repartis en 19 catégories. Choisissez parmi des présélections comme Débutant, Platformer ou RPG — ou créez votre propre palette de blocs personnalisée.


## Simulateur de robot Thymio

Donnez vie a votre code dans le monde réel ! PyGameMaker inclut un **simulateur de robot Thymio** complet avec moteurs, capteurs de proximité, capteurs de sol, LEDs et boutons. Testez vos programmes robotiques a l’écran, puis exportez-les vers un vrai Thymio via Aseba.
