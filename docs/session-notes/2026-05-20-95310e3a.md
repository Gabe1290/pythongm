> what is the git command to synchronise from github?

### Claude

```
git pull
```

That fetches `origin/<current-branch>` and merges it into your working branch.

If you want to keep your local history linear when there are commits on both sides:

```
git pull --rebase
```

If you just want to download remote changes *without* merging them yet (to inspect first):

```
git fetch
```

Then `git log main..origin/main` shows what's incoming, and `git merge origin/main` (or `git rebase origin/main`) integrates them when you're ready.
